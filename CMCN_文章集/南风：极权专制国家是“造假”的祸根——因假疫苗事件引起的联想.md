# 南风：极权专制国家是“造假”的祸根——因假疫苗事件引起的联想

南风：极权专制国家是“造假”的祸根——因假疫苗事件引起的联想

发表于 2018年8月15日 编辑 凌江峰

今年对中共来说也算是一个多事之秋，上海泼墨女孩事情尚未停息，又出了个“假疫苗事件”，闹得全国上下沸沸扬扬，一针假疫苗确实给社会带来不小的恐慌。7月23日有一位叫张凯的律师在网上发表了一篇文章《都在一条船上》引起广泛的关注，据悉此文发出不到17个小时被封，然点击人数达千万，打赏竟高达140万，可见民众对此事的关切和愤慨！近几年来，笔者对国内造假之事似乎听得已经有点麻木了，什么假烟、假酒、假种子、假化肥、假农药都已经司空见惯，甚至每年“315”打假日也懒得上街买东西了。笔者有时在家里沉思，这个国家怎么了？总在高唱“文化自信”为何变得如此没有诚信，原因何在？凡什么东西都得有个来龙去脉，我们是否得从根子上挖掘一下——。

一，   中共这个政权都是靠“造假”取得的。

中国有句俗语，冰冻三尺，非一日之寒。今日社会上的造假只能算一个现象，其根子在中共这个体制上。中共自成立那天起，由小变大、由弱变强，并非是他们所说的代表了真理，代表了正义事业，那都是哄老百姓的鬼话。纵观历史，中共夺取政权，完全靠欺骗和谎言，也就是说靠“造假”取得的。毛泽东就是一个“造假”大师，他在没有取得政权时，将中国的知识分子和中国的老百姓哄得团团转，甚至连老蒋也被他哄骗了。毛泽东在1945年前后说的假话被笑蜀先生全部收集在那本《历史的先声》书中，读者若有兴趣看看就知道了。他先哄骗中国的知识分子，宣称共产党的革命目标是将中国建成和美国一样的民主自由国家，让当时那些民主党派知识分子信以为真而倒向中共.他又哄骗农民，许诺革命成功之后，让农民耕者有其田，所以中国农民才那么支持他们的“革命”。中国的农民是最老实的人，俗话说，哄骗老实人是要遭报应的，中共的革命成功最主要是靠中国农民的支持取得的，陈毅曾说，淮海战役的胜利是靠农民的小推车推出来的。可革命胜利后，毛泽东翻脸不认帐，农民非但一寸土地都没得到，反而被饿死几千万。因为太多了，篇幅有限，不能多举。中国有句话，一个谎言说出要靠十个谎言来圆场。因为中共靠谎言取得政权，维系政权必须靠继续说假话。所以中共夺得政权后，不造假已是不行了。林彪放言，“不说假话不能办大事”，文革中他有四句子名言：“政治斗争无诚实可言，组织斗争结成死党.路线斗争引导敌人犯错误，革命的造谣好得狠！”。笔者就是从那个“造假”时代过来的人，已经练就从中共“正话”里面听“反话”的功夫。凡是他们说“形势大好”，那就是“不好”.如果说“越来越好”，那就是“越来越坏”。三年大饥荒饿死几千万人，他们反而说中国人民是多少幸福。当年全国人民正饥肠辘辘之时，周恩来亲自请来著名画家如黄胄，关山月等人来画“江山如此多娇”。参看（《炎黄春秋》饥饿年代的中国画家）1960年英国蒙哥马利元帅访问中国，因听说中国人民正在饿肚子，所以他要求去北京下面走访一下，他们派人将蒙哥马利领到北京市机关幼儿园，参观孩子们食堂，一下把蒙哥马利骗倒了，他对毛说，我看你们的孩子们都长得不错啊，毛说，我们的孩子们是吃得很饱的。其实毛心里明白全中国多少孩子们在挨饿，这个英国老头回国后马上替中共宣传，说中国人根本不是没有饭吃。

二，   今天习当局又祭起毛的“造假”功夫。

习上台后，别的不乍的，对毛的那一套倒学得蛮熟的，特别对毛说假话功夫学得炉火纯青。他上任后提出的那个24个字即“社会主义核心价值观”的确迷倒了一大批人，我有个朋友是大学教授，曾在文革期间坐牢九年，一直对中共不看好，习提出“社核”后，欣喜若狂，对我说，这下中国有希望了，从来没有哪任中共领导人公开提出“自由、民主、公正、法治”这种字样，看来习近平这下真要走民主道路了。我说，先别急早下结论，再看一段时间。我是被他们哄怕了的，他们十句话难得有一句子真话。果不其然，几年后这位朋友见到我已经无语。我知道，他的无语是被“毕福剑”，被“不准妄议”，被“709大抓捕”，被〈炎黄春秋〉遭封杀，被“刘晓波之死”，被以“煽颠罪”抓进牢里的民主人士等等一系列击得粉碎。他再也不说习有可能成为大陆的蒋经国了。这位夫子还曾经神秘地给我说一件事，习近平把袁腾飞的书送给美国人了。现在他明白习不仅哄骗中国人，甚至连美国人也骗了。否则特朗普不会搞起了贸易战。在这个世界上，凡是专制极权国家，皆是靠哄骗人民来维系的，“造假”是他们维系政权的必不可少的工具。搞独裁专制，必须“造假”，不“造假”难以维系。如今日的北朝鲜，人民至今还被金家王朝哄骗着。他们认为吃不饱饭是美帝国主义造成的.外国人参观只准在平壤，别的地方是绝对不能去的。为什么这些专制国家都爱“造假”呢？这与他们的体制有关，中共想让他们的红色江山万代传。为了保住他们的权位，保住他们的既得利益，必然要欺骗民众。而这个邪恶的体制必然产生邪恶的领袖，如列宁、斯大林、毛泽东、金三代、波尔布特等。有这种邪恶的体制和邪恶的领导人，下面的老百姓不造假有可能吗？俗话说“上梁不正下梁歪”，你上面成天说假话，难道能管得住下面人造假吗？

三，   中共长期打压说真话人士，“造假”必然泛滥成灾。

中共自掌权后，那些正直的人，敢讲真话的人被他们长期地打压，一直翻不了身，毛泽东本来就是一个朝令夕改、说话不算数的政治流氓，怎么可能容得下那些正直的人呢？相反，那些奸佞小人，说假话的人却一直得到他的纵容和包庇。1958年大跃进，毛泽东就是被那些说假话的小人包围，这些佞臣也看中了毛爱听假话，（如河南吴芝圃、上海柯庆施之流）所以上下互动，造成三年大饥荒。而那些党内较正直的人（如田家英、彭德怀）却被他残害。相反，罪大恶极的吴芝圃、李井泉之流反得到毛的保护。还有毛以前的故友梁漱溟因替当时农民说话，被他弃之不用。在中共执政的几十年里，无数的正直的人、有良心的人，敢于说真话的人被他们在一次又一次的政治运动中遭到残酷无情地迫害，如林昭、张志新、逻罗克等，一直到今天，这种迫害仍未停止。习上任后，加大了对敢于讲真话的人的打压，那些秉良心的律师、记者、大学教授和民主人士被关压、撤职、流放，如刘晓波、王全璋、秦永敏、许志永、邓相超、翟橘红等等。因为这些当权者本来就心术不正，所以对那些造假者不会惩办，甚至纵容。如揭露地沟油的洛阳电视台记者李翔惨死，这明明是那些造假商人的疯狂报服，可当局却将他搞成抢劫杀人案，明明可以将此事件作为典型案例在全国造成影响，来维护记者的正当权利和人生保障，打击违法犯罪，可当局却低调处理此事。因为当局从来没有把律师、记者当做自己人，反而时时防范着他们，甚至成为他们的眼中丁。他们若是提高律师、新闻记者的自由权利，就意味着给他们自己制造压力。

张凯律师在《我们都在一条船上》有这样一段话：一个大胆的假设：“如果2010年，那些做报导的记者，律师不是被打压，而是得到荣誉。

“如果那年的疫苗事件，责任官员得到惩处，法院大胆的开庭审理，受害人得到高额赔偿。

“如果那些自发组织起来的NGO组织，可以自由的发挥他们的功效。

那今天会怎么样呢？

不用太聪明也会知道：会产生更杰出的记者，律师，官员……

法官会充满荣誉感，他们会骄傲的说：八年前那个案子是我判的……

NGO组织会自豪的把八年前他们的功绩写在自己的宣传册里，对来访的人夸耀自己在疫苗事件中所做的贡献……

但，这一切都只是假设。

如果把我的这些假设翻译成政治或法律词汇，就是：新闻自由，司法独立，主权在民……

这些已经被历史无数次的验证过的价值，难道我们还需要别出心裁，另辟新道吗？

问题疫苗难道真的只是疫苗问题吗？”

就像张凯这样正直的律师的微信号都被封，难道说这个国家、这个体制没问题吗？西方民主国家是把做人的良知放在第一位的.专制国家是把权位放在第一位的。譬如，毛泽东可以置中国人民死活而不顾，把钱大把大把地送给亚非拉国家，让它们抬着他玩.习当局今天依然学毛泽东“大撤币”激起国人上下愤怒！假疫苗事件发生了，他们可以做做样子，抓捕几个人，但这对他们不是最关心的，他们关心的是他们的权利，是他们的红色江山。中国有句俗话叫做“正人先正己”，己不正焉能正他人？你们成天说假话，干假事，难道可以杜绝下面造假吗？

2018年8月3日
