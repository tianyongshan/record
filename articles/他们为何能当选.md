# 他们为何能当选

他们为何能当选
http://www.sina.com.cn 2007年08月09日15:13 南方周末
□本报记者 苏永通 实习生 廉卫东发自北京
新华社8月2日公布了2217名当选中共十七大代表名单。次日名单刊登在全国各大党报头版头条上。
中共中央党校党史部副主任谢春涛、教授王贵秀等多位党史、党建专家均表示，在党代会前公布代表名单，“这在党史上还是第一次”。
分析人士认为，这表明执政党更加注重民意与民情。“以前，选举仅仅局限在党内，在组织的范围内公布。”中央党校一位资深党建专家说。
谢春涛认为，将公布名单与选举过程中的公示、纪检和
审计介入等措施联系起来，可以看出执政党力图引入公众参与，达到监督和制约代表候选人的目的。
“现在，全中国人民都知道了。”中央党校党史部教授陈雪薇说，“尤其是在党内的情况越来越复杂的情况下，应该增加选举的透明度。”
“怎么当上的”
58岁的陕西靖边“治沙劳模”牛玉琴当选十七大代表，亲戚远远地就问：“怎么当上的啊？”
牛玉琴是全国闻名的老劳模、多年的全国人大代表，拿过的荣誉不计其数。“选不上，我就回家继续干我的。”牛这样对南方周末记者说。
在与南方周末记者的两次通话中，她一说起自己的工作——11万亩荒沙、100多只羊，总是同样的话：“我现在太忙了。”但尽管很忙，她还是一路从靖边选到榆林，到西安，直到北京，当选了四级党代表。
一天，她被告知当上了县党代表；在西安开妇代会时，又增加了榆林市党代表资格——市党代会，她终于到了现场，惟一一张弃权票来自自己；到了西安，她一看选票，自己名字赫然在列，还是想都没想，没投自己的票，结果7票弃权，2票反对，她被高票通过。
党代表的候选人推荐提名采取“三上三下”的办法——由支部提名，再到基层党委、县级党委、市级党委，反复上下征求意见，最后形成候选人初步名单，提交给选举单位。“三上三下”的过程，张晓霞也只是隐隐约约地知道。她是北京奶牛中心种公牛站的技术总监。“上面下来好几次，找我的同事谈话，但是这些我没有参与。考察我，我不应该参与对吧？”
当列入初选人选后，这回市委组织部找了她本人。十多个同事被市委组织部约谈，“一个一个进去”，等到她最后一个进去的时候，已经是下午一点多，“领导们连饭都没顾上吃”。
像牛玉琴和张晓霞这样的一线党代表，据中组部公布的数称，占到了代表总数的28.4％。而从十四大以来的三届党代会，这一比例分别为22％、24.1％、24.3％。
官民比例是近几届党代会关注的焦点，这样的情况在人大选举中同样存在——比如大多数部长和地方党政一把手，同时具有全国人大代表身份。
中央对此作了硬性规定——除中央直属机关和中央国家机关维持不变外，其他选举单位中生产和工作一线代表的比例，一般比十六大时相应提高5个百分点。
他们首先要在提名阶段得到保证。在推荐提名以及考察之后，选举的第三个步骤，即选举单位党委(工委)常委会确定初步人选名单——票数不会是惟一的指标，他们还有一个重要职责，是保证候选人结构的优化。
一线党员代表和“双新组织”(新经济组织和新社会组织)代表的增多，正得益于此。中央要求代表“不仅要具有先进性，而且要同时具有广泛的代表性”。重庆市给了“双新组织”两个名额，由重庆市律师协会会长孙发荣和一个民营企业家获得。
确定预备人选后，最终的决定权是党代会的投票。十七大代表选举的差额为15％，比十六大多了5个百分点。竞争的结果，孙发荣高票当选，而另一人则落选。
也有代表在顺利当选后又被淘汰。本报记者发现，6月1日当选十七大党代表的原中石化董事长陈同海，并没有出现在2217人大名单中。中石化6月22日发布公告，陈同海因严重违纪辞去董事和董事长职务。
选谁不选谁
在安徽省的党代会，红花山集团董事长俞乃平在最后环节被“差额掉”。通知下达时，俞乃平感到很“遗憾”。
从公司支部推荐，以及长时间的“三上三下”，他并不抱有希望。但组织部下来考察，他内心燃起了希望。他和
奇瑞老总尹同耀，是惟一进入正式候选名单的芜湖企业家。
“一旦当上党代表，就能见到市委书记、省委书记，就能反映自己的意见和要求。作为县人大代表，顶多就是和县领导和工商联反映。”他认为，民营企业需要与国有企业平等的机会，包括贷款、立项等等。“重视和支持是不一样的，我们需要切切实实的支持。”
但奇瑞老总尹同耀顺利当选，俞乃平却没被选上。他将此归结为“知名度还不够，缺少宣传”。而且，“党代表的选举是自己争取不了的”。
张晓霞清楚地记得，组织谈话时问的两个问题。“一个是党代表的责任是什么。另一个是怎么参加好党代会。我说，履行职责，认真把大家的心声反映到中央去。”
中央党校党建部主任王长江认为，党代表不只是个荣誉。“党代表首先要有责任感，其次，要有能力，不能只是感谢党感谢组织，必须知道普通党员的心声，并且能够将其反映上去。”
“我不识字，开会不会记笔记，只能记脑子里，上北京开会要讲话的，我说不了什么，”牛玉琴解释自己为何不选自己，“但是，我会实话实说。”
牛玉琴同时也是十届全国人大代表。同样在人民大会堂举行，五年一次的全国党代会，是中国最重要的政治活动。
中国共产党作为中国惟一的执政党，“党的意志”通过人民代表大会，最后上升为“国家意志”，决定国家的发展方向。2002年，中共十六大提出“以党内民主促进人民民主”。
“如果社会民主发展了，而党内民主滞后，那党就落后了，就会很被动。”王长江认为，党内民主搞不好，可能导致决策失误。而民主的决策过程，需要“充分的讨论”和“不同意见的表达”。
代表做什么
“党代会应该成为真正的决策机构。”王长江说。自1988年起试点的党代会常任制，在他看来是一个抓手。
“目前改革已经进入深层次阶段，到底是往前推进，还是放缓，甚至后退，怎么办？这是必须面对的问题。”
全国分成38个代表团——除了31个省、自治区、直辖市代表团外，还包括中直机关代表团、中央国家机关代表团、中央企业系统代表团、中央金融系统代表团、解放军代表团、武警部队代表团和台湾省籍党员代表团。
经过资格审查的党代表们，将在金秋季节走进人民大会堂。他们手中的权力包括：听取中央委员会和中纪委报告，选举包括中共中央总书记在内的新一届中央委员会和中央纪律检查委员会，讨论党内重大问题，以及修改党章等。
按照惯例，具体的选举办法将在会议期间公布，再由大会主席团通过。是否扩大差额比例，甚至将在中央委员以上级别的选举中实行差额选举，是外界关注的焦点。
1987年十三大修改党章，规定“党内选举必须实行差额选举”。在十三大，代表的20％，中央委员5％，候补中央委员的12％实行差额选举，而省一级，则实现了常委差额选举。一位中央书记处书记因接连落选中央委员和中顾委常委，而流下眼泪。
十四大之后，中央委员选举引入预选——由各个代表团先进行预选，选出正式候选人，再进行等额选举。
同样受关注的是，十七大是否在党章中写入中共的创新理论。历次全国党代会均会修改党章，十五大写入“邓小平理论”，十六大写入“
三个代表重要思想”。
参照十四大以来的惯例，十七大可能在国庆节后举行，会期大约7天。议程一般在大会前一天，预备会议后公布。
国外媒体普遍认为，胡锦涛发表的“6·25讲话”，已经为十七大定好基调。自十五大以后，每次党代会前，总书记在中央党校的讲话，总会令外界解读。
