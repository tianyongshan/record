# 以色列拯救被巴武装劫持士兵 谁来拯救巴以和平

以色列拯救被巴武装劫持士兵 谁来拯救巴以和平
http://www.sina.com.cn 2006年07月06日10:08 南方新闻网-南方周末
本报记者 李盛
吉拉德·沙利特，一名19岁的以色列大兵，他也许从未想到会以这样的方式“成名”。
6月25日，巴勒斯坦武装人员袭击了凯雷姆沙洛姆的以军哨所，绑架了沙利特。从那一
刻起，他成了巴勒斯坦武装组织的筹码；而为了拯救他，以色列调集海陆空三军全面突袭加沙。
奥尔默特的承诺
“带吉拉德回家”，整个以色列都在祈祷，不仅为了吉拉德，也同样为了拯救沙利特的亲人——在实行义务兵役制的以色列，几乎家家户户都有亲人在军中服役。以色列领导人无法漠视这样的民意，必须为这个士兵做些什么。
巴勒斯坦武装组织曾经提出过他们的要价，要求用吉拉德交换在以色列监狱内关押的所有巴勒斯坦女囚犯和未满18 岁的少年囚犯，但以色列总理奥尔默特态度强硬，“这不是谈判的问题，也不是讨价还价的问题”。
奥尔默特，没有他的几位前任沙龙、内塔尼亚胡、拉宾那样显赫的军功，此前甚至还被指责他处理安全事务方面“缺乏经验”。这一次，是他上任后与哈马斯的第一次较量，他不能有一丝软弱。他坚持推行撤出在加沙地带的全部犹太定居点和 4个位于约旦河西岸的定居点的政策，这使许多国民对他不满，而面对眼下局势时的强势或许还能缓解他遭受到的不满。
为了证明自己在安全事务上确有作为，在向巴武装组织发出了48小时的最后通牒失效后，奥尔默特在6月28日凌晨下令，开始向加沙地带发动代号为“夏雨”的军事行动，营救吉拉德。在巴以领导层双双更替之后，加沙地区大规模的直接军事冲突爆发了。
哈马斯怎么办
绑架发生当天，巴民族权力机构主席阿巴斯领导的巴勒斯坦民族解放运动(法塔赫)和哈马斯宣布，已就一份暗示承认以色列存在权的文件达成一致。而巴武装人员在此微妙时刻袭击以色列士兵，在阿巴斯的助手看来，是哈马斯的强硬派希望破坏这种共识。
哈马斯政府一直没受到西方承认，一上台就失去了经济援助，政府各派别间的合作也有不足，这让既想坚持武装斗争、又想拉拢民心的哈马斯面临尴尬处境。一旦在这次以军的打击中失去加沙地带的控制权，哈马斯政府的地位将岌岌可危。
哈马斯的选择是与阿巴斯握手言和。6月27日晚间，哈马斯匆忙与法塔赫达成共识，明显“软化”了拒绝承认以色列的立场。但以方对哈马斯的“历史性让步”并不买账，共识没能阻止以军出兵加沙的脚步。
以色列的军事行动日益扩大，在6月29日凌晨展开的搜捕行动中，以军逮捕了64名“哈马斯”内阁部长与议员。 7月2日，以军又把一枚导弹投在了巴勒斯坦总理哈尼亚的办公室。
面对以色列的“公开宣战”，哈尼亚表示巴方不会考虑用被抓走的以军士兵交换被捕的哈马斯官员。“谋杀和逮捕是以色列惯用的伎俩，我们不会害怕，也不会后退，”哈尼亚说，“以色列推翻巴勒斯坦伊斯兰抵抗运动哈马斯　组建的巴自治政府的企图不会成功。”
被牺牲的大多数
“夏雨”行动开始后，以色列空军炸毁了加沙中部连接南北方向的两座桥梁，海军则攻击了向加沙城以及南部汗尤尼斯地区的发电站，切断了电力供应。
在行动中，位于加沙地带北部的拜特哈嫩村庄遭受了数百枚炮弹的袭击，村民马斯里和他的7个孩子躲在被炸掉三面窗户的家里，靠着一盏煤气灯和所剩不多的水过活，炮火逼得他们无处可逃。
除了炮轰和空袭，以军战斗机制造的音爆更是让所有人恐惧，“儿童和婴儿是音爆最大的受害者，这是比恐怖袭击还恐怖的行为”，无辜的居民无力地抗议着。
自6年前巴勒斯坦爆发反对以色列占领的第二次武装大起义以来，已有4000多名巴勒斯坦人和1000多名以色列人在巴方自杀式袭击和以方军事报复的暴力循环中遭遇死亡。
而巴勒斯坦武装组织则威胁说，如果以方行动继续，将会绑架更多以军士兵，甚至故意杀死那些本可以活捉的人。
战争或是和平
7月4日，奥尔默特的强硬立场坚持到了巴勒斯坦武装组织最后通牒的最后一刻，巴武装组织宣布，以色列人质事件的谈判已经结束，不再接受任何方面的介入，以政府将为被劫持士兵的命运负责。
等待吉拉德的将是什么？等待巴以关系的又会是什么？现在的加沙地区，战争一触即发。脆弱的巴以关系面临的可能是一片杀气，也可能是一线生机。
如果哈马斯领导人能够劝说巴勒斯坦武装释放吉拉德，局势也许能够缓和。但面对以军猛烈的攻势，巴勒斯坦武装组织毫不示弱，吉拉德的生命随时处于危险之中。而以色列是否最终会答应巴勒斯坦武装组织的条件，皆大欢喜，也并非没有转机。
但如果吉拉德被杀，以色列势必在加沙加大对巴勒斯坦武装组织的打击，必然造成巴以局势的进一步恶化。巴以关系又将降至冰点，严酷现实与和平理想之间的距离，可能会更加遥远。 来源：
