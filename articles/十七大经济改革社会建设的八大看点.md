# 十七大经济改革社会建设的八大看点

十七大经济改革社会建设的八大看点
http://www.sina.com.cn 2007年10月18日10:36 南方新闻网-南方周末
十七大备受关注的原因之一是，党代会将为未来5年中国在经济、社会和政治领域的改革和发展描绘愿景，提示方向。
10月15日，胡锦涛总书记作了两个半小时的政治报告。报告对中国特色社会主义道路做了系统阐释，重申了“一个中心，两上基本点”的基本路线，并对贯彻落实科学发展观做出了具体部署。在一些至关重要的经济和民生领域，指出了下一步的改革方向。
★经济发展模式的转变：从转变“经济增长模式”改为转变“经济发展模式”，把“现在经济增长主要由依靠投资、出口拉动转向依靠消费、投资、出口协调拉动转变”，“由主要依靠第二产业带动向依靠第一、第二、第三产业协同带动转变”。国内消费提到第一的位置，意味着它对中国经济健康、自主发展的极端重要性。如果要增加国内消费占GDP比重，则意味着社会保障体系将进一步完善，从而把居民储蓄转化为消费。
★收入分配制度改革：首提要在以往强调效率的初次分配上引入公平。这意味着政府将以包括法律在内的多种方式提高低收入者收入，逐步提高扶贫标准和最低工资标准，建立企业职工工资正常增长机制和支付保障机制。
★医改：提出要坚持公共医疗卫生的“公益”性质，强化政府责任和投入，但同时鼓励社会参与，区分营利性和非营利性医疗机构。在医疗管理体制上实行政事分开、管办分开、医药分开。
★教改：坚持教育“公益”性质，加大财政对教育投入，规范教育收费，保障经济困难家庭、进城务工人员子女平等接受义务教育。鼓励社会力量兴办教育。
★住房改革：只提廉租住房制度，意味着政府将加快解决城市低收入家庭的住房困难问题。
★新农村建设：针对过去新农村建设基本由政府主导，且往往变异为新“村”建设，报告提出要 “发挥亿万农民建设新农村的主体作用”。同时强调政府的责任是为农村提供和城市“均等”的公共服务。
★区域经济发展：首提“城市群”概念。报告提出，要“以特大城市为依托，形成辐射作用大的城市群，培育新的经济增长极”。
★重大改革的部门协调：中央的一些重大改革举措往往消解在多头管理的行政管理体制上，在
能源、三农、医疗等众多改革领域，一直都有大部委的呼声。这次报告明确提出，“加大机构整合力度，探索实行职能有机统一的大部门体制，健全部门间协调配合机制”。
