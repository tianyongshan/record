# 当十年治污变成一场天大的笑话

当十年治污变成一场天大的笑话
http://www.sina.com.cn 2007年08月09日15:41 南方周末
记者见地
□黄小伟(南方周末记者)
无锡市民对数月前那场水危机多少仍心有余悸。不过，在太湖治污十多年后，政府似乎终于想明白了，如果不解决污染源的问题，那太湖治污就只能是一个天大的笑话。
因为，这个笑话已持续十多年了，直到最近，中国最值得骄傲的环太湖经济高速增长带被肆虐的蓝绿藻染成了“蓝绿色”。
然而，此次治污风暴仍可能是这个笑话的延续。过去的治污措施，最厉害的无非关停，但十年治污最被诟病之处恰恰就是基层政府管理不善———该关闭的没关，不该批的却上马了。想想看，只要GDP对官员仍然诱惑力十足，就难免有人继续睁一只眼、闭一只眼。
幸好，无锡对于经济增长模式有了新的思维，试图改变过去几十年的“资源依托型”和“投资驱动型”的经济增长模式。几年前，无锡开始将产业转移到制造业的两端——研发和现代服务业。这是制造业中利润最高的两头，也就是经济学上所说的“微笑曲线”的两端。在研发和现代服务业中，大多数企业都是环境友好型企业。他们对资源的消耗比传统产业要低许多。而且，产业向高端转移后，并不会损害地方经济的发展，相反地，在未来几年内，新的产业布局形成后，将带来更高的经济发展速度和良好的环境。
不过，随着初级加工企业开始从环太湖地区向苏北及内地转
移，也许若干年后，人们会遗憾地发现，太湖边上是没有污染了，人们开始污染太湖的水源地。这显然将会是一个更大的笑话。
按照国际经验，人均GDP在5000到10000美元的区间，经济发展和环境的关系会出现一个拐点，从反向关系变成和谐发展。现在无锡市的人均GDP已经超过7000美元，那么无锡究竟是准备迎接这个拐点，还是让笑话继续？
