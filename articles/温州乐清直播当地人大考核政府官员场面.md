# 温州乐清直播当地人大考核政府官员场面

温州乐清直播当地人大考核政府官员场面
http://www.sina.com.cn 2007年11月01日10:37 南方周末
主席台上是市委书记和人大常委会委员，市长和市民代表们坐在台下

“你为什么在违规建设达到50%以上时，才来人大轻描淡写地讲这个问题？如果责任倒查，我看徇私枉法、行政不作为的被告席上，你是第一个！”
一年三“考”副市长
“你把成绩罗列那么细干什么？你对问题的分析呢？我们要听的是这个！”9月27日上午，温州乐清的一名副市长刚埋头做完报告，当场遭人质疑。
而有个副市长说，“这是我工作没做到位”，现场就响起掌声……
这些场景发生在温州乐清市，并进行了电视和网络直播。这是当地人大谨慎发动的一次围绕“权力监督”的改革。
9月27日上午9时，包括这位副市长在内的7名乐清副市长依次“过堂”，在人大常委会委员面前“汇报工作”，这是乐清人大常委会对政府官员的“期中测验”。
乐清市人大常委会主任赵乐强称之为“广场政治”。
一些细节经过特别设计，包括会场的别样安排，人大方面提前两周将一些“别样安排”通知给述职的官员们。
除要求所有委员统一着装外，当地人大官员特别强调了会场格局的改变：
以往，人大与官员会在主席台上围成一团，而这次，政府官员则与人大面对面。主席台的背景是镶嵌有巨大国徽的红色天鹅绒帷幕，人大常委会全体委员和市委书记黄正强坐在主席台上。
台下分成三列，两边是市民代表和机关人员。中间一列是由包括市长潘孝政在内的行政团队，后面是局长们，副市长要在一个独立设置的发言席上演讲。
“广场上的气势出来了，不像以前团团坐，过于和气、没有区分。”赵乐强说。以往会场上的大红幅，这次被换成了两个大屏幕，副市长们在报告时，同步的视频会在背后和面前同时更新。
视频通过媒体向公众传播，“我们的想法就是通过与媒体互动，搭建一个平台，”赵乐强说，这个平台是面向公众、更开放的“广场”，“他们(副市长们)已经在广场上了，甚至面向全世界。”
述职的副市长被安排站在主席台和听众席中间的发言席上，汇报时间也被限定在15到20分钟。
“我们如履薄冰。”赵乐强在会场当着政府官员的面表白。他温和地告诫
人大代表们，人大监督要适度，“要倍加小心”。
有人此时把直播镜头切到了市长潘孝政面前。画面里，沉思中的潘面无表情，拿着笔的手悬在空中，许久，才慢慢落下。
“广场上的政治”
与以往的另一个区别是，这次会议还邀请了四十多个市民代表旁听，并请他们在副市长发言后就副市长们的工作，加入分组讨论。
市民代表、乐清雁荡镇村民戚朝洪回忆，在分组讨论时，多名代表在乐清的
房价虚高问题上发言尖锐。
一名市民代表就乐清楼价直接指责政府在房产市场的监管严重失职，“乐清现在哪块地还在种菜？房子已经卖完了。一套商品房要卖一百多万元，老百姓干一辈子也买不起，房价虚高，社会怎么和谐？”
另一名代表也指责存在房产市场有“猫腻”，“个别干部漠视、纵容违法土地买卖。很多违法案件，都有干部的影子”，以致“违法当事人经济和谐、生意兴隆”。
还有市民代表批评某副市长，“你为什么在违规建设达到50%以上时，才来人大轻描淡写地讲这个问题？如果责任倒查，我看徇私枉法、行政不作为的被告席上，你是第一个！”
也有代表指责说，1.6万多平方米的违法建筑长期存在，与政府拆违经常敷衍了事有关。“敲几个洞，拆违极不平衡，流于形式，是在作秀！”好不容易拆除，却又不清理建筑垃圾，弄得“我们乐清像刚经历过一场地震”。
市民代表发表意见时，人大各工委被要求安排专人在每个分会场做记录，听取与自己有关的市民代表意见，并将此吸收进第二天早上的反馈意见中。“这也是我们设计的一个细节。”赵乐强说，“本来的安排是当天下午反馈意见。我们留出了一个晚上，保证反馈质量。”
赵乐强说，这首先是满足了公众的知情权。同时人大也能更广泛吸取民意，“我们对可解决的重点问题反馈，将保证跟踪监督、跟踪调研。”
吸取市民意见，也许能让委员们更有责任感。第二天，乐清市人大常委会举行现场反馈意见会，各工委主任对官员们的批评异常激烈。“我们有理有据，不由他们不服。”一名工委主任说。
受到市民代表发言情绪感染的法工委主任黄银多，在做治安方面的反馈意见时猛烈抨击乐清糟糕的治安。在电视画面里，黄不时挥动右手，神采飞扬，“说着说着，嗓门就大了起来。”
但作为乐清市的“一把手”，市委书记黄正强仍肯定了人大的监督工作，“人大的调研深入、掌握情况准确、选点比较准，比较客观、比较准确”，应该坚持。
陈峰：人大要有朝气不能有暮气
赵乐强：我们本就该代表人民监督政府
黄正强：永远在原地踏步怎么完善人大制度

12月底副市长们还将接受期末考试
http://www.sina.com.cn 2007年11月01日10:37 南方周末
内在需求
这已是副市长们第二次被考了，今年5月10日上午，他们向乐清市人大常委会汇报了各自分管工作的全年工作思路和工作承诺。根据人大安排，此次为“期中测验”，12月底或明年初，副市长们还将接受“期末考试”。
之所以让黄正强评价为“比较客观、比较准确”，是因为人大要求委员们对官员们的施政进行“独立调查”。
“二考”副市长前，市人大常委会作了安排：常委会的5个工作委员会委员和人大代表一起，用一到两个月时间，对各自对应的工作摸底后，再回到人大的舞台上，和官员们直接“碰撞”。
教科文卫体工委主任吴安武证实，一个月内，他几乎每个工作日都在调研。除与校长、老师和家长开座谈会外，还多次入户，进行个别交流。有时要加班到晚上一点，“人大加班，这是以往从没有过的事。”
为保证调研的真实性，调研时大多要求政府官员回避，教科文卫体工委副主任高娥微回忆，比如，调研社会治安，派出所所长不再被邀请。
翁垟镇人大主席陈峰参加了乐清市人大法工委组织的一次调查，“一开始，派出所的人也在，市民怕得罪派出所，不敢说什么。后来法工委的人说，你们回避一下。结果派出所人一走，警察的不作为、派出所的一些内部问题就被反映出来，社会治安等问题也出来了。群情汹涌。”
这样的安排看起来效果不错。赵乐强说，过去人大讨论，由于不了解情况，往往都顺着政府思路发言，难有不同意见，“现在不一样了。”录像画面里频频有人举起手要求发言，而当天代表的到会率，也创新记录。
为了给官员们“洗脑”，在“期中考试”前，乐清市人大请来了中国政法大学教授蔡定剑给包括四套班子在内的近千名干部讲解监督法。黄正强承认，讲课的目的，就是为减轻改革阻力，“上下形成共识”。
而赵也承认，“上课的象征意义大于实际意义”，最大的支持还来自市委书记黄正强，党委的态度对变革成败的意义不言而喻。
但赵乐强同时认为，此次改革更多出于各方面内在需求。“我只是找到了一个平衡点，迎合了他们。”
黄正强承认，这样的需求确实存在。“当书记的，文山会海、迎来送往，能有多少精力去调研，能了解多少事情？”
而人大的工作，提供了不同于政府的了解渠道，有利把握政府的真实状况。因为政府说的时候，“照例都是说好的多。现在也知道我们政府工作哪些是有成绩的，哪些是做的不够的，清清楚楚”。
作为最直接的受监督者，常务副市长王军对人大的做法颇为坦然。
他认为，副市长们的施政承诺，首先有助于政府改善形象。“现在最不好的，就是群众对政府的公信力缺乏信心”。而副市长们年初在“广场”上提出设想，就等于对公众承诺，只能迫使自己努力兑现。
王军认为，把政府推到“广场”上接受市民审视，至少有两个现实意义：一是为政府搭建民意沟通的桥梁；其次，也能防止政府现阶段由于行政体制原因导致的懈怠。
另外，由于缺乏有效的激励手段，在现有体制下，一大批优秀干部一到45岁，升迁无门，就陷入普遍的懈怠。人大监督的压力可以转化为政府工作的动力，能提高行政
执行力。
“我觉得这样的监督要延伸它的覆盖面。”王军说，市政府是一个整体，分管副市长的工作往往是协调、决策、作部署，“执行更多是在局这一层。如果在这个层面上执行不力的话，我们再急也没有用。”
另外，他主动欢迎“广场”的经常化，“今后能不能因事因时地搭建平台，在重大事项上，让全市的市民都参与进来。这可以弥补政府现行运行机制的缺陷。”
赵乐强说，明年三月的代表大会上，人大将以决议的方式让这项制度明确，“我们希望最终能让它成为一项制度。”
“对这个，我是很自信的。”赵乐强说，“从成本核算来说，这也是最经济的改革。各方面都满意，只有快感，很少阵痛。为什么不做呢？”作者:南方周末记者 吕明合 发自温州
