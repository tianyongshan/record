# 网民抵制家乐福风波全记录(组图)

网民抵制家乐福风波全记录(组图)
http://www.sina.com.cn 2008年04月24日10:01 人民网
4月22日，广州家乐福门口，虽然像平日一样平静，但有保安阻止记者拍照　本报记者 王轶庶/摄

4月19日，武汉街头的抗议群众

发动——“你就是最爱国的中国人”
家乐福声明
2008年4月，通过网络和手机短信，“转发令”重现江湖。“转发20个，你就是最爱国的中国人！”
“最爱国”的中国人传递的是这样一条信息——
“奥运圣火不断受到骚扰，尤其在巴黎。……让全世界看看中国人团结的力量！5月1日,让全国的家乐福冷场！”
4月13日，“水木清华”网友Kittyshelley率先在北京打响“战斗”——带着国旗和展板来到白石桥家乐福门前讲解，“给法国点颜色看看。”
在中国的法国品牌很多，承担全球火炬传递的运载工具都是法国的空中客车，但家乐福不幸地成为“众矢之的”。
“其后台老板路易威登-莫特轩尼诗集团(LVMH)曾多次资助DL集团，支持其分裂中国的罪恶行径。”网络上有这样的指责，接着就号召，“抵制法国货，从家乐福开始！”
借着手机短信和网络，对于家乐福的指责迅速流传。
一份“逛家乐福最新程序”新鲜出炉了：“最大的车，装满冰激凌、雪糕、速冻食品，一小时后走到热的角落，车一扔走人”；“去鱼市，选上等的鱼买一车，让其给切好，最后找个地方一扔，鱼全死全坏”；“拿针将包装食品全部扎孔，放气”……
这样的“程序”并未得到广泛采用。参与抵制家乐福的广东爱国者志愿网网友白原(化名)表示坚决反对，“不允许这样的人存在……无谓的有破坏性的行为我们不认同，不能让他们破坏我们队伍的形象。”
在抵制家乐福的信息广泛流传时，4月16日，以MSN等聊天软件签名的形式，互联网上的“红心”再起。MSN联系人名单上迅速“红心闪闪放光彩”。
此时，在伦敦、巴黎和旧金山等地奥运火炬遭遇干扰之后，4月15日，美国有线电视新闻网(CNN)的评论员卡弗蒂又在节目中爆出负面评价中国政府的言论。
从“反CNN”网站、“抵制家乐福”到“红心中国”，以及留学生和华人华侨的集会示威，中国民众的情绪迅速发酵。据统计，MSN的“红心中国”在一天内超过了200万，在4月20日达到700万。
千龙网就此发表评论《MSN上一片红 感受跳动中国心》，称“一颗红心就是一颗民心”，“红心在跳动，向世人展示着中国人的团结”。
“红心签名”的发起者之一、我赛网CEO陈怀远说，“这简直就是一个奇迹，它创造了一个历史”。
抵制——“心里挺热血沸腾的”
红心与抵制家乐福的信息在网络重叠出现。民众的情绪逐步升高。
巨幅五星红旗堵住家乐福的两入口，抗议者向每一个出入大门的顾客表示鄙视。
4月17日，西安；青岛；4月18日、19日，合肥；4月16日-18日，昆明；4月20日，大连……全国各地，抵制家乐福的抗议、集会和示威此起彼伏。
4月19日，人群堵住了武汉汉口的家乐福，家乐福一度关门谢客。第二天，游行队伍出现在鲁巷家乐福店门前。
“事先也没有很好的安排，现场更没有组织者。大家都是通过网络传递、获得消息的。”同样参加了20日游行的大学生桑帅(化名)承认这次游行“非法”。这是她平生第一次看到游行，“一点心理准备都没有，很茫然”，是“好奇心和天生的爱国心”驱使着她加入了游行队伍。
“爱国心”还打动了一些商人，参加了东莞抵制活动的白原介绍，他们得到了一些厂商的支持，“一家印刷厂免费帮我们做……老板还跟着我们一起摇旗呐喊”。
“一开始是觉得这种活动很有意义，心里挺热血沸腾的……有很多人喊口号的时候哭了。”武汉大学三年级学生周望(化名)参加了19日汉口的游行，但“不像学生”的几个人演说，煽动大家去围攻法国领馆的时候，他就犹豫了，“突然觉得很没有意思”，“砸东西和藏独有什么区别？”
李向(化名)注意到，网络上鼓动去家乐福抗议的帖子没怎么被删去。汉口家乐福门前的游行过后，因为人数太多，公交车无力承载，警察“迅速调来”数辆公交车，将参与示威的学生运回武昌。
情绪燃烧下，不乏吸引眼球的标语口号，武汉的人群里亮出了“一切反动派都是纸老虎”的标语，法国国旗上甚至被画上了法西斯标志；在青岛，愤怒的人群在家乐福门前点燃了法国国旗。
法国驻华大使苏和(HervéLadsous)说，他们对于法国国旗受辱感到很遗憾。
4月19日，示威者也出现在北京的法国驻华大使馆门前，打出“法国佬闭嘴”等标语。不过，据苏和观察，示威人员可能只有6名，而警察却有几十名了。公安和武警筑起两排一字形人墙戒备，示威者在约20分钟后散去。“我们当然得保持警惕，中国政府也会帮助我们进行防卫，”苏和说，“其实我本人上周五(18日)在三里屯骑自行车转了一圈，没有感到不安全。”
十字路口——“卖国贼！你说什么！”
家乐福(Carrefour)在法文中的意思是“十字路口”。在“全球华人空前团结”的时刻，是否应当抵制西方，是否应该把矛头指向家乐福，中国的公众恰恰也处在选择的十字路口。
4月14日，新浪财经发起“争议是否抵制家乐福”的调查。截至4月16日晚9点，46万余人参与调查，其中40.8万人赞同抵制家乐福等法国品牌。几乎同样比例的网民表示，近一个月内不会去家乐福购物。
第二天的外交部记者会上，发言人姜瑜说，“最近一些中国民众表达了他们自己的意见和情绪，这些都是事出有因，法方应很好地深思和反思。”
与此同时，央视主持人白岩松、中国青年报图片总监贺延光、凤凰卫视主播闾丘露薇不约而同地发表博客，反对抵制家乐福，希望大家理智、宽容，不要伤及家乐福的中国员工、中国供货商利益。尖锐的批评随即如潮而至。“你发表过上述言论，还好意思去接那凝聚了13亿中华各族人民爱国情的圣火火炬吗？”三亚网友“泉涌风飘”质问白岩松。贺延光告诉南方周末记者，他的博客留言中大量的语言暴力更加证明自己之前的忧虑是完全正确的，“我甚至收到过死亡威胁”。
搜狐董事局主席张朝阳被网民称作“敢站出来说抵制法货的第一个公众人物”，他因此受到“狂顶”。他称“要让充满偏见的法国媒体与公众感到损失与疼痛，否则他们不会长记性的”。
火炬手金晶在巴黎遇袭，激起许多国人的愤怒，但她的表态颇令一些人失望：赞同网友高涨的爱国热情，但不赞成抵制。昨日的“英雄”旋即被攻击称为“脑残”、“汉奸”。
争议尚未尘埃落定，各地集会抗议已然四起。反对抵制者也在落实自己的观点：昆明工程师马瑞彬组织4名朋友，在家乐福门前打出“创建和谐，反对抵制”等标语，不想立即有一位老伯高喊：“打倒汉奸！”“你们抵制什么家乐福，家乐福里面卖的多半是中国商品……不理性。”反对抵制的朱先生话音未落就遭到围攻。“卖国贼！你说什么！”随后，朱先生遭众人推开，甚至被扔了瓶装矿泉水。
曾参与抵制日货的李向(化名)坚决反对抵制家乐福运动，他认为由于“舆论的不对称”，很多人只是看见网络上一边倒的舆论，看见“多数人”要求抵制家乐福，便去抵制家乐福。

埋单者：民众需要一个目标
http://www.sina.com.cn 2008年04月24日10:01 人民网
参加了抗议的白原坦言，“家乐福很不幸，但是没办法，它是法国企业”，因为“我们民众需要一个目标，在最近各种信息的交流和综合之中，民众自然而然地就形成了这样一种观点和态度”。
家乐福(Carrefour)，法国企业，全球第二大零售业集团。截至2007年9月，家乐福在中国大陆已有分店101家。
3月，中国网络上也曾有过抵制法国货、抵制家乐福、抵制赴法旅游。4月7日奥运火炬在巴黎传递之后，家乐福才很快被中国民众的“激情”推上了风口浪尖。
4月7日，巴黎火炬传递后，人们对法国生出了失望和愤怒，这些情绪随后被大量地指向了家乐福。“3000名警察，居然保护不了一个火炬手？”白原说。
法国驻华使馆新闻参赞贵永华(Oliver Guyonvarch)告诉南方周末记者，火炬在巴黎传递之前，记者无国界组织和藏独分子已在三十多个地点准备了袭击和破坏，甚至准备在埃菲尔铁塔或巴黎圣母院里悬挂标语和旗帜。警察疲于招架，有些顾此失彼。
4月7日，巴黎警方动用了近50辆警车、65辆警用摩托对火炬手形成三层防护。一大早，埃菲尔铁塔对面的桥就被封锁。
金晶是第三棒火炬手。据一位在现场的留学生描述，“那会圣火还没到，大部分警力在后面保护圣火，我们这边警力比较薄弱。”就在此时，有藏独分子冲上去。从某种意义上，藏独分子抢夺空火炬没有意义，却可以让记者拍到警方穷凶极恶的画面。
当天警方共逮捕了十多名藏独分子。在最后一站大学城附近时，警察早早拦在街口，看到疑似支持藏独的人就不让过，除非确信没有藏着雪山狮子旗。
但中国前驻法大使、外交学院院长吴建民的看法是，警察有失职，人们的愤怒可以理解，因为无论怎样，火炬手金晶受到了暴徒袭击。
至于巴黎市政厅的“藏独”标语，贵永华介绍，是一些极端的绿党议员偷偷藏在衣服里带进市政厅并伺机悬挂的。
火炬传递后，法国媒体的表现仿佛幸灾乐祸，《费加罗报》以“火炬在巴黎惨败”为头版头条标题。
4月9日，海外华人论坛纷纷出现号召抵制法国产品的帖子，网友们列出了法国品牌的清单。国内的天涯、猫扑、西祠胡同等论坛里，抵制也得到热烈响应。被抵制的产品包括LV皮包、Dior化妆品、标致汽车以及各种法国香水和法式大餐。甚至有网友调侃提出，要一并抵制“法式舌吻”。
4月10日上午9点，网友“萝雨宁馨”在天涯论坛发帖《爱我中华，抵制法货》，家乐福再次被明确列入抵制名单。
4月12日，有网络文章指责家乐福后台老板路易威登公司曾多次资助达赖集团。LV的光顾者太少，矛头逐步集中在家乐福身上。
自此，这个以“本土化”自诩的庞大企业在全国范围内迅速陷入了一场空前的抵制危机。
传言——最好踩死几个人
“法国政府准备拿出二千万美金，用于家乐福五一降价促销，……要让中国人在促销中挤破家乐福，最好踩死几个人。法国电视台也在积极做准备，拍摄中国人到家乐福疯狂购物”——这样的短信被转发到法国驻华使馆一名中国雇员的手机上。
苏和大使看了后“哭笑不得”，“这很荒唐……同时，我也有点难过”。
也有网友早就对这条传言发出质疑：如果法国政府出钱，为什么用美元而不是欧元？
4月17日、20日，法国LVMH集团董事长贝尔纳·阿尔诺、家乐福集团总裁迪朗终于亮相了，否认关于他们曾经资助达赖分裂集团的传闻。阿尔诺说自己从不介入政治和宗教事务，无论西藏或其他地方。迪朗也称家乐福集团从未直接或间接地向任何政治和宗教活动提供支持。
4月21日，“家乐福澄清资助达赖”的新闻上了中央电视台《新闻联播》，但有网友表示，这是家乐福“强大的”“公关之作”。
外交学院院长吴建民给予了阿尔诺很大的信任。在吴建民任中国驻法大使期间，他曾与阿尔诺有过多次交往。吴建民院长告诉南方周末，阿尔诺“不过问、不介入政治，西藏问题也是这样”，“相信这样一个经济上的奇才，不会因小失大”。
商务部在沉寂许久之后也终于表态。4月21日，一位未署名的负责人接受新华社采访说，“家乐福在华雇用4万多人，年销售额近300亿元人民币，所销产品95%由中国制造。”该负责人同时对家乐福反“藏独”、支持北京奥运会“表示欢迎”。
早在4月16日之前，家乐福就称已经和商务部进行紧急沟通。
修补——“对中法传统友好关系的珍惜和重视”
法国驻华大使苏和现在每天要工作12-14个小时，周末也一样。除了与中国政府、领导人保持不断的沟通与对话，使馆还在对社会工作者、企业和媒体做大量澄清工作。“其实我们透过媒体对广大的老百姓说话，目的是让他们懂得，我们是中国真诚的可信赖的朋友。”
自4月17日起，新华社和《人民日报》连续发表文章，号召人们要理智务实，“把自己的事情办好，就是最大的爱国”。新华社评论更是指出，应“百倍珍惜”这种爱国热情，但必须将其“纳入理性的轨道”，追求“又好又快发展”。
但白原觉得，“我们应该有这种热情，不应该把它收敛起来……该表达的时候就要表达”。
18日，作为胡锦涛主席的特别代表，中国前驻法大使赵进军与法国总统萨科齐会见。“这是在当前形势下，中方采取的一次重要外交行动，体现了我们对中法传统友好关系的珍惜和重视。”外交部发言人姜瑜指出。
21日上午，法国参议长蓬斯莱出现在上海，他向金晶转交了萨科齐的亲笔信函。法国前总理拉法兰等人也即将作为特使来访中国。
然而，对于中法关系将迅速转暖的判断可能太过乐观。在腾讯网的相关新闻留言中，大量网友认为萨科齐“亡羊补牢，为时已晚”，并对萨科齐未“道歉”表示不满。更有网友表示，誓将家乐福抵制到底。
就在法国参议长蓬斯莱看望金晶当天，巴黎市长、左翼社会党人德拉诺埃的提案经市议会投票通过，决定授予达赖“荣誉市民”称号。中国外交部发言人当即表示，“此举粗暴干涉中国内政，严重损害中法关系”。
这个决定并未得到法国政府和萨科齐的支持，法国政府甚至担心，这会令中法关系进一步倒退。
外交学院院长吴建民告诉南方周末，应当将巴黎市政府与法国国家政府的立场区别开来。法国大使苏和也称，“要是一个市政府采取行动，中央政府不同意也没办法。”
但并非所有的公众都愿意这样理解，白原就认为，“不能说只是巴黎支持，法国政府又不支持，巴黎给达赖荣誉市民的称号，跟政府肯定有关系……我们中国民众不可能相信”。
4月22日，中国新任驻法大使孔泉向法国总统萨科齐递交国书，萨科齐指出“法国永远是中国的友好国家”。一国大使到另一国赴任时，均携本国国家元首签署的国书委任状，而递交国书也被看作新大使正式到任的一个标志。
实际上，孔泉3月8日就已抵达巴黎。在此之前孔泉“迟迟”未获萨科齐接见的消息在中文论坛里受到高度关注，有人认为，这是法国在敏感时刻有意怠慢中国大使。
孔泉正式上任后，抵制家乐福事件如何发展还不得而知。但即便家乐福平安度过五一，这件事情也未必就此完结，除了“五一抵制家乐福”的号召，抵制还有一个17天的更长版本——“5月8日-24日，正好是北京奥运会的前三个月，所有人都不要去家乐福购物，……让他们看看中国人和中国网络的力量。请转发。”（作者:南方周末记者 张哲 实习生 刘昊 特约撰稿 司徒北辰） （来源：南方周末）
