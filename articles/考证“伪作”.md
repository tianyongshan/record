# 考证“伪作”

考证“伪作”
http://www.sina.com.cn  2012年04月19日16:57  南方新闻网
考证“伪作”
南方周末记者 曾鸣
南方周末特约撰稿 雍兴中
南方周末实习生 沈颖 张岚
发自成都
“李琢微博门”事件发生以后，其内容真伪引发热议。南方周末记者就“工作餐补贴”、“论文代笔”等问题向成都市计生委求证，其宣传处则以“皮之不存毛将焉附”作为回答，意即该微博既然是冒充之作，里面的内容也不值一驳。
在“李琢19961003”的126条微博当中，共涉及官员25人，其中最高级别为省部级官员，而惦记最多的则是成都市委的主要领导，曾7次提到，颇有微词。
打击对手是最大主题，“李琢19961003”28次提到要打击对手，其竞争对手一共有三人，其中政法处处长宋银邦是其主要对手，宣传教育处副处长王波和政法处副处长冉毅居次。三人中，王波因为有二奶且多次不按时上班，尤其被恨之入骨。
南方周末记者对微博中所涉的部分信息进行考据，以期对个中信息，进行甄别。
人浮于事的会议
这是126条微博中比例最高的部分，每个月都会出现。其中共有十条，媒体有相关报道，南方周末记者一一进行验证，除一条与事实有出入外，其余会务活动都在各级政府网站上得到了证实。
如2012年2月23日孟立联表示，“明天又去金堂开会”，而据成都市人口网2月24日题为《金堂县实施计生困难家庭“1+1看世界”项目》的报道，该天成都市计生委果然在金堂县开展了活动。
该活动针对困难家庭赠送了电视机，以“帮助其了解党和国家的方针政策，获取致富奔小康的相关信息和知识，丰富其精神文化生活，满足其日益增长的物质精神需求”。孟立联描述活动时，十分兴奋，称“又是吃又是拿，天天这样最好，免写材料”。
2012年3月28日，孟立联发博称“领导下乡就是扰民。明天付(傅)市长在高新西区参加活动，要去郫县看看，老子联系了一中午进行安排，现在还要加班写汇报材料”。而据成都市人口网3月29日报道，该日成都市副市长傅勇林果然在郫县参加了名为“生命绿岛-青春驿站”的活动。
2012年2月28日，孟立联发博称“学习十二五规划阵仗大，区县全部来了，几百人。开车，吃饭，租会议室，讲课费，好大一笔开宵(销)。人事处搞的莫名堂的事”。
该日，成都市人口网题为《市人口计生委组织召开人口发展“十二五”规划专题报告会》即披露称“400余人参加报告会”。
2012年3月23日，孟立联称“把老子累惨了。星期一去温江，还有一摊协调事。苦啊，苦”。3天后，据成都市人口网报道，成都市计生委果然在温江区开展了“为青少女提供生殖健康教育服务”的活动。
微博中唯一一条不准确的会务信息写于2012年2月9日，孟立联称“广东要来参观，有什么可交流的，就他妈公款旅游，公款吃喝，烦死了”。事实上，根据第二天成都市人民政府题为《市政协港澳委员视察安仁古镇》的报道，这天来参观的委员来自港澳而不是广东。
微博举报？
网友们戏称孟立联是在“微博举报”。他历时半年的微博中，最吸引眼球的部分，主要包括贪污、送礼、生活作风问题三个内容。
如2012年3月5日称，“老婆说，他们局长郭小鸣贪了至少上千万。狗日的，要这么多钱做什么。”
根据刑法第三百八十三条，“个人贪污数额在十万元以上的，处十年以上有期徒刑或者无期徒刑，可以并处没收财产；情节特别严重的，处死刑，并处没收财产。”
2012年1月8日及1月17日，分别发微博称“今天把领导一网打尽，省市计生委的全部网完，花了不少钱。”“几个区县才送这点，还没有老子在巴中一个县送的多。”
“李琢原来在巴中是副主任，很低调也很有才华。”计生委宣教处副处长王波告诉南方周末记者。尽管无法核实“巴中送礼”这一事实，但一个侧面印证的信息是，李琢的确有过巴中任职的经历。
这两条微博暗指送礼，按照《中国共产党纪律处分条例》第七十四条规定，“党和国家工作人员或者其他从事公务的人员，接受可能影响公正执行公务的礼品馈赠，不登记交公，情节较轻的，给予警告或者严重警告处分；情节较重的，给予撤销党内职务或者留党察看处分；情节严重的，给予开除党籍处分。”
生活作风问题则涉及多人，如：
2012年1月29日称“王三皮，带起二奶到处跑，还请假不上班。狗日的，你还是处长哦。有人告你，看你狗日的咋收场”。
2012年2月20日发帖称“新津县委书记巫敏包二奶：上周五晚上九点五十，巫敏怀拥二奶在羊市街旁若无人激吻”。
2012年2月10日称“鼓动老板做工作把何军弄回来当一把手，这是一个好主意：一是何军是靠男人上来的，能力一般；二是以前相处不错；三是老板这样卖力为她，她应知恩”。
3月29日，称“张艺表达了要我跟她的意思，要慎重，一个被赶出来的能力一般权欲又重的女人，付(傅)市长欣赏她？她哪来的把握会接替老板”。
4月16日，王波向南方周末记者明确表示，他没有包二奶。
吃喝、旅游的业务主旋律
微博中涉及公款吃喝旅游的并不多。孟立联虽然屡次提到喝高，却多半是和朋友，有一次是给副市长代写论文后喝醉；而因公旅行只有一次，即贵州遵义之旅，事实上，这一次孟立联并没有随同事同行。
2011年12月29日，称“昨晚陪检吃饭，喝多了，现还没醒”

    2012年3月2日下午2时，称“还有一个多小时出发去贵州，老子手里的事都忙不过来，真不想去”。
据南方周末记者了解，成都市计生委去贵州遵义市进行红色旅游确有其事。此事后来还遭人举报为“公款旅游”，成都市计生委不得不在系统内澄清，表示贵州之行的主题是红色教育，去的都是革命老区，时间是利用周末，金额没有超支。来源：南方周末
