# 霉事：我给爹妈想出了“离婚”的主意

霉事：我给爹妈想出了“离婚”的主意
http://www.sina.com.cn 2008年01月03日11:46 南方周末
作者:文显(沈阳)
再过几个月，我们家那个小区居委会的“领导班子”就换届了，随着日子一天天临近，爸妈开始为家里的那个“秘密”紧张起来。
我们家是特困家庭，我的爹妈都是没有工作的残疾人，由于行动不便，基本没什么自谋生路的能力，最难的是要供我念大学，家里平日大都靠政府发放的低保金度日，生活很是不易。由于我们家的实际情况，所以每月拿的低保金算是很高的，一家三口，每人每月两百多元，加在一起有八百多元。除了这些，母亲还在我读过书的那所小学附近摆了个地摊，每天能挣十块八块的菜钱，靠着这些，一家子的日子还算过得去。
可问题出在了2007年。年初的时候，城管加大了对占道经营的整治力度，母亲腿脚不好，没法和城管“打游击”，摆了快十年的小摊再也摆不下去了。好在母亲的单位在倒闭前为她交了一部分养老保险，家里又凑钱补齐了剩下的部分，到下半年，母亲到了退休的年龄，得到了养老保险金，每月八百多元。在副食品价格持续上涨的下半年里，这笔钱简直就是家里的“救命钱”。
可是这笔钱是无论如何不能让外人知道的，按规定：特困家庭成员的收入情况必须及时上报，如果特困家庭一旦有了固定收入，那么他们家的低保金就要扣除和那份固定收入相应的金额。也就是说，这笔养老保险金如果报上去，我们家每个月八百多元的低保金就会立刻变成差额后的几十元。父亲的单位黄了十多年了，养老保险有十多年没给他交，我大学里的饭菜价格涨得很厉害，母亲不能再出去摆摊了，这件事一旦“暴露”，对家里的打击可想而知。
还好居委会负责“低保”的李主任知道这件事后并没有“公事公办”，考虑到我们家的实际情况，她私下里为我们隐藏了这个“秘密”。可李主任明年很可能卸任，这个“秘密”还能隐藏多久呢？即使可以继续隐藏下去，整天背负这样一个“秘密”的爹妈，又要承受多少焦虑呢？
国庆节我放假回家，一家人聚在一起又谈到了这件事。“明年街道要的都是大学生，李主任怕是悬了。”说到这，父亲焦虑得一筹莫展。
“实在不行你们俩就办离婚手续，咱们家‘分家’，我和爸过。”我灵机一动，想出了一个自认为“聪明才智”的不是办法的办法。
“也是呵，要能这样，我和儿子还有低保金，可是……”父亲说到这沉默不语了。我知道，爹妈结婚二十多年了，这二十多年来，他们俩为了这个家，什么天大的困难都分担扛过了，如今同样为了这个家，他俩却要被迫离婚，这心中的辛酸和矛盾谁能知道呢？“唉，实在不行也只能钻这个空子了。”母亲说这句话时，我看见她的眼圈一下子红了。那天的晚饭，父亲破天荒地喝了好多闷酒。晚上熄灯后，我躺在床上翻来覆去睡不着，不知为何，我忽然为我的“聪明才智”感到可笑和可悲。
10月28日，这一天是爹妈结婚22周年纪念日。这一天，远在学校的我没有拨去一个问候的电话。因为我知道爹妈在为“离婚”的事而犯难。

“也是呵，要能这样，我和儿子还有低保金，可是……”父亲说到这沉默不语了。我知道，爹妈结婚二十多年了，这二十多年来，他们俩为了这个家，什么天大的困难都分担扛过了，如今同样为了这个家，他俩却要被迫离婚，这心中的辛酸和矛盾谁能知道呢？“唉，实在不行也只能钻这个空子了。”母亲说这句话时，我看见她的眼圈一下子红了。那天的晚饭，父亲破天荒地喝了好多闷酒。晚上熄灯后，我躺在床上翻来覆去睡不着，不知为何，我忽然为我的“聪明才智”感到可笑和可悲。

霉事：我给爹妈想出了“离婚”的主意
http://www.sina.com.cn 2008年01月03日11:46 南方周末
作者:文显(沈阳)
再过几个月，我们家那个小区居委会的“领导班子”就换届了，随着日子一天天临近，爸妈开始为家里的那个“秘密”紧张起来。
我们家是特困家庭，我的爹妈都是没有工作的残疾人，由于行动不便，基本没什么自谋生路的能力，最难的是要供我念大学，家里平日大都靠政府发放的低保金度日，生活很是不易。由于我们家的实际情况，所以每月拿的低保金算是很高的，一家三口，每人每月两百多元，加在一起有八百多元。除了这些，母亲还在我读过书的那所小学附近摆了个地摊，每天能挣十块八块的菜钱，靠着这些，一家子的日子还算过得去。
可问题出在了2007年。年初的时候，城管加大了对占道经营的整治力度，母亲腿脚不好，没法和城管“打游击”，摆了快十年的小摊再也摆不下去了。好在母亲的单位在倒闭前为她交了一部分养老保险，家里又凑钱补齐了剩下的部分，到下半年，母亲到了退休的年龄，得到了养老保险金，每月八百多元。在副食品价格持续上涨的下半年里，这笔钱简直就是家里的“救命钱”。
可是这笔钱是无论如何不能让外人知道的，按规定：特困家庭成员的收入情况必须及时上报，如果特困家庭一旦有了固定收入，那么他们家的低保金就要扣除和那份固定收入相应的金额。也就是说，这笔养老保险金如果报上去，我们家每个月八百多元的低保金就会立刻变成差额后的几十元。父亲的单位黄了十多年了，养老保险有十多年没给他交，我大学里的饭菜价格涨得很厉害，母亲不能再出去摆摊了，这件事一旦“暴露”，对家里的打击可想而知。
还好居委会负责“低保”的李主任知道这件事后并没有“公事公办”，考虑到我们家的实际情况，她私下里为我们隐藏了这个“秘密”。可李主任明年很可能卸任，这个“秘密”还能隐藏多久呢？即使可以继续隐藏下去，整天背负这样一个“秘密”的爹妈，又要承受多少焦虑呢？
国庆节我放假回家，一家人聚在一起又谈到了这件事。“明年街道要的都是大学生，李主任怕是悬了。”说到这，父亲焦虑得一筹莫展。
“实在不行你们俩就办离婚手续，咱们家‘分家’，我和爸过。”我灵机一动，想出了一个自认为“聪明才智”的不是办法的办法。
“也是呵，要能这样，我和儿子还有低保金，可是……”父亲说到这沉默不语了。我知道，爹妈结婚二十多年了，这二十多年来，他们俩为了这个家，什么天大的困难都分担扛过了，如今同样为了这个家，他俩却要被迫离婚，这心中的辛酸和矛盾谁能知道呢？“唉，实在不行也只能钻这个空子了。”母亲说这句话时，我看见她的眼圈一下子红了。那天的晚饭，父亲破天荒地喝了好多闷酒。晚上熄灯后，我躺在床上翻来覆去睡不着，不知为何，我忽然为我的“聪明才智”感到可笑和可悲。
10月28日，这一天是爹妈结婚22周年纪念日。这一天，远在学校的我没有拨去一个问候的电话。因为我知道爹妈在为“离婚”的事而犯难。

“也是呵，要能这样，我和儿子还有低保金，可是……”父亲说到这沉默不语了。我知道，爹妈结婚二十多年了，这二十多年来，他们俩为了这个家，什么天大的困难都分担扛过了，如今同样为了这个家，他俩却要被迫离婚，这心中的辛酸和矛盾谁能知道呢？“唉，实在不行也只能钻这个空子了。”母亲说这句话时，我看见她的眼圈一下子红了。那天的晚饭，父亲破天荒地喝了好多闷酒。晚上熄灯后，我躺在床上翻来覆去睡不着，不知为何，我忽然为我的“聪明才智”感到可笑和可悲。

可是这笔钱是无论如何不能让外人知道的，按规定：特困家庭成员的收入情况必须及时上报，如果特困家庭一旦有了固定收入，那么他们家的低保金就要扣除和那份固定收入相应的金额。也就是说，这笔养老保险金如果报上去，我们家每个月八百多元的低保金就会立刻变成差额后的几十元。父亲的单位黄了十多年了，养老保险有十多年没给他交，我大学里的饭菜价格涨得很厉害，母亲不能再出去摆摊了，这件事一旦“暴露”，对家里的打击可想而知。
还好居委会负责“低保”的李主任知道这件事后并没有“公事公办”，考虑到我们家的实际情况，她私下里为我们隐藏了这个“秘密”。可李主任明年很可能卸任，这个“秘密”还能隐藏多久呢？即使可以继续隐藏下去，整天背负这样一个“秘密”的爹妈，又要承受多少焦虑呢？
国庆节我放假回家，一家人聚在一起又谈到了这件事。“明年街道要的都是大学生，李主任怕是悬了。”说到这，父亲焦虑得一筹莫展。
“实在不行你们俩就办离婚手续，咱们家‘分家’，我和爸过。”我灵机一动，想出了一个自认为“聪明才智”的不是办法的办法。
“也是呵，要能这样，我和儿子还有低保金，可是……”父亲说到这沉默不语了。我知道，爹妈结婚二十多年了，这二十多年来，他们俩为了这个家，什么天大的困难都分担扛过了，如今同样为了这个家，他俩却要被迫离婚，这心中的辛酸和矛盾谁能知道呢？“唉，实在不行也只能钻这个空子了。”母亲说这句话时，我看见她的眼圈一下子红了。那天的晚饭，父亲破天荒地喝了好多闷酒。晚上熄灯后，我躺在床上翻来覆去睡不着，不知为何，我忽然为我的“聪明才智”感到可笑和可悲。
10月28日，这一天是爹妈结婚22周年纪念日。这一天，远在学校的我没有拨去一个问候的电话。因为我知道爹妈在为“离婚”的事而犯难。

遭遇多动症
http://www.sina.com.cn 2008年01月03日11:46 南方周末
作者:潇湘士(广州)
2007不是我的本命年，却依然小灾小难不断。
我寓居在一个单身的小租屋里，几乎每餐都是豆腐干拌鱼干加一大碗米饭。没有热水器，每天冲凉要提前半小时用小电棒烧水，晚上一个人躺在一张很大的硬板床上，借着隔壁传来的电视机里激烈的打斗声来消愁解闷，实在寂寞时便拖着凉鞋走到百米以外的小摊上买一份《南方周末》，有时到天河、越秀几家家教中心给几个学生补习，开始只是两个，现在渐渐多了。这种近乎暗无天日的生活维持将近一个月了。
十个月以前，我还在海珠一所高中领取春节的红包，带高一两个班的语文兼一个班的班主任，尽管工资不敢恭维，但社保每月代办，日子也过得还算安稳。只是办公室的一个湖北女人不太乐意让人清闲，小嘴巴像抹了蜜到处不得闲。有一次小测试，她硬是查出了一处“空改”的小题，便莫名地当着众人又是咆哮又是摔杯子，非要把我怎么样才好，本着好男不与女斗的男儿胸怀，我离开了那个学校。后来听说原先的级长下了，那个女的还志得意满地做了现在高二的老大。
离别海珠之前，我与几个弟兄相聚在学校旁的一家酒店里，正灌得有点面红耳赤之时，她竟也带来了几个学生到此庆功。我故意高声向兄弟嚷道：“只有先主动，才会不被动，才会不被人计算。”
发此豪言时，我刚与一家贵族学校签了约，之前还与中山的一个高中签了约，但我已无心去那里高就了。折腾劳累得筋疲力尽的我，终于在这个暑假去了增城的某中学，接手初三年级的两个班，兼落后班的班主任。孰料世事险恶，校方并没有把原先这个班的班主任休掉，而是让她躲在公寓里原地待命，休养一段时间，当我被整得焦头烂额时，她再底气十足、衣冠齐整的接替我，就这样我又成了另一个湖北女人手里一颗老实的棋子。补课结束后十天左右，我接到校方的一个电话，他居然对我说：“学校决定不再与你续约，你还是另谋高就吧！”电话这头的我真想狠狠地骂他一句！但我还是保持着我一贯的冷静与君子风度，责问他为什么校方不早点通知我。
后来有个同事告诉了我答案，学校没有招到新生，很多老师现在只能带一个班呢！我晕！就这样我几乎卷席而逃。不过这二十来天口袋里最后还是装了六千多，就算是校方给我的一笔不菲的安置费吧！
第二天我便在广州赤岗谋了一个学校，工资千把元，快接近最低生活保障线了，但我还是打算正正经经地给学生们上完这个学期的课，因为我觉得不管学校质量有多差，学生总还是孩子，为师一日，绝不误人子弟。
孰料，第三天东莞桥头镇某学校打电话来叫我去面试，此时我如拨云见日般，自然很快答应。但在去之前我当然也有所顾虑，还特地打电话核实了一下情况，谁知去了才发现，头天开学学生报名数就已满额了，可第二天居然没有招到学生，原本初一扩班的计划就此取消，幸好九年级有两个落后班在合班时搞分裂，大班没凑成，我就这样到九年级办公室上了班……
开学不久我觉得风向不对，师生相处好像找不到感觉，课堂成了我的一人堂。有天下午我上完课，发现惟独此班门窗禁闭，我就凑近门缝细看：不得了，居然学生们在教室里耍牌，男的女的挤成了一堆，恰好值日老师经过，我就轻声地提醒了他一下。后来这事居然闹到了德育室，他们班级还扣了分，学生知道细节后，竟然众志成城地把矛头对准了我，全班集体签名上诉，美其名曰语文老师上课思路不清，要求换老师。
不久，我又去了四会县某实验学校，这里丑话就摆在前头，工资要扣一个月，文凭与职称证书也押下，要走人可以，这些东西全留下。我听了有点胆寒。再加上该校不准师生外出，声称决不让一只苍蝇飞进来，当然也不会允许苍蝇飞出去，总之，很多的制度好像在鼓励多罚款，少手软。我于是卖个破绽又溜之大吉。
人称猪年百姓咋就吃不起猪肉呢？我在猪年遭遇了多动症，鼠年，我还不知啥样呢？

发此豪言时，我刚与一家贵族学校签了约，之前还与中山的一个高中签了约，但我已无心去那里高就了。折腾劳累得筋疲力尽的我，终于在这个暑假去了增城的某中学，接手初三年级的两个班，兼落后班的班主任。孰料世事险恶，校方并没有把原先这个班的班主任休掉，而是让她躲在公寓里原地待命，休养一段时间，当我被整得焦头烂额时，她再底气十足、衣冠齐整的接替我，就这样我又成了另一个湖北女人手里一颗老实的棋子。补课结束后十天左右，我接到校方的一个电话，他居然对我说：“学校决定不再与你续约，你还是另谋高就吧！”电话这头的我真想狠狠地骂他一句！但我还是保持着我一贯的冷静与君子风度，责问他为什么校方不早点通知我。
后来有个同事告诉了我答案，学校没有招到新生，很多老师现在只能带一个班呢！我晕！就这样我几乎卷席而逃。不过这二十来天口袋里最后还是装了六千多，就算是校方给我的一笔不菲的安置费吧！
第二天我便在广州赤岗谋了一个学校，工资千把元，快接近最低生活保障线了，但我还是打算正正经经地给学生们上完这个学期的课，因为我觉得不管学校质量有多差，学生总还是孩子，为师一日，绝不误人子弟。
孰料，第三天东莞桥头镇某学校打电话来叫我去面试，此时我如拨云见日般，自然很快答应。但在去之前我当然也有所顾虑，还特地打电话核实了一下情况，谁知去了才发现，头天开学学生报名数就已满额了，可第二天居然没有招到学生，原本初一扩班的计划就此取消，幸好九年级有两个落后班在合班时搞分裂，大班没凑成，我就这样到九年级办公室上了班……
开学不久我觉得风向不对，师生相处好像找不到感觉，课堂成了我的一人堂。有天下午我上完课，发现惟独此班门窗禁闭，我就凑近门缝细看：不得了，居然学生们在教室里耍牌，男的女的挤成了一堆，恰好值日老师经过，我就轻声地提醒了他一下。后来这事居然闹到了德育室，他们班级还扣了分，学生知道细节后，竟然众志成城地把矛头对准了我，全班集体签名上诉，美其名曰语文老师上课思路不清，要求换老师。
不久，我又去了四会县某实验学校，这里丑话就摆在前头，工资要扣一个月，文凭与职称证书也押下，要走人可以，这些东西全留下。我听了有点胆寒。再加上该校不准师生外出，声称决不让一只苍蝇飞进来，当然也不会允许苍蝇飞出去，总之，很多的制度好像在鼓励多罚款，少手软。我于是卖个破绽又溜之大吉。
人称猪年百姓咋就吃不起猪肉呢？我在猪年遭遇了多动症，鼠年，我还不知啥样呢？

40年友情等于10万元钱?
http://www.sina.com.cn 2008年01月03日11:46 南方周末
作者:闻秀兰(安徽)
李阿姨和母亲是大学同学，又是工作单位里的同事，她们的友谊经历了40年的风风雨雨，尤其退休以后，她们俩更是一起晨练，一起逛街，情同姐妹。
2006年春节过后，李阿姨突然忙碌起来，甚至连晨练也顾不上了。听母亲说她在一家合资企业当上了办公室主任。5月的一天，李阿姨给母亲打来电话，说她所在的那家企业因为要上新项目，暂时资金周转困难，要向母亲借3万元钱，两个月后一定归还。
老朋友张口借钱，母亲自然毫不犹豫地从存折里取了3万元钱借给李阿姨。事后我们都责怪母亲太轻信，母亲却说：“我和你李阿姨交往40年了，她做事有把握，我信任她。”
果然，两个月后，3万元钱还了回来，甚至还拿到了利息，母亲用利息钱请李阿姨吃了顿饭，末了还数落我们：“你们让我提防上当受骗没有错，但是我和你李阿姨是什么关系呀，人和人之间不该只有防范，更要有信任！”
李阿姨依然忙碌，有时我问母亲：“李阿姨所在的企业到底在做什么？厂房、机器、产品都在什么地方？”
母亲也疑惑地说：“听你李阿姨说，好像生产一种销到国外去的药品，工厂在远郊，具体什么地方你李阿姨也没有和我说过。”
2007年盛夏的一天，无意中听见母亲和李阿姨在电话里说什么10万元钱的事情，一再追问，母亲才说出事情的缘由：上年9月份，李阿姨又说企业资金周转困难，张口向母亲借10万元钱，虽然母亲犹豫再三，但是看到李阿姨焦急万分并且信誓旦旦说3个月一定归还的时候，母亲还是几乎取光自己所有的积蓄，凑齐了10万元钱，交给了和李阿姨一起来的那个据称是财务科长的人，那人当即出具了一张借条，还盖上了红彤彤的公章。
事后，母亲也一直没有对我们提及此事。可是，直到2007年5月份，母亲也不见李阿姨提还钱的事，于是，焦急的母亲开始隔三差五地打电话催要钱。而每次，李阿姨总是说要母亲耐心地等等。结果，就这样一直拖到了7月。看到母亲的焦急和不安，我们也不忍心再多说什么。
就在这时，李阿姨不幸被确诊为食道癌晚期，母亲得知后第一时间赶到医院探望，看着李阿姨憔悴的面容，母亲哪里还忍心提还钱的事，只是一味地替李阿姨的病情担忧。
背着母亲，我们对李阿姨所谓的合资企业进行了调查，顿时被惊呆了！这是一家无厂房、无产品的皮包公司，全部资产就是租来的一座办公楼。
当我们把这些情况告诉母亲时，母亲半天没有说话，过了十多分钟，她艰难地说：“我还是不信你李阿姨会骗我。”随即她拨通了李阿姨的电话询问此事，谁知李阿姨立刻就翻了脸，说10万元钱是母亲亲手交给企业财务人员的，又有企业的公章，这事与她无关……
放下电话的母亲嘴唇青紫，身体微微地颤抖，我们愤怒了！要知道这10万元钱是母亲工作一辈子的辛苦钱，养老钱啊！于是我们说：“妈，事已至此，也别难过了，我们现在就报案去！”
泪水顺着母亲布满皱纹的脸流着，她哽咽道：“唉，算了，你李阿姨没有多少日子了，我不想看见她背负着官司离开人世，毕竟40年的情谊啊……只是我怎么也不明白，难道40年的友情就值10万元钱吗？！”母亲说她无论如何也想不明白。
