# 体制优势究竟有多大

体制优势究竟有多大
http://www.sina.com.cn 2007年08月09日15:16 南方周末
体制优势究竟有多大
北京奥组委秘书行政部部长韩子荣：“这种组织优势是任何国家没法比的”。
□本报记者 马昌博 实习生 马会锋 发自北京
“代表国务院在这坐阵”
南方周末：北京奥组委的大量工作是否要和国务院相关部委联络？
韩子荣：有93个国务院部委办是奥组委的委员单位，北京奥组委主要以北京市委市政府为依托－－因为这是北京的奥运会，北京市委市政府是主要领导部门，但很多事情还需要由中央单位牵头。比如奥运会新闻宣传工作协调小组组长就是中宣部副部长，安保工作由公安部挂帅，外事工作由国务委员唐家璇和外交部牵头。应该说，中央国家机关同样是我们的依托。
南方周末：和中央机构协调，有没有困难？
韩子荣：各个部门现在都把奥运作为第一要务。我们搞一次活动，需要请哪些委员，很少有不来的，很多事情只要一重视事就好办，各机构都给我们让路。北京奥组委虽是一个临时机构，但我们经常能和部长们面对面地谈具体的事。
我们租奥运大厦时，
审计署正在审计大厦主管部门，我们一个部长问这个大楼能不能租，副审计长亲自出来接待说，肯定没问题。如果涉及到外交，国务委员唐家璇也经常过来开会。几乎所有的活动和会，国务委员陈至立都会来，这代表国务院在这坐阵。
南方周末：和北京各个机构怎么协调？
韩子荣：我们这个部门是综合协调部门，包括中办、国办、市委市政府办公厅都跟我们像一家人，我们之间都有联络机制。他们的态度是，奥组委的事，快给办。讨论我们的问题，需要上会讨论的，没有挤不上去的会，都给我们让路。比如有的事情必须由刘淇书记决定，他经常把其他会拿掉来保证我们。而我们的事情经常非常急，很多事情有时限要求，今天不报国际奥委会就违约了，我们就经常加会，支持力度就这么大。
南方周末：市里各机构是否都支持你们？
韩子荣：是的。比如为了改善奥运大厦周围的环境，北京市园林局主动把周边的砖铺好，然后把车站给挪了，再把绿化用的树换了。新的树都特别粗，都是北京市园林局专门调来的，他们甚至连屋顶也给绿化了，为这个，海淀区投了大概1000多万。
南方周末：他们跟北京奥组委要钱了吗？
韩子荣：没有，人家觉得奥组委来了，是自己的荣幸。只要你把要求提出来了，各个部门粉皮当成大事办。比如，停车位不够，有关部门就让我们在路的两侧停车，还把配套设施做好，这些工作，我们一分钱都没花，各部门都主动为我们服务，人家就怕不知道我们的需求，如果临时冒出来一件事，人家就急事急办。
“组织优势”的深度体现
南方周末：能否具体概括北京奥组委和北京市委市政府、国家体育总局、中国残联之间的关系？
韩子荣：它们本身都在奥组委里。比如说刘鹏局长，国家体育总局一把手是奥组委的副主席；中残联的邓朴方主席是我们这的副主席，王岐山市长是副主席。一些机构的一把手都是我们奥组委的领导，实际上我们都是一体的。这几家就是一体。
南方周末：作为一个临时性机构，却能得到来自中央和地方的绝对支持，你觉得原因是什么？
韩子荣：这种优势就是组织优势。一些国家办奥运，就是
一个公司来办，它要不断和各方去签约，才可能得到支持，甚至用利益去交换。但我们天然是一体，我觉得，这种组织优势是任何国家没法比的。这么多场馆，政府一声令下就完工了，都交付使用了。各方需求全有保障，这就是我们的组织优势，充分得到了体现。
南方周末：能否再具体谈谈你所说的“一体”？韩子荣：我们是一体的，奥组委是一个对外窗口，是政府跟国际对接的窗口，和政府应该说是一体化。比如政府有一个08工程办是建场馆的，我们这儿有一个工程部，我们的工程部部长是08工程办的副主任，他们是一家。08工程办盖场馆，我们这边提需求，怎么盖，告诉08办，这两家在一块儿办公。我们的部长多半时间要在那里办公。像我们安保部研究国际需求，然后北京市公安局去抓落实北京市公安局局长马振川本身就是安保小组的主要领导，而奥组委安保部部长刘绍武是是北京市公安局副局长，大家都一体化做工作。具体地说，北京奥组委研究需求，别的政府部门来落实。
南方周末：就是说大部分工作他们自己部门就协调了？
韩子荣：无非就是我们各个部门本质上是各政府部门入驻我们奥组委的一个办公室。因为我们毕竟是一个对外平台，国际奥委会不断提要求，这些要求在我们这儿转化成需求，交给政府各部门去落实。然后我们再把落实情况反馈给国际奥委会。
南方周末：中央领导提出要办廉洁的奥运，奥组委怎么保障这一点？
韩子荣：奥组委是一个管理很严格的单位，我们靠制度来规范，没有任何空隙可钻。比如需要买几方长床，就要先立项，立完项再给财务打报告，就纳入预算了。但我又没有权力直接买床，要给物流部下一个函帮我买床，就进入采购程序了，采购又要招标，按照这个程序走完了，这个床才能够拿到。东西到我这了，形成固定资产，固定资产最后由秘书行政部登记造册，然后再给需要的人使用。
如何协调外地赛区
南方周末：跟其他赛区协调是不是也要靠奥组委做工作？
韩子荣：对，主要是前期，我们跟其他城市约定好对方要履行的职责，我们共同制定出一个合作备忘录，然后北京奥组委有相应的财务补贴，其他花费就由所在城市承担。
南方周末：奥组委跟当地去协调，而不是北京市政府跟当地政府协调？
韩子荣：京外赛区的赛事组织机构和当地政府也是一体化的，像青岛奥帆委主席就是他们的市长夏耕，都直接对接的，这个体制就很顺了。同时北京奥组委和京外赛区保持着紧密的工作指导关系，像沈阳把五里河体育馆炸了，就是市长跟奥组委请示，说你们只要答应我们建新场馆，我们会建一个崭新的场馆投入使用。他们得到我们的认可后，回去就把场馆盖起来了。
南方周末：是否有地方政府不太乐意为北京奥运会服务？
韩子荣：没有，奥运会是一个重要的品牌。你看青岛、沈阳就是利用奥运契机，在硬件和软件上提升城市形象，他还可以凝聚人心，过去很多事形成不了统一思想，但因为有奥运，就有了载体，各方都能达成一致。
奥组委秘书行政部简介
秘书行政部是北京奥组委的一个办公机构，是奥组委日常运转的中枢协调机关。随着“好运北京”赛事的展开，指挥体系更加庞大，需要和北京市市委市政府全面无缝对接，要服从市委市政府的领导。同时要符合国际惯例，中国很多行政法律法规和国际要求有差距，但政府又做过很多承诺，经常要经过主管某方面的重要领导出面协调，研究奥运会时候的管理需求，出台一些临时办法或新规定。所以秘书行政部要大量协调国家机关，进行政策保障，通过很多协调会把事情一一落实下来。
