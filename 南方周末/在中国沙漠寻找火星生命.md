# 在中国沙漠寻找火星生命

在中国沙漠寻找火星生命
http://www.sina.com.cn 2007年07月19日15:46 南方周末
在中国沙漠寻找火星生命
中国的沙漠为科学家提供了与火星环境最为相似的实验室。科学家相信，假如生命能够在地球上这些最极端的环境中生存，那么人们也有理由期待在外星球上发现生物
□黄永明
满眼乱石的戈壁中，金百莉(Kimberley Warren－Rhodes)弯下腰，翻起一块直径数十厘米的石头，看到石头的底部有一层微绿的附着物——这正是她想要的。金百莉·瓦伦－罗兹博士来自美国宇航局艾姆斯研究中心。从1999年开始，她已经至少六次来到中国，在中国境内环境最为恶劣的沙漠中寻寻觅觅。
她这是在找什么呢？答案是“火星生命的线索”。乍听上去，这也许会显得有点荒唐，但实际上，中国的沙漠的确为世界上的科学家提供了与火星环境最为相似的实验室。科学家们已经去过了地球上最为寒冷的南极洲，也去过了地球上最为干燥的智利阿塔卡马沙漠，但他们真正需要的是将这两者结合起来的极端环境。
“索尔库里地区位置偏远，景色独特，置身其中，我想我已经感觉到了未来某一天宇航员们登陆火星和返回月球工作时的情境。”金百莉说。索尔库里沙漠位于青藏高原，是金百莉在中国开展研究工作的三个地点之一。全世界只有在这里，科学家才能够找到与火星最为接近的环境。
金百莉找到的那些石头上的绿色附着物，其实是一种生存在极端干旱环境下的蓝藻，它们能够在中国最严酷的沙漠中存活，生命力之顽强超乎想象。
“沙漠中巨大的昼夜温差、湿度差以及强烈的太阳辐射，对生命来说都是巨大的挑战。”金百莉的合作研究者、中科院微生物研究所刘双江研究员说。而蓝藻却在这里成功地生存了下来。
科学家相信，假如生命能够在地球上这些最极端的环境中生存，那么人们也有理由期待在外星球上发现生物。
生命的极限
近些年的科学研究表明，地球上生命的顽强程度达到了不可思议的地步。2003年，美国马萨诸塞州大学的研究人员在《科学》杂志上发表论文，报告在深达两千多米的海底发现了一种能生活在121摄氏度高温下的微生物，他们称之为“菌株121”。研究人员在实验室中发现，菌株121在85至121摄氏度的高温下均能生长。
由于海底的火山活动，海水有可能被加热到很高的温度；同时，在海洋强大的压力下，被加热的海水可能达到400摄氏度而不汽化。菌株121从2003年至今一直是记录保持者，尽管有人曾声称发现了能在230摄氏度环境下生存的微生物，但并未得到证实。
“自上世纪六七十年代布罗克(Thomas Brock)对极端微生物进行开创性的研究以来，人类在研究微生物的过程中对生命极限的理解一次又一次地获得了提高。”刘双江说。极端微生物指的就是那些生活在高温、高盐、高酸、高辐射等极端环境中的微生物。
对沙漠环境中的极端微生物的研究现在还处于起步阶段。“中国有大面积的沙漠和连续的沙漠环境，为研究沙漠中的极端微生物提供了不可多得的条件。”刘双江说。
1999年金百莉第一次来中国时，她对中国的沙漠首先进行了大范围的考察，以寻找最为干旱的地点。从喀什到巴基斯坦边境，从吐鲁番到哈密、花土沟、若羌，再到塔克拉玛干和乌鲁木齐，她的足迹踏遍新疆、青海和西藏，包括甘肃部分地区。
“到目前为止，中国西部沙漠的广袤、力量和美丽是我的经历中印象最为深刻的部分。”金百莉说。
科学家们一度认为，那些偶然附着在火星探测器上的地球微生物无法在火星上幸存，因为火星大气稀薄，表面接收到的紫外线的强度是地球表面的三倍。实验室中，在如此强的紫外线的照射下，一般微生物在短短几秒钟的时间里就会丧命。
然而，有科学家在2005年发现，有一种极端微生物的生命力如此之强大，以至于它们有可能搭探测器的便车到火星上去，并在一些特定条件下在那里存活下来。它就是某种蓝藻。这种蓝藻竟然能够在南极洲和以色列干燥的沙漠中活得很滋润。
研究人员发现，那种蓝藻能够在火星表面存活几分钟，在1厘米深的土壤中存活24小时。假如机缘巧合，它们实际上有可能会从地球移民火星。
“辐射主要会破坏微生物的DNA。”刘双江解释说，“科学家已经知道，某些微生物能够自动修复受辐射而断裂的DNA，而且修复后能够跟断裂前完全一样。”
“目前，科学家已经了解微生物何以在高温和低温环境下生存，但至于微生物是如何适应沙漠环境中的巨大温差的，人们还不得而知。”刘双江介绍说。他以前主要的研究兴趣是热泉中的极端微生物，为此没少在云南跑，而近年来他也开始对沙漠中的微生物感兴趣了。
线索的拼图
金百莉团队在中国选择了三个地点开展他们的研究。托克逊，中国海拔最低的地方，海拔约为－152米，那里又热又干；若羌，与塔克拉玛干沙漠的南缘平行，热但相对湿润；索尔库里，青藏高原上的一个高海拔沙漠，海拔在2500米到3000米之间，它包含了两种环境：冷而干燥，以及冷而湿润。
“有一种微生物群落还没有被很好地研究，就是生活在中国沙漠中那些半透明石头上的蓝藻。”金百莉说。所谓“半透明石头”，指的是石英岩这样的石块。强烈的阳光可以刺透这些石头，照射到生活在另外一面的蓝藻身上，为蓝藻进行光合作用提供了日光。
研究过程中，金百莉等人首先会在他们选择好的地点架设其气候探测设备，详细考察当地的气候条件，包括降水情况、日照强度和土壤相对湿度、温度等等。这些探测是一种长期工作，需要持续多年，以便获得长期数据。
接下来，研究人员在选定的范围内测出所有生活在岩石上的微生物群落的空间位置和特性。最后，研究人员还会把一些石头带回到实验室中，进一步研究微生物的基因差异、形态，以及这些微生物是如何制造糖分来抵御干旱的。“我们的研究结果显示出，水源的不同——比如降雨和融雪——决定了微生物如何在它所处的环境中生存。”金百莉的导师、美国宇航局艾姆斯研究中心首席科学家克里斯·麦克伊(Chris McKay)说。
比如，研究人员发现，在最干旱的沙漠里，微生物倾向于在较大的石块下生存，而不选择较小的石块。他们认为，微生物的这种生存策略为它们提供了最大的“湿度利益”，因为较大的石块能够聚集更多由地下蒸发上来的湿气，比小块的石头更能保持较低的温度，也更能长时间地保持水分。对于微生物来说，一个石块就像是一座大山，不同位置的“气候”是大为不同的。选择大石头为那些在多年无雨或一年只下一两次雨的沙漠中生活的微生物提供了帮助。
2004年，美国宇航局的火星探测器发现了火星曾存在液态海洋的强有力证据。科学家猜测，现在，火星上的那些水一部分已经蒸发，而另一部分可能渗入了火星的地下。假如情况真的如此，那么火星上也许会有一些顽强的微生物仍在地表之下生存着，等待着人们去发现。
金百莉说，她在研究中最为感兴趣的是，当环境所能提供的水量或温度达到生命极限时会发生什么。“理解这些条件如何在地球上对生物产生影响，不仅有助于我们在其他星球上——比如火星上——寻找生命，并且有助于我们理解生物在这些环境下的生存策略。”她说。
在三个研究地点的不同环境组合中，研究人员发现，冷而干的环境对生命存在着最大的冲击。在索尔库里又寒冷又干燥的地方，微生物的数量和种类是所有几个地点中最少的。“这个发现特别重要，因为在全世界范围内，只有在中国才能找到这种既冷又干的环境。”金百莉说，“这是模拟火星环境所必不可少的条件。”相比之下，南极洲虽然是世界上最寒冷的地方，但相对来说它最为干燥的地方还是过于湿润了。
火星的召唤
“非常漂亮的工作。”美国南加州大学细胞与神经生物学博士乔·米勒(JoeMiller)评价金百莉等人的研究，“假如能看看这一环境中是否存在产甲烷生物，将会是非常有趣的事情。”米勒没有参与金百莉等人的研究。
对于火星上存在极端微生物的想法，一个重要但一直处于争议之中的证据是，欧洲空间局的“火星快车”探测器在2004年探测到了火星大气中的甲烷。“的确，微生物的生命活动会释放出甲烷，但发现甲烷是否一定意味着有生命存在？”刘双江思考了一下说，“这我不敢肯定。”
是的，除了生命活动外，甲烷的另一种来源是火山爆发。科学家之所以把“火星快车”的发现看作一个可能的突破，是因为没有任何观测表明火星上存在火山活动，所以这一来源很可能是根本不存在的。
这是甲烷通常的两种来源，排除了其一，是否意味着剩下的一种便是惟一可能呢？美国印第安纳大学的一组科学家在“火星快车”发布它的甲烷发现之后半年，宣布他们在实验室中的研究可能证实了另外一种奇思妙想。有一种理论认为，火星和地球内部自两个星球形成之初就蕴含着大量甲烷。这一理论假如是正确的，将意味着地球内部蕴藏着比现在所认为的多得多的石油，而它的正确性到目前还处于争议之中。
刘双江说，他和金百莉正在筹划着在中国沙漠中开展更多的微生物研究。金百莉研究中国的环境和生态已经有10年之久了。“在我对索尔库里的最初几次考察中，有一次我被一场巨大的沙尘暴所困。我看着那沙尘如一堵巨大的移动的黑墙一样从西边过来，我被这种力量和效果所深深震撼了。”金百莉说，“最重要的是，它让我开始思考，大风如何影响微生物的区域和全球分布。”她打算在未来开展这方面的研究。而人们早就知道，在火星上，强大的尘暴能在极短的时间里席卷全球。
没有人知道，人类究竟会在什么时候什么地点发现火星上的生命，抑或那里根本就没有生命？现在科学家所做的，是在努力为问题的答案寻找点滴线索。
金百莉在青藏高原的索尔库里沙漠中捡起一块石头，石头中央部分的绿色物就是她要寻找的蓝藻
地球上没有哪个地方的自然环境比索尔库里更能够模拟火星上恶劣的环境 (金百莉供图)
