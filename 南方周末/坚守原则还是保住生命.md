# 坚守原则还是保住生命

坚守原则还是保住生命
http://www.sina.com.cn 2007年08月09日15:14 南方周末
韩国尴尬应对人质危机：坚守原则还是保住生命
“我感到非常耻辱，难道我们国家的力量只不过如此而已？”
□本报特约撰稿 詹德斌 发自韩国汉城
青瓦台的不眠夜
7月20日上午10时，韩国外交部的官员们正在静心地等待定于当天闭幕的北京六方会谈的好消息。一条紧急谍报传来——二十多名韩国人在阿富汗遭绑架。办公室的氛围刹那间凝重起来。
外交部一边指示驻阿富汗使馆火速核实情况，一边召开紧急会议，研究对策。由外交部第二次官金浩荣为组长的解救人质领导小组紧急成立了，并在驻阿富汗大使馆成立了现场解救指挥总部。
下午1点半的紧急记者会上，外交部发言人称“接到了有关韩国人被绑架的情报，目前正在确认真伪，但可能性很大”。除此以外，不愿再多说一个字。这样的慎重姿态后来一直贯穿了解救人质的全过程。
韩国外交部也立即寻求“外援”，一边跟美国驻韩使馆和美国政府保持着密切的情报交换，并向美军等盟军求助，外长宋淳则拨通了阿富汗外长斯潘塔的电话，希望阿富汗提供协助。韩国方面还要求阿富汗停止向韩国平民发放签证。
随着韩国人质遭绑的消息被各大通讯社报道，韩国国民的紧张情绪也加剧了。下午4时整，韩国政府召集外交部、国家情报院、警察厅等部门，商讨政府层面的综合对策，建立了24小时紧急工作体制。
当天下午5点零5分，韩国外交部正式公布韩国人遭到绑架的消息。而当晚10时41分，塔利班就发出要求，韩国于次日下午4时半之前从阿富汗撤军，否则就杀害人质。
外交部全面进入紧急状态时，总统府也启动了非常运转机制，彻夜不眠地同有关部门保持实时联系。21日早上8时，总统府决定，将人质事件提至总统府接管。外交部、国防部、国家情报院等各部门收集到的情报和信息必须实时地发送给总统府国家安全保障委员会(NSC)。
在忠清南道鸡龙台三军总部视察的卢武铉总统接到报告后，临时取消原定所有日程，于11时5分紧急飞回总统府。卢武铉决定亲自出面发表紧急电视讲话，并邀请美国有线新闻网(CNN)和国内媒体现场直播，以充分和迅捷地转达韩国政府的诚意。
下午2时30分，卢武铉出现在电视机前：……韩国政府在与阿富汗政府和其它友邦进行密切的合作，……也做好了与有关方面进行对话的准备。驻阿富汗韩军不仅是非战斗部队，且已进入完成任务的最后阶段。
此时，距塔利班的最后通牒时限已经不到两小时。下午6时，塔利班方面有了反馈，称韩国政府的态度很积极，人质很安全。半小时后，韩国总统府召开第二次安保政策调整会议，决定多种渠道接触塔利班，并派遣外交部第一次官赵重杓率对策小组前往阿富汗。
会议结束仅3个小时，塔利班再次提出通牒，如果阿富汗政府在22日23时30分前不释放23名被关押的塔利班囚犯，塔利班将依次处死韩国人质。韩国政府再次陷入困境。22日上午10时，总统府召开第三次安保政策调整会议，讨论塔利班提出的最新要求。韩国人质危机就这样进入了谈判、通牒、紧张、延长谈判时间的循环之中。
生命重于原则
韩国政府开始恪守一条原则，就是不论何时何地都不与绑架势力直接谈判。如果留下恶例，更多在阿富汗的外国人将会成为绑架目标。而且一旦直接谈判，对方会无限度加大筹码；绑匪还可能提出政治性要求。此外，韩国政府也明确知道，正在同塔利班激战的阿富汗和美国政府在此问题上的坚定态度。
不过，在23条无辜生命面前，韩国政府的原则立场显得苍白无力，坚守原则也许就是杀人。
噩耗在7月25日传来：塔利班杀死了一名叫裴亨圭的人质。
卢武铉当晚彻夜难眠，在卧室里随时接受参谋们的汇报。26日零时5分，卢武铉拿起电话，打给阿富汗总统卡尔扎伊。这已经是他第二次就人质问题给卡尔扎伊去电。二十多分钟的电话里，卡尔扎伊对释放囚犯问题持消极态度。
为阻止新的牺牲，卢武铉决定委任安保政策室长白钟天为特使前往阿富汗，敦促阿政府在“释放塔利班囚犯”问题上采取灵活态度。
白钟天抵阿后，同卡尔扎伊举行了50分钟对话，“讲了所有想说的话”，甚至承诺向阿提供政府开发援助(ODA)。
不过，韩国政府的努力再次遭遇严重挫折。塔利班宣布，由于阿富汗政府没有接受要求，于7月31日凌晨处死了第二名人质沈圣珉。
韩国政府不想再看到第三具韩国人质的尸体，转而要求阿富汗和美国政府不要再坚持“决不与恐怖组织谈判”，并采取灵活应对方式。总统府发言人最后还警告：“如果今后再次发生这种杀害我国国民的事情，我们不会袖手旁观，一定会追究责任。”
但阿富汗政府的回应是：不会接受塔利班提出的“人质换囚犯”的条件。
一无所获的特使白钟天在8月3日回国。此时，韩国政府虽未放弃对阿富汗政府所抱的希望，但开始开辟同塔利班的直接接触渠道。
值得回顾的是，外长宋淳7月25日在国会曾向国防部长金章洙转交了一张纸条，上面写着“8＋6＋9”，数字8和6下面都写了“钱”字。韩国媒体推测，23名人质可能被分为三组，每组分别为8个人、6个人、9个人。前两组可能在塔利班温和派手中。韩国政府可能希望促使塔利班组织中比较温和的集团释放抓走的人质。
韩国政府也在做好最坏的准备：如果营救人质的努力告败，绑架事件超过了忍耐限度，也可能会全面讨论军事作战，以“追究责任”。此时，韩国媒体已经在号召国民为此做好心理准备。
从“美国作用论”到“美国责任论”
23名同胞遭绑架的消息传开后，韩国上下无比惊愕。但随着人质基督徒身份的确定，同情、惋惜、愤怒、庆幸、悲痛、焦虑等感情也都夹杂着涌现。
祈祷、烛光集会上，同情和惋惜者不分宗教信仰、政治派别，希望并呼吁塔利班尽早释放人质；但愤怒和庆幸者指责人质不该无视政府劝告而前往“限制旅游”的阿富汗传教，给国民带来不安，给国家带来损失；悲痛和焦虑者则呼吁停止争吵，寻求解救办法。
值得注意的是，韩国国会、媒体、大部分国民同政府，表现出了前所未有的一致。国会在第一时间宣布，撤军计划已开始准备。媒体也按照政府　要求，淡化人质的宗教色彩，克制住报道冲动，不转载外电，不播放塔利班关于人质的录音和录像；网站则删掉可能会刺激塔利班和对人质安全构成威胁的内容等等。人质家属和所在的教会、财团向国民道歉，以博得同情和支持。
即使在第一名人质被处死后，一项调查显示，77.3％的受访者仍对政府为营救人质而进行的努力给予了肯定的评价。
与此同时，韩国国民也感慨外交力量的虚弱，有关“美国作用论”开始在国会议员、学者、媒体和舆论中抬头。
韩国民众认为，从阿富汗当前的政局来看，阿富汗政府无法独立做出重大决定。未经美国和国际安全援助部队的同意，不可能独自决定释放囚禁的塔利班囚犯。
第二名人质被杀后，韩国国民对于韩国的国际影响力表示出失望。上班族金尚敏感喟：“我感到非常耻辱，难道我们国家的力量只不过如此而已？”于是，“美国作用论”进一步演变为“美国责任论”，愤怒情绪开始化为实际行动。
媒体称，韩国为了支持美国的世界政策向阿富汗派遣了建设工兵和医疗部队，向
伊拉克派遣的军队人数仅次于英国，也向黎巴嫩派遣作战兵力。如今，韩国深处困境，美国理应帮助。
8月1日上午，人质家属们前往美国驻韩大使馆，要求美国政府在解救人质方面提供人道主义协助。一部分市民团体甚至称，“人质事件的罪魁祸首是美国”。他们聚集在美国驻韩使馆附近示威，甚至绝食。为敦促美国发挥作用，同时也是为了扑灭反美情绪的火种，大国家党和开放
国民党等5个党派的国会领袖8月1日发表共同声明并于次日组团访美，要求美国政府立即改变态度，出面解决绑架危机。
韩国人质危机在威胁、谈判、杀人的痛苦煎熬中已经进入了第20天。由于阿富汗和美国政府坚持不用塔利班囚犯交换人质，韩国政府只好调整对策，寻求同塔利班进行直接谈判。目前，双方正在就谈判地点进行磋商，但谈判前景仍不容乐观。
