# 城市建设的伦理亟需重建

城市建设的伦理亟需重建
http://www.sina.com.cn 2007年06月07日15:30 南方周末
□肖复兴
近日来，关于北京东四八条的拆迁引起人们的关注。其实，铁镐在东四八条9号院的屋顶砸了下去之前，关于这片街区的拆迁问题就早在争论之中。和它不远的71号院，便是著名教育家和作家叶圣陶先生的故居。在这条胡同里，这样不错的四合院有许多。一条古色古香的胡同，如何保护，确实值得重视，一直在人们关注的视野里。为坚持拆迁有理，东城区的相关部门曾经出示了一些证据，回应人们对拆迁的质疑。
不过，疑问还是存在。东城区文保所所长李仅录表示：被拆迁的这条胡同属于北京旧城历史文化保护区“建设控制区”，而非“重点保护区”。开发商出示了专家论证意见复印件，著名古建筑专家王世仁在意见中却写道：是东四八条危改项目……不在历史文化保护区内。这让我糊涂了，一个说在历史文化保护区内，一个说不在，如此明显相悖却又是非常重要的意见，怎么能够在论证中出现呢？人们有理由怀疑这样论证的严肃性和准确性。
其实，在不在历史文化保护区，在“建设控制区”也好，在“重点保护区”也好，挂牌保护的
四合院也好，没挂牌的四合院也好，同样面对如何拆迁的问题。值得每一个北京人思考，也值得政府相关部门考虑的是，北京老城还剩下多少这样的富有浓郁历史文化含量的老胡同、老院落，禁得起我们如此挥霍？让铁镐和推土机一次次轮番交错地练手？
城市范围和规模越来越扩大和严重的拆迁，其实是对城市建设的伦理的漠视和挑战。我们一直没有重视和研究这一问题，在我们传统的城市建设思路里，城市只是我们建设的对象而已，是为我所用的，我们很少把城市里胡同和四合院也当成赋予了生命的人一样对待；特别是这些由漫长岁月积淀下来而成为历史活化石的胡同和四合院，在因人为的因素越来越少的严峻现实面前，我们依然坚持拆迁的思路，而没有认真反思、检点一下自己，从而善待这些对于我们是财富而并非破烂的胡同和四合院。
城市建设的伦理，还包括普通百姓祖辈居住在胡同和四合院里的情感和利益。如今，“危改”成了当代中国的一个流行词，在电脑里都有词组的储存，两个键就能够打出来。“危房改造”成了帮助老百姓改善居住环境最动听的一个词，却是以把祖辈居住在这里的人赶出市中心为代价，而将新建起来的伪四合院拱手卖给那些有钱人。它同时也成了城市建设最野蛮的一个词，因为城市建设似乎一花独放只剩下拆迁一种模式，好像拆光了四合院、推平了老胡同，就是对老北京最好也是惟一的保护和建设。城市建设的伦理，就这样被我们自己的手粗暴地破坏，劫贫济富而势利地偏向
房地产商和富人，而我们不以为过，却还顽固地以为是城市建设最好最便捷的方式和行为。
因此，重新面对和思考城市建设的伦理问题，对于如何保护和建设北京这样一座绝无仅有的古城，已经迫在眉睫，意义重大。
东四八条今天的命运，并不是惟一出现在我们面前的一次。前不久，崇文区为在前门地区修路，草厂三条和十条各一半胡同的院落被拆掉，夷为平地。而草厂诸条胡同和东四八条一样，都是在8年前被规划在历史
文物保护区内的，是不允许如此大拆大建的。但是，草厂三条和十条两条胡同不是已经如同割去了的半扇猪一样，凸现在我们城市的空间里吗？作为历史文化的老街区，其保护是整体性的。把那些地处市中心的老街区当成肥肉，像卖肉一样想割一刀就割一刀，胡同的肌理就被破坏了，历史文化传承的基因链就断裂了，老北京旧都的风味和魂儿，就会越来越少以致最后荡然无存。城市建设伦理的违背和破坏，到头来，遭到惩罚的是我们的城市和我们自己。
前两天，北京市规划委员会宣布东四八条暂时停止拆迁。这场拆迁风波终于告一段落。但是，我们特别是城市建设的领导者即政府，如果没有勇气和眼光承担起重建城市建设伦理的责任，善待我们城市里不会说话的胡同和四合院，和我们城市里会讲话却没有多少话语权的普遍百姓，那么，这样的拆迁风波还会一起再起。
