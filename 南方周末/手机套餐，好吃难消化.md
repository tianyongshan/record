# 手机套餐，好吃难消化

手机套餐，好吃难消化
http://www.sina.com.cn 2007年08月02日15:45 南方新闻网-南方周末
我的烦恼
纳税者说
□姬晓东
2007年春节，朋友要走我的身份证说送我一部新手机，几天后如愿以偿。手机看起来还算精致，来源于当地移动公司的“优惠”活动。
半个月后，通过“10086”查询缴费信息时得知：我的新手机预缴话费增加了1000元。窃喜又百思不得其解，再次查阅话费更让人惊讶——一夜间，由几十元猛增到303元。此后，我几乎每天都在查询，但话费停留在303元上分文不动，到了下个月的话费也是303元。我只好去查询明细话费，莫名其妙地知道：有一大笔叫做“包月补收费”的，占了我总话费的近一半。
经过两天查询，总算弄懂了“包月费”的由来。原来，朋友拿着我的身份证参加了该公司活动，当场预缴了1000元话费换来这部价值2000多元的手机，然后我就开始“吃”到该公司的“套餐”。这个“套餐”的真谛是“每月必须消费300元话费(另有3元的
天气预报费)，而且必须连续消费要达到5000元以上。如果当月话费超过300元则按实际消费另外收取，花费少于300元则按300元收取”。所以，在3月，尽管我像往常那样使用手机，产生的话费依然是100多元，但剩余的100多元以“包月补收费”的名堂给移动公司做了贡献。
这是何种
霸王条款？一部2000多元的手机，即使我花费2000余元现金购买了，那么按照我目前每月100元的通话费用计算，剩余的钱可以保证我近30个月的话费(因为现在必须达到5000元，才能跳出这个套餐的“套儿”)。可现在我却仅能用16个月，打不完只能给他们补交。
发现猫腻后，我叫妻子去查阅她的电话费用。又是不查不知道，一查吓一跳！她使用了四五年的手机竟悄无声息地一直执行着每分钟4角的通话费。但是，她当初入网时，本地话费全部成为每分钟2角了。带着疑问找到营业员时，人家告知问题出在机主身上，这个号码是前主人销号的，你们拿到这个号码，应该自己办理每分钟2角价格的手续！我哭笑不得，谁都没告诉这个号码属于4角的高价。
我们又查阅了去年预交200元后获得的“每月送30元。总共送720元话费”的套餐结果。一查发现，送的话费必须是本网之间的通话方才有效。可当时交费时，也没人说明这一点。
气愤不已中走出营业大厅，又看到满街的标语宣传该公司改单项收费。这更加令人狐疑了——因为里面附带着多种套餐，比如客户要缴纳60元的彩铃费方可享受到一些单项收费的待遇。所谓的单项收费竟是套餐“套”套餐，说穿了，就是出新招想方设法赚用户的钱！
