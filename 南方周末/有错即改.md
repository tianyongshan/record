# 有错即改

有错即改
http://www.sina.com.cn 2007年01月04日15:55 南方周末
■第1192期061214因技术失误导致部分印点印制报纸出现错漏：头版《吃人公园持续夺命数十条，丧女母亲执著追踪责任者》一文转版提示错误，此文实为头版下转至A2纵深版。
■第1191期061207C17经济版《双面郁知非上海体育大亨的光荣与悲哀》“申花神话”部分第1段，“从1994年至1998年，五个赛季中，三夺联赛冠军，一夺足协杯冠军”有误，这五个赛季的足球顶级赛事中，申花成绩为：1994年甲A联赛季军，1995年甲A联赛冠军，1995年足协杯亚军，1996、1997、1998三次甲A联赛亚军，另外获得1998年足协杯冠军。所以准确表述应为“一夺联赛冠军，一夺足协杯冠军”。
———西北师范大学杨洪远西安诸东辉山东孟德辉
■第1191期061207C24版《尤努斯：实践的理想家》起始部分，“22年前，洪水和饥荒笼罩了孟加拉，这个刚刚从印度独立出来三年、一穷二白的国家……”中有错误。1947年印巴分治，古孟加拉被分为东西两部分，西部归印度，东部归巴基斯坦即所谓的“东巴”。1971年3月26日“东巴”宣布独立。4月17日，孟加拉人民共和国临时政府成立。12月22日定都达卡，标志孟加拉共和国正式建立。1972年1月7日取得完全独立。1974年即遭遇洪灾。因此，原文中的“22年”应为“32年”，“印度”应为“巴基斯坦”。
———北京赵琪
