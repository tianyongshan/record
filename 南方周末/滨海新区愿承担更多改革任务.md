# 滨海新区愿承担更多改革任务

滨海新区愿承担更多改革任务
http://www.sina.com.cn 2007年03月15日10:33 南方周末
滨海新区愿承担更多改革任务
3月6日，在人代会召开期间，天津市长戴相龙就天津滨海新区金融改革试点若干热点问题接受了本报记者专访。
金融改革试点进展如何
记者：金融改革作为天津滨海新区综合配套改革的亮点，它的目标和重点是什么？
戴相龙：金融改革是综合配套改革的首要任务，国务院20号文鼓励天津滨海新区进行金融改革和创新。金融方面的重大改革，原则上可以在滨海新区先行先试。金融改革的目标是建立和北方经济中心相适应的现代金融服务体系，办好全国金融改革试验基地。依据全国金融改革工作的部署和天津的实际，金融改革围绕直接融资、金融企业综合经营、外汇管理体制和完善金融组织体系进行。
记者：扩大直接融资的改革进展如何？
戴相龙：经国务院批准，我们已经进行了产业投资基金的试点。去年底我们成立了由6家出资的渤海产业投资基金，总规模200亿元，第一期已经募集了60.8亿元人民币。同时也成立了基金管理公司，董事长是中国银行的行长，总经理是从香港聘请的专业人士，以后我们还会发展各种投资基金，比如科技风险投资基金、
房地产信托基金，等等。
记者：产业基金在中国是很新的尝试，还在探索和试点阶段，在你看来，滨海新区试点产业投资基金的优势是什么？渤海产业投资基金的设立经历了很艰难的过程，你觉得这个突破有何意义？
戴相龙：滨海新区有三个优势，一个它已经被纳入国家发展战略，产业投资的需求很大；其次，天津曾经是北方金融中心，有这方面的基础；另外，因为紧靠北京，能即时得到国务院有关部门的指导。渤海产业投资基金是国内第一个中资产业投资基金，光是这第一个就足以说明它的突破意义，另外投资方都很有实力，基金的总规模数额很大。最重要的一点，是我们解决了出资人和管理人的利益分配问题。在这个过程中，管理费的确定、绩效挂钩的设计等等最花时间，也是最大的难点。目前我们都解决了，这个搞好了，也可以为以后制定公布产业基金管理法律法规提供实践经验。
记者：作为国内首只中资产业基金，渤海产业投资基金目前进展和你的预期是否相符，有没有遇到什么困难？你对它的前景怎么看？您认为产业基金成功的难点和风险何在？
戴相龙：我觉得，目前的进展还是比较顺利的，各项业务准备还在进行中，现在还在选择投资项目，目前不存在什么困难。渤海产业投资基金有一套风险管理体制，和有关投资选择和管理方面的制度安排，目前我对它的前景比较乐观。
天津还能做什么
记者：作为金融改革的试点，滨海新区在金融企业综合经营方面可以先行，目前有何进展？你怎么看待金融企业综合经营的必要性？
戴相龙：企业集团综合经营决定了金融服务的综合经营，另外，金融企业综合经营也是参与国际金融竞争的需要。金融企业综合经营改革的步伐不能太快。我们国家金融业务总的来说还是分业经营分业监管，但是天津是中国综合配套改革试验区，可以在金融企业的综合经营方面先行先试。现在已经有一批商业银行集团或者保险集团，提出在滨海新区成立全资附属或控股的公司，有的银行在天津成立投资银行部等等，这些机构有的已经成立，有的正在申请当中。
记者：外汇体制改革方面呢？
戴相龙：去年7月，国家外汇管理局也批准天津在七个方面进行外汇管理体制改革，主要是把企业出口所得外汇按一定额度留作存款，其余全部卖给银行，改为意愿结汇，即企业自主决定出卖外汇的数量；把对进出口企业的外汇收支逐笔核销改为汇总核销；跨国集团公司可调度使用所属子公司外汇存款。这样做，更有利于企业的自主经营。
记者：有人认为，滨海新区外汇管理改革可能为资本项目完全开放和人民币完全自由兑换开辟道路，你怎么看？
戴相龙：1993年十四届三中全会关于经济体制改革的决定，就提出了创造条件使人民币成为可自由兑换货币。我认为实现这个目标，需要经历不同阶段。1996年底，我国实现人民币经常项目可兑换。到目前为止，资本项目可兑换范围已经不断扩大，外商投资实行登记即可，外债管理也在放宽，外国投资银行通过设立有关机构也可以从事人民币投资业务。另外，个人购买外汇的额度也大大放宽。
但是，把外币换成人民币直接购买
股票还需要较长的一段时间。目前，国家外汇管理局同意滨海新区进行试点，主要是在逐步放宽外汇管理方面，内容也仅限于上面我讲的几个方面(见上一问答)。随着外汇管理体制改革的进一步深入，我们天津滨海新区愿意承担更多的改革任务。
