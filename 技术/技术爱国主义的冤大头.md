# 技术爱国主义的冤大头

技术爱国主义的冤大头
http://www.sina.com.cn 2007年04月12日15:50 南方周末
□刘洪波
当准备买一只新的智能手机时，我发现很难买到一个称心如愿的产品。
大致上，我希望这只PDA手机能够有一般手机和掌上电脑的功能。作为手机，它能够打电话、发短信；作为掌上电脑，它能够通过各种方式与电脑联接，并且可以高速上网。
这些功能并没有技术上的困难，在境外这些功能已成智能手机的标准配置。但当我寻找几圈以后，知道根据信息产业部的规定，你不可以买到这样的产品。在中国市场上，一台掌上设备有WIFI功能，则不能有手机功能，如果有手机功能，就不能有WIFI功能。
WIFI，就是一般所说的无线上网，可以共享使用宽带网络，但这不能是中国手机的功能，所以中国手机只能通过手机拨号上网，慢如蜗牛。
据说，这中间有“技术爱国主义”的因素。我们有自己的无线网络标准，叫WAPI；又据说，它比WIFI更好。曾经，这个“核心技术”差不多要变成一个强制标准，但因为一些原因搁浅，于是，无可奈何地，现在任何一台
笔记本电脑，任何一台无线上网的台式电脑，都使用了WIFI标准。然而，很有办法地，“技术爱国主义”在手机上得以维持，但我们也没有架设WAPI这种独家的“爱国无线网络”，所以我们的手机上网只好使用慢如蜗牛的GPRS拨号方式。
开始，我以为这只是一种争气的做法，虽然不智，毕竟还表现了气节。后来我才想到，这里面其实大有智慧。如果手机可以无线宽带上网，还有谁肯花百倍的价钱去享受蜗牛速度的服务呢。只要一个手机不准带有某种功能的规定，移动服务商就可以让人别无选择地为上网支付高昂费用。因此，用手机上网时，你可能只是做了“技术爱国主义”的冤大头。
当然，你也有办法来解决问题，那就是购买“水货”，也就是那些供应境外市场的智能手机，往往只要一半的价钱，而且功能齐备。正牌机不如水货机，不许手机无线上网的规定功莫大焉。
现在搞3G通讯，手机上网可以不再慢如蜗牛了，我们仍然有自己的标准，也就是有自己的“核心技术”，大家都说，这样子就不会受制于人，就不会再出专利使用费等等。好啊好啊，只是标准早立，网络一直没有运行，而别人的3G网已开通几年了。但管它的，我们守住了自己的标准，守住了自己的市场，但愿这回守住的不是又一个WAPI。
曾经，我们有过一种“核心技术”，叫“超级VCD”，据说画质等同于DVD，而更加便宜，就是碟片容量小一点罢了，不打紧，看一部影片多换几次碟就行了。一时间，很多人都去买了来。现在，我不知道还有几个人在用这种机器，又有多少节目可供这种“超级机器”播放。不过，可以肯定的是“超级”厂家的产品是卖出去了的，“爱国心”使你支持“爱国技术”，而这种技术甚至一开始就没有准备支持你的使用。
现在，DVD面临升级换代，我们也有自己的“高清DVD标准”，不同于两大蓝光阵营，我们使用红色激光技术，但是据说画质同样地好甚至更好，而且拥有成本又很低，惟一的不足是光盘存储容量只是蓝光光盘的一半乃至三分之一。这话我听起来觉得很像“超级VCD”的升级版推销词，不过这回据说节目内容也有保证了，但我又记起“超级VCD”也曾有过示范碟，也曾有过“片源保证”的承诺。
上世纪80年代，我读过一本书，《第五代计算机》，介绍了日本以模拟技术开发超级计算技术蓝图；在远程通讯方面，日本的模拟技术——传真机占领了绝对市场。数字化浪潮到来，“第五代计算机”已无人再提，传真机更被电子邮件逐出市场。前几年，当日本大力研发
数码相机时，柯达仍陶醉于胶片照相的成就中，差不多也被清场。
我并非为“技术帝国主义”张目，也惟愿“技术爱国主义”发扬光大。但我想，一门心思制定标准，而没有使之变成事实上的标准的能力，或者只知把守市场，不知发展的方向，“核心技术”未必不是“核心麻烦”。我尤其想说的是，“技术爱国主义”要对得起人们的爱国心，不至于让爱国变成上当才好；而且“技术爱国主义”不要搞成“技术赌气主义”，好像非得事事都要“汉家自有制度”才行。
