# 3.如何与法官、检察官交流沟通_1．抓住交流的时机

刑事辩护指导书

3.如何与法官、检察官交流沟通

1．抓住交流的时机

在具体工作上，律师与法官、检察官的交流机会不多，最常见的方式是通过电话交流。在这种客观交流条件下，律师能说的内容不多。

除了少数法院或检察院允许律师直接进入法官、检察官的办公室，很多地方律师都被禁止进入办公场所。而即使律师提出要与法官、检察官面谈，如果没有充分的理由，也不会被允许。

能当面与法官、检察官交流的机会很少，但是，有个场合是法官和检察官无法避开律师的——法庭。

在开庭之前，通常各方都会先入座，然后等待旁听人员到场、被告人到庭，才会开庭.休庭之后，律师要等着签笔录，法官、检察官通常也不会马上离开。因此，开庭之前和休庭之后的短短几分钟时间里，是律师与法官、检察官最好的交流机会，即使法官、检察官在休庭后要马上离开，律师也可以及时叫住他们，表达自己想交流的意愿。

如果律师在案件方面有什么想法，又是法庭上不适合说的，如被告人家属想要代为赔偿损失或退出赃款，案件的背景或双方矛盾的起源，对方当事人在调解过程中的态度与表现，希望法官或检察官可以继续调查某一内容，对案件的个人看法（非当事人立场的本人观点），与本案相关或相似的其他案件的处理情况，等等，一定要抓住这机会与法官或检察官进行沟通。

2．注意沟通的技巧

上文提过，公检法的工作人员每天都面对不同的当事人提出的同样问题，给律师说话的时间是极其有限的，这就很依赖律师的说话技巧。

其一，抓住重点，简洁明了地介绍自己需要解决的问题。

最让办案人员不堪其扰的标准当事人来电是这样：我是 XX 案嫌疑人（或者事主）的家属，这个事情是这样的：在一个夜深风高的夜晚，我们正在 XXXX，这时突然 XXXX，我们就说 XXXX，对方又说 XXXX……

办案人：（打断他）案件事实等开庭的时候说，你打电话来是想说明什么问题？

来电：这个事情发生之后，我们找了对方想 XXXX，对方又说 XXXX，我们只好 XXX……

办案人：（再次打断）你是不是想让我们出面调解？

来电：是的。

办案人：好的，再见。

而如果换成律师来电，就容易交流了：

我是 XX 案当事人的律师，我想请办案单位出面组织当事人双方调解一次。因为 XXXX……

办案人：好的。

在沟通过程中，律师一定要明确自己的问题是什么。是要阅卷，要确定或更改开庭时间，要提起刑事附带民事诉讼，要办案单位组织调解，要求调查证据、通知证人出庭，要召开庭前会议，要求处理或不处理涉案物品，还是其他事由。

在交流时，律师先提出自己主张的内容，再以三四句话简单说明理由，然后视办案人员的回复再进一步提出方案，或者确定具体的时间、地点等内容。如上面要求调解的对话，就可以继续下去：

律师：我想请办案单位组织调解，因为我们提出二十万元的赔偿，对方其实是能够接受的，只是想多要一点，就一直没答应。

办案人：可以。

律师：可以尽快安排吗？我们可以调解时当场支付款项。

办案人：好吧。

律师：我方当事人家属是外地请假过来的，现在住在酒店里等，大概周末就必须回去。麻烦尽快确定调解时间通知我们可以吗？

办案人：好的。

律师：再见。

在电话沟通中，办案人员往往处于被动状态，只会简单表示同意或反对。这就需要律师主动提出下一步更详细的安排，把事情的具体细节确定下来，或者提出解决的方案供办案人员确定。

其二，询问内容要具体、明确。

律师交流的内容应该尽可能具体到某一问题，并且不要过多地询问太基础或常识的法律问题。

不具备法律知识的当事人来电，通常是问：我是 X 案件的被害人，我要对方赔偿我的损失。

答：你去立案庭起诉他吧。

律师可以就与案件相关的法律问题与办案人员交流，但是要问得具体、明确，尽量避免询问框架过大或者与案件关系不大的内容，更不要问缺乏专业常识的问题，会被嘲笑没水平的。

例如：

询问「我能否在刑附民中一并起诉肇事车辆的保险公司？」不要问「我都能把哪些人作为刑附民被告一并起诉？」

询问「残疾赔偿金能否得到支持？」不要问「残疾赔偿金大概能支持多少钱？」

询问「当地盗窃罪的入罪标准是多少元？」不要问「你们实务中对盗窃罪与职务侵占罪的区别是怎么把握的？」

同时也要避免询问要办案人员给出倾向性的意见的问题，如：

本案有罪还是无罪？能不能判缓刑？

我是提刑附民好还是提民事诉讼好？

我要不要把车辆的车主也一起起诉？

这类问题，办案人员是不会作出明确答复的。

其三，做好被拒绝的准备。

律师就具体问题与办案人员交流之前，要对自己被拒绝的可能作评估和准备：我的理由是否充分？对方拒绝的可能性有多大？可能拒绝的理由都有哪些？我要如何应对这些拒绝的理由？

除去理由不够充分肯定要被拒绝或者理由很充分肯定要被同意的因素，如果是可拒绝可同意的事情，比如请办案人员主持调解，要求开庭前会议，要求暂未确定对案件事实是否有影响的证人出庭或证据调查等办案人员的主观因素起决定性作用的，就要事先做好被拒绝的准备，以及思考如何进一步说服办案人员。

这里有个很基础的预估标准。

当前司法系统的普遍现象是案多人少，因此，办案人员通常心态都是不愿意做日常事务之外的额外工作，以防影响或打乱他的正常工作安排。如果律师提出的具体问题会给办案人员带来较大的额外工作量，例如主持调解，补充调查，实地看现场，召开庭前会议，等等，那被拒绝的可能性就会大大增加。

这种情况下，律师就要想办法减少被拒绝的可能。

A、尽量降低办案人员的工作量。如申请调解时，先自己尝试与对方沟通，形成大概的调解意愿，并初步了解对方的意愿与情况，然后把这些情况告知办案人员，增加他调解成功的可能.而不是自己什么都不做，直接让办案人员去说服一个尚未与对方接触并开始沟通调解事宜的当事人。

B、尽量减少办案人员拒绝的理由。尤其是申请调查取证或者排除非法证据之类的与案件有密切关联的问题上，提供的调查信息和调查理由一定要详细。如提供证人的名单、联系方式、目前下落，或者其他确实可供调查的线索，尽量避免办案人员以申请不符合法定形式、与案件无太大联系等理由拒绝。

C、还要注意选择对象和时机。如补充证据，律师在侦查阶段、审查起诉阶段、审理阶段都可提出，但是不同的阶段给办案人员带来的负担也是不同的。侦查人员是调查取证的直接负责人员，而且基于立场上与被告人的辩护人存在对立，对于律师要求补充证据的请求更有可能拒绝.而审查案件的检察官、法官并不需要主动取证，同意补充调查的可能性也更大一些。

D、不要因为被拒绝就放弃说服，在被拒绝时至少再尝试三次。办案人员遇到可能打乱自己工作安排的请求，拒绝是普通人都有的正常反应，不必因为被拒绝就放弃，只要不是绝对不可能同意的事，只要对方没有马上挂断电话，大可继续陈述理由，晓之以理动之以情，进一步尝试说服办案人员。

3．尽量不要夹带情绪，避免不必要的争论

进入司法程序的案件多是矛盾冲突激化的产物，长期接触这类案件的办案人员自然也可能会被负面情绪干扰。因此，老人带新人时总是会提醒：这就是工作而已，不要把个人情绪带入。这话对律师也同样适用，要尽可能调整好自己的心态，避免被负面情绪影响。

4．避开不好说话的人

此外，在刑事诉讼的流程中，不同阶段的办案人员不同，与其打交道的难易程度也不同。一般来说，检察官比侦查人员容易打交道，法官又比检察官容易打交道。所以如果在刑事诉讼前期提出的调查证据之类的请求或者对案件事实、定性提出的异议没有得到对方认可或回应，大可在之后的阶段继续尝试。

5．听得懂话外音

律师打电话给我，问的频率最高的问题是「XX 案什么时候能判」，其次则是「XX 案大概能判多久/能不能缓刑」。

对于第一个问题，我通常都是很坦诚地告诉他，我写案的顺序安排，如果推迟了，原因是什么。

第二个问题，我则通常用「还没确定」、「现在无法判断」、「我也没想好要怎么处理」之类含糊的话应付过去。有的律师不甘心，还要再追问一句：「如果跟被害方谅解好了，能不能给他适用缓刑？」我则继续含糊回答：「我只能说赔偿并取得谅解可以判得更轻，但能不能缓刑现在是给不了答案的。」

这种沟通，就需要律师在沟通方面除了懂得说话之外，具备的另外一个能力，听懂话外音。

案件进入司法程序，处理结果就是机密，法官也好，检察官或者公安人员也好，都不能在公布结果前私下透露。

但是，有时候案件的处理又需要当事人的配合，但办案人员不能当面去教当事人怎么做，否则就有违职业道德。这种情况下，需要以隐讳的方式提醒律师去指导当事人。

比如邻里纠纷引起的轻微故意伤害，双方和解通常都可以适用缓刑，但有时候当事人就是因为几千元的差距不肯让步.或者销售淫秽视频的案件，在立法上标准很低，几百个视频就可能判到十年以上，于是实践中就找犯罪未遂之类的理由减轻并适用缓刑.又或者案件事实清楚、证据充分，情节很轻微，可以适用缓刑，但是被告人非要律师作无罪辩护的。

如果光按法律来处理，案件本身没问题，法律效果很好，但不利于社会影响，这种情况下，法官一般都会提醒律师，由律师指导被告人配合。但是这种「提醒」一般不会说得太明白，律师如果听不懂，就会影响到对当事人的处理结果。

比如我有时候提醒律师：「这种案件的处理本来就不重，而且证据又是很清楚的，承认事实与不承认事实的结果是完全不同的。你再去问问被告人自己的意见，让他考虑一下吧。」明白的律师自然明白这是有可能判缓刑，而不明白的律师就真的去会见了被告人，还做了会见笔录，但对处理毫无帮助。

除了听得懂暗示，还要听得懂拒绝。有些案件是无论如何也不符合缓刑条件的，律师还反复询问「如果赔偿了有没有可能缓刑」。我就会回答：「不要拿赔偿来谈条件，赔不赔你让他自己决定吧。」

不同的法官会有不同的说话习惯，这都需要律师在办案过程和沟通过程中一点点积累经验，慢慢揣测和提高。比听得懂话外音更高一层次的，就是察言观色来判断法官的说话意图了，这需要更多的实践经验了。
