# 专题_作为哲学著作的《资本论》

作为哲学著作的《资本论》

专题

作为哲学著作的《资本论》

主笔◎薛巍

美国学者大卫·哈维

美国评论家、作家埃德蒙·威尔逊

马克思是经济学家、哲学家，他这两个身份大概是密不可分的。

“商品的诗人”

美国学者大卫·哈维说，马克思在《资本论》第一卷中引用了丰富的文学遗产。书中随时会冒出莎士比亚、塞万提斯、歌德、弥尔顿、雪莱、巴尔扎克、狄更斯，以及古希腊哲学和神话、流行文化中的人物（吸血鬼、狼人、下金蛋的鹅）。

埃德蒙·威尔逊的《到芬兰车站》中称马克思为“商品的诗人”，他说《资本论》不但具有科学性，而且也具有艺术性。“真正使这部作品成为一个完美总体的，是马克思的诗人气质，早在学生时代，他就已经显露出非凡的诗才。《资本论》之所以充满迷人魅力，除了其论证贴切中肯而发人深省之外，主要也是因为它基本上乃一本充满想象力的艺术杰作。马克思是个罕见的伟大讽刺家，他无疑是斯威夫特以后最伟大的讽刺天才。马克思和其他许多博学的经济学家不一样，他善于运用游戏诗文。”

马克思创立的是一个完整的理论体系。马克思24岁时，他一个同时代人写到他时说：“想象一下，卢梭、伏尔泰、霍尔巴赫、莱辛、海涅和黑格尔融合成了一个人，你就得到了马克思博士。”《凯恩斯传》作者罗伯特·斯基德尔斯基说：“马克思是那些没有让对他们的希望破灭的杰出年轻人之一。他创造出了最强大、最融贯、最有影响的世俗思想体系，解释了人类的过去、分析了其现在，并预测了其未来。他融合的是关于历史阶段的辩证理论，关于历史的唯物主义理论、对资本主义文明的经济和道德批判、对必将崩溃的资本主义的经济学证明、对革命行动的号召、对共产主义的预测。他的体系最成熟的表述见于他的巨著《资本论》。”

英国布里斯托大学政治学系教授特瑞尔·卡弗曾经说：“从理论上说，马克思在撰写他的杰作《资本论》时处于他的能力最强的时候。不幸的是，这部著作论题广泛但深奥，有很大的篇幅讨论的是现在已经过时的经济学形态。但只要耐心一点，就可以追寻这部对资本主义社会的批判性阐述，它详尽地阐述了资本主义生活形式，包含大量当代和历史性的实例，因此非常可信。因为马克思在研究时不在乎学科界限，他这部丰富的文本触及了我们能列举的、从人类学到动物学等所有科学和学科。在形成这套当代社会理论时，马克思评论了现实的本质、知识、科学、行为、文化、艺术和宗教。”

1939年，英国哲学家以赛亚·伯林出版了《卡尔·马克思：他的生平和环境》一书。他在书中说：“马克思是伟大的新信仰的创建人之一，他用单一、明晰、热情坚持的原理来解释世界。”“《资本论》是马克思一生中的最高（crowning）成就，展示了一种新的、革命性的历史学写法。在这样一个消灭敌人的方法不会因其庄严、缓慢而减损其效率的时代，卡莱尔和叔本华被迫逃避到遥远的文明或理性化的过去，其大敌尼采被逼得歇斯底里、发疯，唯有马克思一直牢不可破、难以对付。跟古代执行上天授予的使命的先知一样，他因为对未来理性社会有着明确、肯定的信念，所以内心很平静，他到处都看到了衰败和灭亡的迹象。在他看来，旧秩序在他眼前摇摇欲坠.他为加速这一过程，为缩短终结前最后的痛苦所做的工作比任何人都多。”

马克思的经济学是为他的政治理想服务的。去年，哈佛大学教授路易斯·米南德在《纽约客》上撰文说：“马克思是一位哲学家。他的哲学致力于人类的自由。马克思是一位启蒙运动派思想家：他希望一个理性、透明的世界，在这个世界中，人类从外力的控制下解放出来。马克思是一个人道主义者。他相信我们能够把世界变成为所有人的利益而生产产品的世界。资本主义社会把生产变成购买的劳动力并用于扩大他人的财富，这妨碍人性的全面实现。资本主义注定会自我毁灭，工人阶级的革命将带来历史的最后一个阶段：共产主义。《资本论》之所以看上去更像一部经济学著作而不是哲学著作、书中之所以充满表格和图表而不是三段论推理，是因为马克思说的，哲学的目标是为改变世界而理解世界。他说黑格尔的哲学是用头立地，他要把它倒过来，用脚立地。也就是说，他要为他的理论找到证据。所以他在大英图书馆花了那么多时间，研究关于工厂状况的报告，关于工业生产的数据，关于国际贸易的统计。他要英勇地证明现实跟理论是一致的。难怪他写不完这本书。”

《资本论》与现时代

美国经济学家罗伯特·海尔布隆纳在1953年出版的《世俗哲人》一书中说，马克思是一位读尽所有经济学家著作的经济学家，一位一丝不苟的日耳曼学究。《资本论》的伟大之处在于，它跟所有道德考虑彻底脱钩。书中的叙述充满怒火，其分析却冷静、合理。他建立了一个所能想象到的最严密、最纯粹的资本主义。在此精粹的抽象体系中，所有现实生活中的明显缺点都被移除。他在这个环境中展开探寻，若能证明，即使是尽可能完善的资本主义也会走向灾难，那就更容易推论出现实的资本主义必然要走上同一条路，只是速度更快。马克思的分析仍是资本主义体系遭遇过的最严肃、最敏锐的检视。马克思主义的革命家沿着道德路线摇头抨击利润动机的不义，但马克思主义的经济学家则不然，对其热情可以加以冷静评估。同理，其严峻的发现也依然中肯。

哈佛大学经济学教授拖德·布赫霍尔茨在《天才的回声》一书中说：“按照马克思的说法，没有一粒原子的价值出自资本家之手。马克思只是简单地想到了女裁缝贾丝明或锤击铁砧的铁匠，认为是他们创造了价值。马克思遗漏了什么吗？他忽视了创意和企业家精神。要创造财富可不只是需要看得见摸得着的投入。企业家精神就是愿意承担投资风险的精神。为什么生活在共产主义体制下的俄罗斯需求美国制造的粗斜纹棉布牛仔裤？不是因为苏联缺乏棉花或生产高品质布料的工人，而是因为他们缺乏想象力、积极性和方法指导。这些无形的因素使得成功的公司和国家脱颖而出。马克思忽视了冒险和等待对社会的价值。因为要投资，资本家放弃了购买物品所能带来的即时满足。他的投资回报就是等待，也就是将他的快乐延迟。”

1980年，哲学家彼得·辛格出版了一本关于马克思的小书。他在书中列举了马克思所做的预测：工人和资本家之间的收入差距将扩大，独立的制造商会被迫降为无产阶级，工资将继续维持在维生的水平，利润率下降，资本主义将解体，在发达国家将发生革命。辛格认为这些预测大部分都错了。到2016年，就很难轻视马克思的这些预测了。

英国历史学家斯特德曼·琼斯认为，马克思和恩格斯痴迷于阶级问题，忽视了其他身份形式的力量。其中之一是民族主义。在马克思和恩格斯看来，工人阶级的运动是国际性的。但今天我们好像在脱欧的英国选民中看到了向民族主义的回归，在美国看到了民族主义的飞涨。
