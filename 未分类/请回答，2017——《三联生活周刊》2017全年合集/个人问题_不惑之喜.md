# 个人问题_不惑之喜

不惑之喜

个人问题

不惑之喜

文◎青衣　图◎陈曦

终于，在生日来临前的几天，生活基本可以自理了。虽然还需要处理药物副作用带来的各种麻烦和小心随时复发的风险，但是心情还是像久旱逢甘霖般舒畅。生病之前买下的裙子和高跟鞋因为错过了季节，崭新地在衣柜里落寞着，染发剂也束之高阁，任由灰白的新发疯长。

这场病其实去年夏天就有所征兆，只不过疼痛尚能容忍，而且两周之内自行痊愈了。所以只是自己贴了几副伤痛膏药，继续踩着高跟鞋每天加班3个小时。作为一个40岁的女人，我实在有些失败。在这座二线小城里只有一套房、一部车和一个孩子，长年累月地加班，从来没有接送过女儿上下学，更别说参加她的任何社会活动。贫困的定义不应该仅仅是金钱，而应该是没钱也没时间。

公司是一家超过万人的国企，进公司12年没有换过岗位，因为工作性质特殊，主要部门都常有交道，部门老总们或多或少给几分薄面，公司诸位老总跟前也常能露脸说话。外人眼里的我，样貌能力学历样样都拿得出手，也成功主持了大大小小多个技术项目，这么热热闹闹拼命三郎似的一个人儿，就是死活升不上去。虽然一直清高地给自己灌心灵鸡汤，但是看着同年的、年轻的呼呼地从身边往上升迁，夜深人静时内心的悲愤还是像喷泉一样往外涌。

2月，又一个心力交瘁的项目结束了，我开始高烧不退，这是去年夏天以来第四次高烧。领导说去北京读个短期班吧，别人都去过了。离开校园多年内心是渴望能再回去的，所以用了大量抗生素把高烧压下去登上飞机。现在想想，万事皆是因果循环，一切冥冥之中自有安排。第一次不是因为工作跟其他部门的同事相处，又因为生病失声而只能听不能说，10天的哑巴生活豁然发现过去的12年我活得太狭隘和孤独，好像在一条泳道里孤身奋力前行，水花四溅，冲刺一个又一个华丽的纪录。现在停下来离开泳道才发现被水花溅到的人感受恐怕不那么好，而我所呈现的工作狂面目也告诉别人“工作就可以满足我，不必把世俗的待遇浪费在我身上”。意识到真相的我五味杂陈，人的心眼瞎起来还真是天地难容。

我的病痛在高烧之后正式来临，先是小腿长出莫名的红包，然后发展到全身关节肌肉疼痛到生活不能自理，抽了几千块钱的血也没说出个究竟。其间工作上发生了几件事情，最严重的是一个涉及运行安全的差错，我的小团队成了最合适的背锅人。新上任的公司老总在我不够级别的干部大会上点了我的名，依照以往我那铁血丹心的性情必然四处申述，闹个人仰马翻。然而病痛让我成功地避开了旋涡，等再回到公司时，时过境迁，真相是什么已经不重要了。在中国人的文化里，弱者往往收获更多的同情，在公司强势了十几年完成多个不可能完成的任务，只有这次才感受到真诚的关怀。人说“江山易改，本性难移”，唯有病痛才能改变一个人的性情，因为身体已无法支持我孤胆前行的灵魂了。

在不得不放缓节奏的三个月里，我慢慢学习成为一个“有城府”的人，凡事慢半拍，给自己和别人冷静思考的时间，考虑合作伙伴的感受和目的。不再是眼里只有任务和时限，减少加班的时间，给团队更多的信任，给自己更多的宽容。水至清则无鱼，物至极则必反，很多糊涂的人未必能力欠缺，也许只是看破不说。感谢上天，对于过去那个一身钢甲、寒光凛凛的我再也看不下眼了，在不惑之年，以这样的方式给我最好的启迪。痛与不痛，得到与失去，未来应该还会继续在过去的惯性中滑行一段，但是无论如何节奏已经改变，时间会带来效果，一切都是最好的安排，欢喜就好。
