# 个人问题_有猫的日子

个人问题

有猫的日子

文◎大路

朋友出差，把猫带了过来，让我帮忙代养一周。送过来的时候，猫被装在一只旅行拉杆箱里。朋友担心它会窒息，拉链拉开了一个小口。看到它时不时把鼻子凑到小小的孔洞，试探性地咻咻呼吸，好像换了环境空气也连带变得陌生。从朋友手上将猫接了过来，瞬间觉得手心沉甸甸，心想一定是只体重超标的猫。

我不养猫，所以对猫的品类不了解也不挑剔。有时觉得如同我的恋爱观，没有特别喜欢某一类人，并要具备某种特质，只信赖感觉。毫无疑问，它是一只让我喜欢的猫。

一个朋友跟我讲他养狗的故事。辗转几个城市，终于在北京落脚。一天，他在南锣鼓巷看到老太太拿一只狗崽兜售，第一眼就觉得那只狗是属于他的，于是当即买下，不曾想就此一起生活了将近十年。他们极其依赖对方，也极其默契，好像再不会有任何人或别的宠物能取代彼此的位置。

猫咪在我家的第一晚，一直藏在床底不出来，我倒出猫粮给它吃，尝试引诱，也无济于事。因为担心，跪坐在地板打开手机电筒，观察床底下它的动静，发现它只是一动不动蹲踞在角落，定定地注视着外面，灯泡一样闪闪发亮的眼睛传达出警惕。终于走开决定不去管它，直到已经遗忘了它的存在，才突然发现它从床底悄无声息幽幽地出现在眼前，跳到床上沙发上，再跳到冰箱上，开始一点一点逐步熟悉它的新环境。这种若即若离的关系，会让人觉得疼惜中又带着痛苦。

下班回来，我一个人窝在沙发上看书，偶尔抬头活动脖颈，发现它蹲在几座默默和我对视，满眼好奇，于是在尽可能不惊动它的情况下拍下它一对困惑的眼睛。有时它静静趴伏在地板上，没有睡觉，它脑子里在想什么我不知道，好像周围一切和它不再有任何关联，于是拍下它寂寥无辜的样子。还有时候，它会因为困倦在我不曾注意到时安心进入昏睡。我看着它，感觉它好像变成了一个柔软的婴儿，让人不忍触碰。给它盖上衣服，然后再次把它眯眼昏睡的各种姿态照进手机。

身边有很多朋友养猫。他们在各种社交软件里频繁更新猫的照片。那种黏腻暧昧的关系，让人心生错觉，好像猫不再是他们的宠物，而是自己的孩子，或者情人，是寂寞孤独时的一个陪伴。

一年冬天，我跟A短暂恋爱，一只英国短毛猫是A的陪伴。下班一进门，因为疼爱，A会抱起猫来转圈，给它拍很多照片。有时又会把猫藏在被子里。我们在电脑上看电影，不时转头看它，它仿佛睡着，藏在被子里一动不动，保持着被塞进去时的姿势。不注意它时，它又会跑过来蹭人的脚，或者肆无忌惮跳上桌子在电脑旁走过来走过去。当时恋爱的情形早已变成一片空白，唯一记得的是那只猫。

中学时看很多小说。爱伦坡、夏目漱石、张悦然，都曾写过关于猫的故事。印象中的那些故事，充斥灵异和惊悚，看得让人窒息。女作者尤擅长把这种诡秘和阴森的气氛用一种巧妙的方式放大，一直记得张悦然的那个故事，结局极其惨烈。一只怀了孕的猫，被父亲打断了牙齿又丢出门外，最后被发现死在没化干净的一场春雪里，身体早已被密密麻麻的虫子掏空，腐烂的眼睛还冥冥地睁着。看完觉得惊心动魄。当时想，以后一定不要养猫，比起人类，寄人篱下的猫的命运会像一张蛛网，随着人类情绪的起伏而飘摇，是如此的无常。有时它们也会从一个人的生活中猝然失踪，像热恋很久的人突然不告而别，让人猝不及防。

凌晨五六点钟，我还没起床，它就跳到我的床上，在床上到处随意走动，拿鼻子蹭我的耳朵，发出呼噜呼噜的声音。我会觉得它吵闹，又疼惜它的柔弱，试图把它搂到怀里。但每次都是在一阵激烈的挣扎后仓促逃走，让我落空。

它被送回去的那天，突然觉得屋子里异常安静。翻出手机看这几天给它拍的照片，看完逐一清空。床和沙发全是它掉落的毛发，拿出滚刷全部仔细粘过一遍，直到终于没有了任何关于它的痕迹。一切又回到原来的样子，好像从来不曾有猫进入过我的生活。
