# 个人问题_气质使然

气质使然

个人问题

气质使然

文◎黑羊（绍兴）　图◎陈曦

从前就觉得孔是个要出国的人，甚至无可厚非地觉得他的做派本就应该完完全全不同于身边的人。孔的一股子“成功家”气质早早在高中时代就显露了出来，虽唯独不在成绩上，但和我这种被老师嗤之以鼻的“烂泥扶不上墙”之流还是有本质区别。孔1.8米多的高个子，颇为帅气，却是皮肤黑得多了点。孔的运动神经好得令人发指，大概也是得益于自修课上得少，篮球场地上的汗水出得多，理所应当地成为以一当多的存在。打球独的人多的是，可是到最后还总是把球传到他的手上，看他一次次地跳进内线用各种奇异的方式把球打进。然后他又怪异地懂鞋子，说得出各种鞋子的款式和价格，我好奇为什么会有这样的爱好，然后他给我轻描淡写地说有一段时间没什么事干就看杂志，然后杂志上有些球鞋的广告，看得多了就懂了。我一边“噢”着一边等他对自己这样的描述做一番评论。只看到他甩甩手，和女生打羽毛球去了。

孔家里有钱。老师什么都知道，以至于训话的时候要分门别类地交流，老师自然不能和他说什么你家里有本事就出国什么的，因为那本事肯定是绰绰有余的。吃完饭去小卖部的时候，在他旁边一起的肯定有丰厚的收获。倒是养成了一批男生守株待兔一般的“好逸恶劳”。其实孔绝不是那种每个班级里都有的刻板人物，也绝不是什么不良学生，不然绝衬不起“气质”二字。要怪只怪学校校风不羁，任学生自由发展，要产生一批奇人。每天6点10分看孔准时到校抄作业，练就一身子神速抄作业的绝技，也没见他考在班级下游.抽屉混乱地层叠各种考试卷，以及丢了不知道多少次的无数张饭卡，但每次都穿戴得整齐，冬天穿一件无比扎眼的纯色艳红羊毛衫也不觉得瘆得慌.每次晚自修偷溜出去上网，他也总是慢悠悠地等一班公交车，然后在下课之前稳重地赶回学校，佯装刚下课去迎接他的妈妈。

抱着一把子的脑容量和极佳的运动天赋，却要把时间“蹉跎”在所谓尘世间表面的“浮华”上，好像是看到现世的达·芬奇穿一身破烂，却赔着笑脸求你赏几个钢镚去玩赌博机的道理一样。孔的行事风格向来具有那种骨子里的所谓大气，或者要衬得出在高中时候就已经成形的我这类市井小民的小家子气。男生总是戏谑着说他装B，其实是对“气质”这东西显示在别人身上就自惭形秽的遮遮掩掩吧。记得年级的篮球比赛，我们班憾负的时候，大抵全场的人都在惊叹他的精彩表现，然而他确是有些失落，极苦闷地去冲了个头。

孔的交友技巧可以用“精妙绝伦”来形容。粗犷的如每逢节假日就塞几千块的巧克力，细腻如打个电话，转身出现在女友身后之流。女朋友陆陆续续地换了不少，貌似现在的那个和最初的那个是一个人。他倒是意外饶有兴致给我讲什么：女人这东西一辈子能喜欢的有很多很多，真爱的可能只有一个。大约只有这事他想要稍微为自己辩解一下。

时至今日，就只有每逢假日无限消磨同学友谊的聚会看得到他了。前一次看见他，“放荡不羁”地搞了个辫子，穿一身嘻哈风格的衣服，倒是有了几分美国青年的味道。晚饭又是他付了账，大概大家都不好意思，这次一起摊了钱转给他。饭桌上聊天，我颇有兴致地问一些美国的事，他还是那副无所谓的状貌来回答我，只挑些琐碎小事，倒是重点谈及什么韩国留学生女孩子。他总还是会“成功”的，我们班的人都一致这样地想，我也不曾敢否定过他，倒多的是仰慕和羡慕，只是好奇他至今为止从来没发过什么朋友圈。
