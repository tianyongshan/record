# 体育世界_是否应该继续嘘加特林？

张斌：是否应该继续嘘加特林？

体育世界

是否应该继续嘘加特林？

文◎张斌

8月6日，2017年国际田径世锦赛男子100米决赛打响，世界纪录保持者博尔特（右）仅跑出9秒95成绩，排在加特林（中）和科尔曼之后，位居第三

没有人怀疑博尔特对于奔跑的热爱，但最近半年间，他不爱训练了则是事实。伤病拉扯着他的脚步，好友罹难让他有理由低沉倦怠。三个月前，博尔特自己便有预警，一再强调100米跑技术难度的精细，稍有不慎便会江山尽失。天赋与欲望这一次没能拯救低落的博尔特，实在跑不动了，是该离开了，尽管再过几天他就31岁了。

9秒95，铜牌，人生最后一战，赞誉不绝，乔丹与阿里此时都要出场，与之比肩伟大与命运结局。人生不完美，看客早有心理准备，但让加特林如此抢戏可是不少人忍不得的，就连国际田联主席科勋爵都毫不掩饰地说：“这不是一个完美的剧本。”何为完美？不是一定要博尔特不输，而是加特林本不该有赢的权利。直说吧，两次兴奋剂检测不过关，理论上要领得12年禁赛期的加特林居然可以逃过终身禁赛，在田径历史的伟大瞬间中去扮演亮色，够荒唐吗？根据安排，主席先生是要亲自颁出这块最具影响力的金牌，够为难的吧！

伦敦，一座对于田径运动似乎充满洁癖的城市。两年前，在北京，加特林出场绝对不会嘘声满天的，关键处还会有掌声。伦敦则不同，去体育场的班车上、热身时、比赛全程，哪怕是加特林走进酒店办手续，都会听到各色嘘声。据说，最极端的一幕发生在酒店自助餐厅内，服务生对着加特林冷冷地说：“你的煎蛋需要添加睾丸激素吗？”也许，这些嘘声如今大多是响在人们的胸腔之间，但世锦赛组织者还是认真地考虑将男子100米的颁奖仪式安排在18点50分，此时恰是晚间赛事全面开始之前，观众上座平平，少来点儿嘘声，谁的脸上都好看些。

以加特林的个性，嘘声前他唯有隐忍，将近6年了，这就是他人生的一部分，未来怎样也不可知。这一次，哪怕全世界都在嘘他，只有一个没有如此，那就是博尔特，一个需要他在获胜后也要情不自禁单腿跪地膜拜的王者。漫长的对抗，博尔特始终是赢家，甚至可以在掠过终点线时，斜睨一眼“世人眼中的骗子”。这一次，恩仇全然无意义，执着的该是世人。

牙买加体育部长就很执着，眼见着“最坏的剧本”，愤怒谴责世界反兴奋剂组织（WADA）不作为，姑息养奸，让田径世界蒙羞，“终身禁赛是唯一可以约束不在体育世界中欺骗的办法”。本次世锦赛，英国田径名宿恩尼斯也从科勋爵的手中领回了一枚原本该属于她的2011年大邱世锦赛七项全能金牌，原来的拥有者俄罗斯选手被查出服用禁药，英国人不依不饶，面对迟到的正义，依旧是一片嘘声。恩尼斯深感遗憾，同时指责WADA如果早就严格执行终身禁赛的法条，何来这庄严时刻的满场嘘声呢？

加特林今年三十有五，绝对高龄。15年前，第一次被查出服用含有违禁物质的药物，自然领得4年禁赛。不甘命运如此，加特林上诉，理由听起来很硬气，自小罹患多动症，长期服用的治疗药物中不慎含有违禁物质。高抬贵手，重返赛道。2006年，又一次倒在药检中。再申诉，又获宽待，再战江湖。虽免于“死刑”，但加特林俨然是“死”过一次，复生的感觉真实而惨烈，最深的痛莫过于再无机会证明自己说辞的真伪，在众人冷漠中，他唯一做到的是，在运动不再需要他的时候，他是如何渴望这项运动的，凡对此漠视的人都是在拒绝认识这个真实的世界。

田径世界的真实是这样的，伦敦奥运会男子百米决赛赛道上，博尔特的7名对手中有5位已经被查出服用兴奋剂。对此，加特林不敢做出任何一句评论，他当下最关心的是，儿子有意愿参加东京奥运会百米比赛，上阵父子兵有可能吗？
