# 体育故事_312场连败之后的胜利

张斌：312场连败之后的胜利

体育故事

312场连败之后的胜利

文◎张斌

卡罗尔学院女子篮球训练计划发起人哈奇（摄于2012年）。在2017全美高中篮球赛事中，卡罗尔学院的姑娘们终于迎来胜利

11月29日，孟菲斯一所私人学校小小的篮球馆里，全美高中篮球赛事中312场的连败纪录悄然终止了。身着白色球衣的一群姑娘泪水奔流，欢腾在一起，她们的球队花名唤作“美洲豹女士队”，从田纳西的亨廷顿小镇奔赴来此，做客于此。毫无准备，这些在尴尬纪录开始蓄积的初年2005年，不过是蹒跚学步孩童的姑娘们这一刻真心地为自己的学校——卡罗尔学院感到自豪。

经过两个加时赛，卡罗尔学院的姑娘们终于迎来胜利，29比27的比分并不能完全将比赛的焦灼呈现出来，第四节最后时刻的一记三分球中的，留住了“美洲豹女士队”胜利的微小可能。过往十几年，每个赛季来临，球队的主教练都会盘算，如何能抢得一场难得的胜利，强行去终止那不断被刷新的尴尬纪录。这一刻终于来了，当球队中的头号球员凯特琳投中那个压哨三分球，把比赛拖至加时赛，全队情感第一次爆发出来，所有人都痛哭不止。

卡罗尔学院由卡罗尔县少年法庭负责运营，颇似早年间我们也有过的工读学校，专门为那些失足或者失去家庭的孩子们提供教育的机会。大约10年前，卡罗尔县亨廷顿镇上的一家服装加工厂关门了，旋即造成这一地区的失业率达至前所未有的高点，同时伴随高涨的还有人口总数，多数家庭举步维艰，家平均年收入不过3万美元，只有15%的人曾经受过高等教育。

贫困快速滋生了犯罪，药物滥用的程度让人崩溃，12～17岁的青少年群体中滥用止痛药物的状况列在全美第六位。更令人心悸的是，这里暗藏的地下冰毒制造窝点一度是全美最多的。在如此糟糕的环境中，任何一个少年都有可能迅速滑向黑暗的深渊。卡罗尔学院初创时的心愿是想庇护那些并不把家庭视为天堂的可怜孩子们，随时都会敞开大门，接纳从县里少年法庭出来的孩子们。

卡罗尔学院校规极严，身着统一校服的学生们每天早晨走进校门前，必须经过严格的搜身检查，以防毒品被带入校园，甚至圆珠笔都不需要使用，因为有孩子曾经在笔管中藏毒。1994年，学院创办四年后，其创始人哈奇发起了篮球训练计划，让孩子们从球场上找到关爱，找到快乐。仅仅一年后，卡罗尔学院被田纳西州中学体育协会接纳为规模最小的会员，自此开始参加区域性比赛，也就有了各种比赛纪录的积累。

学院里男孩子比例高达80%，组队后成绩尚可，但全校女孩子不过十几个，最极端的时候，仅有的9位女生齐刷刷地入队，因此也才有了那个持续12年的312场连败纪录。尽管连战连败，但哈奇依然自豪于学院的篮球选择，用他的话讲，数年的努力，终于让社区的人们意识到，“这些孩子终于又像是我们自己的了”。

最让哈奇绝望的时刻，是那些原本在篮球场上有了些朝气的孩子们被再度发现偷盗家中的毒品带进学校，每逢此时，他都要狠下心把孩子们暂时送到教养所，等她们日后的回归。

“美洲豹女士队”中，球员身材各异，她们当中有的误入歧途，接触过毒品，或者早早就有了身孕。篮球能给她们什么呢？首先，是一个有形的团队，每个人都要心甘情愿置身于此，这对于缺乏家庭关怀和教育的孩子们至关重要。再者，希望孩子们能够意识到，自己也要被别人依靠的，只有相互支撑才有希望。

连战连败，每当比赛结束后，队员和教练们其实是在相互躲避对方的眼神，大家都不想从眼中读到绝望。12年间，没有错失一场比赛，“美洲豹女士队”送走了一位又一位女孩子，她们的人生路也许就此完全改变，也许最终逃脱不了宿命。但在赛场上，当看到对手球员的家长们汇聚在场边时，胜利真的那么重要吗？她们要的是温暖的家。
