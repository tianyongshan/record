# 体育故事_关岛只是个“幌子”？

宋晓军：关岛只是个“幌子”？

体育故事

关岛只是个“幌子”？

文◎宋晓军

2017年8月29日，朝鲜发布的弹道导弹“火星-12”发射现场图片（视觉中国供图）

9月3日下午朝鲜宣布，朝最高领导人金正恩下令进行了氢弹试验，试验取得成功。看到这则消息，我想到了4天前一家朝鲜媒体发文嘲讽美国海军接连两次发生“撞船事件”的新闻。据9月1日美《新闻周刊》网站披露，朝鲜媒体在文中说：当美帝的“宙斯盾”舰沉入海底时，弹道火箭从东方飞向太空，巨大的雷声震撼了世界……想到了这则新闻后，我也就想到了这篇专栏的题目——关岛只是个“幌子”？

8月30日，美“北纬38度”网站发表了一篇英国国际战略研究所（IISS）导弹防御专家埃尔曼（Michael Elleman）对朝鲜8月29日试射“火星-12”的分析文章。文中埃尔曼最担心的是，8月29日朝鲜试射的“火星-12”虽然按正常弹道只飞行了2700公里，但假设朝鲜是在3次分别采用“高吊弹道”试射“火星-12”和“火星-14”后，第一次用“火星-12”测试用于提高弹道导弹飞行距离和命中精度的“末级助推器”（PBV），那就说明朝鲜要发展弹道导弹威胁美国及海外基地是认真的。当然，埃尔曼分析PBV可能没有启动工作，因此“火星-12”只相当于搭载了一枚1200～1300公斤的弹头飞行了2700公里，最后分3段溅落入日本以东的北太平洋海域。而这3段很可能分别是“火星-12”的主火箭壳体、PBV和弹头。埃尔曼认为，也许再过1～2年“火星-12”的技术就会成熟，而这将是朝鲜弹道导弹技术一个“令人不安的进展”。

说到这儿，再说美海军的“撞船”与朝鲜对关岛的“导弹威胁”。根据今年3月17日美国会研究局公布的一份报告显示，到2017年底海军将有34艘反导“宙斯盾”舰。负责西太平洋至印度洋区域的第七舰队有10～14艘“宙斯盾”舰。在“菲茨杰拉德号”和“麦凯恩号”两艘“宙斯盾”驱逐舰先后被撞“失能”后，就意味着第七舰队的海上反导能力下降了14%～20%。考虑到第七舰队的反导舰艇还有维护与训练的非部署时间，第七舰队很可能在朝鲜试射弹道导弹时会出现监视/拦截空白的“窗口期”。虽然现在还不能得出判断，朝鲜是在借助这个“窗口期”故意用向南“射击关岛”作为“幌子”而实际上是要向东对“火星-12”进行PBV测试，但从客观结果上看的确是有这种可能的。如果再进一步推论，假设第七舰队的“宙斯盾”舰不仅没有发生“撞船事件”，而且也装备了可拦截射程在3500公里以上弹道导弹的“标准-3Block2A”拦截弹（美国会研究局“报告”显示在2019年才能装备15枚），此前朝鲜声称要对关岛进行“包围射击”的这个“幌子”还会有吗？换句话说就是，在美海军第七舰队有了足够多的搭载了“标准-3Block2A”拦截弹的“宙斯盾”舰后，朝鲜对关岛进行“包围射击”的这个“幌子”可能就没有那么大的“媒体吸引力”了。

最后我想说的是，虽然前面所说的是基于假设，但朝鲜借助关岛这个“幌子”进行了“火星-12”的飞行试验及随后进行了“氢弹试爆”却是不争的事实。朝鲜第三代领导人接任后，不仅进行了4次“成功”的核试验，而且还进行了远比他父亲多N倍的弹道导弹飞行试验。应该说，朝鲜的“核、导能力”在近两年来有了长足的进展。这不仅说明了奥巴马政府之前的“战略耐心”没能遏制住朝鲜“核、导能力”的进展，而且也说明了特朗普新近推出的“极限施压”也未必就能奏效。那么接下来特朗普政府会接受中方提出的“双暂停”倡议和“双轨并进”的思路吗？如果不接受，他有什么办法止住朝鲜“核、导能力”进展呢？
