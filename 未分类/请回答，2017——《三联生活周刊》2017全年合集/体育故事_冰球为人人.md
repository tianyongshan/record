# 体育故事_冰球为人人

冰球为人人

体育故事

冰球为人人

文◎张斌

2月10日，华盛顿首都队在华盛顿威瑞森中心对战底特律红翼队赛前，阿联酋冰球姑娘阿尔·阿里和华盛顿首都队88号球员合影

2月，体育大事多多，稍不留意，有些努力都留不下痕迹。27岁的阿联酋姑娘法蒂玛·阿尔·阿里2月8日悄然从阿布扎比起飞，目的地华盛顿，巨大行囊里装了全套的冰球装备，这位阿联酋女子冰球国家队的成员心向最爱的球队华盛顿首都队，最心仪的冰球英雄奥维希金正在冰面上等着她的来临，一次注定终生难忘的冰上训练将永远珍藏在这位蒙着面的穆斯林姑娘心中。

NHL为这个月拟订了“冰球为人人”（Hockey is for Everyone）的主题，意在让全球本与冰球或者这个伟大联赛并无缘分的人可以有机会体会运动的快乐。去年11月，《华盛顿邮报》的冰球专栏作家邦德拉在阿布扎比的一座冰球馆里发现了阿尔·阿里轻盈滑行的身影，简短交流后得知在这个沙漠围城的所在居然还有如此热爱冰球的女孩子，随即他向NHL推荐阿尔·阿里出任“冰球为人人”系列活动的形象大使。既然深爱首都队，赞助商慷慨地提供了机票，来一次冰球奇缘好了。

生在阿布扎比，而并非那些敏感的穆斯林国度，这让NHL和华盛顿首都队的公关人员如释重负，海关绿灯放行。落地华盛顿，朝拜首都队的球馆，七家电视台的摄影队闻风而至，穆斯林——时下最热的关键词，镜头齐刷刷地对准阿尔·阿里兄妹两人，黑色的头巾此刻格外显眼，这送上门的好故事，记者们跃跃欲试。群访刚开始时，大家还算克制，聊聊冰球，谈谈偶像，欢声笑语，不知哪位记者第一个忍不住了，开始追问起阿尔·阿里对于“限穆令”的观感。“对此，我非常不高兴的。”阿尔·阿里没有回避。这还了得，就怕被政治热点话题绑架，首都队的公关人员匆忙起身，宣布记者会就此收场。

追在阿尔·阿里身边的不仅是记者，还有来自华盛顿当地穆斯林社区的不少长辈，他们看着远道而来的姑娘心里挺别扭的，好端端的穆斯林女子为啥要倾心于一项男人统治的强对抗运动，冰场上如何才能时刻保有穆斯林女子的礼仪风范呢？阿尔·阿里不会在意身旁的议论，在家乡阿布扎比，她和姐妹们参与冰球训练和比赛早就做到了自由自在，这都啥时代了。

华盛顿的旅程没有几天，阿尔·阿里闹着让哥哥去协调能否再多留两天，多看场球，与偶像多几次见面的机会。首都队足够贴心了，特别安排阿尔·阿里走进球员休息室，奥维希金从天而降，给了一个超级大惊喜，好一顿拍摄，各种社交媒体，穆斯林姑娘根本无法抑制见到运动偶像的激动心情。一起在冰上滑行训练，那更是几世修来的好福气。

10日，首都队主场迎战底特律红翼队，赛前坠球的殊荣自然交给了阿尔·阿里，穿着宽大的首都队比赛服，她乐得像个刚刚拿到新玩具的小姑娘。现场广播中，主持人字正腔圆地渲染着：“她的美国之行再次为我们树立了榜样，一场比赛很简单便可以让我们穿越政治和地理的桎梏和藩篱，让我们在一块平等的运动场上团结在一起。”不知道现场有多少人听清楚了这个长长的句子，阿尔·阿里只顾着在球场中间利用那宝贵的契机与偶像奥维希金和红翼队队长一道来张超级自拍，她飞速掏出手机，问了一句：“可以自拍吗？”得到愉快回复后，连拍数张，搞定了。旋即，冰球迷们就在社交媒体上迅速看到了这张精彩的自拍，点赞连连。此时，现场广播又响起来了：“谢谢你！法蒂玛，祝你在美国旅途愉快！你是受欢迎的！”比赛第二天，阿尔·阿里返回家乡，她真想再回美国，因为那里有她深爱的运动，冰球某一时刻是在为她而闪耀。
