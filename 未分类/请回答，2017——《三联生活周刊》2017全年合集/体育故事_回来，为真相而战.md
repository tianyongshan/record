# 体育故事_回来，为真相而战

回来，为真相而战

体育故事

回来，为真相而战

文◎张斌

2016年10月10日，莎拉波娃在美国拉斯维加斯出席一场慈善网球赛

时光易逝，4月19日，莎拉波娃即将度过30岁生日。再过一周，也就是26日，长达15个月的禁赛期即将结束，斯图加特保时捷公开赛恭候已久，特意在开赛之后的第三天，用一张外卡邀约莎娃入局。从头再来，五味杂陈。

2016年初，曾经的世界第一被兴奋剂事件重重绊倒，反俄罗斯的兴奋剂处罚行动正处高潮，两年禁赛更像是羞辱，人们也因此隐约记住了那种神秘禁药——米屈肼，吃了10年，正邪之间，神鬼难料。莎娃的态度一直很强硬，自认无辜，几经努力，将禁赛期减至15个月，并声言此次回归，一定再为“真相而战”。在莎娃的逻辑中，仲裁法庭既然可以恩准减刑，那至少可以证明，她并非是长期有意服用禁药，无心之过处罚过重了。

禁赛之初，记者终日蹲守，执着地想拍摄到莎娃在海边的一举一动。日子一长，也就淡然了，莎娃也更加自在，难得可以逃离网球的束缚，过上虽不舒心但也闲散的日子，她走进哈佛商学院课堂，像模像样地学起了全球战略管理，还去了伦敦，完成了10天的领导力课程。学以致用，莎娃去了一家广告公司实习了一段时间，甚至还获得了一个令人艳羡的机会，一周时间伴在NBA总裁萧华身边，见识体育领袖的日常工作。与NIKE设计师一道工作一周，继续美化自己的产品线。当然，还会拿出更多的时间去参加自己芳名下甜蜜糖果的各类推广活动。

不知不觉，没有网球的日子过得很快，莎娃声言，自己完全适应了这样的时光，那张外卡邀约让她回到了“应有的日常工作之中”。以上的种种感受相信一并被写进了今年9月份出版的自传之中，每天的时光都有其价值，莎娃自感“被赋予了新能量”。人前风光，背后的训练不曾停歇，莎娃渴望回到赛场，任何训练都无法替代比赛，15个月过后，身心尚有动力，在未来的日子里，尽力成为自我能力所及的最棒的网球运动员。

面对媒体，莎娃小心翼翼地保护自己，更为隐秘的内心感受拒绝分享。记者们只好找到此前同样有禁赛经历的斯特里科娃，这位当下排名第20位的捷克选手2013年因兴奋剂问题被禁赛半年。刑期虽短，但斯特里科娃感触至深，重回赛场的第一场比赛后，她恨不得在床上可以躺3天，哪怕你此前每天训练6个小时，你也休想一上来便可以适应艰苦的比赛，那完全是两回事，紧张的情绪会让损耗极大，因此她担心莎娃复出后会有一段低沉的日子。

禁赛期内，所有受罚者都仿佛困兽一般，不少人含恨在心，需要不短的时间才能找到积极应对的心态，逐渐正常起来。斯特里科娃在那段日子里，先是在家里哭了两个月，抱怨不公平，慢慢地开始自己洗衣服，去邮局取邮件，这才是真实的生活，而日常巡回赛奔忙的日子太不真实了。遭此一难，斯特里科娃自认，最大的收获是看清了谁才是真正的朋友，莎娃也会真切体会到人间冷暖。

但很显然，斯特里科娃不太喜欢莎娃，她不忘提醒记者们注意，莎娃在女子网球世界里是没有朋友的，她实在太高冷，巡回赛期间，不和任何人说话。没有朋友，莎娃的回归之路会更加寂寥，她自己似乎做好了充分的心理准备，持外卡，在第28号场地比赛，没有观众，那时候你才会知道，自己到底是谁，一切都需要从头开始。斯特里科娃复出时，因为排名已经跌至第230名开外，只能在斯图加特公开赛打资格赛。为了积分，她要低下高傲的头，去参加奖金总额不过2.5万欧元的小比赛。好在，复出第一年年底，排名反弹到了第90位，好日子又回来了。需要莎娃隐忍很多，所谓真相也许只有一个——她的确是个相当不错的网球选手。
