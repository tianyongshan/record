# 体育故事_小德：多么痛的领悟

小德：多么痛的领悟

体育故事

小德：多么痛的领悟

文◎张斌

网球巨星诺瓦克·德约科维奇

“最坎坷的道路通向最美的目的地。”对小德充满怜惜的网球评论家们，也许早早就准备好这句话来描绘如今世界第二过往一年的球场内外困顿，以及探寻自我重生的种种企图。

去年此时，小德正在向一个跨年度的大满贯奇迹进军，如日中天绝不为过。但此后一年间，温网第三轮败走.奥运会首轮输球，哭成泪人.以伤病理由，避赛多场，黯淡无光.去年底，将保持122周的世界第一头衔拱手让给穆雷。今年1月以来，大多数的赛事都是止步于八强门槛，与世界排名前20的选手对阵，胜绩寥寥。29岁，拥有12项大满贯冠军之后，小德迎来“断崖式崩溃”。

都说真正的勇士，敢于直面对惨淡的人生。小德没有急着做勇士，他首先要面对的是笼罩四周的猜疑，甚至是守在家门外的塞尔维亚小报记者们的窥探，那些人只想坐实曾经的世界第一对自己新婚妻子的不忠。去年温网恰似拐点，早早出局后，小德依旧气盛，直言：“输球不是因为伤病，也非身体状况不佳。我经历一些私人事务，好在一切都解决了。”一定有着很痛的领悟，“我的处境与所有人都一样，我们都会有些私人处境，面临巨大挑战，我们还会面对很多事情，只有克服，才能像人类一样进化。”

一个人的痛彻领悟，往往不过是在营造众人的想象空间罢了。小德唯有小心护佑着家庭，当妻子再度身怀六甲且悄然现身训练场时，全世界都舒了口气，巨星的家庭总算回归幸福轨迹。但与此同时，属于小德的“网球家庭”破裂了，尽管此前已有征兆，但当小德解散过往10年与之并肩前行的团队时，还是让很多人错愕不堪。小德未曾直接面对公众，而是用一篇《我想再找回在球场上胜利的激情》的长文袒露心情：“我最想重新获得的就是对于胜利的渴望，我是个猎人。”去年最困顿之时，小德曾有如此论调：“我心里承受了巨大的压力，不再考虑冠军的数量了，冠军来了，泰然受之。总之，网球不再是我生命中的唯一。”

此类的感悟，小德的前辈们或早或晚都曾有过，升华者无限接近网球自由世界。但人的境界成长绝非仅仅是金句闪耀，荣耀的光芒是要靠苦难打磨的，有时间成长，才有机会成功。网球一线高手流行礼聘昔日名宿为自己的心灵导师，小德有贝克尔，费德勒曾依赖自己的偶像埃德伯格，穆雷有伦德尔加持，目的都是要重塑心灵，找到网球世界里的综合解决方案。小德颓废之时，贝克尔公开申斥他“有工作道德伦理问题”，并留下一句“成功可不是按一下电钮那么简单”。

道理尽人皆知，克己践行者才终成伟业。此次小德辞退团队，被描写为“清理房间”的举动，但冒着巨大的风险，在一个“独自上场”的运动中，没有人可以独自获胜的。小德坦言，他会找寻一位有着卓越网球经历，并与其有相似人生的前辈作为新的教练团队核心，“在网球世界里，没有太多人打出如此高的水准，有过掌控大格局的经验”。

最残酷的竞技场，需要找到最信任的人托付内心，李娜遇到卡洛斯，就曾找寻到破解心魔的路径。费德勒青春不老，也得益于去年在瑞士深山中的一段苦修。小德自然如是，昔日宛若家人的团队不再有此功效，因此才会有一位名叫艾兹马的昔日西班牙职业网球选手翩然入局。此人也曾征战职业巡回赛，但成绩平庸，退役后以贩售心灵平静之法行走职业网坛。就在小德茫然无措时，两人相遇，冥想之法成了日常功课。在小德球员包厢里，时常会见到这位穿着特异的西班牙人，赛前赛后他会找到小德，相拥一段时间，也许这就是心灵抚慰吧。强者，不得不修心。
