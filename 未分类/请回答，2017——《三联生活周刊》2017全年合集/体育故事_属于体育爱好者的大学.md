# 体育故事_属于体育爱好者的大学

属于体育爱好者的大学

体育故事

属于体育爱好者的大学

文◎张斌

2014年9月26日，洛杉矶市一所大学的预科生们去上体育课

“疯狂三月”让我们心里痒痒的，一句“把体育还给教育”，那简直就是中国体育的未来简史。简言之，教育正常，体育昌明，别无他途。美国有本杂志名叫《金钱》，热心服务家庭理财，但《金钱》绝不仅仅柴米油盐，发布的榜单也很洋气，比如“全美性价比最高大学排行榜”。今春，《金钱》还会拉上《体育画报》“疯狂地”推出“属于体育爱好者的百家最佳大学排行榜”（100 Best Colleges for Sports Lovers），此榜单不枯燥，读来春风拂面。

榜单很讲究，少有烟火气，并非是运动成绩大比拼，你Final Four多少次、我哪些年拿冠军，而是站在体育爱好者的立场上，品味606所大学的运动风尚与积淀。评选遵循两大价值取向，首先是运动表现（占比40%），再者则是校园体育爱好者们的综合感受（占比40%）。当然，有个前提是必需的，学校应有能力提供一流的教育（占比20%），这部分《金钱》积累厚实，24个指标建立价值维度，校友毕业后的成功度也计算在内。

常闻听痛切感悟——“中国就是缺乏体育文化”，至于何为体育文化，定义模糊。《金钱》和《体育画报》将校园体育文化用15项数据分解，逐一打分。在校运动员学生的毕业率排在首位，过往四届奥运会贡献了多少奥运选手、10个赛季里为主流职业体育联赛贡献多少球员、10年里收获多少全国大学生赛事冠军、校际间比赛的运动项目总数、运动员学生获得学位的比例，以上意在考察运动表现。体育爱好者的综合感受则体现为，学校比赛的场均观众数、校内开展运动项目的数量和场景优劣、针对校内精英体育和大众体育的捐助情况、校内有多少体育专业可供攻读、校园40公里范围内学生可以看到哪些职业体育赛事。

该翻看榜单了，斯坦福大学高居第一，运动表现总分第一，爱好者感受第19位，教育水准第10位。要说斯坦福真可以改名体育大学了，四届奥运会，贡献47位选手，过往10年，17项全国冠军，94%的学生运动员能在联邦规定的六年期内读完四年本科。圣母大学列在第二，三项排名都不高，运动表现居然仅仅第21名，但有50名毕业生在顶级职业联赛中闪耀，圣母大学橄榄球很强，场均观众人数8.0795万人，主场外四位前任主教练的塑像巍然屹立。密歇根大学很厉害，教育水准全美第二，学校主体育场能容10万人，场场爆满。去年学校体育募款2530万美元，透着人气旺。杜克大学篮球太有名，77年历史的石制体育场堪为圣地，“蓝魔”在橄榄球方面也是强队，一年的体育捐款甚至可以超过3000万美元。弗吉尼亚大学悄悄来到第五名，10年间夺得51个分区赛冠军，校园内开展56项运动，又一个体育学院。北卡教堂山分校哺育过“飞人”，男篮场均观众1.8328万人，女足21次全国冠军，天下无双，位居第六，基本合理。佛罗里达大学的学生运动员学业表现平平，但几年前成就过篮球和橄榄球双双登顶全国冠军的奇迹，10年16项全国冠军，51人在顶级联赛效力。

名列第八的明德学院很有意思，艺术氛围浓郁，但全校居然有27%的学生活跃在各个校队中，这是何等美妙的运动氛围，10年内获得7个全国冠军。耶鲁大学排在第九位，去年夺得了54年来的第一个常春藤锦标赛的冠军，赛艇和曲棍球很强的。教育水准全美第一的普林斯顿大学在这个榜单上列第19位，过往10年拿下123个分区赛冠军，20%的学生有校队经历。俄亥俄州立大学列第18位，体育爱好者感受全美第一，2.8808万名学生持有学校橄榄球队季票，2500名学生参加60个运动俱乐部活动，去年体育捐款达到3350万美元。
