# 体育故事_岁月留痕_观众做主

岁月留痕 观众做主

体育故事

岁月留痕 观众做主

文◎张斌

2016年6月20日，法国欧洲杯B组第3轮比赛在圣埃蒂安的热奥弗鲁瓦-基查尔球场举行，英格兰对阵斯洛伐克,最终两队0比0战平。图为英格兰队众将黯然离场

岁末，盘点忙，各行各业，各色数据。英国人很细致，铺陈开全年的电视收视数据，给每个节目排排坐。事关体育，结论五味杂陈。2016，体育好年份，以电视版权定义的价值增长继续保持高速。但体育与电视的关系有变，定义价值和贡献价值依然意义巨大的电视正在失去受众，以BSkyB为例，英超本赛季电视收视率下降19%，有的专家们提醒说，19%有些乐观了。

其实，绝非体育电视传播在缓慢下滑，英国电视整体都在承受着类似的苦痛。两年前，巴西世界杯，BBC转播的英格兰队首战，吸引观众1360万人。2016年，虽然也是体育大年，但没有一场直播的观众人数可以超越这个。最佳表现是欧洲杯葡萄牙与法国冠军之争——1230万，威尔士与比利时的四分之一决赛观众人数为1090万人，“三狮军团”小组赛0比0闷平斯洛伐克，勉强达至1020万人。评论家尖酸地提示说，足球虽然还是体育转播中的王者地位，但是追随“三狮军团”的人会越来越少，怪谁呢？成绩难看嘛。

此话听起来好像很有道理，但并不能证明追逐足球直播或者“三狮军团”的观众在发生着断崖式减少。谷歌英国的一个数据告诉我们，2016年中，搜索英超联赛直播网上视频流的明显在增加着，英国广播者观众调查机构给出的结论则是，当下受众更容易受到Twitter和YouTube上播出的赛事集锦的吸引。BT与BSkyB未来三个英超赛季中，将联手贡献超过50亿英镑的版权费，为了争夺受众，他们也会在社交媒体和各自的官方网站上释放进球集锦，久而久之球迷们会依赖于此，逐渐远离整场直播。如今美国的NFL转播也遇到类似问题，球迷们贪恋网络视频，疏离付费转播，最令人匪夷所思的趋势预测居然是：“美国人还会狂热地爱着橄榄球，但是也许再也不看整场比赛了。”

趋势大体如此，但是如此表述还是让人不太能够完全接受，体育传播的进化也许会比想象的更复杂一些，2016年尚不是拐点真正来临的年份，但至少可以慎重地做出这样的表述：“我们正在看到消费者的进化过程，人们正在加快拥抱数字平台的内容，而逐渐断网电视。”还有另外一句话，也是英国人在研究了2016年电视收视数据后得出的：“价值定义的方式将重新来过，BSkyB的付费足球模式将不再无往而不利，人们正在向网络转移着。”

在英国，体育电视的乏力并非始于眼前，但就此唱衰则也有些操切了，电视依然成就大赛和关注度极高的赛事，穆雷温网夺冠时，920万英国人用遥控器签到.里约奥运会，大英军团的夺金高点时，也有820万人聚集.曼联击退水晶宫，保住足总杯荣耀，650万人在电视机前欢腾。需要注意的是，以上数据都是免费模式下的BBC播出。至于说付费平台BSkyB呢，2016年最高点源自联赛杯决赛，曼城激战利物浦，237万人.利物浦与曼联赛季两回合交手，都是接近200万，似乎都不太澎湃，可以达成生意的平衡点，但是对于付费体育电视未来生命力没有提供强有力的背书。

如果数据一再缓缓下滑，英国更为忧虑的是，英超和其他足球版权的价值会不会缩水呢？此时，评论家们的调调是这样的，体育正处于黄金时代，但成熟市场潜力不足了，未来就看新兴市场啦。比如说，中国不久前就以12倍的力度刷新了英超版权合同嘛。不仅是中国，肯尼亚也该被看好，在那里球迷都是捧着手机看英超直播。新兴市场给人无限想象，走着瞧吧，最终还不都是观众说了算嘛。
