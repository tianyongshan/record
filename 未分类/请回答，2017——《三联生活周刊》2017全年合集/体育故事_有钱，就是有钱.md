# 体育故事_有钱，就是有钱

张斌：有钱，就是有钱

体育故事

有钱，就是有钱

文◎张斌

8月26日，梅威瑟（右）和麦格雷戈的跨界大战在拉斯维加斯开启，梅威瑟获胜

周六子夜时分，拉斯维加斯街头，一处处狂欢刚刚拉开帷幔。一辆布加迪2012款运动敞篷跑车夜色中划出一抹浓艳的红色，轰鸣向前。在一家脱衣舞俱乐部门前守候的记者们辛苦没有白费，用镜头从价值300万美元跑车里迎下了驾驶者，同时也是这家俱乐部的实际拥有者——40岁的梅威瑟。梅老板摆下庆功宴，尽情享受人生最后一胜的快意与富足。

梅老板从来都不掩饰奢华与恣肆，因此总也难逃暴发户的恶名。深夜里，他在Instagram上为1780万粉丝不断推送着庆功宴中的劲爆场景，不出所料，又有人冲上舞台，向着下面癫狂的拳迷们大撒美元现钞。这就是梅老板的手笔，“金元大战”的永恒赢家喜欢如此分享成功，但他很不希望外界就此随意给他贴上标签，四个孩子的好爸爸忠诚于家庭才是他最根本的生命角色。

凌晨三点，赌城的夜色里，一家酒吧的大门被重重地推开，麦格雷戈阔步而入，那些从祖国爱尔兰和全球而来的拥趸们已经等候已久。麦格雷戈虽然没有狂撒现钞，但爽快地为今夜在此狂饮的人们埋单，那可是一笔不少的钱，账单显示为77643英镑。这是小意思，几年前还在靠低保活命的麦格雷戈自从入行UFC打一场像样的比赛差不多可以有300万美元的收入，早已锦衣玉食。这次一道现身全球关注度巨高的“体育表演”，29岁的麦格雷戈保底收入便是3000万美元，大约相当于UFC生涯的所有家底了。

把格斗赛当表演看，其实是不错的心态。因此，“金元大战”也被定义为拳手与行为艺术家的对决，大家会心一笑。生涯50战全胜的梅威瑟也两年未曾登台了，终极之战被渲染得像是在拯救地球，保底收入先来1亿美元，最好收成有可能迫近2亿美元。拳手的收入盖世无双，艺术家的终极收入预期可达8000万美元。

有钱，就是这么有钱。一场最终以TKO结局的十回合比赛综合创造价值总计超过7亿美元，PPV（每场单独付费）电视收益超过4亿美元，现场两万观众贡献近8000万美元票房，博彩业收入明暗两道甚是活跃，梅威瑟一人这场比赛的装备赞助金额就接近2000万美元。这是啥概念？比赛当天，全美电影票房收入不过4500万美元，这是由12部电影联手实现的，真是拼不过一场表演。

八月底一向是电影票房最惨淡的时节，假期结束，孩子们得收收心了，一家人出门看场电影不容易。但端坐家中，点击支付99.99美元看一场被渲染得无以复加的搏击赛还是很划算的。目前还没有PPV销售的准确数据，但不会少于500万用户掏出真金白银的，这可又一次刷新了职业拳击PPV纪录。前记录是梅威瑟和帕奎奥2015年联手创造的，由2007年梅威瑟和德拉霍亚联手激发的245万户飞升至460万户，被誉为业界的“比蒙式一跳”。

PPV是北美职业拳击的命根子，好比赛一次性便可收割亿元量级的收视费。梅老板之所以富可敌国，那可真是号召力决定的，PPV销售前三强他都是一号主角，整个职业生涯总计收获近2000万订户，参与413亿美元的价值创造，不服不行的。

不知各位是否注意到了此番梅威瑟与麦格雷戈一战，开始时间拖延了半小时。公开的原因是用户最后时刻增长迅猛，电视付费播出平台对应吃力，UFC TV开赛前索性崩盘了，无奈之下，只好让比赛等等，让每个花了100美元的用户都能顺畅地看到信号，上帝的感受很重要。不只是这次，其实两年前与帕奎奥一役，足足等了一小时，同样在等最后下单的疯狂。比赛是竞技，也是表演，还是买卖，谁埋单，谁是价值核心，都要清清楚楚的。
