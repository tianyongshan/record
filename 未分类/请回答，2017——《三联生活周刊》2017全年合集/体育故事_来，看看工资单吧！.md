# 体育故事_来，看看工资单吧！

张斌：来，看看工资单吧！

体育故事

来，看看工资单吧！

文◎张斌

9月12日，2017~2018赛季欧冠小组赛B组，巴黎圣日耳曼队对战凯尔特人队（东方IC 供图）

过往三个夏天里，大巴黎远渡大西洋，在美国几个城市里与同为豪门的球队一起为球迷们热闹热闹。没想到，热闹之后，大巴黎的心里凉凉的，因为全队上下看不到球迷穿着他们的球衣赶赴盛会，满眼都是别人家的锦衣招摇。恨不能成为纵横四海的全球化顶级豪门，大巴黎下定决心不惜代价买进一位超级巨星。因此，才有了波折不断耗资近6亿美元的内马尔转会，也才让大巴黎在职业团队运动薪酬榜上，一举跃升五位，以650万英镑的平均年薪挺至全球足球俱乐部次席。谁是第一呢？是巴萨。那谁又是全球总第一呢？是雷霆。

内马尔的年薪很透明——3680万欧元，好事者算过，每小时约计3200欧元，足球顶薪了。梅西新合同周薪为50万欧元，年薪不过2600万欧元罢了。顶薪到底意味着啥？专门的研究机构给了我们这样的答案，一份名为《全球体育工资调查》（GSSS）的报告显示，单是内马尔一人的年薪，大致与法国、德国、英格兰、美国、瑞典、澳大利亚和墨西哥七个女足联赛1693名女孩子收入的总和相仿。

这份报告已经是连续第八年发布了，今年首次将女子运动员的薪金涵盖在内，力求达成男女平权、同工同酬的美好心愿。但所有人都知道那不过是心愿而已罢了，NBA与WNBA、英超与女足联赛的薪酬差距基本上都是几十倍的概念。全球最富有的女足俱乐部里昂队为队中的英格兰国脚露西支付的顶薪是14.5万英镑，而英超的平均周薪历史性攀至5万英镑，折合年薪264万英镑，排在NBA（550万英镑）、MLB（343万英镑）和印度板球联赛（299万英镑）之后，全球第五，足球界第一。

英超风光，薪酬最高的前三甲俱乐部还是要让位给巴萨、大巴黎和皇马。去年的第一位曼联（平均年薪520万英镑）走了节俭路线，一干球星走后，没再乱花钱，但就是这样还仅仅是滑落到第四位，领跑曼城一点点。曼市双雄自然是很有钱的，但在全球薪水排行榜上，要排到前20位之后了。雷霆队今年替代去年头名骑士队成为薪金水平最高的球队，平均年薪接近千万美元（929万美元）了，骑士队和勇士队以接近900万美元分列第二和第三位。值得一提的是，前12位中有9支NBA球队，另外三支就是足球三甲。

在全球薪金最高的12个男子职业联赛排行榜上，我们会看到中超的名字，排在前面的有北美四大、欧洲五大、印度板球，还有就是列在第12位的日本职棒。这12联赛跨越三大洲、10个国家、6项运动、257家俱乐部，共有7265名球员从中受益。报告披露，中超总薪金数为5.032亿美元，平均每位球员101万美元。上港队毫无悬念位列中超第一，球员平均薪金达到241万美元，权健紧随其后，209万美元，华夏幸福也属于200万美元俱乐部中的成员。申花队178万美元列在第四，换得足协杯和亚冠双重收获。恒大滑落至第五，近150万美元。百万俱乐部成员还有鲁能以及苏宁，辽宁和延边在这个榜单上最终的位置也一如人们所料。

去年报告中，中超便很显眼，甚至有评论将其定性为“最倾斜变态的体育联赛”，其背后含义很好理解。今年报告中，研究机构认真分析了恒大的财务状况，指出这家中国最为成功的足球俱乐部2016年的收入可以达到6400万英镑，其中电视版权收入占比近五分之一，赞助合作收入近七成，门票收入550万英镑。而维持这支雄心万丈的俱乐部一年的花销接近1.6亿英镑，总之一点，盈利几乎不太可能。按照去年的统计，中超的薪金水平是美国职业大联盟的两倍，是J联赛的三倍。此种走势还会继续吗？答案就在2018年。
