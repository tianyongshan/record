求购C.罗与“金球专家”

体育故事

求购C.罗与“金球专家”

文◎张斌

2016年12月18日 ，C.罗在日本横滨参加“2016世俱杯决赛”

2016年欧洲杯，威尔士火火的，围绕着其主帅科尔曼的是各色励志故事和“心灵鸡汤”。苦心孤诣这么多年，总算扬名立万，科尔曼悄然进入名帅系列，未来绝对不会再愁工作合同了。果不其然，威尔士的斯旺西队江湖告急，欲求新帅，科尔曼根红苗正，岂不是上佳人选吗？美国投资人摇动橄榄枝，得到的回应不过是，科尔曼夫人在Twitter上的婉拒。莫非，斯旺西队本赛季只能对付一名补锅匠了吗？

这条消息夹杂在节礼日大战等爆款新闻之中并不那么醒目，反倒是中国有钱人求购C.罗的消息更突显些，如果足够有耐心，你还会遇到特维斯被中国俱乐部拿下的图文报道，此前的奥斯卡来华已经被迅速更新掉了。面对中国big spenders（大富豪）的邀约，C.罗的回答几乎是标准答案——“钱不是一切。”这条消息不足半天便可登陆所有足球新闻的空间里，中超联赛就是如此震天动地，中国人买矿山、买油田、买品牌之后，如今求购个把巨星，全看心情了。

大家都在猜度给C.罗报价的买家到底是何方神圣，有钱人其实也就那么几家而已。有钱的确不假，但买C.罗此类巨星，总给人要唱堂会之感，抑或是在和谁置气。因此，人人都很关心有钱人的逻辑何在？暂且按下中国有钱人不表，回到文章的开头。斯旺西队的拥有者是两位美国老板，跨过大西洋收购了这家中下游球队。在他们心目中，职业足球终究还是个买卖，因此每分钱的投入都会仔细度量，无论未来谁来补锅，谁来扛大旗，都要在俱乐部中和一位经济学家长期合作，问计于专家，这个角色姑且称为“金球专家”。没错，就是《Monryball》电影里布拉德·皮特扮演的那位，最会算计的人，以小博大，帮助俱乐部买到性价比最高的球员。在配置球探都费劲的中超俱乐部中，花钱找“金球专家”似乎不现实，大多数账还得是自己在心里来回翻来倒去。再说了，有钱人下的一盘大棋，顾问未必能体味深意的。

美国人崇尚数据，一年要出好几本体育数据理论的书，以证明一双慧眼何等宝贵。投资人让斯旺西队插上了理论的翅膀，获封“金球专家”的这位名唤阿尔特曼，哈佛名校毕业，经济学博士，为《经济学人》《纽约时报》和《国际先驱论坛报》等撰写专栏文章。“三狮军团”挫败于欧洲杯后，在《经济学人》专栏中，阿尔特曼博士试图向世界解释清楚，英格兰国家队咋就如此不堪呢？博士的观点其实也不那么受用，“既然英国能诞生出通灵大师达伦布朗，为什么就不能拥有一位伟大的10号球员呢？”这明显是将问题引向死胡同，好似“我就不明白了，13亿人为什么就选不出11个会踢球的？”

文章高明与否，并不影响博士的买卖，他在纽约运营一家数据分析公司，专为俱乐部提供转会决策服务。在美国投资人看来，过往三个转会窗口，斯旺西队转会买卖简直一团糟，如何拨乱反正呢？那就让科学说话吧。据说，斯旺西队体面地结束了上赛季后，博士曾高瞻远瞩地预料到新赛季必然不妙，还连用了好多个“哈哈哈哈”。

用数据说话，提供新的算法，这该是阿尔特曼博士公司的核心价值，可以为用户提供长期的数据分析服务，指导决策，让拍脑袋彻底灭绝。不管文章是否靠谱，但博士有一句话还是对的：“数据并不能孤立地说明问题，足球是一项复杂的运动，任何一名球员的表现都是由队友的支撑才完成的，因此不可轻言结论。”

当下花大钱投资中国足球的各类角色，各怀心事，都是各行各业的精明人，没有“金球顾问”也能做成买卖，特别是要花钱买个镇宅之宝或者门面的时候，问题其实很简单。
