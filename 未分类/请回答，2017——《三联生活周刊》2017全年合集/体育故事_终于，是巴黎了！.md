# 体育故事_终于，是巴黎了！

张斌：终于，是巴黎了！

体育故事

终于，是巴黎了！

文◎张斌

9月13日，（前排左起）巴黎奥申委主席托尼·埃斯坦吉特、巴黎市长安妮·伊达尔戈和法国奥委会主席丹尼斯·马赛利亚等人得知申奥成功后在会场欢呼

两位巴黎市政厅的官员一直在掐算着时间，来自利马的消息总算来了，他们随即扯下了巨大彩色五环上的白色帷幔，远处埃菲尔铁塔如约入画，花都梦圆。早该没有悬念了，但经历过2005年申奥的法国人还是心有余悸，决战日步入投票大厅，巴黎一副胜利者姿态，结果四票败给伦敦，申奥苦情戏中泪雨飘飞。

巴黎角力洛杉矶，前者容易引人疼惜，25年间三次申办折戟，眼睁睁看着100年未再被五环垂青，洛杉矶好在30年前曾办过一届呢。连败连战，巴黎人完全不在意奥林匹克新时代的种种窘迫，任凭别人意兴阑珊，我自壮心不已。探寻胜因，巴黎人的回答笑中带泪：我们最大的财富是失败的教训。此番功成，《队报》头版只有一词——终于！道尽几多甘苦。

12年前，巴黎败走，重要原因是布莱尔和小贝最后时刻的从天而降。反思，再反思，申办奥运不再仅仅是老旧帝国思维中的大国外交博弈，突出运动、烘托运动员才是最好的姿态。两年前，巴黎正式提出申办时，三块奥运会皮划艇金牌得主埃斯坦盖与市长安妮·伊达尔戈一道出任奥申委联席主席，这一姿态并非哪个城市都心甘情愿。法国前总统奥朗德公开表示，政客在申办中应该后退一步，主角该是来自运动世界。

姿态如此，但实操中政治家的作用无可替代。巴黎再启航是奥朗德2012年奥运期间，在伦敦探望法国运动员时第一次提及的。去年里约奥运会，又是奥朗德四方示好，恳请世界认同巴黎的超级能力，空袭之下毫无惧色，将欧洲杯办得生机盎然。新总统马克龙也坚决执行国家战略，入住爱丽舍宫第一天，便约见伊达尔戈市长，力挺首都申办。第二天，又邀请国际奥委会考察团成员为总统府上宾。其实，只要是马克龙当选，国际奥委会便算是服下定心丸，只要不是极右当道，奥运可安。虽然谁也不方便说，洛杉矶领得2028年不也是全拜总统所赐嘛，风险控制不得不做啊。

想当年惜败伦敦，法国人痛切感到了政坛不团结之弊，左翼市长和右翼总统貌合神离，岂有不败之理。团结，在这个人人愿做决断的国度里变成了一致的选择，这一次并非同一政治阵营中的总统和市长识大体，顾大局，给国际奥委会一片祥和的绝佳印象。别小看这个，法国人做到不容易的。

申奥投票总是少数人的事情，说服手中握有选票的委员们即意味着打动全世界。吸取2005年惜败原因，巴黎人将沟通说服工作做得愈加细致，伊达尔戈市长几乎与所有委员都有过会面，甚至利用担任世界城市气候领导力组织主席的契机，也在宣讲巴黎的环保诉求，承诺2020年前减少25%的城市碳排放。事实证明，这很有效。

绝佳姿态之下的勤恳务实，巴黎人做到了。除此以外，还要有不俗的策略，更环保之外，还要更节俭，才更吻合奥林匹克时代主题。巴黎的办奥预算68亿欧元，办赛和城市建设几乎各半，只需新建5%的场馆，其中还包括不少的临建场馆，原则只有一个，能省则省，一些建成近百年的巴黎体育设施将焕发荣光。

奥运会将给巴黎带来荣光之外，还会有哪些实在获得呢？巴黎市政府用规划做出了回应，与悉尼、北京、伦敦和里约几乎一致，用奥运力促城市的平衡发展。2024年奥运会开幕式想必还是在法兰西体育场，该体育场地处圣丹尼斯区，移民众多，沉疴遍地，奥组委规划在那里兴建游泳跳水中心、奥运村和主新闻中心，并承诺新建的游泳中心未来将向社区全面开放，期待着这一举措可以尽快改变法国青少年仅有一半人会游泳的尴尬局面。老百姓要的不多，奥运会最终要实实在在有用。
