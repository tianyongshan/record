# 体育故事_致敬传奇

致敬传奇

体育故事

致敬传奇

文◎张斌

6月10日，在牙买加首都金斯顿举行的2017跑者大奖赛上，博尔特出战100米，本土告别战夺冠谢幕

在一场重大比赛前，在体育场馆内，有人引领着数万人一道郑重地做祈祷，我想绝大多数人一生也不会有此见识。但这一幕果真发生了，就在金斯顿国家体育场，3.5万名狂热民众，持续五小时欢呼、呐喊、鼓噪和起舞，只为目睹“最伟大的牙买加人”博尔特在自己祖国的第84次也是最后一次比赛，一道致敬传奇，目送他奔向今夏退役的终点。

赛前的祈祷词是特别撰写的，无须做任何调整，博尔特百年之后也完全可以刻在墓碑之上。“感谢上帝！你厚爱牙买加。你赐予我们医生，赐予我们荣耀的尤赛恩·圣·莱奥·博尔特，他是运动家精神的化身，他充分体现了牙买加人的积极进取和不可战胜的精神。愿这体育场能成为公平和爱的世界，让你的国度成为地球上的天堂。阿门！”

宗教的仪式感让每一位在场的人都热血沸腾，他们中的绝大多数人都是掏8美元买了门票，端坐在木质的椅子上，等待盛典来临。人的声浪不足以持续将情绪保持在高点，很多人随身携带着类似于南非呜呜祖拉的吹奏神器，齐声共鸣，就好似雄壮的蚊虫军团正在恐袭金斯顿。

这样的夜晚，30岁的博尔特此前也许曾畅想过，在国人面前最后一次如风跑过。2002年，15岁的博尔特迈上了国家体育场的跑道，身边都是比他足足大三岁的对手。枪响，起跑，第一个冲过200米终点，夺得世界青年田径锦标赛金牌，自此名扬天下，岁月就是不断地穿行在起点与终点。

好兄弟骑摩托车丧命，博尔特足足有两个星期未曾训练，他也知道在这场众人精心策划的赛事中，没有人再期许纪录的改写，只期待他用田径场上最短的时间留住无尽的记忆。没有巨星环绕，此番比赛的对手大多是故友，只需要50米，博尔特便进入了巡航状态，我猜这一刻，他格外贪恋如潮的欢腾。所谓退役，就是与这样的景象说再见。博尔特一再强调，定将思念看台上的每一个人。

10秒03，一次罕见的低水平表现，你要知道职业生涯中，博尔特至少50次跑进10秒之内。博尔特坦言，真是糟糕，起跑不好，一切都不如意，两个星期没训练就是这样的结果，但令他异常感动的是，牙买加人集体表达了对他的支持。掠过终点，啦啦队和摄影记者们一拥而上，将博尔特包围起来，早就准备好的烟花照亮了金斯顿的夜空。赛前，博尔特曾经乘坐敞篷车绕场两周，尽享万众膜拜。赛后，他几乎以最慢的速度绕场奔跑一周，努力向看台上的每一位致意，奔向看台高处，与父母相拥，和牙买加舞王酣畅起舞。这个夜晚，就是这个国家能给博尔特最好奖赏，博尔特在Twitter上真诚言道——“感谢！牙买加！”

博尔特今年开始创办跑者大奖赛，虽然奖金很低，但凭借他的号召力，依然是好手如云。国际田联主席塞巴斯蒂安科先生对赛事评价颇高，此番特意赶到金斯顿，与当今田径运动的头牌热烈拥抱，并留下一句话——“你若谦逊，我等赞美如山。”再过两个月，在伦敦，世锦赛现场，博尔特将正式作别，他承诺必将有华丽表演，用胜利为职业生涯作结，但要暂且放下连续四届奥运会夺金的传奇梦想，实在是飞不动了。

今夏过后，世界田径将进入一个“后博尔特时代”，几年内实在难以找寻到合格的接班人。8块奥运金牌、11块世锦赛金牌，完全可以定义神一般的地位，但博尔特其实一直在抱怨，媒体对他不够友善，未来的纷扰也不会停歇的。顾不了那么许多，博尔特急切地要开始自在的日子，牙买加政府许以要职，希望凭借他的力量推动“牙买加宝贝儿”公益项目。
