# 体育故事_莫·法拉赫的反击

张斌：莫·法拉赫的反击

体育故事

莫·法拉赫的反击

文◎张斌

莫·法拉赫在2017田径世锦赛上获男子1万米金牌、5000米银牌（视觉中国供图）

女儿指着电脑里图片问我：“他为什么摔倒啊？”所谓“他”，就是莫·法拉赫。所谓“摔倒”，那是5000米人生最后一战，没能完成最后超越，扑倒在地痛楚充盈于心的一瞬间。我的解释挺苍白，小孩子没听明白，没了兴趣，跑向别处去玩儿了。

真是完美古难全，连续五届大赛1万米和5000米“双金王”，两度伦敦“双金王”以及告别赛道的“冠军结局”都在最后一秒钟化为银牌铸就的残缺壮美。与此前所有比赛的剧情展开全然一致，法拉赫对战非洲军团，暗战惊心，对手们使用一切手段只想阻隔莫·法拉赫接近终点。太多次，莫·法拉赫独骑闯关，最后时刻彪悍的冲击让阻拦者和追赶者一道陷入绝望。博尔特冲刺那是闪电劈开天地，而莫·法拉赫的决胜则好似惊涛骇浪吞噬一切。

行文至此，一定有人会提醒，修辞要注意，尽可能不要用这么多形容词，哪怕是你内心真实感受，也显得不高级，再者给自己留余地啊。直白地讲，莫·法拉赫虽然兴奋剂检测至今清白，但谁能担保在一个有争议的教练麾下缔造出的杰作未来会不会有什么惊天逆转等着让世界猝不及防呢？

26年前，8岁的莫·法拉赫以难民身份逃出索马里，落脚伦敦，自此开始不凡人生。选择奔跑，让他成为旷世的田径英雄，尽管有人如今还再窃窃私语，这个索马里来的年轻人不过天资平平，是那个叫萨拉扎的美国教练才成就了他。至于那个萨拉扎，常年带着队员躲在科罗拉多高原上修炼，莫·法拉赫自2011年大邱世锦赛后的每一块金牌中都有他的塑造之功。也就是这个萨拉扎最近两年一直在接受美国反兴奋剂组织的严密调查，身有嫌疑，但始终没有定论。

两年间，莫·法拉赫一次次狂飙般掠过终点，但必须一次次面对责难，很多人的头顶上都多了一个巨大的问号。曾在萨拉扎训练营中寻求成绩突破的美国女选手卡拉·高彻告诉BBC记者，萨拉扎给过她甲状腺药物，并声言此种药物具有“治疗用途的豁免权”，也就是说一旦阳性是有退路的。卡拉拒绝了萨拉扎，坚称自己并不需要。

这一鲜活细节被反复提及，加深了不少人对于萨拉扎军团的怀疑，莫·法拉赫的一次错过飞行药检则几乎让想象插上了翅膀。本届世锦赛1万米决赛后，莫·法拉赫仅接受了对于自己友善的媒体采访，这进一步激怒对立面，氛围诡异。战完5000米，莫·法拉赫总算解脱，第二天登上伦敦塔，极目远方，直抒胸臆，向对立面开火了。“历史永远不会撒谎的。过往这么多年，我所取得的成就，众人引以为傲。你可以爱写什么就写什么。事实上，我的成就都是艰苦付出和信念换来的，为我的国家而奔跑。”

针对萨拉扎，莫·法拉赫声明，过往两年，他一直都是在英国田径队的训练营中度过的，与恩师之间交集不多，只是接受一些遥控的指点，到了当下的人生阶段，他自己完全可以掌控训练和比赛。另外关于他与长期用药的另外一名教练关系不凡的传闻，莫·法拉赫愤懑地回应，所有写出的新闻都靠想象，而非事实，真正的事实是这么多年了，自己付出太多辛劳，承受了别人不能想象的伤痛。这些靠联想写稿子的记者就是要毁掉他的成就，针对这些人，莫·法拉赫撂下一句话：“如果想证明我做错了什么，那就请拿出证据。”

明年，莫·法拉赫将正式转型为马拉松选手，他拒绝透露是否还会继续师从于萨拉扎，但对于更长距离的竞赛，他本人充满了期待，因为那是异常开阔的赛道，可以躲避开来自于对手的干扰。由此可见，这些年来跑道上的心酸实在不少，如今挥别，就留与后人评说吧。
