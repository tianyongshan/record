# 体育故事_谁都可能需要这金色的肩膀

张斌：谁都可能需要这金色的肩膀

体育故事

谁都可能需要这金色的肩膀

文◎张斌

8月17日，在美国亚利桑那州斯科茨代尔国家高尔夫俱乐部，菲尔普斯与哈克特（前）

不久前，亚利桑那州坦佩城一间嘈杂的餐厅里，人们几乎没有注意到城中最有名的居民菲尔普斯正在身旁饱食人间烟火，而与其对面而坐的另一个大个子，那就更不会有人愿意多看上一眼了。其实，两个大个子正在进行一场十分有趣的对话，日后等孩子们长大了，他们将如何解释自己曾经的荒唐举动。那个面孔陌生的大个子是澳大利亚两块1500米自由泳奥运金牌得主格兰特·哈克特，菲尔普斯曾经的对手，孙杨最初的榜样。

这场讨论不过是两位游泳巨人心理健康治疗再聚首中的一个小环节而已，37岁的哈克特飞越13000公里，不止一次投奔菲尔普斯夫妇，只为从黑暗的日子里早早脱身。在澳大利亚，哈克特曾经拥有恰似美国最成功四分卫般的国民拥戴，悉尼和雅典奥运会两块1500米自由泳金牌实在炫目。功成身退，迎娶娱乐明星，转年便拥有一对龙凤胎。投身商海有人助力，很快便有了企业家头衔，大学里轻松修得学位，电视台还为其准备好了游泳池边最重要的那个话筒。简直了，就是完美人生。

但光彩未能彻底包裹住哈克特滥用药物和酗酒的不堪，家庭破裂，争夺一对儿女抚养权的官司败北，人们曾经目睹国民英雄在墨尔本一家酒店大堂里酩酊大醉，穿着狼狈至极。今年2月，哈克特的父母报警，称自己的儿子情绪崩溃，酗酒不休。澳大利亚人通过电视新闻看到了哈克特被警方移送行为治疗中心的过程，举国震惊。离开治疗中心后，哈克特失踪了一段时间，父母通过媒体向儿子喊话，“快回家！亲人愿意提供所有帮助。”

当时正与妻儿一道在巴黎欢度情人节的菲尔普斯闻听好友哈克特出事了，旋即短信规劝躲在酒店房间里的哈克特敢于面对残酷现实。菲尔普斯妻子诚邀哈克特来家里做客，努力走出黑暗。伦敦奥运会后，菲尔普斯经历了炼狱人生，吸食大麻被英国小报曝光，两度酒驾临近牢狱之灾。

在心理治疗中心的八周时间，让菲尔普斯重获新生，自此他对心理健康颇有心得，与妻子一道致力于帮助一切可以帮助的人。一位11岁的加州游泳少年求助菲尔普斯，称自己近两年间一直与自杀的念头对抗。菲尔普斯如实相告，自己第二次酒驾被抓住后，也有过自杀了断的念头。倾心交谈后，少年从焦虑中缓解过来，并玩笑说：“我终于知道了自己与菲尔普斯之间能有这么多的相似。”

当初，菲尔普斯走出治疗中心后，面对公众说的第一句话便是：“我在泳池中曾经不可战胜，但我最终还是个人罢了。”巨星往往会被想象成为无所不能的超人，但重压之下，往往早已发生扭曲，平日里按照团队的设计，以最可爱的面孔、最合理的谈吐、最恰当的姿态与世人见面，久而久之，总会认定是别人变了，其实是自己开始不正常了。因此，菲尔普斯最能理解运动世界里苦苦挣扎的同类。伍兹此番出事，虽非醉驾，但滥用药物之名算是坐实了。第一时间，菲尔普斯托人找到伍兹，表示愿意帮其解除心中病痛。在菲尔普斯看来，伍兹与哈克特倒是在家庭分裂、痛失孩子抚养权后才真正坠入心理深渊的。

伍兹在如何挣扎，外人无人知晓。菲尔普斯拯救哈克特的过程清晰可见，他推荐好友进入专业机构治疗，陪着彻夜深聊，在泳池里一道轻松与水交流，在高尔夫场上欢声笑语。据哈克特本人讲，如今按照菲尔普斯的指引，他逐渐可以找到自己生命中的舒适区了，脱离澳洲鱼缸般的生活，当下更平衡了，他也从心理医生那里找到了向自己孩子解释荒唐行为的具体做法，新生活即将开始。

哈克特认定，无论你是伍兹也好，还是白丁也罢，每个人总都有人生的问题需要面对，他的幸运即在于有菲尔普斯这个金色的肩膀可以靠一靠。
