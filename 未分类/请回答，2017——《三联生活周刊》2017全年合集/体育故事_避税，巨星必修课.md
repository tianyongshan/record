# 体育故事_避税，巨星必修课

体育故事：避税，巨星必修课

体育故事

避税，巨星必修课

文◎张斌

曼联队主教练穆里尼奥涉嫌逃税遭起诉

英国下议院议员玛格·希勒女士，履新公共财物特别委员主席，马上便盯上了曼联队主教练穆里尼奥。她公开指出，鉴于西班牙税务部门已经启动对于穆帅的税务调查，英国方面自然也要相应地启动程序，希望可以证明，2004～2007年间，在切尔西执教期间，今年54岁的葡萄牙名帅是否采取同样的手法，进行不合理避税。

这是今夏以来爆出的第二起足球税务案，C罗更加喧闹些，甚至威胁要因此永离西班牙。很显然，穆帅与C罗同属一个经纪公司，门德斯一手打理两人的财税事务。C罗被爆在前，穆帅涉案金额不多，不过290万欧元，但即便如此，如果不能在法庭上洗清罪名的话，依法可判处5年徒刑，1740万欧元的罚款。去年，梅西深陷税务诉讼，最终律师辩词中，强行将父子两人切割开来，以梅西完全不知父亲所为为由，让球王避祸，其父则难逃刑期，但法院也算法外施恩，给了21个月的刑期了事。西班牙法律规定，初次犯法，刑期在24个月以下的，可以免除牢狱之灾的。

既然，西班牙税务部门一年来，连续高调地喊打喊杀，英国税务局自然不会放过已经被撕开的突破口。来自于英国税务局措辞严厉的函件不久就会寄达穆帅家中，《星期日泰晤士报》近日透露，2004～2007年间，穆帅逃避缴纳个人形象收入所得税1000万英镑，去经纪公司将收入先是转账至爱尔兰，进而又汇到英属维尔京群岛的一家壳公司，而这家壳公司的是由穆帅名字创办的一家新西兰基金控制的。在西班牙税务部门公布的材料中，通过这家离岸公司，穆帅为自己在皇马期间的收入至少避税100万欧元，凭此线索，英国税务局呼啸而至。

去年底，一家名为“足球解密”的网站便四处散播C罗和穆帅逃税的新闻，德国《明镜》周刊率先引用“足球解密”的材料，透露说，C罗近1.5亿欧元的个人形象代言收入被转至所谓的避税天堂，轻松地仅仅付出了564万欧元的税款，税率不足4%。“足球解密”和《明镜周刊》在正式发布消息之前，曾将所有材料提交给门德斯方面，希望能够得到合理的解释。最终，他们得到的是一份律师函，坚称无任何罪责，经得住税务调查。

曾几何时，西班牙可是足球巨星的税务天堂，皇马“银河舰队”初创之年，贝克汉姆得到了意想不到税务优惠——长达6年的不超过24%的税率封顶，而西班牙6位数的年薪基本上是要一半被税务局拿走的。2010年后，被称为“贝克汉姆法条”的最优税务政策被停止，60万欧元年薪以上的都要老老实实遵照西班牙税法纳税。

巨星们的收入何止薪水，巨额的代言费占比越来越高，在球员经纪公司一厢情愿认定，只要散步在全球的收入不进入西班牙境内，即可逃脱西班牙税务部门的“魔爪”。显然，他们过于天真了，西班牙税务机构最近几年终点稽查运动员收入，无论球员的收入得自于哪个海外国家，都必须证明与自己所签署的西甲合同没有关联。即使没有关联，也需要积极申报的。最让巨星们绝望的是，一位西班牙税务部门官员曾说：“我看所谓海外收入，只要是穿着西甲球衣，我看都该算是与原有合同相关，必须纳税。”

纳税，天经地义。避税，人人所愿。但在合理与非法避税之间，往往只有那么一根细细的红线。但请球迷放心，西班牙税务部门往往高高举起，轻轻放下，为巨星们准备的刑期都不过24个月，他们要的是税款，不要人犯。未来离岸公司避税法将不再那么流行，经纪公司还会继续努力去寻找我们这个星球上的避税天堂，魔高一丈，可不是随便说说的。
