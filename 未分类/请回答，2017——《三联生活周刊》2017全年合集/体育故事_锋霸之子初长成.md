# 体育故事_锋霸之子初长成

锋霸之子初长成

体育故事

锋霸之子初长成

文◎张斌

足球运动员贾斯汀·克鲁伊维特

即将上演的欧联杯决赛居然比往年多了几分情趣，曼联的欧冠路径如此奇绝，阿贾克斯的青春风暴扑面，夜深人静起身看球，平添理由。细数两队众将，上个月刚刚年满18岁的贾斯汀·克鲁伊维特甚是惹眼，端详面孔，那可真是昔日锋霸亲生的啊，与父亲帕特里克·克鲁伊维特一样一样的。只是，在球衣上签名时，新新人类用的是@justink_，时刻提醒21万球迷们到Instagram上去追寻他的脚步，未来属于贾斯汀的世界里，拥趸何止百万，他想追上偶像C.罗的亿级量级。

1995年夏日，18岁的克鲁伊维特欧冠决战，登场不过23分钟，便以一记轻巧破门，帮助阿贾克斯登顶欧洲。如今，小克鲁伊维特有机会复制父亲传奇，这一切来得是如此之快。贾斯汀的星途平顺，自小便在光环下成长，有些常人无法想象的压力他已经可以安然面对，习惯常人将自己与父亲随时比对。“决赛我不会紧张的，虽然我在考驾照时考了两次才过关，但这种状况在足球场上绝对不会出现的。”自小，贾斯汀就要选择是否在球衣上印着家族姓氏——克鲁伊维特，因为那份荣耀的背后就是压力，就是看客们时刻的苛责。巨星后辈再有巨星之名，难上加难啊。

贾斯汀在家排行老二，将门三虎子，哥哥今年20岁，也是阿贾克斯青训产品，后来转投其他球队，寂寂无名。老三鲁本15岁，还在业余球队中历练，别小看那支球队，队中传奇之后比比皆是——古利特之子马西姆和克鲁伊夫外甥艾斯特尔是其中最有名的。直到今年1月8日，贾斯汀在阿贾克斯逐渐获得出场机会，屡屡有惊世进球，老爸乐不可支地在社交媒体上与人分享爱子的进球。看过贾斯汀踢球的人一定会知道，虽然与锋霸父亲有着几乎同一张面孔，但其身体特点还是有所不同，他速度更快，更适合扮演锋线精灵，小罗和C.罗都是其未来样板。贾斯汀自己也时常感慨，并非所有巨星之后都是幸运的，只有从父亲身上成功遗传优良基因的才能走上球星路，否则不过是足球学徒罢了。

贾斯汀的爷爷当年也是苏里南球星，他的妈妈也是很像样的女足球员，他血液中流淌着家族的足球基因。父亲做球员时，带着一家人转战四方，贾斯汀儿时记忆中父亲场上的身影有些模糊，但他会一遍遍去看父亲欧冠决赛的进球，找寻门前的灵感。克鲁伊维特对贾斯汀极其关爱，一路呵护其成长，每场比赛后都要帮着爱子复盘，虽然因为一份高薪合同让其离开阿贾克斯到大巴黎出任体育经理一职，但并不妨碍他继续引领儿子向前。贾斯汀说，爸爸给了他“101个建议”，最为重要的只有一个——“门前要自私一点儿！”

如今，C.罗是贾斯汀的活榜样，他渴望自己也能有那样的足球生活，每周进球，完美身材，成为世界上最棒的球员。虽然身为阿贾克斯王朝后代，但埃因霍温休息室里悬挂的一句话一直在激励着贾斯汀：“勤奋完全可以击败天才，只要天才不努力。”显然，贾斯汀极其懂事，自我驱动力很强，他期待自己每天都能有进步，不仅作为一名球员，更重要的是作为一个完整的人。

克鲁伊维特天分很高，但因此也承受了伤病的梦魇，31岁便草草退役，留下了些许遗憾。贾斯汀人虽小，但已经悟到，唯有专注，才能成就不凡，你背后印上的任何家族荣耀不能最终护佑到巅峰的，全要靠自己走出独特的路径。新星耀眼，贾斯汀的名字逐渐可以跟在姆巴佩后面被列为下一代巨星候选了，阿贾克斯不过是孵化器而已，决赛在等待，豪门在等待，让心灵与身体一道成长，让巨星可以成就巨星。
