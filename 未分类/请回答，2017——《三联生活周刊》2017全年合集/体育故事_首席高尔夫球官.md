# 体育故事_首席高尔夫球官

首席高尔夫球官

体育故事

首席高尔夫球官

文◎张斌

2009年4月24日，美国总统奥巴马和副总统拜登在白宫草坪上打高尔夫球

老罗斯福总统1908年对他的继任者谆谆教诲，身为总统，骑行在马背之上的形象是完全可以被拍摄的。打网球？不可以。打高尔夫球？万万不可的，那是灾难、致命的。老罗斯福与媒体打交道素有手段，开总统形象塑造之先河，深谙公众人心冷暖，高尔夫球还是不碰为妙。

100年前的教诲，奥巴马算是没听进去，八年任期，下场打高尔夫球306次，这数绝对不虚，起居注一笔笔记着呢。与后任特朗普打嘴仗时，把柄算是被抓住了：“他比伍兹打了更多的高尔夫。”还有一句酸酸的：“我可没时间打球，我们得工作啊。”同时，配合爆料，2013年，奥巴马飞到佛罗里达，约着伍兹打了场球，据政府问责署的公开信息显示，这一趟可是花费了纳税人350万美元，“空军一号”展翅飞翔，海岸警卫队将球场团团围住，花老钱了。奥巴马未做回应，总统也是人嘛，休假打球已经充分考虑了节俭原则，在华盛顿、在夏威夷，都是选择邻近军事基地旁的球场。

前任小布什也爱高尔夫球，那可是非常克制的，同样八年在位，有数的24次下场，2003年更是全年停杆，生怕媒体给自己贴上“荒于嬉”的标签，那一年美国大兵们在伊拉克和阿富汗双线支撑，战局吃紧。近当代的美国总统对于高尔夫球大多采取敬而远之的策略，尼克松、肯尼迪、里根和克林顿都是高球迷，据说如果这些前总统能够来一次关公战秦琼般的下场对决，克林顿预计会以超出标准杆10杆，拿走冠军，高手总统们谨记一条，任内少打，越少越好。

都知道奥巴马深爱篮球，当年还是小伙子身板时，常常打球归来，浑身小伤。妻子米歇尔遂建议丈夫，索性开练高尔夫，一辈子的好运动，为什么不呢？从芝加哥一个二手高尔夫俱乐部起步，奥巴马全情投入，据旧友回忆，他训练非常认真，面临挫折耐性十足，始终心绪平稳，但这并不影响其澎湃的获胜诉求，挂点儿小彩，动力就更足了。只可惜入主白宫前，进步并不快，“差点”徘徊在17左右。八年任期，奥巴马毫不避讳旁人闲言，不惧“首席高尔夫球官”的名号，以饱满热情下场打球，访英期间，还曾与老友卡梅隆交手，争取让全球看客能够感知亲民、活力与自信。到辞别白宫前，“差点”已进步到了10，各项技术可圈可点，如果不是两党时下交恶，来场奥巴马对决特朗普的高球赛，那将是对权柄交接的最好纪念。

如果按照任内的下场打球次数计，奥巴马其实很难是头号“首席高尔夫球官”，艾森豪威尔先总统任内打球超过800次，赞誉高尔夫为“最佳运动”。威尔逊先总统更是狂热，超过1000次，人生目标便是，定要在球场上超过夫人，成为更好的球手，也许一生心愿未了呢。

对于高尔夫球，奥巴马看似随意自在，其实心怀原则还是很鲜明的，打球不仅是个人爱好，更是重压下逃避纷扰的最好方式，他自己说：“只有打球的时候，你才会不置身于糟糕的情境，偷得清闲。”看过《指定幸存者》后，对于号称世界上最有权势的人日常状态也算是有个美剧版的了解吧。好运动，小逃避，能理解，能理解。奥巴马还有另外一个原则，任内打球尽量不邀约有可能误导公众的政要或利益关联方一道下场，球友都是真实的旧交故友。

真热爱一项运动，总统的身份再特殊，公众差不多也都会体恤理解，借此将自己还原为真性情，还会自然加分的。政治家不缺智慧的，让所有人看到你在运动，看到你也需要偷得浮生半日闲。
