# 军事夜话_“印－太战略”的军事切入点

宋晓军：“印－太战略”的军事切入点

军事夜话

“印－太战略”的军事切入点

文◎宋晓军

12月1日，美国国会参议院将对共和党税改法案进行辩论及全体表决。美国参议院以51比49通过税改法案，这将是31年来国会首次修改税法（视觉中国供图）

12月2日，以色列战机空袭了叙大马士革以南14公里处的基斯瓦镇（Al-Kiswah）附近一处疑似有伊朗军事顾问的基地。同一天，驻日美军的6架F-22A隐形战机抵达韩国，准备参加将于本月4～8日进行的代号为“警惕王牌”的美、韩空军联合演习。同时，美参议院在当天以51票对49票通过了参院版本的减税方案。正是这三条新闻，让我想到了本期专栏的标题——“印－太战略”的军事切入点。

其实用这三条新闻诠释文章的标题很简单，因为在前两条的军事新闻中，一条与濒临印度洋的伊朗有关，另一条与濒临太平洋的朝鲜有关，而第三条新闻则与美国悬而未决的2018财年国防预算有关。简单说，在第一条军事新闻的背后，显然是以色列（包括沙特等海湾国家）希望特朗普“印－太战略”的军事切入点向濒临印度洋的伊朗方向倾斜，而在第二条军事新闻的背后是韩国（包括日本和澳大利亚）更希望特朗普“印－太战略”的军事切入点向濒临太平洋的朝鲜和中国方向倾斜。至于第三条新闻，虽然46名民主党参议员全部投下了反对票也没能阻止减税方案的通过（有两名独立参议员和一名共和党参议员也投了反对票），但接下来民主党很可能在12月8日前后在参院狙击包括近7000亿美元国防预算在内的2018财年联邦预算的通过（提高预算上限需参院三分之二的票通过）。那么在国防预算不确定的背景下，特朗普的“印－太战略”军事切入点在哪里呢？

众所周知，在特朗普上任后废除了奥巴马的“亚太再平衡战略”，并推出了他所谓的“印－太战略”。虽然这个战略还没有十分清晰的路线图，但如果从纯军事的角度看，有两组数字可以提供一定的线索：一是截止到今年9月30日，特朗普政府将奥巴马政府在阿富汗的美军驻军从8400人提升到了1.53万人.二是从今年1月截止到10月30日，在特朗普执政期间美军共向塔利班目标投下了3554枚炸弹，而在奥巴马执政的2016年一年期间美军只向塔利班目标投下1337枚炸弹。除了上述这两组数字外，据11月28日《华盛顿邮报》报道，美军驻阿富汗司令尼克尔森（John Nicholson）表示，在特朗普政府决定向阿富汗增兵和扩大美军参与军事行动范围后，美国计划到2019年将阿富汗政府控制的国土从目前的64%提升到80%。换句话说，特朗普“印－太战略”的第一个军事切入点应该是在阿富汗。

如果以上分析成立，就可以看出特朗普的选择既是现实的，又是符合“政治正确”原则的。也就是说，在军事资源有限的情况下，选择印度洋与太平洋之间的阿富汗作为其“印－太战略”的军事切入点是现实的.同时选择共和党政府发动、花费了近6万亿美元打了16年多的反恐战争的起点——阿富汗作为军事切入点先找回“政党自信”，又是符合“政治正确”原则的。当然，选择在与伊朗和中国都接壤的阿富汗先立足维持军事存在，也不排除特朗普政府有从长计议再慢慢勾勒出“印－太战略”完整路线图的用意。现在的问题是，与阿富汗都接壤的伊朗和中国对此有什么反应呢？

值得注意的是，12月1日中国最高领导人在“中国共产党与世界政党对话会”上说：面对日益复杂化、综合化的安全威胁，单打独斗不行，迷信武力更不行。而同一天，伊朗外长扎里夫（Mohammad Javad Zarif）在巴库举行的“阿富汗问题伊斯坦布尔进程”第七次外长会上说：阿富汗的外国军事存在不仅不能解决恐怖主义和极端主义的问题，而且历史证明这反而会成为煽动者招募恐怖分子的借口。
