# 军事夜话_“基尔库克迷局”的谜底

宋晓军：“基尔库克迷局”的谜底

军事夜话

“基尔库克迷局”的谜底

文◎宋晓军

10月21日，伊拉克政府军与库尔德武装在基尔库克发生交火。图为库尔德士兵在观察战况(视觉中国供图）

10月20日，美国企业研究所学者鲁滨逊（Michael Rubin）在《华盛顿观察者报》上发表了一篇题为《为什么伊朗“恐怖大师”苏莱曼尼会在伊拉克发布自己的照片？》的文章。鉴于鲁滨逊曾于2002～2004年在国防部长办公室担任过伊朗和伊拉克项目主任，同时还在伊拉克库尔德地区的三所大学——苏莱曼尼亚大学、萨拉赫丁大学和杜胡克大学担任过访问学者，因此可以说他文章的观点，是揭开四天前发生的“基尔库克迷局”谜底的一种版本。

所谓“基尔库克迷局”，指的是10月17日凌晨伊拉克军队和什叶派民兵兵不血刃地夺取了由库尔德武装——“自由战士”控制的石油重镇基尔库克后，媒体上疯传是伊朗革命卫队圣城旅司令苏莱曼尼（Qassem Soleimani）将军一手策划了此次行动。而鲁滨逊认为，这是伊朗故意公布苏莱曼尼将军在基尔库克的照片对其能力夸大的一种宣传战。他的主要理由是，2014年“伊斯兰国”打到库尔德地区伊拉克军队溃败时，伊朗革命卫队提供了帮助。而当时奥巴马政府出于对伊拉克马利基（Nouri al-Maliki）政府的不满，提供空中打击的行动迟缓了一些，加之北约成员国土耳其之前在叙利亚与“伊斯兰国”的关系，因此伊朗一直就制造美国支持“伊斯兰国”而只有伊朗才坚决打击“伊斯兰国”的舆论。这次伊朗又利用库尔德“公投”机会，试图把美国卷入自己培训的伊政府军与“自由战士”的内斗。

虽然鲁滨逊的版本不是没有道理，但奥巴马政府晚于伊朗出手打击“伊斯兰国”、“库民党”（KDP）领导人巴尔扎尼（Masoud Barzani）执意公投及“库民党”与“库爱盟”（PUK）的长期不和却是事实。如果没有这些事实，伊朗怎么会挑唆美国出资几百亿美元提供装备并培训的9.6万伊拉克军队与2.3万库尔德“自由战士”内斗呢？再说，特朗普政府明知伊拉克总理阿巴迪（Haider Abadi）“丢了”石油重镇基尔库克就会输掉明年的大选，为什么之前不全力阻止“库民党”领导人强行举行公投呢？而这些“情节”，似乎都像是为把苏莱曼尼将军塑造为“传奇将军”的铺垫。事实上，在伊拉克军队和什叶派民兵轻而易举控制了基尔库克后，一些美国学者及以色列媒体都认为美国输给了伊朗和苏莱曼尼将军。

值得注意的是，在“基尔库克迷局”的真相大白之前，“迷局”的效应正在发酵。伊拉克军队和什叶派民兵正沿着基尔库克一路向北不断“攻占”之前库尔德“自由战士”控制的地域，已接近了伊拉克与土耳其边境。同时，在10月20日最担心库尔德独立的土耳其与伊朗签署了“本币贸易协议”。而就在同一天，一群库尔德人在美国驻埃尔比勒领事馆前打出了“美国人民：你们允许伊朗人用你们的武器侵略吗？”“伊朗支持的什叶派民兵（PMU）正在用坦克攻击库尔德人民！”等横幅，抗议特朗普政府对基尔库克“陷落”后的沉默。

英国中东问题专家斯凯（Emma Sky）2003～2004年曾担任北约联军驻基尔库克协调员，并于2007～2010年担任美军驻伊拉克司令的政治顾问。她10月18日发表在《大西洋月刊》上的一篇文章《美国在伊拉克已经变得可有可无》中给出了“基尔库克迷局”谜底的另一个版本：巴尔扎尼又被美国出卖了。她的主要理由是，1975年伊朗国王与伊拉克达成协议突然中断了对巴尔扎尼之父创建的反政府武装的援助，条件是伊拉克承认伊朗的领土要求。当时他父亲老巴尔扎尼（Mullah Mustafa Barzani）转向美国寻求援助，但被当时的国务卿基辛格拒绝了。
