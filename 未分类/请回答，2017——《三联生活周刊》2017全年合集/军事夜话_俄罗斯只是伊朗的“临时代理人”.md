# 军事夜话_俄罗斯只是伊朗的“临时代理人”

军事夜话：俄罗斯只是伊朗的“临时代理人”

军事夜话

俄罗斯只是伊朗的“临时代理人”

文◎宋晓军

6月23日，俄罗斯国防部称，俄海军“埃森海军上将号”“格里戈罗维奇海军上将号”护卫舰和“克拉斯诺达尔号”潜艇向叙境内的恐怖组织“伊斯兰国”设施发射了6枚“口径”巡航导弹

6月23日，俄罗斯三艘舰艇在事先未通知美国的情况下，从地中海向“伊斯兰国”位于叙利亚哈马省阿奇巴（Aqirbat）地区的弹药库发射了6枚“口径”巡航导弹。事后，一位读过我在2017年第25期《三联生活周刊》上发表的题为《美国与伊朗的“代理人战争”》一文的媒体朋友马上就问我：俄罗斯和伊朗到底谁是谁的代理人呢？对此我回答说：俄罗斯只是伊朗的“临时代理人”。

为了解释这个回答，不妨先回顾自6月18日至24日发生的一些新闻事件，以及叙利亚和伊拉克战场态势。6月18日，伊朗时隔30年首次向境外的叙利亚代尔祖尔地区“伊斯兰国”军事设施发射了6枚弹道导弹，同一天美军在叙利亚拉卡南部击落了叙军一架苏-22攻击机。6月21日，沙特国王突然宣布废黜现任王储而立了自己的儿子本·萨勒曼为新王储，同一天特朗普的女婿库什纳作为中东特使开始出访以、巴，而美军在叙南部的坦夫附近再次击落一架伊朗制造的无人机。6月22日刚从伊朗访问归来的伊拉克总理说几天内将宣布解放被“伊斯兰国”占领的伊拉克第二大城市摩苏尔，同一天美防长告知土耳其若美国支持的叙库尔德武装——民主军收复了拉卡就会收缴美国为他们提供的武器。6月23日，俄罗斯向叙境内发射巡航导弹，同一天沙特等逊尼派国家要求卡特尔10天内必须满足降低与伊朗外交关系等13点要求。6月24日，以色列发射导弹袭击了在戈兰高地附近的叙政府军。

当然，仅从上述信息还不能看出俄罗斯要充当伊朗的“临时代理人”。但以色列情报网6月23日发的一则消息则称，俄罗斯已在大马士革东南叙政府军几天前收复的比尔卡萨巴区（Bir Qassab）的一个村庄开始建立军事基地，该基地距大马士革50公里，距约旦96公里，距叙、以边境85公里，距目前美军与叙反对派武装控制的坦夫185公里。同时，俄罗斯调解叙利亚危机总统特使拉甫连季耶夫（Alexander Lavrentiyev）在约旦首都安曼与美方开始谈判，其大致内容是：在包括叙、以和叙、约边境的叙南部建立一个非军事区，俄确保黎巴嫩真主党和什叶派民兵适时撤出该地区后，由美、俄双方在该地区建立共管机构。但美方出于两个原因拒绝了俄方的建议：一是俄无法确保黎巴嫩真主党和什叶派民兵不进入该地区.二是美军支持的叙反对派的目标是沿叙、伊边境北上赶在叙政府军和什叶派民兵之前去抢占进攻叙东部的最大城市代尔祖尔的前进基地。

如果以色列情报网的消息是可信的，那么在美拒绝了俄充当美、伊之间“协调人”的角色之后，俄向叙境内发射巡航导弹之举就表明其要为伊朗充当“临时代理人”了。其实，自2015年9月出兵叙利亚后，俄罗斯就一直在“协调人”与“临时代理人”两者之间徘徊。而两个角色的转换，则取决于美国在北约和乌克兰问题上对俄的态度。但从近一周来中东和东欧方向的各种变量上看，特朗普政府在国内“通俄门”的持续压力下，显然不可能以牺牲北约和乌克兰的利益与俄在叙利亚问题上做交易，因此他只能选择伊朗的宿敌——沙特和以色列作为“代理人”，联手摆出了一副要与伊朗“对决”的姿态。正是在这一背景下，俄罗斯才发出了要从“协调人”转换到“临时代理人”的信号。不过，仅从伊朗和俄罗斯导弹打击的目标上就断定俄罗斯要一直在叙战场上为伊朗充当“临时代理人”，还为时过早。因为毕竟沙特的新王储与以色列总理内塔尼亚胡和普京也有非同一般的关系。
