# 军事夜话_德黑兰用了谁的剧本？

宋晓军：德黑兰用了谁的剧本？

军事夜话

德黑兰用了谁的剧本？

文◎宋晓军

6月20日，伊拉克亲政府民兵团体“人民动员阵线”下属的什叶派武装在北方古城尼尼微附近投入战斗（视觉中国供图）

9月7日凌晨2时42分，以色列空军战机在黎巴嫩空域对位于叙利亚中部迈斯亚夫（Masyaf）一处叙军的科研中心（CERS）发射了多枚导弹。几个小时后，韩国空军50多架战机举行了对朝鲜“关键军事力量”实施低空突防轰炸的演练。看到一个“真打”和一个“假打”的两则新闻后，我首先想到了当天美国保守派杂志《标准周刊》网站上一篇文章的内容及这篇专栏的题目——《德黑兰已经熟读了平壤的剧本》，文章的作者是美国民主基金会资深研究员鲁吉罗（Anthony Ruggiero）。

文章大意是，鉴于美国介入朝核问题先于伊核问题，而曾在2005年担任“六方会谈”美方防扩散顾问的鲁吉罗认为，伊朗至少从朝鲜身上汲取了三个方面的经验：1.可以在核交易上作弊.2.利用保留核能力的有限核交易.3.可以发展军事和非核能力。比如第三个方面，鲁吉罗就列举，朝鲜自1994年与美国达成“框架协定”4年后至少5次发射了运载火箭（SLV），并于2012年和2016年将卫星送入太空轨道，而这些SLV为朝鲜发展可将核武器投送到美国的洲际导弹提供了帮助。例如，今年朝鲜至少试射了5次可以打击关岛的“火星-12”弹道导弹。而伊朗则在2015年7月与六国达成了“伊核协议”后，试射了两次SVL和14枚弹道导弹。鲁吉罗最后认为，朝核问题的路线图充分说明了“有限核交易”最终是如何导致美国受到核威胁的，而来自伊朗的核威胁只是时间问题。

巧合的是，9月7日凌晨以色列空军对叙利亚的空袭，恰恰也是因为伊朗在第三个方面的进展。按“以色列情报网”透露，以色列国防部长利伯曼（Avigdor Lieberman）在空袭后说：我们绝不允许从伊朗到黎巴嫩的“什叶派走廊”的形成。根据该网站透露的信息，伊朗在伊拉克的代理人“人民动员部队”（PMU）已进入了叙境内，一旦叙军及什叶派武装夺取了东部最大城市代尔祖尔和下游的边境口岸阿布卡迈勒（Abu Kamal），就意味着叙利亚与伊拉克600公里的边境线完全被伊朗的代理人所控制。而以色列情报部门认为，遭空袭的叙军CERS不仅生产化武，而且还可能正在利用来自伊朗和朝鲜的支持生产炸弹和导弹。一旦这些武器被送到已坐拥15枚火箭弹的黎巴嫩真主党手里，将对以色列安全构成巨大威胁（2006年“黎以战争”期间真主党共向以色列发射了4000多枚火箭，造成以军民160多人死亡）。

很显然，以色列的空袭及释放出来的信息，从一个侧面证实了鲁吉罗的观点。也就是说，虽然伊朗没有像朝鲜那样不断触碰“核、导红线”，但伊朗却充分利用了“有限核交易”并未限制其在地区扩大军事影响这个“有利条件”。事实上，这不仅是近来伊朗想方设法不让特朗普政府找到其违反“伊核协议”蛛丝马迹的真正原因，而且也应该是“德黑兰已经熟读了平壤的剧本”的一种结果。

最后我想说的是，在特朗普政府试图找到伊朗“违约证据”之时，伊朗却成功地利用了近期的朝核危机加快了在地区范围内扩大其军事影响的行动。这其中的道理很简单，如果特朗普政府找不到伊朗“违约证据”而强行“毁约”，那么看到了这一幕的朝鲜可能就会像俄罗斯总统普京之前所说的：就是吃草也要搞出能打到美国本土的核武器。当然，只要美国不针对伊朗强行“毁约”，朝鲜的“要价”也就有了希望。从这个意义上说，有几十年外交经验的朝鲜“二号人物”金永南8月初对德黑兰为期10天的访问是否带去了“平壤的剧本”，倒是很值得琢磨的一个问题。
