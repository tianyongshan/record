# 军事夜话_特朗普政府是“纸老虎”吗？

特朗普政府是“纸老虎”吗？

军事夜话

特朗普政府是“纸老虎”吗？

文◎宋晓军

2月10日，在中、美领导人通完电话后，英国《卫报》和美国《外交政策》发表的评论文章中，都将特朗普承诺美国政府继续奉行“一个中国”政策的表态比喻为是“纸老虎”。看到这个比喻，我马上想到了1月31日美国国防部长马蒂斯签发的重建美国军队的备忘录。那么马蒂斯的这一纸备忘录与英、美媒体说特朗普政府是“纸老虎”有什么关系呢？

1月31日，美国新防长马蒂斯根据4天前特朗普在五角大楼签署的重建美国军队的行政令，代表国防部签署了一份备忘录。备忘录实际上是一份分三阶段落实总统国家安全指令的预算指南。三个阶段分别是：提高作战准备、解决紧迫的能力不足实现计划平衡及建设更强大、更得力和更致命的联合部队。2月7日，美国海、陆、空和陆战队负责军队建设的“第二把手”在众院军事委员会作证时，分别用陈述事实的方式提出了各自在2017财年“提高作战准备”上的预算缺口。据美联社报道，海军的预算“缺口”为120亿美元，陆军为82亿美元，空军为62亿美元，陆战队为42亿美元。那么，这306亿美元“提高作战准备”的预算缺口与3天后特朗普改口承诺奉行“一个中国”政策有没有关系呢？

1月30日，美国总统特朗普签署了一项新的行政命令，要求“每推行一项新的监管，就必须撤销两项监管”，然后由白宫来进行审查，而涉及到军事和国家安全的规定则除外

回答这个问题先从美国的国防预算说起。美国法律规定总统必须在2月的第一周向国会提交下一财年的联邦预算案，其中自然也包括国防预算。鉴于目前总统换届以及奥巴马提交的2017财年联邦预算案并没得到国会的批准（目前执行的是以2016财年开支为标准的延时拨款决议），所以特朗普才会指示马蒂斯以“重建美国军队”为原则，在3月1日前拿出一份2017财年的国防预算修正案，在5月1日前拿出2018财年的国防预算案，在2018年初拿出2019～2023财年未来5年的国防预算规划。而这三个预算文件应是与前面提到的备忘录中的三个阶段相对应的。因此，从2月4日国防部长马蒂斯在东京说南海问题要通过外交手段解决，到2月10日特朗普承诺继续奉行“一个中国”政策，至少表明特朗普政府目前在“提高作战准备”的第一阶段，不想公开在军事上挑战中国设置的“安全底线”。

接下来的问题是，特朗普政府会不会在重建美国军队的第二、第三阶段挑战中国的“安全底线”呢？如果仅以国防开支作为一个衡量标准，这还要观察特朗普政府重建美国军队的计划执行得是否顺利。而其中的一个关键点，就是看美国国会能否尽快取消2011年通过的“预算控制法案”（BCA），进而把重建美国军队的钱给到位。比如按照BCA设置的上限，2018财年基础国防预算最多只能支出5490亿美元，而最近无论是官员、国会议员还是学者都认为这无法满足重建美国军队的需要。参院军事委员会主席麦凯恩最近甚至建议重建美国军队要将2018财年基础国防预算提高到6400亿美元。换句话说就是，国会取消当年众院以268票对161票、参院以74票对26票通过的BCA，才是特朗普政府改变其“纸老虎”形象的前提。

最后我想说的是，按照2016年1月1日发布的《中央军委关于深化国防和军队改革的意见》中的时间表，中国军队在2016年完成全军编制体制阶段性的改革任务后，将在2017～2020年的4年时间里，对相关领域的改革做进一步调整、优化和完善。可以看出，中国军队的改革时间表与特朗普政府重建美国军队的“三阶段”有很多重叠。从这个意义上说，接下来中国军队的改革进程是否顺利，也可以说是一个衡量特朗普政府是不是一只真的“纸老虎”的指标。
