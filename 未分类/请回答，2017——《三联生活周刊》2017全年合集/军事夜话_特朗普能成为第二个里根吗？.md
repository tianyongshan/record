# 军事夜话_特朗普能成为第二个里根吗？

宋晓军：特朗普能成为第二个里根吗？

军事夜话

特朗普能成为第二个里根吗？

文◎宋晓军

12月13日，美国驻联合国代表黑莉在位于华盛顿的“斯提亚－波灵联合空军基地”的一个机库里举行的新闻发布会，公布了伊朗向也门胡塞武装提供导弹攻击沙特的证据（东方IC 供图）

当地时间12月13日，美国驻联合国代表黑莉（Nikki Haley）在位于华盛顿的“斯提亚－波灵联合空军基地”（Joint Base Anacostia-Bolling）的一个机库里举行的新闻发布会上，展示了伊朗向也门胡塞武装出售弹道导弹、违反联合国安理会2231号决议附件的“证据”。两天后的12月15日，美国最早的网络杂志《石板书》上发表了一篇题为“美国可能同时陷入两场不同的战争”的短文。看了这篇短文的大致观点后，我想到了一天前与朋友聊天的内容。

先说《石板书》上短文的大致观点。文章作者基廷（Joshua Keating）引用了资深共和党参议员格雷厄姆（Lindsey Graham）对朝鲜开战概率的猜测和黑莉的“证据”可能会导致明年1月特朗普撕毁“伊核协议”的推断，认为特朗普政府对朝鲜和伊朗发动战争的可能性在增大。他在文章的最后写道：如果局势继续朝着目前的方向发展, 接下来的问题与其说是特朗普是否会发动新美国战争，不如说是会在哪里发动战争。其实基廷的这段话，恰恰就是我一天前与朋友聊天时朋友向我提出的问题。我当时的回答是：应该是在叙利亚，也许特朗普真的想成为第二个里根。

接着我解释说，特朗普政府不仅在减税和要求大幅增加军费上与当年的里根政府有相似之处，而且里根任期内在积极介入“两伊战争”的同时，还在1982年借埃及与以色列达成“土地换和平协议”之势，通过支持以色列对黎巴嫩发动战争恢复了中东地区所谓“美国治下的和平”。而特朗普政府在阿富汗、伊拉克、叙利亚开始增加驻军的同时，也正在竭力推进由沙特王储萨勒曼（Mohammed bin Salman）和特朗普女婿库什纳（Jared Kushner）策划的所谓“以巴和平B计划”（说服巴勒斯坦领导人阿巴斯同意巴勒斯坦在加沙地带建国，并从西奈半岛北部并入大片土地。据传埃及已基本赞成这一计划，但巴勒斯坦恢复1967年之前边界的要求难以满足），为下一步孤立打压伊朗、黎巴嫩真主党及哈马斯创造政治条件。而且值得注意的是，就在黑莉指责伊朗向也门胡塞武装出售弹道导弹的两天前，与沙特和以色列都保持良好关系的俄罗斯突然宣布从叙利亚撤军的决定。这一幕，也不免让人想起1982年以色列发动对黎巴嫩的战争时苏联的“观望态度”，以及当时叙利亚政府的“独立无缘”。

说到这儿，有了解中东的朋友问：35年后已物是人非，且不说阿巴斯会不会同意由沙特王储小萨勒曼和特朗普女婿库什纳积极策划、推动的所谓“以巴和平B计划”，就是如今黎巴嫩真主党的实力以及伊朗通过打击“伊斯兰国”在叙利亚和伊拉克的军事存在和影响力也不容小觑。对此我回答说，这正是美国一边增兵阿富汗并强调不会从伊拉克和叙利亚撤军，一边在积极寻找伊朗违反联合国安理会2231号决议“证据”的真正原因。

最后，要想缕清特朗普能否成为第二个里根的种种线索和决定性变量，是一件十分庞大和复杂的工程。我所能提供的只是一个军事上的观察点：特朗普政府在顺利推进其“减税计划”的同时，能否在明年1月如愿以偿地像当年里根那样得到大增幅的国防开支。因为以美军目前的战备状况看，要在叙利亚协助以色列及沙特支持的逊尼派武装与处于顺势的伊朗支持的什叶派武装和黎巴嫩真主党抗衡，还有很多不足之处需要增加开支来弥补。当然，还有一个题外话的前提就是：即便朝鲜半岛再出现一些状况，特朗普政府都会接受中、俄两国不能在朝鲜半岛动用军事手段解决问题的建议。
