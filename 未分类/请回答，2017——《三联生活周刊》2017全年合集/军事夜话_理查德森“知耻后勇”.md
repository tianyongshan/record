# 军事夜话_理查德森“知耻后勇”

宋晓军：理查德森“知耻后勇”

军事夜话

理查德森“知耻后勇”

文◎宋晓军

8月21日，位于新加坡的美国导弹驱逐舰“麦凯恩号”和一艘商船在马六甲海峡附近相撞。图为驱逐舰左舷受损（视觉中国供图）

9月19日，是理查德森（John Michael Richard-son）担任美国第31任海军作战部长第四年开始的第一天。这天他在参院军事委员会接受了议员及第七舰队两艘发生“撞船”事故驱逐舰遇难水兵家属的询问。在持续了两个多小时的询问过程中，理查德森对在西太平洋部署的舰艇状况表达了一种“知耻后勇”的态度。而这一态度是非常值得重视的。

据美海军学院新闻网9月20日报道，尽管理查德森在询问中承认，自2006年第七舰队的舰艇从20艘增加到目前的40艘以来，在西太平洋实施前沿部署的舰艇普遍缺乏“认证”，但他并没有确认美政府问责局（Government Accountability Office,GAO）在9月份提交的报告中舰艇“认证”过期的具体数字。可同时，他不仅表示了要改变舰艇“认证”过期比例过高的现状，而且还要改变舰艇指挥官的一种习惯——即在水兵和舰艇没有准备好的情况下就接受舰艇的作战部署任务。例如，当他被问到为确保“认证”合格而延迟了舰艇的作战部署后是否会对舰艇指挥官产生负面影响时，理查德森明确地回答道，他不仅不会指责舰艇指挥官，反而给他们发勋章。他表示：这正是我们管理一支安全、有效海军所需要的诚实和透明的态度。

那么理查德森不愿意承认GAO报告中第七舰队舰艇“认证”过期的具体数字是什么呢？9月7日，GAO发布了一份28页题为《针对舰队持续维护保障、训练和其他挑战，美国海军应当采取的行动》的报告。报告中指出：驻日美海军11艘驱逐舰和巡洋舰的“认证”过期状况从2015年5月至2017年6月上升了5倍。如有8艘舰艇的反潜训练作战证书已经过期，而有7艘舰艇的巡航导弹作训练战证书已经过期。具体到发生“撞船”事故的“菲茨杰拉德号”驱逐舰，其10项关键作战训练证书已全部过期，另一艘“麦凯恩号”驱逐舰，已有6项证书过期。很显然，这些数据又说明了第七舰队舰艇“认证”普遍过期并被迫“超负荷”部署，大部分都发生在理查德森任海军作战部长期间。

那么作为海军最高指挥官不确认这些数字，是否就表明理查德森对其中的隐患没察觉呢？事实上，在今年5月16日理查德森签发的《未来海军》白皮书中，就已明确指出：美国海军必须摒弃过去优势所带来的安逸或自满的心态，即刻开始建立一支规模更大、能力更强、行动更迅速的舰队，未来舰队必须时刻准备好执行各项作战任务。时间就是一切！可见，在第七舰队接连发生“撞船”事故的前夕，理查德森不仅意识到了隐患的存在，而且也提出了解决的设想。巧合的是，他在白皮书第一章中设定的前提是：要应对来自中国、俄罗斯、朝鲜和伊朗的挑战，而中国海军是排在“挑战”第一位的。也就是说，理查德森其实知道第七舰队存在的隐患，但按白皮书中说的中国海军“舰队现代化正以不可思议的速度推进”的现实，他只能明知不可为而为之了。而他不愿意确认GAO报告中的那些数字，也许是出于保密或别的什么原因。

最后我想说的是，理查德森在参院听证会上所表达的“知耻后勇”，无疑与中国海军近几年的活动频度增加和活动范围扩大有关。值得注意的是，就在理查德森任海军作战部长在参院听证会上表达“知耻后勇”的前一天，中、俄海军“海上联演——2017”第二阶段演习刚好在西太平洋北部的鄂霍次克海和日本海拉开了帷幕。对于中国海军而言，我想理查德森签发的《未来海军》白皮书里最后的一句话“时间就是一切”也应该是适用的。
