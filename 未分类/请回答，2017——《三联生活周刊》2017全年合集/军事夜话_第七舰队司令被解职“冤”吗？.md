# 军事夜话_第七舰队司令被解职“冤”吗？

宋晓军：第七舰队司令被解职“冤”吗？

军事夜话

第七舰队司令被解职“冤”吗？

文◎宋晓军

被撞后的美国海军驱逐舰“菲茨杰拉德号”（视觉中国 供图）

9月24日，美海军作战部副部长莫兰（William Moran）上将签署了对海军舰艇海上事故频发进行为期60天综合审查的备忘录，审查重点是今年来第七舰队发生的几起损失惨重的撞船事故。看到这则消息，我想起莫兰今年1月10日在“海军水面舰艇协会”年会上的讲话中的一段比喻，比喻的大意是：纽约市目前有250支消防队，试想若纽约市仅有200支消防队，会有多少生命在火灾中失去？而我们海军的情况很类似。在我想起莫兰的这段比喻后，又难免不产生一个疑问：第七舰队司令奥库安（Joseph Aucoin）因接连发生撞船事故被解职是不是有点儿“冤”呢？

要回答这个疑问，得先从8月21日“麦凯恩号”驱逐舰与悬挂利比里亚国旗的商船相撞当天美海军海上作战部署的情况说起。根据美国海军新闻网公布的海军舰艇部署情况，8月21日，美海军的276艘舰艇中，共有96艘舰艇在海上执行作战部署任务，其中第三舰队0艘，第四舰队2艘，第五舰队25艘，第六舰队16艘，第七舰队53艘。如果以第七舰队约70艘舰艇的数字计算，也就是说美海军第七舰队只用了占美海军舰艇总数约25%的舰艇执行了海军超过50%的海上作战部署任务。这样看来，作为第七舰队司令的奥库安是不是相当于莫兰比喻的手下只有200支消防队的“纽约市消防总队总队长”呢？

说到这儿，再看一份2015年5月29日美政府问责局（Government Accountability Office,GAO）发布的一份报告中披露的数字。这份题为《海军力量结构：降低海外部署舰艇长期风险的可持续发展规划与综合评估》的报告长达80页，是GAO受国会军事委员会委托对美海军海外部署舰艇成本与收益分析的报告。报告中有一幅2015年2月制作的图表，图表对驻扎在本土、日本横须贺及西班牙罗塔的海军舰艇的时间分配做了直观的对比描述。如在本土部署的舰艇以27个月为一个轮换周期推算，其维护时间占19%，训练时间占41%，作战部署时间占22%，可机动时间占19%.而西班牙罗塔基地部署的驱逐舰与巡洋舰以24个月为一个轮换周期推算，其维护时间占33%，训练时间占17%，作战部署时间占50%，无可机动时间.在日本横须贺部署的舰艇同样以24个月为一个轮换周期推算，其维护时间占33%，作战部署时间占67%，无训练和可机动时间。由此可见，奥库安2015年9月7日开始接手的驻日第七舰队无疑是一支得不到训练和休息时间的“疲惫”的舰队。

奥库安指挥一支“疲惫”的舰队，除了每年要完成上百次的双边演习、数百次港口访问及十几次“航行自由”行动外，还要不时派舰艇应对朝鲜半岛危机。舰队在“疲惫”的状况下接连出现撞船事故奥库安就被解职，也许是有点儿“冤”。但说到“冤”，7年前同样指挥着一支征战了9年的“疲惫”军队的美军驻阿富汗司令麦克里斯特尔（Stanley McChrystal），因对奥巴马之前宣布的阿富汗撤军时间表公开表达不满而被迫辞职算不算“冤”呢？值得注意的是，就在奥库安被解职的当天，特朗普总统宣布了他的阿富汗新战略。同时，与奥巴马2009年12月就宣布了阿富汗撤军时间表一样，特朗普之前也宣布废除了奥巴马的“亚太再平衡”战略和美国主导的贸易框架——《跨太平洋伙伴关系协定》（Trans-Pacific Partnership Agreement，TPP）。从这个意义说，探讨麦克里斯特尔将军和奥库安将军的“冤”与“不冤”实际上已经不重要了，重要的是：美海军第七舰队的频密行动与中国东海与南海的主权权益密切相关，而阿富汗则是与中国陆上接壤的14个国家中美军驻军最多的国家。
