# 军事夜话_美国又要抛弃叙利亚库尔德人吗？

宋晓军：美国又要抛弃叙利亚库尔德人吗？

军事夜话

美国又要抛弃叙利亚库尔德人吗？

文◎宋晓军

11月22日，土耳其总统埃尔多安在俄罗斯索契会见了伊朗总统鲁哈尼（左一）（视觉中国供图）

当地时间11月24日，土耳其总统埃尔多安与美国总统特朗普就叙利亚问题通了电话。对此土耳其外长在新闻发布会上说，特朗普明确告诉埃尔多安他已下达不向叙库尔德武装——“人民保护部队”（YPG）提供武器的指示。事后，白宫发言人并没有质疑土外长的表述，只是说随着拉卡打击“伊斯兰国”作战的结束，美向叙境内合作伙伴提供军事支持的计划调整了。那么特朗普政府是不是继伊拉克后又要在叙利亚抛弃库尔德人呢？

要回答这个疑问，首先要看两则消息和两组数字。第一则消息是，继美国国防部长马蒂斯强调美军将留在叙利亚后，五角大楼官员承认美军在叙利亚大约有2000人的驻军.第二则消息是，继俄罗斯、伊朗和土耳其总统在索契就叙利亚问题会谈后，沙特在利雅得不仅召集了叙反对派组成“高级谈判委员会”准备参与下一步日内瓦会谈，而且明确提出了“叙总统阿萨德必须下台”这样与“索契会谈”主旨完全不一样的主张。两组数字中的第一组数字是，到2017年11月，叙政府控制叙利亚18.5万平方公里国土中的约9.7万平方公里.由叙逊尼派东部部落武装和YPG组成的叙民主军（SDF）控制了约5万平方公里.叙反对派武装控制了约2.5万平方公里.“伊斯兰国”控制着约1.3万平方公里.土耳其控制着约2000平方公里。第二组数字是，据土耳其媒体报道，美军在叙利亚有13个驻扎地，而俄罗斯有5个军事基地。

很显然，从以上两则消息和两组数字里可以推测，如果美国将YPG从SDF中剥离，从而获得坚持认为YPG是库尔德工人党分支的土耳其的支持，并转而利用沙特在逊尼派中的影响力和财富整合、赎买占叙人口约70%的逊尼派中所有反对派武装，还有可能在叙利亚战略态势上对冲俄罗斯和伊朗已经取得的优势。也就是说，以牺牲只占叙人口约10%的库尔德人利益为代价，换取土耳其与俄罗斯、伊朗的“不合作”，并重组叙逊尼派反阿萨德武装，也是一种选择。其实以YPG为主的SDF占据的约5万平方公里土地，特别是占叙约40%的油气资源，大部分都是叙东部逊尼派部落的传统领地。值得注意的是，2008至2010年美国驻土耳其大使杰弗里（James Jeffrey）11月24日在接受《华盛顿邮报》采访时，似乎证实了上述推测。他说：如果我们想继续留在叙利亚，就必须修补与土耳其的关系。

现在的问题是，即便是上述推测成立，就美军在叙利亚的驻军人数与同样是外来者的俄罗斯和伊朗支持的力量相比还是处于劣势。而根据五角大楼的数据显示，从2017年7月至9月的第三季度，美军在中东的驻军增加了33%，其中在叙利亚从1251人增加到了1723人，在伊拉克从8173人增加到了9122人（含文职人员）。在“伊斯兰国”即将被剿灭之时，美国不仅强调不会从叙利亚撤军，反而开始增加在叙利亚和伊拉克的军力投入，难道真是在为特朗普的“后伊斯兰国”叙利亚战略做准备吗？有意思的是，YPG在埃尔多安与特朗普通电话后并没有做出明确的回应，他们只是在社交媒体上展示了其士兵沿着幼发拉底河东岸一路向南突进靠近了叙、伊边境重镇阿布卡迈勒（Albu Kama）的照片。那么YPG急于靠近由俄罗斯提供空中支援、由伊朗代理人——黎巴嫩真主党和叙什叶派民兵控制的阿布卡迈勒，又要向特朗普传达什么信号呢？莫非他们要与美国在叙利亚打击的主要对象——伊朗结盟来“要挟”特朗普不要“抛弃”他们吗？总之，埃尔多安与特朗普的通话让未来叙利亚的局势更加扑朔迷离了。
