# 军事夜话_耶路撒冷与叙利亚战场

宋晓军：耶路撒冷与叙利亚战场

军事夜话

耶路撒冷与叙利亚战场

文◎宋晓军

12月7日，以色列总理内塔尼亚胡出席外交部发布会时发言，就美国总统特朗普承认耶路撒冷为以色列首都一事表达赞赏（视觉中国供图）

当地时间12月8日，当媒体铺天盖地地报道特朗普认可耶路撒冷为以色列首都所引发的各种反应时，美国有线电视新闻网（CNN）记者斯塔尔（Barbara Starr）和克劳福德（Jamie Crawford）发出一条报道称，多位美国官员告诉CNN，美国防部长马蒂斯正要求其参谋们寻找到对付伊朗的行动计划。几小时后，路透社又从约旦发布了一条题为“尽管对认可耶路撒冷为首都的决定很愤怒，但沙特看起来与美国的和平努力是一致的”的报道。看了这两条报道后，我就想到了本期专栏的标题：耶路撒冷与叙利亚战场。

我之所以想到这个标题，根据之一是，据此前《华盛顿邮报》报道，美国务卿蒂勒森和国防部长马蒂斯开始并不赞成特朗普的决定.根据之二是，上述CNN两位记者的报道.根据之三是，在路透社的报道中引用了约旦政治分析家兰塔维（Oraib Rantawi）的一句话：沙特与美国有直接交易，希望达成一项对巴勒斯坦不公平的协议以换取美国的支持，为海湾国家与以色列合作对抗伊朗铺平道路。也就是说，虽然国务卿蒂勒森和国防部长马蒂斯开始并不赞成特朗普的决定，但他们与为特朗普策划决定将耶路撒冷定为以色列首都的幕僚们的最终目标却是殊途同归——都是要遏制伊朗。现在的问题是，以马蒂斯为代表的官僚与以特朗普女婿库什纳为代表的幕僚实现“遏制伊朗”这一共同目标的路径交汇点在哪里呢？

简单说，以马蒂斯为代表的官僚们的路径，似乎是通过增资重整美国在中东地区的军事态势并保持在叙利亚、伊拉克的军事存在，然后再谋划、实施遏制伊朗的具体行动。而以库什纳为代表的幕僚们的路径，似乎是立即通过“强制站队”迫使阿拉伯国家在对抗伊朗与支持巴勒斯坦建国的实际行动上必须做出“二选一”的选择。很显然，特朗普在评估了国内中期选举和预算僵局等政治因素后之所以选择了后者，可能还有后者比前者见效更快的原因。因为马上可以检验“遏制伊朗”这一共同目标效果的具体地点，就应该是在叙利亚战场。换句话说，前者与后者路径的交汇点就在叙利亚战场。

值得注意的是，按照12月8日CNN报道中引述总统安全事务助理麦克马斯特（Raymond McMaster）的话说：大约80%的阿萨德武装分子是伊朗在叙利亚的代理人，他们正在建立通往地中海使我们面临在以色列边境有一支伊朗代理人军队的大陆桥……我们必须解决伊朗及其代理人日益增长的实现称霸地区目标的能力。若把麦克马斯特这段表述与特朗普的决定结合起来看，以逊尼派穆斯林为主体的阿拉伯国家会坐视代表什叶派穆斯林的伊朗“控制”逊尼派人口占多数的叙利亚吗？比起特朗普政府承认已被以色列实际控制的耶路撒冷为其首都而言，代表什叶派穆斯林的伊朗对叙利亚的“占领”孰轻孰重呢？

事实上，按照联合国秘书长叙利亚问题特使德米斯图拉（Staffan de Mistura）12月7日宣布的消息，12月10日叙政府代表团将再次重返日内瓦。可以预计的是，与他们面对面的拿了沙特援助资金的36名逊尼派穆斯林叙反对派代表，并不会因为特朗普的决定而接受阿萨德继续留在他目前的位置上。从这个意义上看，特朗普认可耶路撒冷为以色列的首都并开始搬迁美驻以使馆的决定，除了有国内政治因素考虑外，至少是把一大部分宝押在了变幻莫测的叙利亚战场上。因为作为特朗普政府上任后要竭力遏制的目标——伊朗，早已经把大量的人力、资金和武器装备投入到了叙利亚的战场上。
