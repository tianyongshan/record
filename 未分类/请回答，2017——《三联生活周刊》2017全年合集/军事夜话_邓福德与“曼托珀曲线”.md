# 军事夜话_邓福德与“曼托珀曲线”

宋晓军：邓福德与“曼托珀曲线”

军事夜话

邓福德与“曼托珀曲线”

文◎宋晓军

美参联会主席邓福德（Polaris/视觉中国 供图）

9月27日，美国有线电视新闻网（CNN）驻五角大楼记者布朗（Ryan Browne）以《高级将领：2025年中国将成为美国“最大威胁”》为题在CCN网站上发了一篇报道。报道的大意是，一天前美参联会主席邓福德（Joseph Dunford）在参院听证会上回答夏威夷州日裔参议员广野庆子（Mazie Hirono）提问时说：我想中国可能会在大约2025年前对我们的国家构成最大威胁。虽然邓福德在证词中也谈到了俄罗斯、朝鲜的威胁，但邓福德对“中国最大威胁”的预测，还是让我想起了20多年前一个美海军战略分析师的预测——曼托珀曲线。

简单说，所谓“曼托珀曲线”的由来是：1991年秋天，美军知名战略分析师、海军情报办公室副主任曼托珀（William Manthorpe）在给一群海军及陆战队将军们做有关“后冷战”海军战略的报告时，给出的一幅题为“假设”的曲线图表。这幅曲线图表要表达的是，随着冷战的结束，像苏联那样的大国挑战美国国家利益的意图会快速下降，而一些“不受控的中、小国家”的挑战会逐渐上升，但到了21世纪开始后，像俄罗斯和中国这样的大国对美国国家利益的挑战会快速上升，而到2010年时会达到一个高值。为此他建议美军应该瞄准俄罗斯和中国这样的大国而制定军事战略和军队发展规划。当时听曼托珀报告的，刚好有参与制定冷战后海军战略的五位海军及陆战队将军。随后这五位被称为“五人帮”的将军，在1993年就制定出了被认为是可以应对所有国家挑战的、将美海军派往抵近欧亚大陆近海实施前沿威慑的“从海上”战略。

随着时间的推移，“曼托珀曲线”被提出来的10年后“9·11”事件发生。小布什政府将海、陆、空及陆战队集结在一起投入到了中东。很明显，小布什政府是试图一次性地用其强大军事能力彻底解决来自各类国家及非国家行为体的挑战。但直到2008年金融危机爆发，小布什政府的解决方案未见效后，才由奥巴马政府接手开始对其战略进行调整。8年后，鉴于奥巴马政府在调整中对军方作用的“忽略”，才被再接手后的特朗普政府在口头上承诺要“重振军威”。而这，正是邓福德9月26日在面对参院质询时再次提出与“曼托珀曲线”相似想法的大背景。只是当广野庆子问他是否还像2015年在参院作证时那样认为俄、中、朝是对美国威胁的排序时，他才貌似将“中国威胁”的量级提高到了俄罗斯之前。有意思的是，曼托珀在1991年描绘他的“曼托珀曲线”时，认为俄罗斯崛起向美国利益发起挑战的可能性更大。当然，26年后邓福德也为他的判断设定了一个解决方案，即未来5年美国每年的国防开支必须要增幅3%～7%，才能在应对中、俄的军事现代化时保持竞争优势。为了强调这一点，他还专门提及了中国国防开支从2007～2016年扣除通胀后平均增幅为8.5%。

最后我想说的是，26年前曼托珀推出“曼托珀曲线”后，最终导致了美国海军“从海上”战略的出台。而眼下邓福德在他第二任期开始时“修改”了他3年前刚接任参联会主席时的“威胁判断”，将中国设定为“2025最大威胁”，又会导致美国有一个什么样的新军事战略出台呢？特别是在特朗普政府废除了奥巴马政府针对中国意味明显的“亚太再平衡”战略之后，美军将如何重置其在亚太地区的兵力结构和前沿部署呢？当然，作为参联会主席，邓福德最终还是要把总统的政策意图通过规划和计划变成一个个可管理的具体任务去落实。若乐观地想，他第二任期开始落实的那些任务，不正好为中国军队正在进行的改革提供了一些更具体的参照吗？
