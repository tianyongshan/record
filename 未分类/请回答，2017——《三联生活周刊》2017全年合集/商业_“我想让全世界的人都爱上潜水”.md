# 商业_“我想让全世界的人都爱上潜水”

商业：“我想让全世界的人都爱上潜水”

商业

“我想让全世界的人都爱上潜水”

——专访PADI全球总裁、首席执行官德鲁·理查德森

文◎程磊

PADI（国际专业潜水教练协会）的全球总裁德鲁·理查德森（左）与新设立的大中华区总裁楼彦

11月15日，作为全球分布最广、持证人数最多的潜水训练机构，PADI（国际专业潜水教练协会）宣布成立大中华区。目前全球已经有超过2500万人在PADI完成潜水课程并获得执照，他们中的大多数都宣称，潜水改变了自己看待世界的方式。中国以潜水人数连续5年50%的增长，成为PADI当前最关注的市场。潜水这件事，连接着怎样的可能性？我们专访了PADI全球总裁、首席执行官德鲁·理查德森（Drew Richardson）先生。

问：

中国市场在PADI的战略中处于什么位置？

理查德森：

在潜水培训市场，PADI无疑是世界上最大的潜水训练机构。创建51年来，我们在全球拥有超过6400家潜水中心和潜水度假村，由近14万名专业教练来对学员进行潜水培训并颁发相应等级的潜水技能证书。现在PADI每年发出的潜水执照中超过10%在中国。对比全球市场的增长，中国是增长最快的区域，最近5年中国持证潜水员的增长超过50%，我们预计未来中国的潜水人口将超过4000万。我们需要为这个蓬勃发展的市场匹配更多服务，所以我们来了。我们希望能将中国潜水的基础设施做得更好一些，同时也帮助不懂中文的海外潜店，去做好对中国市场的服务。

问：

今年上半年PADI完成转手，这个过程中为什么会出现被中资收购这样的传言？

理查德森：

在此之前，PADI的实际控制人是美国著名投资机构普罗维登斯股本合伙人公司。现在的实际控制人，是一个由家族基金和捐赠基金组成的财团Mandarin。Mandarin直译为“普通话”，可能是这个名字让人产生联想。

问：

从你们掌握的数据来看，什么样的人在潜水？

理查德森：

自1966年PADI成立以来，我们在全球共发放了超过2500万张PADI潜水证书，从男女比例来看，男性占了65%，现在希望女性也能更多地参与进来.年龄方面，男性的平均年龄是28岁，女性是27岁。如果以这个年龄段来观察中国，中国的女性占比很高，占55%。全球的潜水员画像是这样的：他们是一群性格开朗的人，受过良好的教育，收入水平较高，注重精神生活和生活质量，喜欢冒险、刺激一点的运动。

问：

中国目前没有像样的潜水旅行目的地，要发展这个产业，需要做哪些工作？

理查德森：

地方政府需要意识到，作为旅游产业的一部分，潜水本身对经济的带动虽然有限，但却是一个可以启发更多人参与到海洋环保中来，同时兼顾经济利益的一项运动。除了在海岸区域挖掘美丽的潜点，基础设施的搭建也可以更大胆一些。迪拜可以在城里造一个人工滑雪场，在中国建造人工潜水地当然也是可行的。我们可以看到，在海洋馆里开展潜水体验很常见。在欧洲，出现了人工建造的潜水深池.在台湾地区的台中，也有一个21米深的池子专门提供潜水体验服务.墨西哥的坎昆，有一座建在海底沙地上的雕塑博物馆，吸引了大量潜水爱好者。巨大的人工珊瑚湖、人工潟湖，都在陆续问世，一切皆有可能。

问：

很多人觉得潜水是一项危险系数很高的运动，实际接触下来，其实比想象中要安全，你觉得是什么原因让人们对潜水留下了危险的印象？

理查德森：

一个密闭的空间，相对陌生，水下还有一些貌似令人望而生畏的生物，想象起来，似乎令人畏惧。当然，这也是人们好奇之所在。其实，潜水发生事故的比例是很低的，远排在常见的足球、篮球等运动之后。而PADI存在的价值，正是将有可能发生风险的可能性降至最低，我们将所有的风险，分解到教学的每一个细项，休闲潜水就是在已经对水下情况足够了解的情况下进行。过去50年我们已经证明，只要遵循培训条款，风险是完全可控的。

问：

你个人的潜水是怎样的履历，有哪些毕生难忘的经历？

理查德森：

我十几岁就开始潜水，后来成为潜水教练，再后来加入PADI，把工作和兴趣结合起来，这是非常美妙的经历。我潜遍全世界，看过很多风景。令我印象深刻的是在南极，这是一片陆地上几乎看不到任何生命迹象的区域，只有冰山。但水底下却是一个生机勃勃的世界，有非常完整、丰富的生物链，有和印尼四王岛一样漂亮的软珊瑚，你不得不感慨大自然的奇妙。如果要说毕生难忘，现在全球热播的BBC纪录片《蓝色星球2》，他的重要推动者大卫·爱登堡曾回答过这个问题，他说他最难以忘怀的，是他第一次水肺潜水的那三分钟。我想，那三分钟，可能也是每一个热爱潜水的人毕生难忘的。

问：

有人喜欢水摄，有人喜欢看动物，也有人追求深度。在海底世界，你为之兴奋的是什么？

理查德森：

我喜欢海底那个三维世界带给我的沉静与自由，每一次下潜都会有新的发现，可能是庞大的鲸鲨，也可能是隐藏在珊瑚里的丁点小的豆丁海马。每次都不一样，每次都感受新鲜，让我每一次都有期待。你可以很休闲，像泡在热带的巨大鱼缸里.也可以很刺激，去挑战一些东西，去某个海底洞穴探险。每一种潜水，都有吸引人的地方。

问：

很多潜水员都认为潜水这件事改变了自己看待世界的方式，你呢？

理查德森：

人类大部分认知，建立在只占地球面积30%的陆地之上，当你接触到更加辽阔的世界你会怎样？潜水让我感到这个世界的希望。我父亲出生的时候，地球上是15亿人口，我出生的时候地球人口是20亿，现在这个星球已经住了75亿人。人口带来的环境压力是有目共睹的。如果大家都去潜水，能够深入接触孕育地球的海洋，就一定会理解海洋是地球的源泉，会和身处的世界有了情感上的联系，相信我，这是自然而然发生的。这继而会改变一个人对周围环境的认知和行为方式，如果可以的话，希望全世界的人都可以潜水，这也是我珍视这份事业的动力。

问：

如果希望让人们接受这项运动，你会如何向他人推荐？

理查德森：

这是震撼人心的运动，人类花了几千年时间才爬到了食物链的顶端，却只生活在地球30%的面积上，如果有好的、安全的工具，你又有足够的好奇心，为何不去探索那生机勃勃的70%的世界呢？潜水并不是极限运动，而是一种生活方式，是一种认识世界、审视自己的生活态度。潜水和旅行是美妙的组合，探索未知而奇妙的世界，结交志趣、性格相投的“地球”好友，在旅途和每一次下潜中，学会与这个世界相处，遇到让自己内心变得更美的人。

问：

气候变化，海洋变暖，海洋环境朝着坏的方向发展，你认为潜水这项运动的未来发展会怎样？

理查德森：

如果你问我，什么时候才是开始保护环境的最佳时机，我的回答是：当下，在这个想法出现的第一时间就付诸行动。我在PADI工作了30年，全球海洋环境确实遭受着极大的压力。更可怕的事情是，针对现状，我们没有行动起来。在PADI成立50周年的时候，我们倡导把更多的责任放在对海洋的保护上，我们不断指引潜水员，鼓励他们探索水下世界，培养他们保护海洋生态的意识。潜水群体将会是这个世界解决海洋问题的驱动者，他们会分享自己对那片深蓝的爱，他们的使命与对环保的热情，会让他们成为解决海洋星球威胁的最好人选。
