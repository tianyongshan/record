# 商业_从“好车”到“正确的车”

商业：从“好车”到“正确的车”

商业

从“好车”到“正确的车”

主笔◎李三

观致CEO刘良

即便把合资车也纳入视线，观致也都是上乘之作，但遗憾的是销量始终不如人意。这个一路磕磕绊绊走来的高冷品牌，何时会有转机？刘良就任观致CEO之初，有人说他有点临危受命的意味，几个月来，他治下的观致开始出现复苏的迹象。举手投足间，刘良传递出一种自信，这也许和他的成长经历有关。

“人生就像一盒巧克力，你永远也不知道下一颗吃到的是什么。”这话出自《阿甘正传》。对刘良来说，他开始品尝巧克力的时候，阿甘还不知道在哪里。

刘良1977年参加了高考，当时他高中还没毕业，他过了分数线，但在体检环节没有过，“因为小时候有中耳炎，对听力有点影响。考听力很搞笑，坐在隔壁，墙那边说打倒蒋介石，你听出来就好，听错了就说明耳朵有问题”。刘良读高中在乡下，那个年代念书不易，学校很烂，他是那一届的绝唱。

刘良一直觉得他的幸运之神是第一次参加高考开始的，“如果没有恢复高考，像我们这些农村娃，是出不来的。我父亲一辈子就念了三年半书，母亲文盲，但他们有一点很坚持，哪怕是勒紧裤带，也要让儿子上学。我念书时成绩一直很好，等到1978年高中毕业参加高考，我是县里的状元”。

大学还没毕业，研究生制度恢复了。他在纺织厂一边实习一边做毕业设计，并准备考研。那时国家教委有一个计划，选一批尖子生去国外留学，刘良真就考上了出国研究生。那算是幸运之神第二次关照到他。

1982年，原本准备去美国读研的刘良，突然又遇到了一个插曲，网球运动员胡娜去美国比赛，滞留不归，“胡娜事件”致使中美文化交流一度被暂停。学校辅导员跟他说，留学这事还是有，美国是去不了了，日本怎么样，日本还可以去。“我一个农村娃，哪里都可以，巧的是我在毕业那年鬼使神差选修了日语。

在日本，刘良读完了硕士，接着又考上东京工业大学博士生。东京工业大学是日本的麻省理工学院，他是机械工程硕士，高分子材料博士。在日本学完了，就去了美国，念了一年的博士后。不久，他就进入汽车产业。

到今天，进入汽车行业已有27年的刘良，曾任职威伯科公司卡车、客车、轿车事业部全球总裁，有着伟世通、福特等知名企业高管的供职履历。是什么吸引他来到了观致呢？

在较长一段时间里，刘良觉得汽车行业已经是一个夕阳产业了，但过去三年汽车行业开始不一样了，自动驾驶、车联网、新能源车等等，有些改变根本就是颠覆性的，这一切给刘良一种跃跃欲试的冲动，汽车行业要变成一个高科技很有吸引力的产业，他又有激情了。

三联生活周刊：

为什么会选择观致作为职业的落脚点？

刘良：

我对观致一直很关注，来之前也做过一些案头调研，我真心觉得一群曾经在奔驰、宝马等企业任职的全球汽车精英人士，聚集到观致这样的平台，以聪明著称的犹太人也为此投资，这是一个很有意思的事。还有奇瑞，作为民族品牌的标杆，一心要打造新的中国品牌，这是很有想象空间的事业。

三联生活周刊：

去年9月进入观致，今年3月出任CEO，你是怎样的心态？

刘良：

以前我服务的企业都是成功、成熟的跨国公司，观致是一个非常年轻的初创型企业，它的性质决定了企业文化和以往我经历的有较大差别。总体来说，我觉得在中国要成功，要对中国市场比较了解，要有能力制定策略并使之落地，我自认为我这方面有优势。我以前的职业生涯也遇到过很多困难，我觉得那些吃苦的经历，在观致说不定会用得上。我自信我能够带好一个团队，能够和股东进行充分沟通。

我来观致前，对困难是有预估的，来了以后现实情况应该说比我想象的挑战还要大，但我觉得这是个机会。也许当时观致“新高端”这个概念生不逢时，经过几年磨合，我觉得，只要现在找准方向，而且有足够的资金支持，观致一定会取得成功的。

三联生活周刊：

你觉得观致亟待解决的问题是什么？

刘良：

在我看来，观致首先缺子弹，产品就是汽车公司的子弹，没有合适的产品，拿什么去市场上竞争？我们快速决定做了一款和奇瑞平台共享的SUV，这个产品今年底就会亮相，我来之前是没有这个计划的。SUV是中国市场的宠儿，我们已经有观致3都市SUV、观致5 SUV，今年将会推出副品牌Model Young的一款紧凑型SUV。

三联生活周刊：

在产品认知上，观致是否需要一些调整？

刘良：

观致是好车，但它也许不是一款正确的车。以德国人的标准，更多看重操控性、制动距离等，但以中国市场目前的消费需求看，更多的还是看重空间、动力选项、内室色调等等，当然还有价格定位。之前，观致的产品只做对了一半，做了一个好车，相对我们的目标客户来说，不一定是正确的车。

所谓做正确的车，出发点不是工程师的梦想，出发点是市场需求。未来的新车型将按照这个思路去做，既要做好车，更要做对的车。

三联生活周刊：

你对观致的未来有信心吗？

刘良：

我来观致后的最主要工作之一，就是确定观致品牌的核心价值。我认为，中国汽车市场处于消费升级过程中，“新高端”品牌定位因此有了市场机会。观致的未来在于消费升级，汽车消费者买第一辆车时，由于财力不够，用小几万块钱买了一个自主品牌的车，几年之后，当这些消费者开始购买第二台车的时候，观致应该在他们的消费清单上。

三联生活周刊：

你所说的“正确的车”，是否应该包含一些流行元素，比如移动互联？

刘良：

我对车的定义从几方面来考虑：一是造型，观致在这方面有独特的设计，我们要继续优良基因，更年轻化、更动感.再就是配置，其中一类是智能互联，观致逸云在2014年刚出来时是行业领先的，未来新款开发，将重新梳理它的功能和能力，了解市场需求和同行的开发态势，我们要做差异化，确保在新车出来时，观致逸云能够重回行业领先。

当然首先要保持品质第一，然后是智能互联，将它变成一个亮点，动力要有非常大的提升，全驱的动力选项会出来，内饰也将重新审视，这些措施在驾驶员辅助驾驶技术、安全技术等配置方面都会加强。最后，我们会非常慎重地审视价格定位。

三联生活周刊：

怎样从一个好的产品变成一个“好+对”的产品？

刘良：

从一个好的产品变成一个好的加对的产品，要求是很高的。品牌方面确实是很大的一本书，首先品牌就是品牌定位的问题，我们的定位就是新高端。观致的目标群体为“80后”“90后”，这批新的汽车消费者，他们会越来越多地看中独特的时尚造型、一流品质以及智能互联带来的体验。时尚的设计、一流的品质我们已经有了，在产品体验以及智能互联方面再进一步加强，我们有能力做得更好。

这是很有挑战的，一个产品要不断地投入，既做好又要做对，品牌也要打造，而且可能不会带来立竿见影的效果，需要一些时间。另外，尽管现在中国市场对新能源车的追求方兴未艾，但我认为，对观致来说，提出两栖战略是正确的，传统车的量需要尽快上来，一旦达到基本的销量开始盈利，传统车业务就会变成观致新能源车发展的助推器。而这个助推器一旦出现，就变成差异化的优势。
