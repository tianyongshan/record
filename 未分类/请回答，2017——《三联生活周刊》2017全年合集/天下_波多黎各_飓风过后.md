# 天下_波多黎各_飓风过后

天下

天下

波多黎各　飓风过后

9月23日，波多黎各圣胡安当地居民坐在屋外乘凉。波多黎各20日遭飓风“玛丽亚”袭击，全境断电并实行宵禁，部分地区遭遇洪水。

美国　抗议

9月24日，坦帕湾海盗橄榄球队和明尼苏达维京人橄榄球队在明尼阿波利斯的美国合众银行体育场比赛前，坦帕湾海盗队队员在奏国歌时下跪，以示对美国总统特朗普抗议。去年9月，美国NFL（国家橄榄球联盟）的非裔球员科林·凯普尼克在奏国歌时，拒绝起立以抗议种族歧视，引起轰动，后来有更多球员效仿科林的行为。

土耳其　释放

9月26日，土耳其新闻评论家卡德里·居尔色里（Kadri Gursel）出狱后亲吻妻子。7月，土耳其对政府持批评态度的《共和报》的记者和管理人员接受法庭审理，17名该报工作人员被控“支持恐怖主义”。评论家卡德里·居尔色里包括在内。

也门　革命纪念

9月25日，也门童子军参加在首都萨那举行的1962年革命纪念活动。也门1962年9月26日爆发革命，革命军推翻了哈米德家族的统治，并在也门北部地区建立了阿拉伯也门共和国。

叙利亚　战争

9月24日，叙利亚民主力量（SDF）的一名成员在车上安抚受伤的同伴，他们前往拉卡的一家医疗中心接受治疗。近期，美国支持的叙利亚民主力量向拉卡发起进攻，已经占领了80%～90%的地区，并将剩余的武装分子围困在北部的几个街区。

沙特　解禁

9月26日，沙特国王萨勒曼发布有关交通系统改革的国王令，其中包括允许女性驾驶车辆。这也意味着，全世界唯一一个禁止女性驾驶车辆的国家，对此解禁。据《纽约时报》报道，新规将于2018年6月生效实施。
