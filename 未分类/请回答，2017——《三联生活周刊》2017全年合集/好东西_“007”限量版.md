# 好东西_“007”限量版

好东西

好东西

“007”限量版

欧米茄海马系列“指挥官”限量版腕表设计以詹姆斯·邦德的海军制服为灵感，主色调为红、白、蓝三色。蓝色陶瓷表圈的潜水刻度由Liquidmetal技术打造，前15分钟的刻度饰以红色橡胶。表背可见摆陀上饰有的英国皇家海军中校军衔徽章的“三道杠”图案，以及007的“子弹”标识。

剩饭转化器

来自英国的一位设计师本·沃森（Ben Cullis Watson）结合日本百年的堆肥方法，设计了一款可以将剩饭剩菜分化成液体肥料的垃圾桶Bokashi。垃圾桶内分有2个隔腔，可将食物降解。垃圾桶开合部分还安装了橡胶密封圈用来防止气味逃逸。由于食物的分解会产生大量的恶臭，它内置了臭氧除臭系统，可以净化内部空气。

蔬菜种植器

Ogarden是一款主打绿色蔬菜品牌的摩天轮蔬菜种植器，它有20根种植管组成，可种植80株蔬菜。将蔬菜种子播种在由椰子壳碾磨而成的压缩土壤中，底部的水槽可以定时定量补充水分，同时中间的120瓦照明设施可以给植物提供生长所需的光源。运行时慢转速可以让蔬菜均匀向上生长。

简易喷泉

Ankway是一款太阳能简易喷泉装置，它无需铺线铺管，只需放在水面，在“太阳神”的照耀下即会喷出五彩斑斓的喷泉。这样的喷泉可以增加水中的溶解氧，同时也可以让池塘变得更加生机勃勃。太阳能水泵每小时最多可以输出40加仑的水，并将水喷射至18英寸到空气中。

智能吹风机

Sylph是一款智能吹风机，能以磁性方式挂在墙上，其重量只有150克，外观采用无扇叶外形，内置冷、热、智能风三种模式。长时间使用的吹风机可能因吸入灰尘和其他碎屑而影响设备的性能，Sylph的自动清洗模式可从出风口吹出灰尘和碎屑。

会“开花”的伞

这款外形清新的雨伞会在雨水的冲刷后，在面料上“开”出小花。由于面料的特殊性，雨伞的布料遇水则会将原本隐藏着的可爱印花图案显露出来，在恼人的雨雪天气里看到温馨的效果。

立体感线条

「上下」“揽月”系列限量版看似中国竹篮，实以皮革模拟竹篮的肌理与构造，编织过程中不同程度的拉紧度处理赋予线条愈加鲜明的立体感。

带刺的镜子

Pols Potten品牌的Prickle墙面挂镜，带铜色金属涂层的铁刺包围着直径81厘米的圆镜，不同长度和深度的金属杆构成了放射状的3D造型。

心动时刻

I Do心动系列白金男女对戒以恋人的心跳波动为设计主题，以白金与玫瑰金的对比，心电图的律动波纹图案与钻石一起被嵌入戒指中。

彩色花瓶

法国设计师布鲁莱克兄弟设计的Ruutu花瓶系列包含5种尺寸和7种不同色彩，制作时需要将玻璃吹入钢模并用手工抛光，花瓶底部还刻有设计师的名字。

金属明式椅

Driade品牌的Mingx座椅是从明代圈椅的经典造型转化而来，中国的木椅被改换成钢管材料并加上皮质坐垫，但它的原型仍然清晰可辨。

黄铜落地灯

De Castelli品牌的Ribot落地灯，手工制作的抛光黄铜灯罩如同一件雕塑品，LED灯光照射在它的表面，产生液体般波光粼粼的效果。

刻纹腕表

经典的Boucheron Reflet系列腕表，黄金表壳上雕刻着精细的圆模刻纹，与白银表盘上的线条相互对应，4个时数位置用小颗钻石来标记。

动感线条

Zaha Hadid设计公司为Perrin Paris品牌设计的限量版手袋，白色小牛皮搭配羊皮内衬，前盖上装饰了不对称形状的金属环扣，动感流畅的线条尤其引人注目。
