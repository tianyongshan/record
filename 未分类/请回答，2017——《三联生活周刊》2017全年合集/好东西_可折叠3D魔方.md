# 好东西_可折叠3D魔方

可折叠3D魔方

好东西

可折叠3D魔方

Infinity Cube是一个可以无限翻转折叠的3D魔方，由8块较轻的铝合金正方体连接而成，可以从各个方向进行折叠并翻转且不会卡机。它可以用来缓解工作间隙的焦虑和烦躁。平铺时，其厚度仅为20毫米，小巧而轻便。

传递思念的珠宝

由意大利“珠宝诗人”Marco Dal Maso设计的TOTWOO智能珠宝，其半球弧面造型分镶黑色和白色两种天然玛瑙。两个配对首饰可以互传心意，除了敲击首饰传递默认的“我想你”之外，用户能在APP里选择四种“爱的密码”发送，对方首饰则会闪出相应的光。

陶瓷潜水表

防水深度达600米的欧米茄“碧海之蓝”腕表，表壳由整块蓝色陶瓷打造而成，其陶瓷表圈上有以Liquidmetal技术而饰的潜水刻度，排氦气阀门则采用橙色橡胶注塑填色，齿槽状旋入式表背采用陶瓷Naiad锁扣技术。腕表通过了由瑞士联邦计量研究院（METAS）设立的八项测试。

省力厨房剪刀

Clever Cutter是一款自带砧板的厨房剪刀，它的独特之处在手柄之处的弹片设计，剪下去后弹片会受力反弹，可以缓解手腕用力。相比于平时用刀在砧板上切菜，Clever Cutter可以直接在锅边就把菜剪入锅内，便捷省力。

折叠蜂箱

这款折叠蜂箱能够自动吸引自然界的蜜蜂。蜂巢的入口是隧道式泡沫材质，模仿空心树木的缝隙，底部的网状结构可以保证其通风效果。吸引蜜蜂主要依靠箱体底部的费洛蒙信息素垫。蜜蜂捕捉到这种信息素时，会以为蜂巢内有蜜蜂群而陆续飞来。

多功能花盆

Parrot Pot是一个智能多功能花盆，内置了四种传感器，可以实时监测阳光、土壤肥沃度、温度以及土壤水分这四种数据。盆口分布着四个喷水口，从花盆底部的蓄水池抽水灌溉。用户可以通过手机远程读取植物状态，并远程控制浇水灌溉。

便携防水收纳包

Droplet XL水滴收纳袋是由防水材料制成的超轻型收纳包。它配有一个水滴形的硅胶储存盒和铝合卡扣，适用于保持任何电子设备的干燥，也可用于冲浪后存放潜水衣。20升容量适合大量的装备，但折叠后只有手掌大小（9厘米×7厘米），方便随身携带。

银莲花开

法国Lalique工坊的银莲花主题系列包括桌面装饰品、醒酒器等，磨砂缎面仿水晶和抛光仿水晶制作出花朵的美态，红色仿水晶花蕊上还装饰了白色珐琅。

白金动偶钟

帕玛强尼Hippologia自动钟以白金阿拉伯纯种沙漠马的形象代表时间的流逝，母马和小马驹被设计为座钟的机芯，底座的机械分别连接动偶装置和时间显示装置。

迷你手袋

卡地亚2017年春夏系列迷你C de Cartier手袋用色彩鲜艳的小牛皮制作，包身上只有明显的包边缝线和银色雕刻字母作为装饰，优雅中透出青春活力的感觉。

线条与圆点

出自巴塞罗那设计工作室Goula/Figuera的“线条与圆点”照明系列，一组由相互关联的图形元素组成的模块化吊灯，容纳LED光源的电源线构成它的枢轴。

Q系列扬声器

英国音响品牌KEF发布的第八代Q系列包括书架式、落地式和中置式扬声器，专属Uni-Q同轴共点技术配以全新高音负载阻尼管，高频下段的声音尤其表现出色。

太空铝椅

意大利品牌Altreforme的Cyber铝椅如同一件太空时代的家具，鲜艳的金属色彩结合接近雕塑的独特形状，其中也包含了向戴维·鲍依的《太空怪人》专辑致敬的因素。

孔雀腕表

宝格丽Divas's Dream37毫米玫瑰金镶钻腕表，珍珠母贝表盘上伫立着一只华美的孔雀。在罗马神话里天神朱庇特之妻朱诺拥有圣鸟孔雀，它是丰饶与永恒的象征。
