# 好东西_多功能手机配件

好东西

多功能手机配件

科技创新公司Mutants DesignLab推出的这款Wondercube数码配件外形小巧却功能强大，它集合了数据线、充电电缆、读卡器、OTG USB、紧急电话充电器（9V）、应急手电筒、电话支架、钥匙扣等多种功能于一身。

蓝宝石表壳腕表

宇舶表将原材料氧化铝与过渡金属铬一同加温至2000到2050摄氏度，进而结晶为彩色蓝宝石，打造出首款Big Bang Unico蓝宝石腕表。表壳中层、表圈和表背均切割自整块蓝色蓝宝石晶体，具有耐磨的高硬度。透过表壳其自产机械机芯清晰可见。

旋转钻石

萧邦Happy Hearts系列戒指将可以旋转的单颗钻石与镶嵌钻石的玫瑰金心坠相结合，并点缀了绿色渐变的心形孔雀石，各种心形元素大小不一，交错排列。

高清无人机

Micro Drone 3.0无人机在机身底部配置了一个单独的720P摄像头和万向调节装置。虽然身材小巧，仍具有每小时45英里的时速，在500英尺的范围内可以进行飞行倒置翻转等特技。由于摄像装置并没有嵌入机身中，因此在娱乐性飞行中，也可以将摄像头拆除以减轻重量，保持续航稳定性。

迷你心电扫描仪

Kardia新推出的这款迷你心电扫描仪可以精准监测用户的心房颤动频率。只需要将两根手指分别放在Kardia手机表面安装的电极上，驱动器即可对指纹进行扫描，显示心率参数，并在参数超出正常范围时发出警告。它由CR2016硬币电池供电，足以维持200小时的正常使用。

自由折叠扬声器

TransSpeaker是一款可以变形的支架扬声器。设计师尼尔·李（Nier Lee）受到变形金刚的启发，将四个扬声器单元拼接组装，并实现自由折叠。展开后的扬声器还可以作为iPhone或者iPad的支架使用。

风球空气净化器

这款Windball风球空气净化器，剔除了传统净化器多余的空间设计，将核心部件HEPA过滤片浓缩在Windball的球体中。有空气净化和冷风对流两种运行模式可选。当冷风对流模式开启时，顶部HEPA过滤部件抬起，房间内的空气将呈现对流模式，内置风扇用于冷空气循环。

海星吊坠

Jewelvary品牌的Starfish吊坠以海星为设计元素，珊瑚、彩色蓝宝、月光石、沙弗莱石、绿松石、祖母绿以及珍珠等融合在一起，构成海星的绚丽色彩和枝形结构。

斜面水晶视窗

播威Virtuoso Ⅷ腕表采用沿着12点至6点轴线的斜面水晶视窗，第一个盘面显示动力储存，第二个显示小时、分钟和大日期视窗，机芯可提供10天以上的动力储存。

极简酒吧凳

Cappellini品牌的Alodia酒吧凳以简约大胆的设计实现支撑功能，凳子的概念被减少到最低限度的材料，由此提供一个清爽的轮廓。

空间之鸟

KEF品牌的Blade共点声源扬声器搭载4个精密的9英寸低音单元，它的外形设计源于现代艺术家布朗库西的《空间之鸟》，连续的骨架线勾勒出浑然一体的曲线。

金属色彩

宝格丽2017春夏配饰系列中的Serpenti Boxy Dragone系列晚宴包，金色山羊皮包面上装饰着银色漆面小牛皮的蛇形图案，镀金黄铜拉链上同样搭配了珐琅蛇头及吊穗。

明与暗

Holly Hunt品牌的Summit吊灯以黄铜材质实现光明与黑暗的组合，深色青铜外观的灯罩对比明亮的抛光青铜内饰，反映出美丽的光线。

红狐狸

Iittala品牌与艺术家克劳斯·哈帕尼亚米（Klaus Haapaniemi）合作的Vulpes红狐狸玻璃摆件，二维图形被转换为三维物体，结合冷热加工技术的手工制品每个都是独一无二的。
