# 好东西_彩色吸音墙板

好东西

好东西

彩色吸音墙板

西班牙工作室MUT Design创造了一系列受甲虫壳启发的彩色吸音墙板。这一系列包括12种不同的设计，每种设计都模仿昆虫壳上的图案和颜色，通过橡木制成的支架将其固定在墙上。

可拆卸珠宝

千叶珠宝“燃雪”系列戒指以桃心形托爪密镶钻石而成，可拆卸的戒指花头可作为吊坠使用，简约的戒圈可作为一款独立的时尚配饰。

非充气式轮胎

非充气式轮胎Superelastic是NASA为火星探测车准备的最新成果。它表面布满了用镍钛合金制成的金属丝，编织方式与锁子甲十分相似。拥有形状记忆功能的镍钛合金网面使其在布满杂石的路面上运行良好。

宠物冲水马桶

宠物专用的冲水马桶Catolet内置了运动感应器，能感应猫猫狗狗是否站在上面上厕所，并能辨别排泄物是小便还是大便。如是尿液会顺着凹槽流出，如是大便会被磨碎机磨碎，随着尿液与水流入下水道。

清漆保护膜

无味油漆Belay能不留痕迹地从表面上剥离下来。清漆般的光泽能保护表面免受划痕和污渍的破坏，可应用于曲面和不平整表面，并且不受木材、金属、树脂、瓷砖以及石材等材质的限制。

钢琴键衣架

设计师Patrick Seha设计的这款壁挂式衣架Piano，采用了类似钢琴键式的结构。挂在墙上的一根根橡木条犹如墙面装饰，轻轻将挂钩拉出便可悬挂大衣外套，收纳背包、雨伞等，实现了灵活的收纳策略。

翡翠绿大明火珐琅

宝珀在机芯设计上将双轨制历法的农历与单轨制历法的公历结合，显示出生肖年、天干地支、五行、时辰和月相。这款中华年历孤品腕表不仅所有显示都以金色涂饰加以高温炉火焙烧，表盘更采用了翡翠绿的大明火珐琅。

实木吧台凳

Kristalia品牌的Sharky凳用聚氨酯椅座搭配实木椅腿，构成舒适的带靠背吧台凳。作为椅架和底座之间的连接部分，两个鳍形连接元件清楚地保留在后部。

莫里斯印花

LOEWE William Morris手袋系列重新演绎了工艺美学大师威廉·莫里斯的经典印花主题，从他的作品中选择森林、莨苕和金银花等图案，印制在大象包、Puzzle几何包等包袋上。

百年马克杯

芬兰最古老的品牌之一Arabia为庆祝芬兰独立一百周年，推出了十款不同图案的马克杯。杯身上的装饰图案由独立以来的每个十年衍生，讲述着各个时代的故事。

红纹石戒指

意大利珠宝品牌Pomellato的Ritratto戒指，用作戒面的大颗桃红色红纹石上点缀着祖母绿。宝石用纵向切割的方法，让表面呈现出不同色调的变幻图案。

彩色便携灯

Marset品牌多彩轻巧的便携灯Bicoca用彩色聚碳酸酯制作，灯柱部分形似铅笔，是圆形和三角形等基本几何图形的叠加，可倾斜的灯罩用来调节光源的方向。

珐琅彩绘

蒂芙尼Metro腕表系列的设计灵感来自新艺术运动时期的彩绘玻璃，珍珠母贝表盘上以珐琅彩绘工艺绘制灵动的花朵图案，表壳和表链上镶嵌密集的圆形切割钻石。

劲黑钢笔

万宝龙与设计师马克·纽森合作的新款Montblanc M系列劲黑钢笔，亚光黑色树脂笔杆中间点缀了跳跃的亮橙色，衔接笔帽与笔杆的磁性设计使笔帽不易丢失。
