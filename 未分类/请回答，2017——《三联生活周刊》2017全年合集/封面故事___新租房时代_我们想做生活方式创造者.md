# 封面故事___新租房时代_我们想做生活方式创造者

封面故事：我们想做生活方式创造者

封面故事 | 新租房时代

我们想做生活方式创造者

——专访自如CEO熊林

记者◎王珊

自如网 CEO熊林（于楚众 摄）

三联生活周刊：

6年前开始做自如的时候，是基于一种什么判断？为什么会采用自如这种模式来做租房服务？

熊林：

首先是宏观层面看大势。当城市化进入到中段，比如60%左右的城市化率，有一种情景就会发生，那就是大城市里面会有超过30%～40%的人口需要租房住。像北京有2000万人口，至少要七八百万人需要租房子。我们过去所有的房子，都是为了销售而建造的，从它的设计、定价，都不是为了租赁。当时链家租房这块业务，虽然只占到整个业务的10%，但每年都有40%以上的增长。在美国，专业化租赁机构大概占到市场的60%左右，在日本的话能够占到87%以上，但是在中国这个比例还非常低，不足3%。说明这块市场需求是很大的另外一个是微观层面的真实感受。链家董事长左晖在北京租了近10年的房子，我自己刚工作那会儿也是租房子，各种类型的房子都住过。当时，我去做了很多一线的调查，跑到通州、苹果园等地方，去看年轻人租房的情况，听到的都是吐槽。

市场有需求，也有痛点，但具体要怎么做？我们需要找到一种业务模式。这时候我就和左总去了日本，回来后我又自己去了美国、英国，去看他们的模式。房屋租赁公司有几种，一种是单纯的经纪公司，做一个信息服务的中介平台.也可以做一个集约式的房地产公司，比如自己开发或收购房屋用于租赁.当然还有第三种，就是自如这种模式，我们既做产品，用互联网和标准化的装修把房子和人连接起来，同时又做好服务，把保洁、维修、搬家、装修这些事情统统做起来。我们认为第三种模式，是最适合中国的一种模式，既能让房主和租客双方省心，又能以一定的速度扩张。我从一个顾问的角度，强烈地建议进入这个市场，最后左总拍板，自如就这么干起来了。

三联生活周刊：

作为第一家专业化租房机构，是不是挺难的？因为好多服务链条或者业务模式都是要从零建立的。

熊林：

对。你说的这个是特别实际的问题，当时我们去看美国、日本的公司，管理十几万间房子，市值都是几百亿美元，你会觉得你对标的企业都跟在天上一样。中国的市场土壤和他们完全不一样，中国没有这么大量的集中的物业服务来让我们去把它进行规模化租赁运营，我们是分散式的，一个城市里几百万套房子，可能有几万个物业公司，当时在市场上找一家员工超过20人的保洁公司都没有。我们都知道年轻人需要带Wi-Fi的房子，我们就去跟联通谈，跟电信谈，但根本谈不下来。我们想为出租房配备家电，去跟国美谈苏宁谈，他们一听不是集采，而是要挨家挨户送到，就不理我们了。

三联生活周刊：

那大概到哪一年的时候，你意识到这个模式成立了，这个规模是可以做得起来了？

熊林：

我们大概是做了一年半，到2013年的时候。当时最明显的事儿是，我们的家电采购可以做集采了。前面那些不理你的供应商，也都愿意参与了。通过那一次集采，一下子把家电的成本就降低了10%以上。我们马上就把这个省下来的钱，给自如客配置的家具家电做了提升，那一年我们发布了自如友家的3.0版本，房子里开始配了沙发、微波炉。2013年，自如房源突破了5万间。那时候我们就感觉，这个事情看到希望了。

三联生活周刊：

这几年下来，自如的租客有哪些变化，他们对房子的需求发生了哪些变化？

熊林：

这几年，我们发现租户的年龄在变大。为什么呢？因为越来越多的租户不仅选择在自如租住，还会在自如恋爱、结婚。我们发现了这个需求，然后认真地去满足，也针对性地推出了整租服务。我们现在有一居室、两居室，也有家庭居住的三居室。我们希望产品能够覆盖年轻人生活需要的各个阶段。

他们对房子的需求也在不断提高，主要是对质量和服务的要求在不断提高。因为很多人一毕业就住进了自如的房子，他们没有经历过之前的黑中介，起点是高的，所以后边要求肯定会逐步提高。比如家具家电的质量、设计的人性化、服务的细致程度。所有这些，倒逼我们必须不断提高自己的产品品质和服务质量。

围绕业主、客户、房子三方，我们不断挖掘需求，推陈出新。从自如友家、自如寓开始，先后推出了自如整租、自如驿、自如民宿、业主直租等产品，基本涵盖了城市租房族的各种需求。2016年6月，还推出了“自如优品”，用户可以在自如平台上购买到家具家居等多种商品。我们希望能把租房生活从标准化迈向个性化。在不久的将来，人们更愿意装点自己租来的房子，会意识到即使住在租来的房子里，也能享受到品质生活。

三联生活周刊：

互联网对于租房这种特别传统的业务，有哪些战略意义上的价值？

熊林：

我之前做过企业里面的ERP系统，也做过企业的战略咨询，所以我来看这个业务的时候，我们最开始就确定了四个最重要的方向，到今天自如依然是这么说：首先我们就是一家做产品的公司.第二个是做服务的公司.第三个我们用互联网O2O的服务方式，来向客户去交付我们的产品和服务.第四个，我们希望我们在全世界，哪怕不建一间房子，但是我们能管理一个全世界最大的社区。

那为什么要去做互联网？因为你想做产品，但是这个产品是一个重度投入的，每套房都要花很多的钱来装修，你有没有一套系统，真的把每一套房子的装修、投了多少钱、今天进行到什么程度、什么时候可以租给客户，把这些事情给管起来很重要，所以你必须要有系统，靠Excel管1000间是可以的，但是你管10万间你就管不过来，那更不用说你管100万间。

另外你会发现，大城市里面不管是客户找房、买房还是租房，大家都越来越从网上去找，所以我们说自如做的时候，那为了让大家很方便地去找房，去获取信息，不能再用门店的方式，所以我们一开始说自如就不建门店。我之前做过顺丰的咨询，顺丰是中国第一家给快递员配移动终端的公司，所以我非常清楚顺丰为什么能承诺，我的收派员在半个小时之内一定可以到达你那儿去收件，他的收派员很早就用了手机终端，我们一定要用信息化的手段来提高服务人员的工作效率，所以我们的保洁阿姨、搬家师傅、维修师傅，全是用移动终端来接单的。

三联生活周刊：

现在自如已经有多元产品。对于未来的发展，你还有什么设想？

熊林：

自如希望成为一家全世界范围内受人尊重和喜爱的企业。前提是我们所做的事情必须是专业的。我们必须要客观地承认，房子的投资属性在下降，像以往30%、60%暴涨的年代已经一去不复返。未来人们生活的半径越来越大，不会因为一套房子绑在一个城市、绑在一个单位。人们开始体验生活更多的可能性，我们就是在满足这种可能性

我们希望再过一两年，能够将管理的房屋达到80万间，这是目前世界上最大的公寓运营商的规模。我们过去花了6年时间，使客户达到100万的数量，希望再花不超过18个月的时间，这个数量能够翻一番，达到200万。并且无论规模多大，都始终为客户提供高品质的产品和服务。对大中小型业主来说，自如是专业的资产运营管理商。对自如客来说，自如是品质生活方式的创造者。
