# 封面故事___物质审美_段召旭：仪式感在殿堂之外

封面故事：段召旭：仪式感在殿堂之外

封面故事 | 物质审美

段召旭：仪式感在殿堂之外

记者◎傅婷婷

段召旭（王旭华 摄）

段召旭是一位钢琴演奏家，也是一位古典音乐教育家。在他的琴房中，整整一面墙是数千张他收藏的古典音乐唱片。每次练琴结束抬起头来，他会看到对面墙上的小提琴形状的钟表，上面有巴赫的头像。时钟的旁边，则是他画的一张素描，那是他的男神钢琴家李斯特。

客厅两面墙的书架上，大部分关于音乐，小部分关于文学。偶尔在一排书里，会出现“超人”或“正义联盟”的大部头，显得跳跃又不失情趣。而他收集的美国漫画角色“超级英雄”的模型，使宽敞的客厅显得过分丰富，却也还是放不下的，尚未拆封的规整地放在盒子里，整齐地摞在储物架之上，还有一部分则占据了卧室的一面墙。

“物”的丰富性，体现了段召旭在职业身份内外的生活方式。古典音乐很容易让人联想到音乐会上演奏家和听众的“仪式感”，但真正的仪式感，却不在殿堂之上。玩具模型有从购入、动手做到高级玩家的层层递进的挑战，是段召旭在生活上属于自己的仪式。而古典音乐的普及，听起来深入浅出的网上课程和三、四线城市的巡演，段召旭则经历了“先破后立”的内心的仪式。

从建立到消解

段召旭是古典音乐圈子里的人。在穿着上，这一行有一些讲究。

在比较高规格的钢琴演奏音乐会上，演奏家的着装应该是三大男高音演唱会那样，燕尾服、小翻领白衬衫，并且搭配一个白领结。还有腰上的腰封，以挡住腰带。在相对轻松的音乐会上，演出服装则比较多元化。演奏家可以用领带代替领结，甚至连领带也放弃，穿T恤衫和休闲西服，就上台弹奏。

学生时代的段召旭也曾经每逢演出，都是小领衬衫和白色的手打领结，弹独奏时如此，弹伴奏也是如此。“当学生的时候，肯定要证明我很专业，我很懂。那时候如果穿错了，别人可能觉得你不懂，太幼稚了，就过来教你了。”段召旭定制的第一套礼服，是在香港弥敦道上一个巴基斯坦人开的店铺里。当时和他同去的一位老指挥跟他说：“这得来一身儿，咱们就靠这个。上台就那几分钟，这衣服最重要。”

时至今日，专业水平提升后带来的自信，反而消解了段召旭在着装上的仪式感。对他来说，形式上的仪式感已经不那么重要。演出时，他选择服装也在符合自己审美的前提下，更加多样化。但段召旭一直保持着搭配手打领结的习惯。他对手打领结的特殊情感，源自钢琴家霍洛维茨。霍洛维茨有收藏领结的爱好，每次演出永远打花领结，他一共收藏了800多个领结，还因此被国际领结协会授予了全球最优秀打领结奖。段召旭也同样对领结有特殊的情感，比起成型的领结，手打领结更加灵动，并且多了一层仪式感，“那时我也觉得演出就必须要系领结，也养成了收集手打领结的习惯”。日常生活中，段召旭也曾在着装上很专一于某一个品牌。那未必是最好的，却一定是某段时间里无法代替的。段召旭青年时代的偶像是阿兰·德龙，阿兰·德龙有一个品牌，那时候，段召旭的皮鞋、衬衫、领带、西装、袜子、香水，全都是这个牌子，其他的他不看也不买。只是这个牌子并没有做得很好，撤出北京时，店里的代理专门给段召旭打电话：“我们要撤了，大甩卖，你来不来？”段召旭跑到西直门，买了店里所有的自己那一码的衬衫。

衬衫穿上了，装衬衫的盒子还留着，鞋穿上了，鞋盒还留着。段召旭喜欢到一种痴迷的程度。直到四五年前，他才把它们珍藏了起来。“对这个人和这个品牌我还是很热爱，但是生活里不会只是它了。”就好像他崇拜的钢琴家李斯特，以前他专注于弹奏李斯特的曲子，而现在也不会只弹李斯特了。

从物品到内心

段召旭从小持续收集美漫角色“超级英雄”的漫画和模型。

段召旭的家里有各种各样的模型。变形金刚、圣斗士、街头霸王、超人、高达……这些模型在客厅和卧室里分别占据着一面墙的陈列。他现在最喜欢组装的是高达模型，因为高达能够带来制作的快乐和审美的享受。模型虽然只有二三十厘米高，但零件比较精细。段召旭在卧室的阳台设置了一个“喷漆”专用工作台，“喷漆”是高级玩家的玩法。为了这个爱好，他配置了排风扇、防毒面具、喷漆专用工具箱。

“喷漆”是把塑料零件根据自己的审美去设计颜色。他最喜欢的白扎古，是他第一个使用“喷漆”技术的模型，通过喷漆，他做出了黑色战争的痕迹，枪和炮的金属光泽，用银色点出掉漆的做旧风格，用锈液点出生锈的感觉。

在两年前的一个暑假里，段召旭去动漫店学习了“喷漆”。他很怀念那段每天跑到动漫店，一待就是一下午的日子，只是专注于自己的兴趣，不用想别的。在他看来，夏天可以有的最理想的生活，就是先给自己喜欢的模型喷上颜色，然后再出去撸串。在学习这项技术的时候，段召旭曾经跟动漫店的那群开店玩模型的孩子经常用行话讨论玩法。

投入其中的段召旭似乎有点痴迷：“喷漆的时候闻多了，感觉油漆还有一点甜味。”喷漆让段召旭有很多的联想，有关于生活的，也有关于钢琴演奏的。

“比如，喷漆技术的素组就是把零件剪下来，打磨，然后一拼即可。这就跟弹钢琴的第一步一样，把音符都弹下来，节奏都卡准。有的人弹琴就如同素组，虽然没有什么情感，但可以把复杂的曲子弹得极其精准，也很不容易。而更进一步的染色，高手会做出渐变的颜色。钢琴里的情感和句子的润泽，比如哪个音抻长一点，哪个音短一点，和这一点也有相似之处。”

“喷漆和弹钢琴一样，永远不可能对每一个细节做到真正的控制。比如喷漆时，为了制造‘硝烟感’的效果，需要先拉阴影，再喷主要的颜色。而拉阴影的时候，会有粗有细，面积有大有小。其实这才是艺术。”

练习技术的时候，段召旭也会想到“超级英雄”的无所不能——准确性、速度、力量。而不被任何一种技术难住，也是演奏家在键盘上的理想状态。

从学术到审美

从殿堂到普及，这种外在仪式感的打破，其实是内心仪式感的一种建立。

“我曾经在20多岁的时候，把李斯特的‘超级练习曲’一套都给练了，当时想开一场音乐会。后来读完音乐美学就放弃了。弹奏成套的曲目，基本上是成全自己。既然表演是在大众面前的展示，你就不能不考虑观众。很多艺术家，考虑的还是内行的眼光。这是一个怪圈，可能最后的结果是内行觉得不错，但是外行根本听不懂。所以就看你是想为了证明自己，还是考虑观众的感受。”段召旭说。

段召旭现在常常去演出，频繁的时候一周2场，不频繁的时候，一个月1～2场。有时是在省会城市，比较多的是三、四线小城市。在很多地方，之前从来没有开过钢琴独奏会，连音乐厅都没有，他们就会到酒店里演出。“其实那个地方从来没开过独奏会，我去开，我觉得蛮光荣的。有的还把我载入当地的地方志的音乐史册，说是首场钢琴独奏会。小城市没有人去演，更需要我来传播。”

前几年，他在中央音乐学院读了音乐美学，开始重视在演奏中和听众的互动。有一次，段召旭跟一位教授聊天，觉得古典音乐式微，搞不过流行音乐，喜欢古典音乐的只是小众。但教授投入在自己的学术领域里，并不觉得自己的领域只是小部分人的爱好。段召旭当时觉得很惊讶。“我们的父辈，可能会去追求一些他们认为高级的东西。现在人们不这样了，哪个让我爽，我就听哪个。我们不去管你高级，你就自己高级去。你曲高和寡，你阳春白雪，那随便。老百姓都承认，你们这是高级的。有时别人问我干什么职业的，我说我是搞古典音乐的，他说，哎呀，你那高级。完了就没话了。”在段召旭看来，音乐审美和接受美学是比学术和理论更重要的东西。体现在演奏会上，就是音乐本身是不是好听、听众是不是爱听。他一直有一个观点，就是艺术领域的内行要自己去做普及工作，不要让外行占据。

“现在我们的古典音乐会就处在这么一个大部分的音乐会都是内行去听，而且很多都是免票的。所以我当时搞这个巡演，我为什么觉得挺好？就是我起码为古典音乐挣一份尊严，我们也是要卖票的，不是免票来听的。在学校的音乐会，常常是免费还怕人不来。我就觉得这很遗憾。”

也是在这段时间，他意识到，在影视圈里，哪怕只是在中戏进修的一个小演员，活儿都不断，甚至比古典音乐圈里的名人事儿都多。而古典音乐圈里很出色的人，很可能没事儿干，就在家教教学生。“我也没有多么大的使命感，但就是觉得这行业怎么就弄成这样儿了，可能得改变一下吧。”

“传统音乐会的标配，是开场先弹点巴赫或者斯卡拉蒂这种早期的作品，接着是贝多芬奏鸣曲，上半场才完整。下半场选一些浪漫的，最后一首李斯特，有效果地结束。更专业的，就是曲目更有学术性，但半小时的曲子，观众多半受不了。很多人甚至会觉得古典音乐挺可怕的。”

在各地巡演的时候，因为主要面向的是大众，段召旭选择曲目的标准是既有专业水准，又有可听性的曲目。他的常备曲目是钢琴家李斯特的《钟》，这也是他拿手的。此外，还有肖邦的《英雄波兰舞曲》、老柴的《胡桃夹子》、勃拉姆斯的《匈牙利舞曲》。段召旭说：“我还弹《黄河》，还有《彩云追月》。好多同行跟我说，《黄河》，大俗曲儿。但是在我看来，曲目不分专业和非专业，只有演奏水准有专业和非专业之分。”

段召旭喜欢听相声，他的古典音乐理念也受到了郭德纲的影响。虽然，古典音乐这种艺术形式一直是殿堂音乐这一脉，而相声一直是民间艺术的一种，但段召旭还是希望做一些普及工作，他觉得能听古典音乐的人可以更多。“郭德纲振兴了相声，让这种艺术形式在社会上继续保持活力，而不只是晚会的点缀。我一直觉得，郭德纲干的事情跟我在古典音乐里想干的事情一样，当然，我没有他干得好。”

郭德纲有一个段子广为流传。“别人说郭德纲，说相声只顾搞笑，不顾品位。郭德纲回答说：我跟主流相声有个协议，我负责搞笑，他们负责品位。如果相声不搞笑，那就太搞笑了。”段召旭认为，古典音乐也是如此。如果音乐不好听，那也很搞笑。

段召旭认为古典音乐和流行音乐的关系，就像是白酒和可乐。“喝惯了高度白酒的人，再让他整天喝可乐，他肯定受不了，觉得太不刺激了。经过这么多年的古典音乐的学习、弹唱和表演，现在让我痴迷地去弹流行音乐，我肯定做不到。因为我觉得不够过瘾，不够复杂，不够刺激。但是人也不能天天喝白酒，适当喝软饮料也很正常，所以我平时也会听流行音乐。”

段召旭很明白，任何一种艺术都不可能受到所有人的喜爱，古典音乐更是如此。他无意过多去教育本身对古典音乐没有兴趣的人，但是，他也不想错过那些真正想听的听众。在中国很多城市，演出资源比北京、上海要少很多。因此，在段召旭的古典音乐普及课的音频专栏里，常常有听众问他什么时候到他们所在的城市演奏。“其实很多想听的人，不知道该怎么听。我们如果能做到让想听的人能来听，这就已经很不错了。我认为，首先应该展示音乐的美好，而不是去展示它的深度，也不想灌输说人们有多么地离不开古典音乐。”

在斯特拉文斯基的传记片里，《春之祭》首演时下面的观众都是穿着燕尾服、白衬衫去看这部舞剧。这种美好的仪式感，段召旭也很喜欢。他把它当成一种中国古典音乐未来的画面：“如果真的有一天，很多人都爱听，大家知道你在弹奏什么。那时候你把他们带进音乐厅，穿上那样的衣服，那才行。”
