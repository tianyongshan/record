# 封面故事___这些年，我们正在追的网剧_《无证之罪》：

封面故事：《无证之罪》：彼岸的悬疑、此岸的推理

封面故事 | 这些年，我们正在追的网剧

《无证之罪》：

彼岸的悬疑、此岸的推理

记者◎张月寒

网剧《无证之罪》拍摄现场

从爆鳝面（杭州特色）到“南岗老砂锅居”，从江南式小混混到东北粗犷的黑帮特色……宁波人紫金陈最初在杭州构筑的那些推理手段，在剧中大多被保留。同时网剧主创充分发挥自己的北方地域情结，为这个作品焕发了新的生命力。

2013年，一个从网络公司辞职、学水利专业的宅男，看到了这样一则社会新闻：某地发生了一起强奸杀人案，现场只留下指纹这个证据。警方采集了镇上上万名男子的指纹进行比对，最终抓到了凶手。

他脑海中突然咯噔了一下，然后一直在回味，这个，不是可以作为写推理小说的一个很好的素材吗？

“杀人是为了找人。”他觉得，这个故事点，挺有意思。

于是他打开电脑，开始构思。这个故事最终成了今年9月6日大家在爱奇艺上看到的网剧。场景从作者生活了很多年的杭州变为了导演、编剧熟悉的东北。观众们也透过屏幕，依稀闻到了老江桥下黄毛被杀后的血腥味，凝固在哈尔滨冬天的冰与雪的酷冷中，这么飘了过来。李丰田的不寒而栗，让人看这部剧的时候，无来由地心里一紧。剧中扮演朱慧如的邓家佳认为，哈尔滨的冬天特别适合这部剧“冷淡”的气质。

不用说了。此剧就是今年一路口碑很好的网剧《无证之罪》（以下简称《无证》）。那个喜从社会新闻中寻找灵感的理工科宅男，就是《无证之罪》小说的原著作者紫金陈。

在《无证之罪》的实体书出版之前，紫金陈先在网上免费连载自己的作品。之前在网络公司工作过的他，深知互联网公司都是先免费后收费的策略。他也知道刚开始紫金陈这个名字没有任何人气基础，一切要从零开始。于是在连载时，他先免费刊载四本小说，希望第五本可以出实体书。幸运的是，最后事情发展果然按照他当初预想的那样。

写推理小说这件事本身，也不是紫金陈某个深夜灵光一闪的冲动。辞职后，理科出身的他冷静分析国内出版业现状，“把市面上所有国产小说的题材都罗列出来”，发现推理小说写的人最少。他再纵观全球图书市场，发现推理小说并不是小众读物。于是紫金陈就想，既然国内写的人少，那么自然竞争压力小。后来写着写着，摸索出了自己的风格和技巧，他认为也算“曲径通幽的一段历练吧”。

《无证之罪》导演吕行

辞职写作的紫金陈是幸运的。这段生涯他仅度过了两年。2014年，《无证之罪》的实体书就出版了，并也迅速被买走了影视改编权。当时买这部小说的公司，就是华影欣荣影业，也是《无证之罪》网剧的制片人齐康、联合出品人何俊逸当时所在的公司。最初齐康想将《无证之罪》拍成电影。2015年下半年，经韩三平推荐，爱奇艺找到何俊逸、齐康所在的团队。这时爱奇艺手中已有紫金陈的《坏小孩》和《长夜难明》两部小说的改编权，说希望与他们合作，开发成一个系列。

当时，同类题材《心理罪》和《灭罪师》已创下不错的成绩，也让齐康一众人在网剧领域看到了希望。至此，《无证之罪》网剧这个项目，正式成立了。

爱奇艺副总裁、自制剧开发中心总经理、《无证之罪》的总制片人戴莹告诉本刊，早在爱奇艺真正决定做这个项目之前，导演吕行就已经几次跑去哈尔滨采景。做了非常多的前期准备。当时她就感觉这个团队非常用心，更坚定了与他们合作的愿望。

《爱情公寓》里的“小姨妈”邓家佳在《无证之罪》中扮演朱慧如

场景由原著的杭州迁移到剧中的哈尔滨，一直以来是观众集中争论的点。一些原著粉有些抵触，也有很多观众认为这一场景的变换恰恰是全剧的一个亮点。迁移原因每个主创都略说了一些。导演和编剧首先是东北人，尤其是导演吕行，本身就是哈尔滨人，对这个城市有着自己根深蒂固的回忆.编剧马伪八认为“冰天雪地的肃杀气氛更符合故事想要传达的感觉”.制片人齐康在2010年和吕行为一个作品做前期筹备的时候去了哈尔滨，对这个城市留下了深刻的印象。

平台方面，戴莹也是哈尔滨人。所以当吕行跟戴莹说场景将从杭州转换到哈尔滨时，她脑海中一下就勾勒出那些熟悉的场景，很是亲切。之前戴莹负责操盘的几部罪案剧，无论是《心理罪》（无锡）、《美人为馅》（南京）、《灭罪师》（无锡一个摄影棚内），都是在南方拍摄的推理剧。所以，“这次他们能把案件重新拉回到北方”，她自己从情感上是兴奋的。

事实证明，这部剧播出以后，也正是其间看上去自然但又力透纸背的东北特色，成为该剧一大亮点。编剧马伪八告诉本刊，东北孩子有一点与别的地域稍有区别的是，中学时就可以看见一个“五脏俱全的社会模型”。虽然呈现的形态有可能比较幼稚，但是“面子、暴力、利益和人际关系等种种成人社会所谓的阴暗面，都会以一种恰到好处的方式完成东北少年的人生启蒙教育”。

所以我们在剧中看到的无论是黄毛等小混混，还是张兵、老火等“大哥”，在粗粝的冰碴中似能感到某种属于东北人的行事风格——第一集，有点痞气的警察严良直接在派出所内就用折凳打人.老火放高利贷的银行卡、密码居然直接附在卡后.张兵把火哥这些要命的卡放在黄毛身上忘拿走时，这么重要的东西也没勒令其立刻送过来。

从爆鳝面（杭州特色）到“南岗老砂锅居”，从江南式小混混到东北粗犷的黑帮特色.从一个三本大学毕业的孱弱的程序员（没有黑化），到有着那种“卑微的恶”的实习律师。宁波人紫金陈最初在杭州构筑的那些详细的推理手段，在剧中大多被保留，同时网剧主创充分发挥自己的北方地域情结，为这个作品焕发了新的生命力。

除去东北特色，《无证之罪》另一让人印象深刻的是其间塑造了一个形象很丰满的配角——李丰田。个人认为，国产剧领域如若能塑造出类似美剧中的T-Bag、小剥皮这类极度瘆人、深入人心、几代人难忘的变态角色，是电视剧发展史上的某种进步。这一点，李丰田接近了。

“袄哥”李丰田，当他穿着破棉袄、头发很油、话不多地第一次出现在剧中的时候，我突然心头一凛。接下来，他迅速将一个不知轻重的小混混“爆头”。面对镜头，演员宁理将瘆人感塑造得扑面而来。通过弹幕看出来，李丰田这个剧集中段才出场的配角，迅速吸引人的眼球，圈粉无数。

爱奇艺副总裁、自制剧开发中心总经理戴莹

编剧马伪八作为一个cult片的粉丝，在构思这样的情节时，他坦言“是会发生一点点额外的热情”。而李丰田在剧中诸多打人、杀人的细节，都是原著中没有的。这个变态、冷漠、话不多的杀手角色，是网剧的一个全新的创造。

第12集众人震惊的将要有滚烫铁水浇筑在朱慧如身上那段戏（“小姨妈盖浇饭”），来源于马伪八当时去景地的时候，正好赶上工人在炼铁，突然来了灵感。而让观众毛骨悚然的李丰田暗杀邵海那段戏，马伪八说是演员宁理“根据造型和服装的一种发挥”。

我觉得李丰田的那种“瘆人”，大都来自于他的“无底线”。你永远都不知道他下一秒钟会做出些什么，这才是整个角色最可怕的地方。当他杀人的时候，你能透过屏幕感受到他那种对生命的漠视。是真的漠视，且是一种天性，像德州链锯杀人狂那种，是天生的长在骨子里的。有人杀人是为了快感，有人杀人前会有挣扎，但李丰田这种，是完全不会有任何感觉的麻木。这才是真正可怕。

剧中朱慧如的扮演者、演员邓家佳告诉本刊，刚看剧本时，她真没有料到李丰田这个人物会是这么丰满。“宁理老师确实给这个角色赋予了很多鲜活的部分。因为当初看剧本时，只觉得李丰田这个角色特别狠，是在人群里看不见的一个人、隐形杀手。最后看剧，演员确实为这个角色加分了。”

通过这次各个不同角度的采访聊下来，发现整个《无证之罪》的拍摄过程非常“细”。邓家佳回忆，拍摄时她穿了五条裤子。面对冬天的哈尔滨，她感觉最困难的不是身上的冷，而是拍黄毛那场戏时，由于情节中必须要有留下指纹这个关键的点，所以在零下30℃的哈尔滨，那场戏导演要求她必须不能戴手套。在那样的温度中，导演对于演员们的要求依然很高，也让人看到这个网剧团队背后的用心，是丝毫不亚于拍摄一部电影的。

“现场基本上我们没有怎么改剧本。”邓家佳说，“有些剧本你会发现有一些不是很合理，因此在拍的时候会有一些演员的二度创作。但《无证》不同，因为剧本已经呈现得够好了，所以基本没有人改。”

“朱慧如在剧里很柔软、很隐忍，一直不想跟男性世界抗衡。当时揣摩角色的时候，我重点是放在对这个角色合理性的理解。”邓家佳介绍。

剧中，当哥哥为保全自己而死，朱慧如在小砂锅店远远地看见哥哥僵硬的尸体坐在那儿的时候，她被警察拦着，但已知道发生了什么事。相比于以往电视剧刻画丧亲场面的声嘶力竭、痛哭流涕，这一段邓家佳的处理方式，却是那种“收着”的。心碎、惊恐，但又是平静得不敢相信。

《无证之罪》中朱福来朗读的那封非常感人的遗书，就是那段平实的、指导妹妹如何如何关电闸的那些家常的遗言，是朱慧如哥哥的扮演者荆浩重新写的。“我演那一段的时候，镜头对着我，其实我是看不见里边哥哥那个尸体的。当时我还说，浩哥你帮我坐在那演一下。浩哥非常好就答应了。要不有的演员哪会做这种事呢，镜头又拍不到。我们这个戏似乎真的聚集了一帮爱演戏的人。”邓家佳说。

《无证之罪》的导演吕行，曾担任过一些电视剧的副导演，也执导过一些数字电影，但《无证之罪》是他第一部真正意义上的、影响力较大的独立作品.《无证》也是马伪八的第一部编剧作品。这也成为近两年网剧发展的现象：随着市场的火热，有越来越多的新人导演、新手编剧的出现。戴莹认为这个现象很好，是“注入了新鲜血液”。问及如何放心将一个大的项目完全交给新人？戴莹说：“我们还是要相信艺术是有天分的。有些人是在大量的实战中形成自己对艺术的理解。”据戴莹介绍，爱奇艺最初考量一个导演，如果他（她）之前没有独立的影视剧作品，就会看他们的短片，去看他们的“视听语言和审美能否和他们达成一致、是否适合即将要做的这个项目的气质”。

从平台角度来说，爱奇艺认为具有整合能力和审美判断的制片人也非常重要。他们推出“幼虎计划”，去扶植和培养一批优秀的制片人。“我们基本是以制片人为核心，然后让他（她）去搭建整个团队。”戴莹说，“大多数时候制片人会找到他认为合适的编剧团队，去完成初期的整个故事的构架及撰写。当然我们也有自己的文学版权IP库，也会在其中寻找特别好的编剧，去为我们做剧本转换。当有了剧本以后我们再找合适的团队，去完成拍摄。”

国产推理剧，既不像欧美、日本那样，有历史悠久的、繁荣的推理文学作为支撑.也不像上述国家在推理文学的影像改编道路上，介入较早，很多已有自己的体系或形成传承。我们的推理文学、推理作家，尚处于新和摸索的阶段。目前推出的这些推理剧，虽已有惊喜但和西方、日本那些真正精湛的作品比起来，还是有差距。但是，中国网剧领域此类题材的尝试毕竟让人看到了希望，以及众人的努力。目前受众对于国产剧整体的审美品味尚处于宽容，这也为网剧进一步发展，提供了必要的激励。“国产网剧的发展，还是要找到国剧的特色，在这些特色的基础上，再去突破。这是很关键的。”戴莹最后总结。
