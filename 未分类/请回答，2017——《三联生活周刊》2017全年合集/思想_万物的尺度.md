# 思想_万物的尺度

思想：万物的尺度

思想

万物的尺度

主笔◎薛巍

物理学家杰弗里·韦斯特和他的著作《标度》

美国物理学家杰弗里·韦斯特提出，标度定律能够解释从植物到人类和城市等各种东西的发展过程。

非线性比例关系

我们一般都听说过规模经济这一概念：经济的规模扩大、产量增加后，产品的平均成本会降低。这是一个人类活动领域的规律，也许可以用到自然界？美国理论物理学家杰弗里·韦斯特则是反着推，在动物界，动物的体型越大，它的效率越高，然后城市跟动物类似，城市越大，效率越高。但城市的规模能不能无限扩展下去？他认为，维持城市规模扩大的是技术创新的速度，随着城市的增大，对创新速度的要求越来越高，最终可能难以为继。

韦斯特在《标度》一书中阐释，标度定律是决定有机体、城市和公司生与死的普遍定律。生物体几乎所有的生理特征都服从这一定律，无论是基因组的长度、主动脉的长度、树的高度、大脑灰质的数量、演化速度还是其寿命。按照这个定律，一种动物如果它的大小是另一种动物的两倍，它包含的细胞也是后者的两倍，但每天需要的食物只比后者多75%，而不是100%，也就是说它的效率提高了25%。比如，一位体重120磅的女士每天不活动、不工作的话，大约需要1300卡食物。她的牧羊犬，体重是她的一半，即60磅重，细胞数量也是她的一半，如果是线性减少的话，这只狗只需要650卡热量的食物，但实际上她的狗每天大约需要880卡热量的食物。这一规则说明，个头越大的动物效率越高，因为它需要支持其组织所需的能量越少。

孩子服药的剂量跟体重是什么比例关系呢？一些药物的说明书上推荐的剂量是线性减少的，比如婴儿体重6磅的话，服用40毫克，婴儿体重36磅，就是240毫克。但如果是三分之二非线性增长的定律，6磅重的婴儿服用24毫克的话，36磅重的婴儿只要服用大约120毫克。

所有哺乳动物一生中心跳的次数都是一样的，大约15亿次。鼩鼱的心跳大约是每分钟1500次，能活大约两年，而大象的心跳每分钟只有大约30次，所以能活75年左右。只有人类是例外，现代人类平均心跳接近了25亿次。

韦斯特保证，你告诉他一种动物的体型，他就能推断出它需要吃多少东西、心率是多少、多长时间能长大、能活多久、繁殖多少个后代。这就是规模效应，城市也是如此，当一个城市的大小增加一倍时，它需要的加油站、道路和电线只需增加85%，工资、GDP、专利申请会比预计的多增加15%，不过污染、盗窃、传染病也会多增加15%。韦斯特说：“在大城市，疾病传播得更快，生意的生生死死更频繁，贸易交易得更快，甚至人们走路的速度都变快了。在人口超过100万的城市，人们走路的平均速度是几千人的城镇的两倍。”

韦斯特解释了动物为什么到了一定的地步之后就会停止生长。所有的哺乳动物和许多其他动物都有着同样的生长曲线，生物学家称之为有限生长，以区别于鱼类、植物、树木的无限生长，后一类会一直生长到死亡时。但即使是无限生长的物种，在达到一个稳定的大小后就会死掉。有限生长的物种到了一定阶段即使继续吃，其个体发育的生长也会停止，是因为代谢能量的供应要用于维持既有的细胞和创造新细胞。维持既有细胞所需的能量跟既有细胞的数量成正比，即随着有机体的质量而线性增长，而供应能量的代谢率是非线性的。假如一个有机体的大小增长一倍，细胞也增长一倍，维持生存需要的能量增长系数是2，但代谢率的增长系数只有1.682，这样维持生存需要的能量的增长速度大于能量供应的增长速度，导致增长所需的能量减少，最后降为零，生长停止。生物出生之后生长速度快，这是因为出生后几乎所有能量都用于生长，很少被用于维持既有细胞，而成熟之后，所有能量都被用于维持、修复和替代既有细胞。

生命的极限

为什么几乎所有东西都会死掉？韦斯特说，死亡在进化的过程中扮演了一个核心的角色，因为这使得新的适应、设计和创新得以出现和壮大。从这个角度来说，个体会死掉不仅是好事而且很关键，虽然它们本身不会因此而感到高兴。这是意识的过程。我们都知道自己会死掉。其他有机体没有背负这种有意识的知识。无论是细菌、蚂蚁、杜鹃花还是三文鱼，都不在意甚至不知道死亡.它们活着然后死掉，参与到把基因传给后代的生存斗争中，玩着适者生存的游戏。我们人类也一样。但是过去几千年间，我们形成了进化过程中的意识和良心，开始了思考其意义的探险，提出了道德、关心、理性、灵魂、精神、宇宙之神等概念。韦斯特说，他16岁时去看了英格玛·伯格曼的电影《第七封印》之后，意识到“生活中除了钱、性和足球之外还有更多东西”，开始对形而上学和哲学思想感兴趣。

自然选择只需要确保个体中的大多数能够活到生育出足够的后代、把它们进化出的适应能力最大化就行了。一旦实现这一目标，它们就完成了自己的演化义务，它们还能再活多久就不重要了。所以人类进化到了能至少活40年，这样可以生育十多个孩子，其中至少一半能够活到成年。但为了确保有足够的个体可以活到这个年纪，我们进化到了有些过度的设计，所以很多人都能活得更长。可以跟汽车比较一下。出于各种社会经济学和技术上的原因，汽车演化到了至少可以开10万英里的水平。根据生产过程中的波动和保养、维修的水平，有些汽车能够使用更长时间。实际上，只要持续地保养、维修和更换零部件，汽车能使用相当长时间。人类如果吃得好、过得好也一样，定期体检、讲卫生、偶尔更换一下身体的部位。但我们不可能像保养汽车那样，让自己的身体无限地坚持下去，因为汽车相对简单，而“我们的身体是高度复杂的适应系统，尤其是，它不只是可替代的部分简单地相加”。

英国科学作家菲利普·鲍尔在《自然》杂志上肯定了韦斯特这种大一统思路的价值。他写道：“柏拉图和亚里士多德代表两种类型的科学家：一种是看到事物之间的相似性、共同的模式和普遍性，另一种是看到差异、变化和特别之处。两种观点我们都需要。物理学强调前者，生物学强调后者。杰弗里·韦斯特采取的是柏拉图式的观点。他在《标度》一书中解释了许多复杂系统的大小和特性之间的关系，包括自然界的系统和人体系统。每一个系统都有一些可以根据其大小加以预测的特性：一头大象就是一个增大的老鼠.纽约是一个增大的圣塔菲.沃尔玛是杂货店的扩大版。如果你觉得这种规律性让你感到诧异，你本质上就是一位柏拉图主义者。如果你对它不以为然，你就是一个亚里士多德主义者。书中许多内容以前都被普及过，但是韦斯特揭示了这些规律背后的深层法则。它们是复杂系统不可避免的方面。你可以忽视它们，但逃离不了它们。”
