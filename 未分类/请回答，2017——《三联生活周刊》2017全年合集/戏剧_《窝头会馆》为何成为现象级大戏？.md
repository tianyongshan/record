# 戏剧_《窝头会馆》为何成为现象级大戏？

戏剧：《窝头会馆》为何成为现象级大戏？

戏剧

《窝头会馆》为何成为现象级大戏？

记者◎驳静

《窝头会馆》剧照，因为扮相特别，很多观众都看不出来这是濮存昕（韩军 摄/视觉中国供图）

宋丹丹扮演的房客和何冰扮演的房东苑大头正在“掐架”。这部戏以台词密、长为一大特点。此次时隔三年再演，记住大段台词算是明星演员们面临的难点之一（吴平 摄/视觉中国供图）

齐集五大台柱的《窝头会馆》在北京人艺上演这段时间，遭遇空前追捧，抢票态势比上半年《茶馆》更激烈。该剧此番热度因何而起，往后是否还可抵达更高的高峰，乃至它能否脱胎“献礼剧”成为真正常演不衰的经典？

《窝头会馆》的故事发生在1948年。北京南城一处小四合院里住着几户小市民。何冰扮演的“苑大头”苑国钟是房东，为得痨病的儿子操碎了心.濮存昕扮演前清举人古月宗，他是前房东，但赖在这“窝头会馆”，白住了20多年.另有两位女房客，宋丹丹扮演的厨子的媳妇儿和徐帆扮演的前清格格，二人从头掐架到尾不消停.此外，杨立新扮演保长肖启山，他算是街坊，但常到会馆生事儿。小市民们刻薄、自私、贫嘴不停，用北京方言将建国前夕的老北京和市井浓缩在这一小方院子里。

2009年9月《窝头会馆》第一次在北京人民艺术剧院上演时，正值建国60周年，当时就十分受欢迎，首轮演出36场，票房1020万元，打破了人艺自己的纪录。然而，还是比不上今年抢票的场面——开票前一天下午就有观众排在了售票窗口前，截至剧院大门关闭的晚上11时，排队已有80余人。凌晨陆续又有观众加入队伍，等第二天早上8时半开票，队伍已长到拐弯进了首都剧场旁的小鹁鸽胡同，队首和队尾互望不到头。原网络购票和电话订票途径最终也因现场排队人数过多而取消。这些细节本身就成了媒体热烈报道对象。

一部看似情节和人物都老派的剧，也不是第一次演，这样的热度到底从何而来？

《窝头会馆》是部什么戏

2009年是张和平到北京人艺当院长的第三年，他跟本刊记者回忆，实际上他从2007年接到任命那几天开始，就动了念头，要在人艺做一部新剧。“1961年，我考入北京艺术学院话剧表演系，梦想就是去北京人艺，直到46年后，我真能到北京人艺工作，没有泯灭的梦想成了真。”所以他在2007年国庆节期间接到通知后，就约了作家、编剧刘恒。这部剧也成了刘恒的话剧处女作。

在那之前，刘恒最为人所知的是电视剧《贫嘴张大民的幸福生活》，之后又凭冯小刚导演的《集结号》拿到第45届金马奖“最佳改编剧本”。当然，张和平对刘恒的了解远多于上述作品。他在北京市文化局工作过多年，而同一时期，刘恒正好在《北京文学》，二人的单位在同一栋楼。张和平在紫禁城影业公司当总经理时，已经请刘恒写过《云水谣》等电影剧本.2008年北京奥运会，王安忆和刘恒是张和平为开闭幕式请来的文学顾问。

所以即便刘恒此前并没有写过话剧，张和平认为，他们二人互相之间有足够的认知和了解，确立了合作的重要前提。“可以说《窝头会馆》的启动甚至早于我去人艺，2007年刚有为人艺创作剧本这一想法时，还比较宽泛。但我对国家重大时间节点都比较在意，眼看着建国60周年来临，我们才逐渐把创作方向往‘建国’这个命题上靠。”张和平说。

刘恒告诉本刊，“为人艺写剧，当然要展示人艺风格，其他限定条件也挺多，这是基本条件”。确立了这些目标后，接下来的事轮到作家操心。刘恒第一反应是“写什么呢”。查资料、逛图书馆、蹲档案馆，然后，他盯住上了这“会馆”。他久居南城，在大杂院生活多年，个人生活体验里原本就有“老北京的杂陈，底层人的算盘，市井里的幽默和粗鄙”。

及至终于开始动笔，已经是2008年春天。刘恒把自己扔到怀柔一处院儿里，“那一个多月几乎无雨，阳光很好。又靠山，写累了出去遛遛。为了减肥，也为了保持思维连贯，我不吃午饭，多年如此，写《窝头会馆》也一样”。

刘恒说在那样一个与世隔绝的环境里写作，挺顺利，“一气呵成，边写边大声读，过瘾”，后来到剧本朗读前，一字未改，只在排练中小改几处，主要也是做减法。

到了剧本朗读会，张和平记得刘恒“激动得哭了”，那是读到了主人公死。刘恒说他悲从中来，“写到这儿时也落泪了，后来不少观众看到这儿也哭了，这就对了”。更激动的是张和平，为了建组，他跑到唐山给徐帆送剧本，当时她正在拍《唐山大地震》。宋丹丹当时人在上海，张和平又飞到上海送剧本。宋丹丹在他们一同回北京的飞机上读了，“读完激动得一塌糊涂”。后来宋丹丹见谁都夸剧本好，她后来在舞台上对表演层次和节奏的把握也让刘恒有“知音之感”。

逐个说服演员后，接下来就交给“大导”林兆华。不像以往京味儿剧目的慢吞悠闲，林兆华为《窝头会馆》寻找到的是“呼噜呼噜吃面条儿的快节奏”。整部剧台词量大、语速快，刘恒说自己“写台词上了瘾，有时候单一角色单一台词量过于膨胀，给表演出了不少难题”。舞台千头万绪，到处是障碍，“好在大导在节奏上做了处理，他对舞台全局的掌控令人钦佩”。

“五星级”阵容

除了名编、大导和五个明星演员，其他人也是铆足了劲。为《窝头会馆》担任灯光设计如今也是戏剧导演的易立明告诉本刊，“舞台设计和灯光这两份行当，这么多年下来，无论什么戏，我都自己一个人做，毕竟这样更有艺术连续性。但《窝头会馆》，我还是退了一步，想踏踏实实做灯光，舞美由同事曾力承担”。

事实证明，这部举剧院之力的戏没有辜负谁，首轮演出上座率达95%，比人艺彼时平均上座高了近40%，1020万元票房更是破了纪录。

两个月不到，《窝头会馆》进行了第二轮27场的演出，阵容仍然集齐五位明星演员。紧接着2010年6月，又演出9场。换句话说，不到一年时间里，《窝头会馆》进行三轮演出共计72场。此后直到2014年，虽然每年仍然上演，演出场次却逐年减少到大约6场。

今年《窝头会馆》为何重新如此火爆？原因除了上一轮演出距今已有3年，而且本轮仅6场而“奇货可居”外，张和平认为，这是“多种因素合力的结果”。

“首先是它的全明星阵容，除了演员，导演、编剧、舞美和灯光全是最好的，它在首演时就被称为‘五星级’。这得感谢人艺传统，平常一部话剧，能有一个腕儿就不错了，而这戏不只有五个，每一个还都有‘一棵菜’精神，一走进剧院，都是戏比天大。另一个原因，当年一些不太重要的角色还不太成熟，八九年后他们也成了腕儿了，当年的青年演员，成了台柱子了。这就是满台都是，一开始飙戏，节奏、默契各方面都趋于臻熟，这些都成就了它现在的盛况。”

但这种盛况可复制吗？张和平觉得这会“越来越难”，因为“再往后要集齐五大明星，恐怕不易，再加上如今选择那么多”。

当年的青年演员何冰，今年经过电视剧《白鹿原》里的鹿子霖等角色，更火了，一些观众冲他去。观众当中不乏冲着五位明星去排队的。事实是，2009年至今，五位明星演员也鲜少缺席。张和平认为，明星演员当然促进了它成为今天这样现象级的一部话剧。“客观地讲，如果没有这些明星演员的号召力，这戏不会这么火。这部戏里，有点戏和角色合一的意境了，换别的演员，味道就不对了，到了这个份儿上，大家都不愿意戏往下掉。”《窝头会馆》曾经也有过若干B角，后来逐渐就看不到了。

追星还是追戏

因为今年抢票情况前所未有的激烈，人们由此互相探询两个疑惑，一个向戏剧人提出：为了票房，戏剧是否要用更多明星？另一个则面向观众：他们是追星，还是追戏？

第一个疑惑在易立明看来并不成立。“戏剧作品启用明星，首先这是很正常的现象，全世界都这样。从戏剧史的角度来看，实际上只有‘演员’的概念，明星只是因为他的社会影响力，到了舞台上，看的仍是他的舞台表现力，从商业角度，明星加盟或许能引起更多关注，但如果舞台表演不那么好，票房恐怕也不会对此有太积极的回应。而《窝头会馆》恰是因为剧中的五位主演正好也是明星，换句话说，其实他们已经是‘艺术家’范畴。这跟一般明星还不太一样。”

“凡热捧都让我感到幸福，”追星抑或追戏，刘恒觉得这无法割裂开来看，“好演员是有光辉的，那是艺术散发的神性。所谓追星，追光而已。作家是殉道者，我们追寻的永远是艺术神圣的光芒。”

这种艺术光芒最后的结果是征服观众。易立明觉得观察购票环节的激烈程度，不如观察观众看完戏后的反应。“如果观众原先是冲着明星去的，出来后没说，啊呀我受骗上当了，如果是冲着戏去，肯定不会失望。最佳状态就是，兴高采烈地进去，欢天喜地地出来。”

实际上，张和平、刘恒和易立明都感慨今年观众对《窝头会馆》之热切，也都到现场再次感受。易立明坐在一层最后面，他观察到，年轻人很多。刘恒则感慨“观众似乎越来越年轻，主要是自己老了，看谁都年轻。天冷且天晚，岁数大的人出不来”。刘恒回忆2009年首演时，自己很兴奋，一连看了5场。今年，兴奋又多了一层，“放了3年还能火，放8年呢？20年呢？有奢望了”。

他说的“奢望”，是指作为经典剧目流传下去的可能性。一种观点是，从今年购票的激烈程度来看，《窝头会馆》正在逐渐成为观众认可的经典剧目。首轮演出时，人们曾将《窝头会馆》对标茶馆，认为某种程度上这是一出翻版，它身上有老舍原著、焦菊隐导演的《茶馆》的影子。

张和平并不记得当年有太多这种声音，起码不认为这是主流声音。易立明认为“如果说火爆程度，是当年茶馆翻版”，但他从戏剧角度分析，“戏剧结构和舞台呈现，都不是《茶馆》那个时代了。从舞台技术、演员整体表现，《窝头会馆》其实还是在往现代戏剧上着落”。所以刘恒特别盼望这两个剧能前后脚演，让观众挨着看，让观众自己品。

这部戏在制造了现象之外，会成为经得起时间的经典作品吗？

易立明觉得，目前毕竟演出场次仍有限。“有些戏演了1000场，票房两个亿，你说它不好也没道理，至少说明它跟观众有共鸣。可惜《窝头会馆》目前演过的场次还不够多，还不能说它对市场产生了巨大影响。今年是火爆，但这6场的票房对北京市今年的演出市场产生了深刻的影响吗？不能这么瞎说。它的影响是，在话剧表现非常多样化的情况下，它严谨、技巧高超地表现了历史的一个片段。从这个意义上讲，北京市今年话剧市场影响大的，就要数《窝头会馆》。”
