# 手艺_奢华，来自唤醒身边的传统

手艺：奢华，来自唤醒身边的传统

手艺

奢华，来自唤醒身边的传统

文◎钟晓武

前段时间，我采访了一个电影海报圈的设计师黄海。一间贴满电影海报的小房间里，陶器中煮沸的白茶正发出咕嘟咕嘟的响声，满室茶香。黄海告诉我，这里也是他构思创意的空间，为了激发创作灵感，他每过两周都会重新倒腾一遍房间的布局。前段时间，他设计了一张全球政党高峰论坛的海报，一个让很多设计师感到头疼的政治主题。也是在这房间，他一边喝茶一边琢磨，当目光停留在茶杯上时，灵感忽然涌现：喝茶。茶，能让整个世界安静下来的东西，同时代表着中国传统文化的气度。于是，一个通过茶杯映射的世界：“共饮一泓水”，这一在身边被唤醒的传统，成为了高峰论坛海报的核心创意。

传统，往往蕴藏在五行八作的日常之中，在过去，他们有一个共同的称谓：手艺人。只是，在以工业化为基础的现代社会中，手艺人的身影变得幽而不显，只是在一些特定的情境中，人们才会想到他们，感受到那些难以磨灭的传统印记。

上周末，在三里屯橙色大厅雷克萨斯承办的“天工开物·非凡匠艺——中国新匠精神”展览上，我便有这样的感受。正如展览学术顾问、艺术家邬建安所说，手艺如同动物身上的毛，其下的皮和组织却是在这个时代逐渐瓦解掉的传统民俗。幸赖发展并不均衡的国土，在一些相对偏远的地方，手艺连同它赖以生长的民俗，依然顽强地生存着。只是，对于多数都市人来说，那些凝结着人类双手智慧与精巧的手艺似乎正在渐行渐远，手艺真的不再有生命力了吗？在邬建安看来，并非如此。只是，在传统手艺的精华与都市人群的需求之间，需要一批对两者同样敏感的艺术家与设计师，他们正是新匠精神的承担者。

一直以来，都有一种说法，传统的消失，在于其本身已经丧失了生命力，正如民国初年鲁迅等人批评的国粹，本不足惜。事实在于，我们需要传统，传统的重要性，正如诗人艾略特所说：“对于任何一个超过25岁仍想继续写诗的人来说，我们可以说这种历史意识几乎是绝不可少的。这种历史意识包括一种感觉，即不仅感觉到过去的过去性，而且也感觉到它的现在性。这种历史意识迫使一个人写作时不仅对他自己一代了如指掌，而且感觉到从荷马开始的全部欧洲文学，以及在这个大范围中他自己国家的全部文学，构成一个同时存在的整体，组成一个同时存在的体系。”

传统技艺，需要有心人的保存。但真正使其成为传统而非博物馆历史，正在于对其所做的现代转化，在现代生活中召回传统。大约两年前，我曾经在西安采访过一位皮影制作大师。对他来说，那个辉煌的年代似乎已经过去。至少在几十年前，农村的婚丧嫁娶场合，皮影总是不能缺少的角色。随着生活节奏的加快，传统的婚丧仪式日趋简化，皮影在诸多娱乐形式中早已成为古老的记忆。

改革开放初期，皮影的制作与演唱已渐趋分家，皮影戏中的人物造型，开始成为旅游外贸市场的紧俏商品。之后，随着机器皮影的出现，手工制作的皮影，虽然造型优美、制作精良，却因为成本高逐渐失去了上述市场，它们再次蜕变，成为博物馆藏品和各种文化活动的展示品。当我在一个陈旧的单元楼里，听着有“碗碗腔之王”称号的皮影艺人李世杰拉着月琴，咿咿呀呀地唱着“我朝也曾有几位好汉爷”时，我有些悲哀地发现，皮影戏，作为中国民间的一种特色剧目，似乎正在走入历史。然而，不久之后，我发现自己的判断并不准确。一次，一个来自河北的皮影剧团邀请我去现场看戏，剧团的团长充分借鉴现代戏剧的手法与元素，通过跨界，赋予皮影戏新的生命，使它再次拥有了自己的舞台与观众。

不唯皮影，当人们看到木版年画、手工折扇时，似乎很难想象，这些在今日看来像艺术品一样精美的物件，曾经就是我们的日常。这种对比，多少让人有些尴尬，社会的文明水平虽在提升，人们的富裕程度也在提高，但这一切却似乎多少以日常生活的粗鄙化作为代价。

其实，周作人早在1924年的北京便有过相似的感受，他感叹在当时的北京已很难吃到好吃的点心，感慨于生活的极端干燥与粗鄙，因为作为一个人，“我们于日用必需的东西以外，必须还有一点无用的游戏与享乐，生活才觉得有意思。我们看夕阳，看秋河，看花，听雨，闻香，喝不求解渴的酒，吃不求饱的点心，都是生活上必要的——虽然是无用的装点，而且是愈精炼愈好”。

周作人文人式的理想，于今日的商业社会似乎越来越为奢侈。但不能向自然索求的，人们还可以向商品索求.不能在传统中沉湎过去，人们还可以在现代中发现传统。一切，需要用心的创作，需要切当的情境。

正像画家彭薇说的，从来没有抽象的传统，只有当它与我们的生命体验发生关联，传统才能被激活，才能成为真正属于我们自身的私人传统。当我们读到一本用传统装帧式样制作的卷册或书本，何尝不感到跨越千载的传承？当我们用宋代的抹茶法亲手制作一杯手工茶，又怎能不体会到陆游笔下“矮纸斜行闲作草，晴窗细乳戏分茶”的那份意境？

甚至于，当我们乘坐一辆凝聚现代科技的豪华汽车时，如果仔细体察，却于目视之处、指尖所触发现传统技艺和审美的印记，怎能不唤起内心深处的惊喜与悸动？全新雷克萨斯LS正是在这个意义上诠释了源自传统手工艺的豪华艺术，将那些听起来稀松平常的物料：玻璃、木材、布匹，用传统手艺幻化成为杰出的艺术品，突破工业制品的冰冷，把温度和感动带进豪华轿车的内室。

与其他豪华车上的木饰不同，全新LS的实木饰板从全球范围精心挑选理想原木，然后运用传承百年的Takumi工匠技艺，创造出融合自然之美和手工技艺的独特装饰材质——艺木。无论是传统的薄木切片工艺，还是融合先进的激光雕刻技术而成的天然木纹与金属几何线条交织的高级饰板，都能凸显“手艺”精妙的感染力。

让光流动起来，简单的折射和反射也让人惊叹不已。玻璃内饰是全新雷克萨斯LS另一个直击人心的耀目之作。以日本江户时代绵延至今的Kiriko切子玻璃工艺为灵感，手工切割出颜色清晰而精致的玻璃线条，再经过反复抛光打磨，造就晶莹剔透、光彩夺目的凌光切子内饰。

大师级工匠清水裕子（Yuko Shimizu）在京都的工作室，通常会以传统折布工艺，设计和制作高级定制时装与婚纱。为了实现雷克萨斯为全新LS内室所提出的设计要求，她带领六名团队成员，经过长达数年的不断尝试，最终用一块完整的布料，通过极其精密的折叠、定型，完成了被称为鹤羽折布的车门饰板，具有漂亮“L”形立体图案的折布实现了随光的角度改变呈现不同视觉效果的动人体验。

其实，传统从未远离，一直蕴藏于我们生活的角落中，等待被召唤。如果说越来越多有志于创作的艺术家和设计师，正承担着将民间手艺与现代商业结合的新匠精神，那么诸如全新雷克萨斯LS这样的豪华轿车，无疑正是这种精神的载体。豪华的艺术，源自世代传承的传统，亦将成为未来新的传统。
