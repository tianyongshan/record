# 投资物语_全国“通勤”

全国“通勤”

投资物语

全国“通勤”

文◎邢海洋

这是去年12月份的一个早晨，我们一家人从杭州去上海，没有坐高铁而是普列。满员118人的车厢算上我们三口只有9位乘客。淡季的火车真是惬意，上车前直接买票，说走就走，卧铺总有空余，恍惚间，中国真的进入了铁路通勤时代。

这个冬天，雾霾肆虐，北方大面积雾霾的时候，“逃”到海南的人，一旦行程结束，经常面临着飞机无法降落的问题。邮轮出行，海面上的浓雾也会把船阻在港口之外。汽车出行，更可能遭遇高速封闭。水陆空遭遇全面堵截的时候，也只有火车风雨无阻，成为全天候的交通工具。这个意义上，高铁是投资“大跃进”送给中国人的礼物，没有了畅达的铁路，无论南北都可能物流梗阻陷入瘫痪。

2017年春运前夕，全国最长的高铁线路沪昆高铁通车了，同时通车的还有广州经南宁的广昆高铁，边陲一隅的春城连开两条，一下子被纳入了京沪8小时生活圈。2016年，全国铁路投产新线3281公里，至此，全国铁路营业里程达12.4万公里，其中高速铁路2.2万公里以上。更震撼人的是“四纵四横”规划行将完成之时又横空出世了“八纵八横”高铁网络规划，到2020年，铁路网规模达到15万公里，其中高速铁路3万公里，覆盖80%以上的大城市.到2025年，铁路网规模达到17.5万公里左右，其中高速铁路3.8万公里。

铁路修到这个份上，除了春运那几天，铁路运能紧张状况已经基本缓解，瓶颈制约消除，中国人国内的“通勤”实现了。铁路运能的宽裕，从货运上就能旁证出来，过去铁路的主要收入来自货运，运力过于紧张，这里也是腐败的高发部门。可自2012年铁路货运量遭遇连续4年下滑，2015年竟同比跌去一成。本来高铁大发展为普通线路腾挪出了更多的运能，按理说释放出的运能该转化为收入，可并非如此。货运和客运此消彼长，到了2015年，双方的关系终于逆转过来，客运收入首次超过了货运。

但铁路带来的便利性却是由铁总债务的急剧扩张实现的。2016年铁路投资8015亿元，此前的几年中每年的投资也在8000亿元上下，而每年铁路都是在盈亏线上运营，很少有盈余能够用来平衡资产负债表，结果就是2009年，铁道部共有1.3万亿元负债.2010年，负债直线上升，增加到1.89万亿元.到了中国铁路总公司成立，铁总背负债务2.7万亿元，到2015年底，铁总负债达到4.14万亿元。当2016年数字出来的时候，公众肯定会倒吸一口凉气。因为这笔庞大的开支，如果说是为公众的便利而投入，具有公共设施的属性，这笔债就应该全国人民一起背负.若排除了公益属性，铁路或应大幅度提高票价。

自1995年10月，旅客票价基价率从硬座每人每公里0.03861元调整到0.05861元，这一基准票价率一直执行至今，20余年间并没有发生改变。其实，只要铁路客货运价格涨上一个很小的幅度，企业就能大幅度提高盈利水准，毕竟每年铁路客货运收入近万亿元，在此基础上几个百分点的增幅就甚为可观。但作为最平民化的出行方式，普速列车客票承担着超乎寻常的社会责任，也因为敏感，铁路部门一直试图用高铁的市场化增量消化普通列车的负担。可中国高铁的版图上，却是东部人口密集经济发达地方线路赚翻，中西部地形复杂修造成本高的线路却盈利能力甚微。2015年京沪高铁全年盈利66亿元，而穿越漫长喀斯特地貌的贵广高铁每年还给银行的利息就要30亿元，可开通第一年的车票收入才10亿元。世界上绝大多数高铁都是亏损的，中国东部高铁群的经济状况算得上奇迹了。

可如今，当我们说走就走，漫游在曾经“难于上青天”的内陆地区的时候，除了对铁总的大刀阔斧心存感激，是不是也有一些淡淡的不安？
