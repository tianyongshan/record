# 投资物语_环保与物价

投资物语

投资物语

环保与物价

文◎邢海洋

铜、铝为原料的建材制品价格涨了30%，木材涨了10%～30%，包装纸翻番。这个秋天，装修的人遇到了不大不小的难题。两年来物价一直在温和上涨，煤铁等黑金涨幅甚巨，化工制品则在夏秋之际突然飙升，白色家电则是稳步上行，如今则轮到了被环保风暴波及的家具建材以及新闻纸、包装纸等纸质用品。问题于是归结为：环保收紧是否会令稳健的物价“脱轨”？

夏秋之际，环保督察组进驻前后企业关停、产品价格飞涨已经成为“环保－物价”互动的常态模式。典型案例，8月环保督察组即将进入染织业重镇浙江绍兴之前，这里已经有企业慑于环保压力关门停产，染料价格即启动上涨模式。国内的染料产能高度集中于浙江，且以绍兴上虞为中心，这里分散染料产能占全国的80%，活性染料产能约占全国的38%。环保督察的结果是，一大批企业被责令停产，表现在市场上，如染料分散黑ECT300%的价格一个多月内5次提价，价格从2.4万元/吨，涨至目前的4.5万元/吨，涨了近九成。

今年环保督察处罚力度可谓史上最严。来自环保部的数据显示，截至今年6月底，7个督察组向地方交办的共计31457个环境问题举报均已基本办结，共处罚金近3.7亿元，立案处罚案件8687件，责令整改24299件。年内环保督察将实现对全国所有地区的覆盖，前三个批次环保督察名单中没有出现的浙江省，一下子成为环保督察的焦点。

10年前国际市场分散染料价格曾每吨接近20万元，当中国企业进入这个市场，价格攻势下欧美国家已无力招架，但我们抢占市场的代价也颇为惨烈，染料产业污染较为严重，容易产生大量废水废气，屡次曝出环保问题，为各种化工厂环绕的太湖水质一度劣五类，蓝藻大面积爆发。可以说此前2万多元的价格是以生态环境、居民健康为代价的，是市场主体恶性无序竞争的结果。执行严格的环保标准，淘汰小散乱污的生产状态，实现集约化生产是解决问题的唯一可行办法。

当环境选项计入成本，很多工业产品似乎只有涨价一条路了。三次环保督察以北方地区为主，小散乱污企业被全面清理整顿，可北方的企业多处于生产链的上游，即使螺纹钢和煤炭价格涨翻了天，消费者却感受不深。当环保督察深入到南方轻工业重镇，物价压力一下子迫近了。纸制品行业，一年时间瓦楞纸已经从2200元/吨飙升至5000元/吨左右，上涨一倍多，而瓦楞纸正是快递包装盒用纸，关系到网购链条上的所有买家卖家。8月到9月的仅仅一个多月时间，国内新闻纸的价格从4000元/吨涨到5500元/吨，这直接增加了传统媒体的运营成本，给这个江河日下的行业以重击。

严厉的环保政策的“蝴蝶效应”似乎已经波及到了那些并未污染环境的局外人。当一家德国汽车零件商发出紧急求助函，呼吁对一家供货商的关停可能影响300万辆汽车的生产，进而造成3000亿元损失时，环保之于宏观经济运行的负面意见达到了高潮。环保真的会使实体经济的运行失控吗？观察产业上下游的运行，最好的指标是产品的价格，当供给侧改革的风暴刮向污染严重的炼钢厂，钢材价格的确经历了持续近两年的长周期复苏，可还没有恢复到它开始下落时的高点。同样在化工领域，尽管很多散乱的小厂被封存，化工产品作为整体也只比两年前的低点涨了不到一倍。环保风暴面前，总有合规企业在正常生产，即使一个行业都存在治污难题，也会有更贫穷落后的国家希望把产能转移过去，走上“先污染再治理”的道路。物价的急涨只会是暂时的，一旦企业整改合规后，产能自然会释放出来。

四次环保督察，今年来CPI始终处于2%之下，说明在新经济与旧经济的“接力”中，整个社会已经有足够的弹性吸纳环保成本。自然，当8月CPI同比增长1.8%，消费者已经感受到压力。
