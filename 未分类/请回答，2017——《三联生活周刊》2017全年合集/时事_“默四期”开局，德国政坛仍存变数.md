# 时事_“默四期”开局，德国政坛仍存变数

时事：“默四期”开局，德国政坛仍存变数

时事

“默四期”开局，德国政坛仍存变数

文◎刘怡

2016年6月，伊拉克难民拉希德·贾米拉赫一家被临时安置在柏林一处体育馆。涌入德国的上百万中东难民给默克尔的选情带来了微妙的变数

8月25日，联盟党候选人、现任德国总理默克尔在巴伐利亚的基辛根温泉与支持者握手，呼吁民众在大选中投票支持本党

优异的经济成绩和德国人对稳定性的偏好使得默克尔的连任几乎不曾遭遇波折。但大选结束后“新右”与“新左”政党的坐大，以及欧盟和欧元区改革计划的变数，都在给她的最终执政成绩单投下阴影。

没有香槟、游行、红毯，来不及做太多好高骛远的期许：拿到第四个总理任期“准入证”的这个星期，安格拉·默克尔女士很忙。9月25日，新一轮英国“脱欧”谈判已重新启动，德国的态度尤为举足轻重。欧盟委员会将发布过去两年申根区边境控制成效的评估报告，德法两国皆赞同继续延长边控期。9月29日，欧盟数字高峰论坛将在爱沙尼亚举行，默克尔要出席开幕式，并和与会的各国领导人做非正式会晤。另外，希腊财政部长将会见欧元集团主席，就明年纾困计划到期后的变化展开磋商.法国总统马克龙将公布他的欧盟改革计划.与柏林关系微妙的普京则要飞往安卡拉，密会另一个在西欧形象不佳的政治强人埃尔多安。所有这些事态，都需要身为欧盟实际仲裁者的德国总理做出关注或回应。相比之下，赢得大选反而要容易得多。

作为老牌中右翼政党基民盟/基社盟（CDU/CSU，亦称联盟党）的候选人，默克尔是继阿登纳和科尔之后本党第三位享受总理“四连庄”待遇的领袖。事实上，偏爱稳定多过变化的德国民众，在大多数时候都乐于给予联盟党足够慷慨的机会。1969年和1998年两次导致“改朝换代”的大选结果，主要是因彼时联盟党不得人心的经济政策所致，而搞经济恰恰是默克尔的长项：彭博社在9月第二周向48位经济学家征集的调查显示，2017年德国GDP预估增速为2.1%，创造2011年以来的最佳表现。在世界经济论坛最新发布的《2017～2018年度全球经济体竞争力排行榜》上，德国力压英国和日本，高居世界第五。无论是1990年以来最低的失业率，还是连续三年实现收支平衡的财政报表，都比默克尔的竞争对手马丁·舒尔茨（Martin Schulz）的如簧巧舌来得更有说服力。后者即使是在风头最劲的第二季度，在平均支持率上和女总理依然有着两位数的差值。

9月25日大选结果揭晓后，德国选择党的两位主要领导人亚历山大·高兰（左）和艾丽丝·韦德尔（右）拥抱欢庆。这个创建不过4年半的极右翼民族主义政党在此次选举中取得大捷，一跃成为国会第三大党

9月24日，社民党主席舒尔茨（右）与妻子在北莱茵—威斯特法伦州的维尔塞伦结束投票后，携手走出投票点

然而从另一个角度说，9月24日这场联邦议院选举，默克尔和舒尔茨都是输家。与4年前相比，他们所代表的联盟党和社民党（SPD）在最终得票率以及所获议席的数量上都出现了大幅下滑。“攻擂”失败的社民党的得票率较2013年跌去5.2%，议席数量由193席降为153席。执政党联盟党的得票率更是由41.5%暴跌至32.9%，创造了1953年以来的最低纪录，在709个议席中仅获246席。相反，传统两大党之外的4个小党派齐齐奏响凯歌：鼓吹经济自由主义和亲欧洲立场的中间派政党自民党（FDP）收获了历史第二好的成绩，拿下80个议席。继承原东德统一社会党衣钵的左翼党（The Left）和中左派环保主义党绿党（The Greens）都守住了原有阵地，议席数还略有上升。尤其令人侧目的是，以平民主义、欧洲怀疑主义和反移民、反伊斯兰教作为政纲的极右翼新党“德国选择”（AfD，亦译“另类选择”）在建党不过4年半的情况下，豪取12.6%的选票，以94席成为议会第三大党。传统两大党支持率下滑，四小党势力相对均衡，形成了远为复杂的新格局。

新格局造成的第一项直接影响，便是“默四期”内阁构成方案的变数。在“默三期”选择加入政府的社民党已经表态将专注于头号反对党的角色，而极右翼的德国选择党和立场过于“左”倾的左翼党都不在默克尔的结盟考虑范围之内。如此一来，要想在国会实现简单多数，联盟党只能选择和自民党以及绿党联手，组建所谓的“牙买加联盟”（三党标志的颜色分别为黑、黄、绿，与牙买加国旗相同）。鉴于这三个党在历史上并无合作组阁的先例，且默克尔的左臂右膀、现任财政部长朔伊布勒将会改任联邦议长，新一届政府在应对内外考验时取得一致的难度势必大大增加。而选民已经通过投票给极右翼发出了警告：在欧洲财政和货币问题、中东难民危机以及国际立场上，政府无法拒绝表态。对素来倾向于回避争执的默克尔来说，考验才刚刚开始。

“老左”暗弱，“新右”崛起

在2013年的上一届大选中，联盟党挟经济复兴之利，豪取631个议席中的311席，只需与任一小党结盟即可组建内阁。而身为头号反对党的社民党同样表现不俗，拿下193席。为求得尽可能广泛的一致，方便制订和推行政策，默克尔决定与社民党组建左右联合政府，将外交、经济、司法、劳工、环境等六个重要部门的部长席位让渡给后者。此举使社民党领导人信心倍增，有意在积累经验之后，于2017年最终问鼎总理府。

为备战此次大选，社民党这个“老左派”盟主所做的准备不可谓不充分。由于党内大将、前外交部长施泰因迈尔在今年2月当选为联邦总统，领导层最初有意征召现任副总理兼外交部长加布里尔挂帅出征。但鉴于过去4年间加布里尔曾直接参与“默三期”内阁的诸多决策，特别是在有争议的希腊债务纾困和难民安置问题上与女总理步调一致，他本人认为自己难以形成有辨识度的竞选政纲，于是转而推荐时任欧盟议会议长马丁·舒尔茨。后者被认为存在三大突出优势：首先，舒尔茨在欧洲议会有过不俗的任职经历，支持自由贸易政策和一体化立场，不至于引起选民的疑虑和恐慌.而他在国内并未担任过重要行政职务，与默克尔政府过往的失误无关，有可能被当作气象一新的改革者。其次，针对饱受诟病的收入不均、失业福利下调和养老金问题，舒尔茨做出了针锋相对的批评，以呼应本党作为社会公平这一传统左派价值观维护者的形象，在若干州颇受好评。最后，和形象呆板乏味的默克尔相比，不修边幅的舒尔茨能言善辩、擅长利用社交媒体，在青年群体中极受好评。有评论家认为，舒尔茨将扮演美国大选中桑德斯的角色，造成一股新的“左派旋风”。

然而，现实很快令舒尔茨失望了。最初几个月的新鲜感消散之后，社民党的支持率开始节节走低。过于强调“左派价值感”的竞选纲领被认为难辞其咎：舒尔茨原本是希望打一张平民牌，将过去十几年经济繁荣期里被边缘化的弱势群体、老年人和低收入者团结到社民党的旗下，但这一做法恰恰构成了对“议程2010”的否定——这项改革方案是由社民党历史上最成功的领导人之一、前总理施罗德在2000年提出的，意图在10年内降低政府在就业和福利政策上背负的财政包袱，增加德国的出口竞争力。默克尔的经济和福利政策大体延续了施罗德的思路，德国在2008年全球金融危机后的迅速复苏也证明了“议程2010”的成效。而舒尔茨擦枪走火、把矛头指向党内前辈，无视改革带来的效率提升和经济红利，却只喋喋不休地批判其负面影响，自是令人质疑其是否要恢复“大锅饭”的做法。多位经济学家对此提出了公开质疑，令舒尔茨先失一阵。

对社交媒体和青年学生团体的倚重，在选战开始后也被证明是噱头大过实效的做法。由于舒尔茨在重要的贸易政策和继续推进欧洲一体化的问题上与默克尔并无太大分歧，单靠宣传手段上的新意根本不足以造就差异战略。作为欧美大国中社交媒体影响力偏弱的“脸书（Facebook）洼地”，8267万德国人中仅有3100万脸书用户，占比显著低于美、英、法三国。60岁以上老年人群体的投票意向基本不受社交媒体的引导，而这部分人士（占合法选民总数的1/3左右）恰恰是联盟党的稳固票仓。结果，虽然观察家公认舒尔茨在9月初电视辩论中的表现稍微优于默克尔，立场更加强硬和鲜明，但仍不足以扭转大势，最终以一场惨败而收场。

既然舒尔茨的挑战被证明是雷声大、雨点小，而联盟党的得票率也在迅速缩水，那么剩余的选票最终流向了哪里？答案是来势汹汹的四小党，特别是身为“非主流右翼”代言人的德国选择党。事实上，以年初的荷兰和法国大选作为参照，选民的投票率越是上升，分流到小党的票数越会增加。这是因为西欧存在一部分政治冷感较为突出的中年“表态型”选民，他们平素对政治不甚关心，但一旦认定国家正面临重大考验，往往会直接用选票伸张自己的诉求。而在过去4年的窗口期内，席卷整个欧洲的中东难民危机不可能不对这部分选民有所触动。具体到德国，本次大选的投票率较2013年上升了4.4%，创造2005年以来的新高。但表态型选民一来并无习惯深究各大党洋洋万言的竞选纲领存在的深层差异，二来更倾向于从动机推断执政前景，因而更有可能被口号明确、与当下社会问题联系更紧密的新党所吸引。“老左派”社民党与“新右派”德国选项党得票率的此消彼长，原因即在于此.后者也是荷兰的自由党以及法国国民阵线在中欧的平行镜像。

2013年才在黑森州诞生的德国选择党，历来仅被视为无足轻重的极右翼民族主义小团体。在2013年大选中，该党仅仅获得4.7%的有效选票，未能满足跻身联邦议院所需的5%门槛线。但难民危机的持续发酵显然给了该党可乘之机：2014年欧洲议会选举中，德国选择党一举赢得206.5万张选票，在欧洲议会占据了7个席位。在2016年的地方议会选举中，该党在萨克森－安哈尔特、莱茵兰－普法尔茨和梅克伦堡－前波美拉尼亚三个州都位居第二。尽管该党关于废除欧元区、抵制移民和难民流入、对大国企实行私有化的政纲听起来像是半个世纪之前的产物，但的确呼应了一部分弱势选民特别是原东德地区民众的现实关切。与前特朗普时代的美国不同，工业部门在德国国民经济中所占的份额以及中小企业的蓬勃发展，使得大部分德国传统产业工人自视为全球化和自由贸易政策的受益者。但在波罗的海沿岸的北部州和东部边境地区，流动程度较低的农民依然觉得自己被执政党冷落了。而德国选择党提出的口号是“我们关注你们”——既是从收入上，也是从归属感和国民感情上。这对身处非理性恐慌和迷茫情绪中的表态型选民尤其构成一种诱惑。

默克尔尚未把该党当作值得担忧的对手。但到了今天，她已经不得不正视“小把戏”的影响：尽管联盟党拒绝把极右翼视为合作组阁的伙伴对象，但跻身联邦议院的德国选项党将拥有更大的舞台和更充足的资金去传播自己的理念。过去因为惯性而投票给“老左派”和“老右派”的消极型选民，正在被“新右派”悄无声息地改变观念。

“默”守成规之惑

琐事缠身的默克尔或许还没能静下心来回味这场似是而非的“胜利”：既然执政党的经济表现超出预期，头号反对党又遭遇惨败，何以联盟党的最终得票率不升反降？答案与法国和荷兰政坛的重新洗牌显然有共通之处：执政党对于内外两大挑战——贫富分化和难民危机——的回应不尽符合选民的预期。具体到德国，这又和默克尔惯于回避和淡化政治矛盾，将注意力集中投入到发展经济以及输出软实力上的长期政策有关。

尽管长期被视为“两德统一之父”赫尔穆特·科尔的政治接班人，默克尔乐于秉持的却是一种经济优先于政治、“闷声大发财”的国际观。德国军费开支占GDP的比例远低于“北约”成员国的一般水平，在国际事务上的表态也延续了全球化初期的传统，不当头、少站队、和稀泥。与日本、印度等国乐于借助安全政策上的调整“修正”后冷战时代世界秩序的立场相比，同样志在联合国安理会常任理事国席位的德国态度往往含混不清。面对特朗普意在颠覆美欧伙伴关系和全球权势版图的挑战，默克尔虽然勉为其难地做出了批评，却始终拒绝亮出关于“最坏时刻德国将怎么办”的预案。她的目光仅仅局限于欧洲：至少表面上装作如此。

当全球某地突然爆发了人道主义灾难、核危机抑或大规模军事冲突，政治家会打电话给中、美、俄领导人寻求帮助。但会有人向默克尔求助吗？默克尔领导下的欧盟倾向于假装看不到欧洲以外的世界：除非一个国家问题可以被“坍缩”成欧洲问题，否则它就不存在。在这一思路驱动下，乌克兰危机被解读成了“欧洲内部争端”，尽管欧盟始终需要攀附“北约”和美国来分担俄罗斯造成的安全压力。千万人级的难民潮被标记为“暂时性的人道主义和社会危机”，原因是德国不想介入利益关系更复杂、责任成本更大的叙利亚内战。默克尔甚至公开宣称，指望她出头领导西方世界纯属“异想天开”。

是的，的确存在这样一批“异想天开”的人。今年春末，美国皮尤研究中心在全球37个国家进行的民意调查显示：有42%的西方民众相信默克尔在危急时刻能做出更正确的决断，而特朗普的支持率是22%。但默克尔更乐于把她的精明强干用到吮吸东欧、呵斥南欧以及与亚洲做生意上。德国政府每每热衷于在全球共识大体分明的非传统安全事务和经济领域内贡献意见，但对更严峻也更需要大国展现担当的“旧”问题则高高拿起、轻轻落下。德国拥有硬实力，但德国无意使用它——对世界秩序中的固有受益国来说，这当然是足够乖顺的姿态.但对全球化初期希望将欧洲打造为政治“第三极”的规划者们来说，则是彻头彻尾的退步。

甚至在欧洲内部事务上，默克尔的表现也很难说让足够多的人感到了满意。柏林－巴黎双核素来被视为欧盟的动力来源，但法国经济长期无法走出低谷，遂使危机意识极强的德国总理生出了一种“寓利人于利己”之中的古怪理念：德国需要将本国的经济优势置于一切问题之先，随后方有兑现国际承诺的空间。唯有从这个角度，方能解释默克尔一系列首鼠两端之举的心理动因。2015年中东难民危机爆发之初，德国为维护其在人道主义问题上的一贯正面形象，率先表态支持接纳叙利亚和伊拉克难民，当年即安置百万人之多。然而一旦察觉到轻率决策造成的族群冲突和财政负担可能影响德国本身的繁荣，默克尔立即“变脸”，力主通过边控条款、限制难民和非法移民经巴尔干进入西欧。德国在希腊债务危机中的表现同样应作如是观：默克尔最初有意抛弃希腊，但又担心其他“欧猪”国家步雅典的后尘、使欧元区的声望和稳定性遭受毁灭性打击，遂授意其心腹、财政部长朔伊布勒（Wolfgang Schaeuble）拿出一份极为苛刻的纾困方案，强迫希腊方面接受。结局遂使得援助国不堪其扰，受援国依旧不领情：而这都是汲汲于“德国优势”、将经济与政治高度捆绑的后果。

承平之时，如此机会主义的路线尚有可圈可点之处，但在危机年代，恪守成法的结果只能是一步慢、步步慢，徒劳地根据事态发展修修补补。爱惜羽毛的默克尔不愿主动向外输出硬实力，便只有继续维持欧盟的门面，以经济实力吸引一众小国作为羽翼。而在国内，对经济数据的强调使联盟党内阁亦步亦趋地追随其社民党前任拟订的“议程2010”，在价值观上日益平庸化，部分失望的民众遂转而倒向极化色彩更明显的“新左”“新右”小党派。倘若放在25年前，德国选择党鼓吹的“回归传统价值”“重振民族自豪感”之类复古口号根本不会掀起多少波澜.但在2017年，对死保经济之“术”已经渐感厌烦的选民已经有12.6%投票给了这个极右翼新党。默克尔曾以为自己的国家可以免遭席卷全球的平民主义浪潮的侵袭，但最终只能接受现实。

前途未卜的欧洲

舒尔茨或许会为他在参选之初做出的一项决定后悔：由于预判对执政当局的不满将主要来自左派，该党提前声明不管大选结果如何，都不会和联盟党组建联合政府。孰料开票结果一公布，中左派和中右派齐齐成为受害者，“默三期”的结盟关系也不复存在。假使联盟党掌握的议席数依然与2013年时相当，则它只需吸纳任意一个小党便可成功组阁，在政策自由度上自不必被盟友束缚。然而社民党退盟在先，默克尔又不愿对左翼党和德国选择党妥协，结局遂只能是“牙买加联盟”，与同属中右翼的自民党和偏左翼的绿党联手。

微妙之处在于，这两个有望成为“造王者”角色的小党派，恰恰是对方的宿敌。以中右翼保守派知识分子、自由市场推崇者和欧洲主义者为基本盘的自民党，在两德统一之前曾多次参与联盟党发起的联合政府，欧洲一体化进程的重要倡导者、曾连续18年担任德国外长的根舍便是出身自民党。但进入90年代后期，随着以和平主义者和反核能环保主义者为基干的绿党的兴起，自民党逐渐走向衰落。此番卷土重来，自民党主席林德纳表态赞成默克尔的减税和欧元区改革计划，并要求取得朔伊布勒离职后空出的财政部长一职。但该党在外交和财政问题上立场更为激进，不仅表态赞成英国“脱欧”以及和莫斯科修复关系，还质疑法国总统马克龙提出的欧元区各国应设立统一财长制、共担财政风险的计划的必要性。与之相反，绿党历来是更强大的欧盟体系的鼓吹者，在要求增加公共开支和实行更严格的环保法规方面也和默克尔的主张颇有分歧。即使三党联合政府能在几个星期内组成，默克尔在未来4年能享受的政策自由度也将比之前三个任期大大缩减。

9月25日开始的这个星期，全欧洲都怀着惴惴不安的心情关注着德国发生的一切。希腊已经开始对其主要银行系统的不良贷款率和承压能力进行内部测试，以为明年8月纾困计划的结束做准备。在这一问题上，自民党坚决主张将该国切割出欧元区。即将离任的朔伊布勒对法国提出的建立欧元区统一预算制度和常设欧洲货币基金的动议做出了积极回应，但他未必会赞同马克龙在几天前的索邦大学演讲中鼓吹的激进方案：建立欧盟共同军费预算、反恐预算和多国联合的军事干预部队。作为双轴心之一，默克尔认为收支平衡条款同样适用于法国，因此拒绝为马克龙的增加开支计划开绿灯。英国女首相特蕾莎·梅也有她的担忧：为了向以德国选择党为代表的疑欧主义者显示信心，德国在英国“脱欧”谈判中的立场或许会变得更加强硬。

没有人怀疑默克尔身为政治“消防员”的才能：过去12年间，女总理曾经在金融危机、欧元危机、克里米亚危机以及难民危机的交替考验下屡次转危为安。她的绝佳心理素质和实用主义手腕都是毋庸置疑的长处。但本次大选的结果已经显示：与美国、法国、荷兰相比，德国并无任何不变的“特殊国情”，一样会遭受全球平民主义大潮的洗礼。倘若不以预见性见长的默克尔无法在其第四个任期内完成在制度上重塑欧盟，同时继续维持德国的经济优势的双重目标，已经释放出不佳信号的“向右转”和“向左转”还会进一步加剧，最终彻底颠覆全球化时代西欧最保守、稳定的政党体系。
