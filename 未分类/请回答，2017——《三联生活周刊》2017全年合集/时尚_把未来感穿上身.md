# 时尚_把未来感穿上身

时尚：把未来感穿上身

时尚

把未来感穿上身

记者◎杨聃

荷兰设计师艾里斯·范·荷本的3D打印高级定制服装系列

生物科技与时尚的结合改变了穿衣的意义。

自从看到迈克尔·福克斯在电影《回到未来2》中穿越到当时还是未来的2015年，影迷们最大的心愿就是拥有一双像电影中那样鞋底发光并带有自动系带功能的球鞋。虽然现在看来发光的鞋底有点傻，不过自动系带功能还是挺酷的。2005年，汀克·哈特菲尔德（Tinker Hatfield）决定处理这个“伏笔”，10年后，首批89双自动系带球鞋如期而至。更新的HyperAdapt 1.0智能跑鞋更注重舒适性，一踩进去，脚后跟的传感器被激活，运动鞋就能自动调整松紧。

可见对于未来我们会穿什么，穿成什么样，科幻电影还是有借鉴意义的。正是如此，8月底即将上映的《星际特工：千星之城》，导演吕克·贝松为了让故事里的相关人物形象更“合理”，2015年就开始在网上征集创意。最终可以看到蕾哈娜饰演的舞女泡泡秒变造型了，一摘礼帽就能从俱乐部女郎的性感黑色连裤袜换装为优雅的金色礼服裙。

为了配合这股“未来感”的设定，女主角卡拉·迪瓦伊在好莱坞首映礼上穿了一件3D打印的高级定制。这件被调侃为有“钢丝”裙视感的礼服来自荷兰设计师艾里斯·范·荷本（Iris Van Herpen）的Aeriform系列，在今年巴黎高级定制展中初次亮相。

具有未来感的技术设计

与她此前的“跨界联合”一致，“钢丝”裙是和建筑师飞利浦·比斯利（Philip Beesley）共同设计的，它真正是用金属材质制成，细看之下可以发现在礼服和迪瓦伊的皮肤之间有一层肉色的网衬，至少不用担心皮肤直接与金属摩擦的问题了。事实上，舒适度毋庸置疑，用他们的话来说，这是一件“缠绕在身体上的金属蕾丝，质地如羽毛般轻盈”。以水和空气为主题的Aeriform整个系列都有涟漪般的视觉效果，最初的灵感来自于丹麦的水下乐队Between Music。

作为第一个应用3D打印技术的时尚设计师，艾里斯·范·荷本的设计面料包括了树脂玻璃、塑料、刚性黑色和白色不透明材料、金属织物、反光纱线、闪光箔片等。从小学习传统芭蕾的经历让她更加注重身型与律动中的体态变化，这些共同构成了她设计中的形态创新和“流动感”。2011年，艾里斯·范·荷本把水花飞溅的瞬间定格下来穿在了模特身上。那件具有科技与自然融合感的作品属于她和伦敦建筑师丹尼尔·威德里格（Daniel Widrig）的合作系列，被当年《时代》杂志评为年度最伟大的发明之一。

之后的艾里斯·范·荷本一直不忘尝试点新东西，2014年的“Biopiracy”系列作品使用了比利时公司Materialise开发的热塑性弹性材料，打造了首个具备柔韧性的3D立体印花裙.2015年，在她与意大利建筑师尼克罗·卡萨斯（Niccolò Casas）和3D Systems合作完成的“Magnetic Motion”系列中，一件透明的水晶礼服采用了光固化快速成型技术.2016年秋冬高定压轴登场的泡泡连身裙，是她用透明硅胶包裹了数以千计的玻璃泡使之坚实成型。

在接受英国版《Vogue》采访的时候，艾里斯·范·荷本在众多设计中，挑了两件对她尤为重要的作品：一件是“共感觉”（Synesthesia）金色裙子，它的皮革装饰了金色箔片，经过激光切割以及浸胶涂漆处理后，可以反射、移动、振动，甚至发出声响。冰岛歌手，曾经的“糖”乐队主唱比约克在其专辑《Biophillia》封面中所穿的就是那件。自此之后，比约克就成了艾里斯·范·荷本的超级“粉丝”，无论开演唱会、领奖，还是拍摄音乐MV都会穿上高定系列。

另一件是白色骨架裙。“这件的感觉像是把隐藏在皮肤里面的线条外穿出来，它结合了跳伞和芭蕾元素。”她说，“骨架裙代表了自由和不完美，我认为美正是存在于这两者之间。”

艾里斯·范·荷本的设计看起来有些怪异，并非突出传统意义上的女性美，其方式也恰恰和所谓的高级定制背道而驰——没有强调传统工艺和手工制作，而是跟艺术家、建筑师以及科学家合作，采用新晋的科技手段。因为她觉得：“高级定制并不是昨日颂歌，在当下快速发展的数码时代，它同样可以在制作、材料甚至是可持续方面保持创新。”大都会艺术博物馆服装学院院长安德鲁·博尔顿（Andrew Bolton）也看好像她这种联合不同领域专家以各自所长共同创作的模式，在博尔顿看来这才是时尚的未来。

技术结合时尚的烟雾裙

荷本的Aeriform系列高级定制服装

当然，时尚不只有衣服，还有配饰。当Applewatch在手表业激起了比谷歌眼镜在行业内更大的波澜时，智能穿戴成了科技和时尚都在追逐的方向。科技大公司如苹果、谷歌、三星，小公司如Jawbone、Misfit、Pebble都瞄准了这一市场。时尚品牌也有顺势反应迅速的，如万宝龙、泰格豪雅、宝格丽等。然而三四年之后，科技公司的急流勇退和时尚公司的大幅跃进，让智能看起来没那么重要了。

可穿戴设备领域最大的一家硬件公司Fibit过得并不好，信息技术研究和分析公司Gartner调查了来自美国、英国和澳大利亚的9500名消费者，通过在线问卷的形式了解他们对智能手表、智能手环以及VR眼镜的消费态度。结果显示，智能手表的弃置率是30%，而智能手环的闲置率也高达29%。经过两年的快速增长期，到了2016年第三季度，整个可穿戴市场的增长只有3.1%，几近停滞。其中智能手表的下跌尤为明显，Applewatch也只卖了110万只，出货量减少了71%。

摩托罗拉最先放弃了智能手表，其全球产品开发负责人表示短期内不会再开发新的Moto 360产品，因为“市场上目前并没有为新款智能手表留下足够的空间”。最近，据CNBC的报道，跟泰格豪雅、Fossil等公司合作智能手表的英特尔决定不做了。要知道可穿戴设备曾一度被英特尔看好，它先后买下了健康手表公司Basis和自行车头显设备公司Recon，还为时尚品牌提供芯片。与此同时，英特尔还做过类似像智能耳塞、健康手镯等设备。然而，英特尔在7月初已经砍掉了3年前收购来的Basis。据说，发起这项收购的新技术集团（New Technologies Group）现在更关注的是增强现实（AR）领域。

相比之下，时尚品牌却显得前仆后继。

根据International Data Corporation的预测，可穿戴市场在2021年将扩大一倍，其中智能服装为主要增长。早前日本东京大学教授染谷隆夫率领的研究小组开发出了一种新型传感器，将其印刷到衣服布料上，它就能通过肌肉运动时产生的微弱电信号来观测手臂的运动状况，这种传感器还有望用于测量心搏数和体温等。

“我们现在做衣服的方式和100年前没有太大的区别，这很不光彩。”Under Armour创始人凯文·普兰克说。Under Armour从2015年开始研究智能服饰，包括可以调节皮肤温度的衣服。普兰克也承认虽然智能服饰一开始只会吸引到极少部分的人，但发展空间很大，他看好互联网健身（Connected Fitness），毕竟当时就已经有超过1.5亿用户注册了其健康APP。普兰克想做一个信息数据平台，无论传感器是Applewatch、Fitbit手环还是未来的智能面料，“不管你用什么设备，我们都是数据的中央存储器”。

诸如此类的生物科技和服装，当然不应该限定于运动服装的结合，改变了穿衣的意义。就像英特尔开拓与创新战略总监托德·哈珀尔曾畅想的那样：“如果我母亲上了年纪，我想随时了解到她什么时候害怕或是焦虑，怎样才能在不打扰她的情况下实现呢？又或者说，如果我要照顾患有自闭症或自闭症谱系疾病的孩子，怎样才能知道他什么时候注意力集中，什么时候没有呢？这些问题都能通过服装来解决了。”
