# 热点_八达岭野生动物园熊咬人事件，粗放经营的乱局

热点：八达岭野生动物园熊咬人事件，粗放经营的乱局

热点

八达岭野生动物园熊咬人事件，粗放经营的乱局

记者◎刘畅

2016年8月25日，八达岭野生动物园因发生老虎咬人事件，停业一个月后恢复营业，但东北虎园暂时谢客（刘畅 摄/视觉中国 供图）

2016年10月3日，八达岭野生动物园在“十一”长假迎来客流高峰。司机开大巴车带着游客进入猛兽区并提醒游客不要投喂食物（黑克 摄/视觉中国 供图）

八达岭野生动物园又生事端。继2016年7月23日老虎伤人事件之后，2017年8月18日，陈先生在马来熊区投喂食物被咬。

近几年兴起的野生动物园的运营模式，是尽可能满足人与动物近距离接触的需求，但以最近事故频出的八达岭动物园为例，猛兽在自驾车周围自由活动的管理模式屡屡犯错。记者在调查中发现，谁来监管动物养殖场，却也是各方推诿、扯皮的糊涂账。在城市的空间里，我们该如何与野生动物相处？

马来熊伤人事件

“自从上次有人被老虎咬了，我们这儿更火了！”本刊记者走访八达岭野生动物园时，一位工作人员自嘲地说，“都是找刺激的人，想证明自己比别人聪明。”

这一次出事的是来自北京的陈先生。因为最近他被熊抓伤的视频流传后，他本人也处于舆论的风口浪尖上，所以陈先生拒绝了面对面的采访，只同意用微信这种方式来沟通。他告诉本刊记者，两年前，他曾被八达岭野生动物园的广告吸引来过，想实现自己与野生动物亲密接触的梦想。看到园中动物的生存状态并不安逸，猛兽又能在私家车旁随意走动，觉得无趣，也十分危险，并无再去的打算。但去年发生老虎咬人的事件后，他认为园方一定进行了大规模整改。于是抱着再看一看的心态，8月18日那天，与朋友驱车故地重游。

八达岭野生动物园包括野生动物游览区、山地观光区、生态保护区、古文化区、休闲区五大功能区，不过在大门入口处的图示上对此五个区并无明确标识。园区提供给游客的示意图上，自驾车的游览路线包括两个部分：猛兽区和温顺区。路线有些许交叉，但一进门主要是刺激的猛兽区，一路上山，至西边方舟广场的环岛掉回.下山是温顺动物的路线。

据陈先生讲述，同两年前一样，他先是排队买票，也同大家一样签好了协议。至于他有没有确认协议内容，陈先生对此没有给出明确肯定。陈先生说他在入口处买了些投喂动物的食物后，穿过两道自动开合的铁门，驶入园区。

“最先途经的白虎区令人满意。”陈先生评价。在去年的老虎咬人致死事件后，猛兽区各处竖有严禁开窗、下车的警示牌，白虎卧在路旁，车道两边有电网隔离，陈先生觉得十分安全。

继续前行不远，严禁下车的路段之后是一个名为“野性天地”的休闲区域。这里在形态上与普通动物园无异，中心区域是临时停车场，四周是关在笼中的熊、虎一类的猛兽。园区工作人员在笼边售卖投喂的食物，爆米花和鸡肉块。爆米花是长长的那种，鸡肉块用树枝扎着，可伸进笼子投喂。陈先生说他们一行人稍作停留，就继续前进，重新进入不可开窗、下车的猛兽区。

第一站便是后来他出事的马来熊区。陈先生面前的熊都分散在园区里，几乎每辆车周围都有熊围着乞食，比两年前来的时候更甚。当时有两头熊也朝他的车走来，据陈先生对记者讲，他和朋友看马来熊憨态可掬，见别人也在喂食，还查了一下网上的资料，确认马来熊性情温顺，觉得没有危险，就从车窗的小缝朝外扔食物，陈先生的朋友坐在副驾驶上拍照。

一只熊叼着食物在一旁吃，另一只熊则扒上了他的车窗。因为车窗的玻璃有防夹功能，熊爪伸进来，玻璃受到阻力自动下降，门户洞开，熊就扒着车窗把头伸了进来。陈先生和朋友都吓蒙了。他使劲儿扣玻璃升降按钮，可毫无反应。突然，熊一口咬在了他的左臂上，剧痛钻心。他本能地挣脱，加速往前开了一段。从后视镜中，他看到熊被挣脱后继续追赶车辆。

据陈先生车后目击者拍摄的视频，在熊又迎到陈先生的车上时，园区巡逻的吉普车轰鸣着赶到，从陈先生的车左侧穿过，把马来熊驱走。但是，吉普车刚开过，躲到土坡上的熊第三次来到陈先生的车前。陈先生见熊过来，复加速向前开了一段，方才完全摆脱熊的纠缠。

陈先生告诉本刊记者，他被熊咬伤，园区的巡逻员第一时间并不知道。“我刚要喊他们，就因为车窗关着，巡逻车没有听到，驶过去了。”等他们折返时，陈先生把车窗降了个小缝，使劲儿喊，他们才停下。“我恳求他们告诉我该怎么办，他们却让我自己去医院。”陈先生说，当时他心都凉了。

没从巡逻车得到帮助，陈先生联系门票上的求助电话。电话接通后，对方听说陈先生被熊咬伤，十分意外，口气却和巡逻车上的工作人员差不多。没有近路和指引，他只能自己开车走完了游览区后半程的路。回到山下的救护中心接受简单的消毒后，他被打发到延庆县医院。陈先生不放心，打算直接回到市区。出发前担心在路上会发生意外，陈先生就给110打了电话，把刚发生的情况做了备案。民警的反应令陈先生感动，从市局到分局，再到属地派出所，都纷纷给他打电话询问情况。

但由于当时心里紧张，路况也不熟，陈先生一行人走错了路，到达市区时已是晚上。医院的急诊看不了，本来只能第二天再看。而医生得知他是被熊咬伤时，非常惊讶，表示没有相关经验，建议到昌平区中西医院就诊。此时，八达岭野生动物园的一位副总经理向他打电话询问情况，表示关心，陈先生说他当时稍感欣慰。

医生为了谨慎起见，建议他们给北京市疾控中心打电话咨询。北京市疾控中心也表示不清楚，让陈先生遵医嘱，先注射狂犬疫苗和破伤风疫苗，再观察和查找相关的治疗方案。

投喂需求屡禁不止

事后，陈先生感到很委屈，在马来熊区，他说不单那一天看到有不少人投喂，觉得那里向来有投喂的历史，所以他认为那是一片园区默认的投喂区域。

中国科学院动物研究所的张劲硕博士告诉本刊记者，虽然马来熊是体形最小的熊科动物，而且它的食谱中植物性的食物比重很高，但作为熊，它仍是猛兽，猛兽就应与人隔离。

大兴野生动物园在北京城南，是北京另一家大型的野生动物园。根据他们的经验，幼兽长到三个月以上，牙齿和爪子就有能力伤害到人，不但饲养时需要着重训练幼兽的行为，展示时也要保证不能与人直接接触。

本刊记者屡次联系八达岭野生动物园了解情况，均未得回应，但上网查询，不难发现近几年八达岭野生动物园类似陈先生遭遇的这类事故并非例外。根据其他媒体曝光的视频，陈先生出事前不久的7月22日，在几乎同样的位置，一只马来熊的头伸进了一辆黑色SUV，有攀爬之意。在车中巴望了数十秒后，才把头收回，幸而无人受伤。果壳网的李旻是动物学专家，他向本刊记者介绍，野生动物的本能是躲避，像八达岭野生动物园这般与人亲近的行为，是长期互动中产生的结果。

如果游客投喂屡禁不止，这样的需求是否应该得到满足？

“就像人不会在大街上随便喂陌生的小孩一样，这不是一种尊重的行为。”北京动物园饲养队副队长张恩权曾在接受本刊采访时称，投喂这种行为对长续保存动物群体、对公众接受自然教育都没有任何帮助。

李旻告诉本刊记者，动物园组织游客投喂也是不应该的，这会误导游客。“凭什么动物园卖的食物就可以喂，我们自己带的就不行？”这让游客认为投喂是一种生意。动物园更需要做的，应该是展示动物的自然状态。

他向本刊记者介绍了新西兰惠灵顿动物园展示马来熊的方式。惠灵顿动物园依山而建，最低的地方靠海，园内还有流水。马来熊馆的设计便依托了山势。在山势高的地方引入了流水，在馆内做了个小瀑布。溪水旁边是一座爬架。这个场馆有两个观察点。一个在爬架背后的山上。在这里，游客能看到爬架的上层和坡顶，以及部分缓坡，而缓坡上有灌木会遮蔽一部分视线。从上观测点看完，顺山路向下，瀑布的正前方又有一个观测点。走完这两个观测点，游客就能看到几乎整个场馆的所有部分。只要熊出来活动，就可以看得见。但在每一个点，熊都不会被人360度围观，都能找到遮蔽处，熊不会感到心理压力。

本刊记者到大兴野生动物园探访。姜利成经理向记者展示了大兴野生动物园分布在不同区域的两种截然不同的游览方式：为满足投喂的需求，在步行区的马来熊馆被设计成一个大坑，游客可以向坑中的熊投喂园区售卖的饲料。为了满足游客与猛兽近距离接触的猎奇心理，他们这里同样有猛兽区，但不允许自驾，游客坐在动物园改造的笼车上，笼车轱辘用挡板护住，经由专用车道，穿越动物的领地。姜经理告诉记者：“绝对不能用自驾车。抛开游客可能会下车不说，老虎、熊的爪子冲击力非常强，如果它们愿意，轻轻松松就可以把汽车的玻璃拍碎。实在太危险了。”

何为真正的野生动物园

但是，舆论重又在同一个沸点沸腾起来，对陈先生遭遇，“作死，活该”的评论不绝于耳。

“那些人只有亲身经历时，才能明白自己真正面对的是什么。”说这话时，一年前被虎咬伤的当事人赵菁女士已经从舆论的风暴里走了出来，平静地坐在本刊记者面前。她重新留起了长发，右脸的伤疤恢复不少，但长势不一，疤痕较重的地方还需再做整形手术，不过光线暗时并不十分明显，也不影响她笑。

与本刊记者约见之前，她刚刚经历了一次职场的挫折。今年7月，她经过笔试和两轮面试后，重新找到了一份在延庆区内的工作。重拾旧业，在一家私企负责对外的项目。起初，一切顺利，即使面试官提起她脸上的伤疤，她说是因为车祸，也没有引起怀疑，直到她曾被媒体曝光过的那辆出事的速腾车被认了出来。公司前一天临下班还在给她安排工作，第二天就跟她摊牌了。“他们说需要总和区政府打交道，怕惹麻烦。可能也是受舆论影响，觉得我是个不负责的人。”她无奈地笑笑。

8月18日，陈先生在八达岭野生动物园马来熊区投喂食物时被咬的视频截图（图片来源于网络）

老虎对她造成的伤害仍旧挥之不去，而记者又将她引回事发地的场景。东北虎园在马来熊区后面，是一条700米长的U形路线。赵女士被咬后，猛兽区竖起了电网和警示牌，以东北虎区最多。记者驱车探访时，发现注意力被山路分散许多，而且同动物园里每个展区一样，路两旁是浓密而单一的草木，猛兽的所在并不明显。记者驾车转过弯后，抬头偶然在左侧的电网上看到“坡上有虎”的警示。循着警示牌细看，才看到一只东北虎卧在牌子后面的9米高土坡中的草丛里。

赵女士回忆，在其他园区看猛兽的视线基本是平的，而这里需要特意仰视才能找到老虎。当时，他们一家一路搜寻了半天，一只东北虎也没见到。“儿子还问，大老虎去哪儿了？他姥姥说，大老虎回家睡觉了。”在距出口19米的地方，一车人环视四周，未见老虎，也未见出口的提示，旁边尚有一部巡逻车，晕车的赵女士打开了车门。

业内的相关人士认为，这仍是管理理念上的混乱所至。“中国没有真正意义上的野生动物园。”张劲硕向本刊记者介绍，“野生动物园”的英文对应的是safari，原指欧洲人到非洲大草原上的游猎活动，后来发展成为在野生环境下的动物园。“真正的野生动物园是指东非的野生动物园，或是美国的黄石公园一类国家公园。在那样完全的野外环境中，人需要严格受约束。一旦违反规则，安全自负。”

中国的野生动物园不过是借了“野生”之名来人为设计。场地范围、游客密度、动物园的环境，都与野生环境完全不同。“既然是人为设计，能够改造环境，就应该从规范管理者的角度来要求经营者，引导游客游览。”一位业内人士告诉记者。他的观点与大兴野生动物园的姜经理不太一样，在他看来，野生动物园的功能是开放、互动的生态体验。这其中的核心就是自驾游览区。“自驾区的目的就在于人能够无屏障地观察动物，欣赏动物在自然生境中的状态。”业内人士向记者分析道，为了能够近距离直接观看动物相对自然的状态，应该将动物展区集中规划，并把动物的运动场与自驾车的游览道完全分开。

8月23日，延庆区旅游委通报熊咬人的初步调查处理结果，责令八达岭野生动物园立即整改，严格限制自驾游进园车辆数量，进一步加强对游客的宣传告知力度，继续加大园区内巡视检查力度，确保游客安全。

整改要求发布后，记者再次来到八达岭野生动物园的马来熊区探访，车流未见减少，却恰好见到电网发挥作用的一幕：马来熊被电网围了起来。笼车里的游客仍向网内投食，但食物没有扔远，落在电网下，熊趴在网下用爪子够时，“啪”的一声。熊被电后，在网内狂逃。

但是，电网是不能用作最终隔障的。

北京动物园饲养队副队长张恩权解释过，一旦停电，电网就不能用了。而且，电网会对幼年动物和敏感动物造成动物应激，甚至死亡。动物处于极度兴奋或者恐惧的状态下，还会忍受电击冲破电网。所以，一级隔障必须是壕沟、墙体一类的物理隔障，保证动物在任何情况都出不来。

据北京市质监局近期透露，野生动物养殖场所安全生产等级相关规范已开始征集意见。意见表明，游客自驾观览的场所应设置游客车辆观览专用通道，通道应采取隔离网、架空、强化玻璃幕墙等方式，确保车辆不与猛兽有任何接触可能。野生动物园的建设似乎将有明确的标准。

未来有了规定，谁来监管？据了解，我国野生动物园归林业部主管，有自己的中国野生动物保护协会。但是，行业协会不具备执法权限，只做业务指导、分享业内资源。同时，野生动物园为经营性质，国企和私营的情况都有，各自归属十分复杂。动物园区的管理与园中的动物的保护也分属不同的部门。这又将是一个复杂的问题。
