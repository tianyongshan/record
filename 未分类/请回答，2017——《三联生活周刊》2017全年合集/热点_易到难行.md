# 热点_易到难行

易到难行

热点

易到难行

文◎齐夕

4月18日，被爆出“易到巨额资金被乐视挪用”后，易到用车上海办事处遭围堵，近百名司机登记要求提现

易到用车创始人周航（王攀 摄/东方IC）

曾经的中国第一家预约车服务平台“易到用车”如今商业信誉危如累卵。中国特色专车大战里，烧钱成了第一生产力。不肯烧钱补贴的周航错过融资机会，不得不“卖身”乐视，跳进充值返现的黑洞。60亿充值到账就意味着60亿负债。钱不够烧、风控失守、网约车新政、与乐视的分歧……导致易到身陷绝境的原因不止一个而且早有端倪。行路难，易到还能走多远？

讨债危机爆发

中国技术交易大厦，位于北京海淀纯白色肃穆的基督教堂。它见证了时代的更迭：它曾叫第三极大厦，是全球最大全品种书店，第三极盛极而衰倒闭后，互联网巨头腾讯也曾在这栋楼办公。如今，在全国千千万万的易到司机和乘客眼里，这个楼是他们可以讨债的地方。从2016年底，易到欠汽车租赁公司管理佣金的消息就被媒体证实，2017年，欠债从租赁公司逐渐蔓延到易到司机和乘客。

4月19日的技术交易大楼附近广场，气氛与往日不一样。穿着特勤服装的保安紧张地观察着，见到陌生面孔就会上来盘问。大楼东边大门外停着两辆警车，连带四下张望的几个特勤，让周围空气一下子紧张起来。

4月19日下午接近5点，18层易到前台的女员工已经接近崩溃，说话有气无力。前台上A4纸打印的告示上满是司机涂抹的红手印，打眼看去有些触目惊心。窄窄的前台桌子后又挤了两位男保安，从脸色到语气充满气愤怨怼。有人问前台乘客的钱现在要怎么办，女员工欲言又止，保安回了一句：“爱咋办咋办。”

根据要求，第一次办理提现的司机被引导到19楼，排号再等叫号。整个19楼曾经是易到“小而美”的王国。装饰风格被评价为“年轻而潮”，办公环境舒适。如今19楼办公室门禁森严，门前不大的走廊摆了三排折叠椅，三个彪形保安严阵以待。易到员工非常警惕，发现记者就会请出去让他们联系公关部门。在微信和QQ上，成百上千无法提现的易到司机和乘客在集结。有人甚至在群里建议，“去易到搬点固定资产”“打砸一下引起政府介入”。

“还是建议打010-12345北京市政府热线。”北京的程伟是乘客中的理性派，在群中一直劝大家合理合法解决。

程伟平时打车比较多，因为易到有充值返现活动，他认为易到在当时性价比是最高的。2016年6月，程伟第一次充值2000元，第二次充值10000元。截至现在，账户里面余额显示还有2万多元，算本钱还有1万多元。

乘客群里的多数人都赞同，使用易到起初体验非常好，司机素质、叫车速度都很不错，也很少加倍，即便是加倍也较为合理。但从2017年开始加倍频繁，价格大涨，3月开始忽然就开始打不到车。程伟说起易到现在的服务都是恨铁不成钢的语气：“客服跟我承诺，4个小时内我们专员会跟你回电话。4天过去了，也没有人给我回电话。”程伟认为，像他这样一线城市打车需求大的用户，存个一万两万很正常，这样的乘客不会少，还有大量的存几千、几百的乘客。担心发酵下去，会形成对社会的一个不稳定因素，程伟给市长热线、交通部门、网信部门等都打了电话。

目前，全国只有北京可以线下提现，多数外地司机只能在APP上试手气。在讨债司机群体中，各色提现秘籍在流传：近期有接单，要上午10点准时点，一天不能点提现超过两次。如果有人截屏显示提现成功，群里就会沸腾一会儿。其他司机们要么调侃他应该去买福利彩票，凭这个运气肯定能中500万元.要么认为提现成功的人是易到的卧底、图片是PS的。不过群里司机们多数对易到评价不错，不希望易到倒下去。很多易到司机发现无法提现后都会跟乘客商量能不能现金支付，但是乘客想的是把余额尽量花完。于是，虽然叫车的单很多，司机不接，乘客也打不到车。

4月20日，在一片质疑声中，易到发布了“关于公司正常运营的声明”。对于无法提现的问题，易到的解释是，因正在与国家有关部门监管信息交互平台进行数据对接，系统或会出现暂时性不稳定情况。

充值返现黑洞

很多了解租车行业的人士都知道，易到身陷债务危机至今已有半年。但没想到，来自内部的人士以一种出人意料的方式将这个半遮半掩的问题彻底公之于众——4月17日，易到创始人兼CEO周航以个人名义发出一份声明称：“据我所知，易到当前确实存在着资金问题。而这个问题最直接的原因是乐视对易到的资金挪用13亿。”周航表示，易到所面临的并非简单的债权债务纠纷，而是可能会引发妨碍社会稳定的群体性事件。

而易到与乐视控股对周航的指摘，在第一时间以联合声明的形式予以回击，称周航恶意诽谤，“堪称农夫与蛇的现代版”。一方面，易到与乐视控股严正声明从未挪用过包括用户充值在内的易到任何资金，而且已投入近40亿元资金及大量生态资源，支持易到发展。

另一方面，声明承认了确实有个“13亿”问题，并非挪用。声明称，2016年11月，在易到单独贷款困难的情况下，乐视控股以名下乐视大厦作为抵押物，以易到为主体取得的一笔14亿元联合贷款中的一部分。当时双方已明确约定，该笔资金1亿用于易到，13亿用于乐视汽车生态。声明强调，周航本人不仅知情，也在相关的董事会文件上签字确认。

据媒体报道，声明由乐视创始人贾跃亭连夜反复修改，亲自敲定。但易到副总裁杨芸否认周航签过字，并告诉媒体，“周航起先并不知道这笔贷款”。易到与乐视控股的声明，没有让周航有丝毫退让。相反，他还在朋友圈向“正在通宵达旦忙碌的贾总和乐视所有PR同事们”喊话称：“如果，向我泼脏水能解决司机提现的问题，能协调好充值用户打不到车的问题，可以解决司机围攻易到办公室的问题，那好，可以尽管泼多点。”其对方情绪可见一斑。

4月20日，易到创始团队与股东矛盾彻底公开化：易到CEO周航、易到联合创始人杨芸、易到联合创始人汤鹏宣布从易到正式辞去所有相关职务。

13亿元的资金流转涉及复杂的银行和公司财务流程。从目前已有信息，背后真相尚不清晰，争论双方也没有解释“挪用”的定义。但从长期原因看，易到今天的危机皆因大规模充值返现而起。

在2015年专车大战中，整个行业烧掉200多亿元人民币。用周航的话说：“拿打火机烧钱都没有这个行业烧得快。”

周航曾表示，他厌恶补贴，也不希望股权稀释太多。因此在2014年上半年，虽然曾有资本敲门，但他没有接受。易到前员工冯成在2014年2月入职。据他回忆，当时大部分时间都很轻松。冯成唯一感受到的压力是“上级主管部门”。当年8月，北京市交委下发《关于严禁汽车租赁企业为非法营运提供便利的通知》，“易到被认定非法”的新闻闹得沸沸扬扬。彼时，Uber中国起步不久，滴滴、快的还没有在专车业务发力，政策压力全部指向了易到。当时一度疯传百度要投资易到，但由于政策原因，融资并没有达成。

据冯成讲，进入2015年后，尤其是3月之后，易到公司内开始起变化。公司内部开始疯传钱快烧完了。公司出现了一拨离职潮。到乐视公布投资前，公司比较动荡。根据易观智库发布的2015年Q2专车行业报告，滴滴、Uber、神州分别以82.3%、14.9%和10.7%的活跃用户覆盖率名列前三，而易到仅有2.7%。

“补贴用户是一件很幼稚的事情，易到绝对不参与补贴大战。”这话是周航说的，但他后悔了。周航向媒体这样形容他和易到的2015年，“如果说创业是一个战场，易到由于缺兵少药，就躲在壕沟里面，听着隆隆的炮声，却连出头的机会都没有”。之后，周航承认在专车大战中判断失误。

终于，援兵来了。2015年10月，乐视以7亿美元收购易到70%的股权。周航在演讲PPT上写下了“起死回生”四个大字。按照资本的游戏规则，周航话语权的弱化此刻已经被注定。

2016年2月，乐视派出彭钢进入易到，担任总裁。周航的职位仍为创始人兼CEO。之后，充返活动更加凶猛，充值礼品包括乐视手机和乐视电视。

充值返现的效果立竿见影。根据易到提供给媒体的数据，2016年1月7日至9日三天，易到完成近10亿元的充值额。到2016年7月，总充值金额超过60亿元。

冯成认为，充返本质上是一个资金杠杆，“你在赌用户不可能挤兑和快速消耗。它会成为一个资金池，如果善加利用会是好事情”。冯成离开易到后成为易到、滴滴和优步的供应商。据他观察，从风控角度上看，易到做法堪忧。“易到是有补贴但无科学补贴”，他认为，易到光顾着补贴，没有弥补漏洞，产品上很多本来的优势反倒成为坑。比如日租造成司机钻空子，“他可以不用跑1公里，坐在家里就能把充值的钱套出来”。

曾经的网约车行业战略人员周旋所在的公司在2016年秋季曾经组建过团队研究竞争对手易到，但很快就解散了。“它自己没钱了，没必要研究了。”根据她的数字，2016年5月，充返力度最大的时候，易到订单数也发展最快，峰值达到一天58万单。等到2016年10月易到企图再掀起一轮充值大潮时，用户应者寥寥。

从第一到出局

“那些成功的明星公司，回溯当时，都好像曾经存在或多或少的‘硬伤’，却在前行过程中逢凶化吉，几经转型峰回路转般一一化解。”周航认为运气可能是最大的变数。易到曾经是行业公认的先驱，但却起了个大早赶了个晚集。

（插图：张曦）

1973年出生的周航在创立易到之前就已经实现了财务自由。1994年，21岁的周航大学刚毕业就创办佛山天创电子企业有限公司。2000年后，天创已经成为国内规模和影响力领先的专业音响公司，为周航赚了人生第一桶金。2010年，周航创立中国第一家预约车服务平台——易到用车网。动因周航讲了无数次——去上海出差时，虹桥机场等出租车等了近一个半小时。被戳到痛点的周航想到“走到哪都有专车”的模式。据说周航招揽汤鹏只用了一份PPT。那是2010年，汤鹏还任职于雅虎，在周航展示商业规划PPT的次日，汤鹏辞职并加盟易到，成为易到的首席技术官。徐小平的真格基金成为易到的天使投资。

从2010到2014年，网约车平台只有易到一家。但周航并不是一个狼性的创业者。在易到公司内部，员工们都称呼周航为航叔。周航的标签更多的是理想主义、自由主义，他关心时政，喜欢写作，他的个人公号曾经因为点评时政而被封。

周航曾在参观Airbnb和Uber之后表示更喜欢Airbnb，在他看来，Uber一切都往效率的方向走，冷酷，Airbnb是温暖的。因而，周航更重视产品的用户体验。与滴滴、Uber系统派单不同，易到是司机选单。“这本身就是牺牲效率追求用户体验。”周旋说。她认为易到的业绩并不是全靠充值返现，易到的司乘满意度确实很高。

2014年后，成立才两年的滴滴很快后来者居上。2014年1月滴滴完成C轮1亿美元融资，12月滴滴完成D轮7亿美元融资，2015年7月融资额达到30亿美元。滴滴的投资者阵容中包括苹果、阿里巴巴、腾讯等互联网巨头。滴滴与易到的融资规模慢慢就不在一个数量级了。

冯成认为，2015年时滴滴优步现金还比较充足，易到的补贴大战没必要再继续了。充返活动反映到企业报表上意味着巨额的负债。之后，滴滴最先收紧，之前平台都是双向补贴，补司机又补乘客。滴滴先降了乘客，然后抬高司机奖励标准，Uber策略也是类似。“大家都太痴迷于重返带来的虚假繁荣了，运营上失误非常多。”

在网约车行业，易到内讧严重也是公开的秘密。滴滴与Uber合并后，不少Uber员工跳槽到易到，但很快发现，虽然跳槽时给出的允诺很高，但领导层面很多战略都不明确，导致员工也无所适从。此外，按照冯成的说法，易到内部不止两派，有前易到派、乐视派、阿里帮，后来又有了优步团的说法。易到创始团队与乐视派在很多问题上观点不一致。

周航在4月17日晚的个人声明中提出，网约车新政之后，市场经历过疯狂补贴大战后，又重新回归到理性和以服务为核心的新阶段。“这对于一贯以品质为坚持的易到，本是重大的利好消息，易到也理应抓住这一机遇实现更好的发展。”事实上，有观点认为，网约车新政是易到和乐视之间关系决裂的一道分水岭。

从新政征求意见开始，整个网约车行业的心情就一直在坐过山车。即使是对滴滴这样的行业老大，新政之后，资本也谨慎了很多。滴滴也不得不开始调整组织架构，开始讲“出海”“金融”故事。新规要求，网约车平台需拿到“线上能力认定”、《网络预约出租汽车经营许可证》、司机的《网络预约出租汽车驾驶员证》和车辆的《网络预约出租汽车运输证》四张牌照才算合法合规。虽然易到在声明中强调，3月24日向“北京市交通委员会”提交全国线上、北京线下牌照的申请，至今仍没有拿到线上牌照。而且，网约车平台中，易到是唯一一个还没有拿到线上牌照的。

周航在个人声明中点明：“很遗憾由于乐视众所周知的原因，也不可避免地殃及了易到本身。”从2016年底开始，乐视一直处于资金链恐将断裂的阴影之下。旗下手机、汽车、体育等板块相继出现资金问题，这给易到新一轮融资造成了不小的影响。

4月20日晚，贾跃亭发布了致股东的公开信坦言资金压力，光是乐视大屏就亏损6亿多元。公开信的说法是：“通过接连几年全部扩大，乐视生态中的单个非上市系统呈现了期间性的资金压力，其间有些乐视网相关公司运营受到了影响，掣肘了乐视网的赢利完成。”但贾跃亭宽慰股东们，“乐视历经‘飞升之劫’，所幸祸兮，福之所倚”。

这不是贾老板第一次自爆资金链危机。2016年11月6日，贾跃亭就曾发表“反思信”。信中承认，乐视蒙眼狂奔、烧钱追求规模扩张同时，全球化战线一下子拉得过长。相对应的是资金和资源其实非常有限。乐视汽车前期投入巨大，陆续花掉100多亿元的自有资金，直接导致贾跃亭对LeEco的资金支持不足。为此贾跃亭还承诺，即日起，自愿永远只领取公司1元年薪。然而市场并不买账，2016年12月7日，苦苦找钱没有进展的乐视网股价大跌，紧急停牌。

虽然资金、供应链捉襟见肘，贾跃亭执意要赌智能电动汽车这个风口，并企图以此超越BAT。然而，不仅自动驾驶技术和环境尚未成熟配套，汽车这样的重资产行业的烧钱程度似乎超出了贾跃亭的掌握。乐视投资的法拉第未来（Faraday Future）在美国内华达州的工厂项目被曝出拖欠供应商货款、停工等消息。就连斥资150亿元驰援乐视的融创中国董事长孙宏斌也表示“乐视汽车我还没看懂”。

在贾跃亭看来，易到是乐视汽车生态的流量入口。贾跃亭在推出超级汽车计划时勾勒了汽车共享的蓝图。但不少分析怀疑，乐视看上易到的原因就是易到用户充值是一个资金入口。由于乐视早已自顾不暇，周航曾试图进行自救。消息称，周航曾联合雷军的顺为资本企图对易到进行回购，但最终没有谈拢。

冯成并不同意“成也乐视，败也乐视”的说法。他认为即便是乐视没有收购易到，易到也不至于关门，但会非常艰难。航叔应该也会到处找其他投资。“但和现在不同的是，虽然欠钱，但易到没有拖欠过费用。充返补贴带来的虚假繁荣，今天都会加倍偿还。当时就觉着有钱就好，算渡过难关，其他没多想。”

4月20日发布的2016年年度审计报告显示，2016年乐视网营业利润是亏损3.37亿元，比业绩快报预测数减少了超过850%。审计后的利润总额为亏损3.29亿，比预测数少4019万元，减少幅度超过900%。同一天，易到官方召开“新易到 新团队 新未来”的管理层大会，宣布核心管理层的新名单，由原易到总裁彭钢接任CEO。易到、乐视以及它们的债主们都在等待着掀开故事的下一页。
