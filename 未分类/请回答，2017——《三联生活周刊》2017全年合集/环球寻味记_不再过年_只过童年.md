# 环球寻味记_不再过年_只过童年

不再过年 只过童年

环球寻味记

不再过年 只过童年

文◎余泽民　插图◎张杰客

黑夜。寒风。飞雪。25年前晚秋的一个凌晨，当国际列车驶出二连海关的时候，我将额头抵在灰冷的窗玻璃上盯着站台上几盏稀疏、光秃的路灯光缓缓沉入黑暗，感觉一块大幕落下。那一刻对我来说，绝不仅是地理上的离乡和不知归期的出走。接下来是电影里的镜头：经过10天险象环生的颠簸我终于抵达布达佩斯东火车站，拖着比我体重还重的行李跳下站台，我根本不知道谁或什么在等着我。站台上的人散尽了，我开始感到恐慌，这时候一个陌生的高个子年轻人手里捏着我在北医读书时的一张照片朝我走过来，不太敢肯定地问照片上的人是不是我……随后，他带我去了西火车站，把我送到200公里外的南方城市——塞格德，把我一个人扔在了那里。从那之后，我再没有回国过过年，无论人家的年，还是自己的年。

刚开始漂泊的那几年，乡愁还是挺重的，在离塞尔维亚和罗马尼亚边境都不太远的那个小城里，我何止举目无亲，甚至没什么机会讲母语。每年，日子一过到阳历的一月底二月初，一股黏稠的感伤就开始在心底泛滥，粘连了五脏六腑，在国内过年时的热闹记忆在大脑里上千亿的神经元中无序地苏醒，不可名状的寂寞就像青春期囊中的精子，即使天天往外排，也排解不清，于是会写一封信询问家人：“今年过年是哪一天？”

上世纪90年代初还没有互联网，一封手写的纸信要背着几张盖了戳的邮票经三个星期的风雨才能到北京，经常是，当我终于撕开一封在返途中又走半个多月的家信时，才知道春节已过。母亲要么在信里问我：“你那边怎么过的年？”要么就不怕啰唆地告诉我他们共串了几家亲戚。如果赶上国内的邮政旺季，这一去一来，有时要用大半个季度。

回信大多是由母亲执笔，像外文一样流畅的小字，医生特有的连笔.父亲的字迹只会在信纸四周的空白找到，规矩，工整，一笔一画，是一个左撇子被强制改用右手后创造的字体。遥遥万里，他们根本不知儿子的生存状况，即使想关心也无从开口，只能乏味地问“你在那边怎么过的年？”每年他们都这样问，可我从没有具体回答过。因为对我来说，春节只是一个普普通通的日子，我只好敷衍地回信讲：“都好，放心吧。”热闹的除夕，只是在记忆惯性中留下的一个时辰的影子，如果说“过”，也只能在心里。

那时候没有微信，打国际长途是件很奢侈的事，一分钟通话费约1.5美元，更不要说在我连房租都交不起的日子里。偶尔下决心给家人挂一个长途，必须事先做好一系列准备：拨号之前，在电话机旁放一块手表，表下压一张字条，字条上写好准备说的几句话，恨不得连问候语都要写上。即便如此，只要电话一拨通，一听到电话另一端父母的嗓音，我照样会“我－我－我”地结巴好一阵，想好了要祝的“过年好”或拜年的话，也从来没有说出过口。一是出于紧张，二是太不习惯，因为从小就缺少与父母直接表达感情的方式和训练，所以话到嘴边换成了貌似心不在焉的寒暄：“过年出门了吗？”或“北京有没有下雪？”撂下话筒后，不仅没有预期的释放感，胸口反而填了更浓的乡愁。

那些年，我在国外是靠吃百家饭穿百家衣睡百家床熬过的，一无所有，但又拥有许多，许多的时间、精力和大把爱我和我爱的朋友。有一次，一个叫“艾迪特”的女孩照例找我上中文课，微笑着送了我三根筷子，并用一根细细的彩绳系了一个蝴蝶结。“中国的月亮新年快乐！”她用匈牙利语跟我说。那是我第一次知道中文“春节”的匈文说法。要知道，当时中国货在匈牙利还很稀罕，只靠淘金者搭乘火车一包包地扛。在塞格德，像碗筷、健身球、折扇之类的小物件只在果戈理大街内一家半地下的草药店里当礼物卖。有趣的是，那里的筷子是按根卖的，原因是他们认为筷子很珍贵，万一丢了或折了一根，可以再配。艾迪特之所以送三根筷子，肯定也出于这个考虑，真是一个暖心的女孩。作为回礼，我用毛笔写了一个小篆的“狗”字送她，其实那是女孩索要的，因为这一年是她男友的本命年。

还有一年，元旦刚过，我接到作家克拉斯诺霍尔卡伊·拉斯洛打来的电话，邀我去他家帮助张罗一个朋友聚会，一是为他庆生（他是1月5日的生日），二是过“中国的月亮新年”（那几年他痴迷中国文化）。拉斯洛便是《撒旦探戈》的作者，2005年的国际布克奖得主，1993年我们就成了朋友，但那时候无论他还是我，做梦都不会想到我后来会成为他的译者（我刚把《撒旦探戈》的译稿发给了译林社的编辑）。拉斯洛第一次去中国是1990年，从那之后就写过《乌兰巴托的囚徒》《北山，南湖，西路，东河》和《天空下的毁灭与忧伤》等好几本关于中国的书，还娶了一位学中文的年轻妻子……拉斯洛的电话让我喜出望外，要知道，我出国之后，这是第一次有人邀我过春节。

拉斯洛的家在距布达佩斯30多公里的一个僻静山乡，在老树掩映的半上腰上，他请一位著名的建筑设计师为他盖了一幢东方味的小木屋。朝山谷的那一侧视野开阔，坐在露台上正好看日落的风景。在客厅的柜子上摆着我为他写的一幅唐诗书法和他从中国带回来的二胡，在黑色三角钢琴上有几本关于中国的书，书架、矮柜、墙上、门口，随处可见来自中国的小挂件或小摆设，一个黄铜的千手观音像是我带他从潘家园“淘”来的，我记得很清楚，那是我砍价最成功的一次，从3000元砍到180元……大桌上摆着我烧的中国菜，西餐的刀叉旁躺着中国的竹筷。客人们热热闹闹地挤了一屋，两条狗也在生抽王、老陈醋和香油味的刺激下撒开欢儿地在几十条腿间钻来跑去。录音机里放的是梅兰芳唱的《宇宙锋》，虽然在场的谁都听不懂京剧，但那咿咿呀、铿铿锵的怪异响动，还是帮主人达到了营造春节气氛的目的。拉斯洛很喜欢梅兰芳，称他是中国最伟大的声乐大师，说有着跟法瑞内利一样任何女性都不可能具有的清脆、有力而又甜美的歌喉。法瑞内利是欧洲历史上最有名的阉人歌手。

晚饭后，客人们散坐在客厅里，晚餐变成了读书沙龙。拉斯洛捧着书念了几段有关春节的典故，汉学家谷兰回忆在北大读书的岁月（她翻译过多部元杂剧，还译过莫言的《酒国》），老作家麦瑟吉聊起他读过的中国诗歌（他说包括沃洛什·山多尔在内的许多匈牙利诗人都翻译过李白的诗），文史学家海尔奈谈起他编辑《易经》《道德经》的感想（他至今钟情于中国文化，不久前刚出版了《山东汉画像石汇编》），当时担任《匈牙利之橘》杂志主编的瓦格沃尔基则绘声绘色讲他去香港拍片的见闻……这是一个国内人想象不出的场面，在东欧一个僻静的山乡里，一群外国人为中国聊得热火朝天。凌晨，客人们告别回城。尽管下山时车开得很慢，但沉闷的马达声和轧雪声还是骚扰了寂静的山乡，山坳里回响起稀稀落落的狗吠。许多年后，我偶然看到摄影家希拉吉·兰凯在那晚拍摄的一张合影，才知道作家艾斯特哈兹那天也在场，命运的安排，如今我也当了他的“中国声音”（几年前我翻译过他的《一个女人》和《赫拉巴尔之书》，现在正在译《和谐的天堂》）。

又过了几年，先是有了网络电话，而后有了QQ和微信。不过，发达的通讯技术奇迹般地缩短了地理上的距离，但也冲淡了游子内心的乡愁。现在，国内刚刚敲响除夕的钟声，我在布达佩斯的手机就叮咚不停地接到拜年短信。于是，我埋头舔屏，回复，留言，发表情，过年虽然同步了，但再没有了过去从寄信到收信之间漫长等待中的幽深思念。以后，我们都再难接到家书了，再难看到亲人的笔迹。

如今过年，对我来说最有意义的或许是，在回复完所有短信之后，安静下来，会聚心神，让孩提时代的黑白记忆慢慢变彩色，尽管是那种褪了色的彩色，许多早被遗忘了的熟悉场景像默片里的镜头在眼前浮现：灯笼，炮仗，年糕，懒龙，一年一度凭户口本才能买到的花生瓜子，烤在蜂窝煤炉旁小板凳上自酿江米酒，外婆的白发、父亲的唠叨和母亲的缺席（母亲是位妇产科医生，即使逢年过节，也总被叫去上手术台。在我家房檐下有一个用罐头盒做的铃铛，用一根铁丝连到四合院大门道外一个只有大人站到门墩儿上才可以伸手够到的电瓷瓶，那就是专为医院来的人设计的创意门铃），点花炮用的白色线绳烧焦的气，和一首我从不懂事时就会背的北京童谣：新年来到，人人欢笑.姑娘要花，小子要炮.老头儿要顶新毡帽，老太太要块大年糕……

时光倒流，空间穿越。能这样入神地回忆过年，一年也只有这一次机会，与其说过年，不如说过童年.还是说，人到中年后迷恋回忆，不再过年，只过童年？
