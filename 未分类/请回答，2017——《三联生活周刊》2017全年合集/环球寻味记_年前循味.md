# 环球寻味记_年前循味

年前循味

环球寻味记

年前循味

文◎张添智　插图◎张杰客

甘肃谋食两年余了，两个月能回次东北老家已算奢侈，但12月回家时就有不同，因为最多一个多月，就可以回家过年，想想脚步都轻快起来。妈妈一天未发微信给我，我知道，她和爸爸已经在张罗明天迎接我的晚饭了。手提箱打开合上若干次，装得下许多行李，却装不下对家的惦念，年愈近，心愈切。拉上手提箱的拉链，这声音已经在与父母当当的剁馅儿声相和了。

是的，每次回家，父母都要过年似的给我包上一顿饺子。这顿饺子要属牛肉蒸饺最具代表性。爸爸的牛肉蒸饺一吃就是三十几年，而且味道绝不会有任何差池，尽管每次我吃下第一个时他都要手搭桌沿儿看似漫不经心地问我一句：好吃吗？每次看爸爸调馅儿，到兑腐乳汤的时候他都会跟我解说，包牛肉馅儿的饺子一定要放腐乳汤，如果没有，把腐乳少加水卸开也行，不加是出不来那个味儿的。腐乳的鲜甜吊出洋葱胡萝卜的鲜甜，加上不同于水饺的爽快口感，这一锅牛肉蒸饺就会一如既往的干、香、鲜，一如既往的味道让我一个接一个地将它们不迭地填进口中，算是回应爸爸照例的询问。家和年的味道，都会由他们发散开去，这时我可以放下任何芜杂的心态与琐事，彻底放松下来，如同靠岸的船家将缆绳抛向码头的一瞬。

东北人吃饺子，往往要配锅粥，谓之“溜缝儿”，为什么呢？因为实在的东北人认为光吃饺子饿得快，喝碗粥就圆满了，如同饺子是墙砖，而粥是泥灰，缺一不可。我家往往是配苞米（玉米）面粥。虽是粗粮，却经过妈妈的精心熬制。最接地气的做法是放入干白菜，满碗都是干白菜清甜的香.或者是胡萝卜切丁，菠菜去梗儿，一把夏天剥好冻起来的青豆，稍加薄盐调味，爽口无比，只是出锅颇烫，心急不得，慢慢挨着碗边小口吸啜，边配上饺子，吃得饱嗝儿连连，二老算完成了接我回家的任务，心里熨帖得很了。

说到东北的年货吃食，酸菜不得不再说说。曾有人做过比喻：东北游子返乡，火车一过山海关，似乎就可以闻到酸菜的味道了。这并不夸张，酸菜和饺子一样，虽然常见，但却是百家百味，每个人的记忆都有不同。每次回家，父母一定要问问，想不想吃酸菜？然后便会操持起来。儿时对酸菜最深刻的印象有二：一是看爸爸切酸菜，酸菜是整棵的大白菜腌制而成，吃的时候要把菜帮一片片撕下，只见父亲将一片片菜帮横铺在砧板上一层层铺平码好，然后用菜刀从右至左平行下刀来将菜帮“片”成若干薄片，然后切丝，整个过程像变魔术，我至今看时还和小时候一样想，什么手法让一片片白菜帮子可以片出那么多的薄片而不错位？一堆堆酸菜丝切好后过水挤水团成团子，就可以等着下锅了。二是酸菜可以生吃，随意地撕下一条，放进口中慢慢地嚼，冰凉酸爽，没有一丝多余的味道，这往往是小时候的开胃菜，由于正餐外一点额外的奖励而异常满足。酸菜味道朴实，大抵都是用骨头这些脂肪味道较为浓郁的食材来配，排骨、大骨棒最为常见，也有人家用鸡骨架来炖，不一而足，辽南沿海还会在骨头、五花肉之类的基础上加入“海蛎子”（牡蛎），这样一来酸菜的酸、骨棒的香、海蛎子的鲜就和谐统一，相得益彰了。

光写到这些，口中已是阵阵潮润。这些美食就是沟通我与家的最快通道，父母就是用这些最朴实的食物为材料，用牵挂和爱，在我的心中搭起一个难以割舍的家，无论走到哪里，只要这些食物的香气蒸腾开来，便是召我回家的密码，指引着我，不会迷路。我想我也会渐渐继承这些牵挂与爱，同样给我的孩子搭起一个永远接纳他们的家，将这份漂泊与牵挂，传承给他们。父亲常跟我说：人从小吃什么，便对什么有感情，无论多么粗鄙，日后都会常常想起。是的，我很感激他们为我建立了这样的情感，让我的心永远有一个角落保持调柔与善良，当与周围起了矛盾时，选择及时的释怀与原谅。
