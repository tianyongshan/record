# 环球寻味记_年就是心里的家

年就是心里的家

环球寻味记

年就是心里的家

文◎英珂　插图◎张杰客

2015年，18岁的女儿决定去美国读书。这一年的春节我们一起做了很多我记忆中小时候的年菜，她帮我打下手，我给她讲我们小时候的年。因为在我的潜意识里，是觉得也许从今往后可能会有相当一段长的时间我们一家三口很难在一起团聚过年了。因为她需要时间读完她的大学，她需要时间去适应新的工作，她需要时间为自己的人生目标付出努力。而这些时间都不可能为我们的年让路，所以，2015年的春节我们家过得比较隆重。我记得我做了她喜欢的素什锦、油焖虾、干烧鱼、冰糖肘子，以及家传的全家福大砂锅。记得我跟她说希望她以后无论在哪儿，逢过年还是要做点像样的菜庆祝一下，无论是在美国还是在哪国，年就是心里的家，年必须是年。

2016年的春节，我们在北京，她在纽约。我们这边的年三十晚上是她的同一天的早晨，我们视频通话，她看到我们餐桌上的菜，激动得大叫。等到了她的晚上，我们的初一早上，她发图片给我：“看，我们的年菜。”我看到了几个姑娘分别做的红烧鸡翅、煎三文鱼、炖菜、回锅肉……也是满满的一大桌，以及几个姑娘欢快的笑脸。尽管我们过的年对于她们来说只是一个平常的日子，吃完这顿，睡一觉，明天还要继续上课或继续写她们的论文，可是即便是远隔千山万水，气氛还是一样能传递的。年还是要过的。

前几天跟在纽约的女儿通电话，她突然冒出一句：“好想吃北京的糖葫芦，也好想念大砂锅。”我问她，你想家了吧？她说还好，只是累了的时候会想念家里的食物。估计是因为纽约人开始准备过圣诞了，连带着让她想到了过年和大砂锅。

人的很多美好的记忆都是跟过年有关的。

对于我们这波生于上世纪60年代的人来说，小时候过的年更是充满了美好、幸福和新奇。那是一年中最重要的时令节日，是一年中最快乐的庆典。因为它是由很多平日里吃不到的肉菜、鱼菜等稀罕的美味佳肴，以及只有那几天才被允许拿着副食本去购买的花生、瓜子、香油、芝麻酱、富强粉、好大米组成的。这些稀罕的物品在平时是没有的，副食本上也只有过年的那个月的一栏里才会有这些内容。这些物品是以每人半斤为计量单位的。

当然，在孩子们的记忆中，也许还会有新衣服、新衣服兜里揣着的小鞭炮、提着纸灯笼去邻居家讨要糖果等等这些令人兴奋不已的事情。在没有电视和手机的年代，这些内容会是一整年的期盼。因为相对于众多的普通又乏味的没有任何色彩的日子而言，那几天的美好是可以受用一整年的。

在那样的年代里，过年是念想，是期待，同时也是收获。

上世纪90年代初，我在日本读书，适逢阳历新年，我被朋友邀去东北的秋田县乡下过了一个日本的新年。那一年12月30日我从东京坐夜行巴士前往秋田县的大曲小城，近700公里的路程大巴车正好走一个晚上，31日的早晨，也就是日本人说的“大晦日”（除夕）的早晨我到了大曲。我的朋友加藤先生是一位基督教的牧师，平日里他的家只有他和太太两个人，因为过年，两个在仙台工作的儿子也都带着家眷回来了。这个年，加上我这个远道而来的外人，一共有八口人在一起辞旧迎新。

加藤先生家的年菜是祖辈传下来的数十款蒸煮的菜品。洋子夫人在12月25日过完了圣诞节，就开始分批分类地做起来了。日本人的年菜口味非甜即酸，据说这是在没有冰箱的年代里便于保存。红糖煮黑豆是家家必做的。除此之外，还有诸如蜜煮鱼干、煮芋头、醋藕、糖醋红白萝卜、豆羹、煮鱼子、昆布卷等等，多到数十种。所选食材要遍布山川湖海，象征五谷丰登、长寿不老和子孙繁荣。这些菜肴所寓意的事物都很有趣，比如煮黑豆，代表的是勤奋，每天勤奋地干活到被太阳晒得黑黑的.煮鱼子代表的是子孙繁荣.煮大虾表达了要活到腰都弯成大虾那样的意愿，代表长寿.糖醋红白萝卜表达的是做人要清清白白、干干净净。

所有的菜品分别烹煮后，在12月31日，把食物按照食材和颜色搭配妥当分别摆放进漂亮的“重箱”（精美的大漆食盒）里，然后就等待着正月一日的早晨全家一起享用。重箱分三层、四层、五层不等。每一个日本人的家里至少都会有一个这样的食盒，一般都是用大漆制作的。因为一年中只有新年这几天会用它来盛食物，所以日本人很看重它。高级的重箱以大漆莳绘的工艺，配以松竹梅椿仙鹤的纹样居多。光鲜亮丽，精美绝伦。同时，重箱中一层层的年菜，一个个的寓意，都代表了农耕社会里人们最纯朴的愿望。

因为我来自饺子的国度，因此，我责无旁贷地为加藤先生家的年夜饭增添了这道中国的饮食文化。

那个年，在加藤先生家中，伴着他们一家老小，看了日本的“春晚”（红白歌合战），听了12点钟敲响的108下的钟声，吃了象征着长久的过年荞麦面，大年初一的早上踩着厚雪去一个几百年的小神社“初诣”（新年参拜），喝了神社供给的热米酒，买了一个平安符……在寒冷的东北秋田县过了一个温暖的新年。

人到中年以后，时常会回忆起小时候那些难忘的事情，其中也包含了出自母亲之手的很多味道，特别是那些过年时吃过的。比如，我妈常在过年的时候炸芝麻排叉，把关东糖溶化后加入花生末和芝麻，制作成另一种甜食，用金属材料的粥勺做全家福砂锅里用的鸡蛋饺，炸豆腐丸子，自己炒江米，再用擀面杖擀碎做米粉肉，等等。一进入腊月，我妈就开始一道道菜有序地做，然后一盆盆、一碗碗地整齐摆放在阳台的角落里。年三十的晚上，我们更多的时候是做一个全家福的大砂锅，砂锅里整整齐齐地摆放了平时难以吃到的烧肉、大虾、蛋饺、豆腐丸子、炸猪皮、香菇和大白菜。这么一大锅，我妈会用鸡汤慢慢煨个把小时。

这些年，我也开始做全家福大砂锅了。砂锅里边的料也都尽可能地自己做。在我家过年三十，吃过我做的砂锅的朋友，似乎都还满意这一大锅的实实在在。

前几天，在一篇文章中看到一句话，深有感触。

“古代文化中有一种仪式，一种庆典，即一年中要有一次机会让人做点非同寻常的事。”而这个“事”，我觉得就应该是“过年”。

近些年来，人们常挂在嘴边，也是说得最多的一句话就是：“现在日子好过了，每天都跟过年似的。”

在对于仪式和庆典这样的事已经渐渐地淡漠，生活中让人兴奋的东西越来越少了的今天，至少我们应该好好做顿年夜饭然后好好过年。因为它既是一年的结束也是一年的开始，用一顿美好的餐食来做一年的承上启下，不是一件很浪漫的事情吗？
