# 生活圆桌_岁月嘌呤

生活圆桌

生活圆桌

岁月嘌呤

文◎张彤　图◎谢驭飞

我去医院时特地花19块钱挂了一个外科专家号。脚趾莫名其妙地疼了两天，我想大概是前天陪女儿打网球时不小心扭伤了。

我一瘸一拐地走到大夫眼前时，他都没怎么抬头，我描述了一下症状，他终于看了我一眼，不怀好意地说：“痛风吧？”

我怎么没有想到呢？在我们这座海滨城市，在我工作了20年的媒体圈，热爱啤酒与海鲜的壮士们，有多少早早地就背负了痛风的苦难，而我却可以逃脱吗？

“去化验一下血吧，我估计你是痛风。”大夫说着举起了左手，“我开始也是脚，现在上这儿了。”我看他各个关节都挺粗，当然也许外科大夫就应该有这么一双灵巧而粗大的手。

等我两个小时后，手捧尿酸指数616的化验单，再到他诊室门口时，这位姓李的大夫正在给一个黑人朋友检查踝关节。我在门口站了一会儿，细读了LED屏上他的简历，原来，他的专长并不是骨科，而是消化道恶性肿瘤手术。

李Sir看到我的化验单时，情绪突然高涨起来：“我才516，你比我整高100。”他看起来高兴极了：“你喝啤酒吧？”“喝一点。”我不好意思地说。“肯定不止一点。”他一边说话，一边在病历上龙飞凤舞。“你去内科的风湿专科吧，我真不知道他们现在都用什么药。”我问：“那么您发病时吃什么药呢？”“实在疼了就吃点止疼药，一般也不吃，忍几天就好了。616啊。小伙子。”他合上了病历，仍然赞叹不已。

去内科路途遥远，我一瘸一拐地走了好一阵子，期间，我把化验单照片发到朋友圈，病友们一分钟之内都冒了出来，他们给我列药方，开食谱。除了本地的啤酒主义者，还有广东的浓汤死忠粉、云南的菌菇爱好者、江南的竹笋豆腐美食家等等。他们给我的药方，除了两种众所周知的处方药，还有日本的“步行王”和澳洲的芹菜籽，当然，每个人在最后都会说，关键，得忌口。

走出医院时，发觉满街都写满了嘌呤。所有的餐馆超市，在我眼里都像《超级玛丽》里的蘑菇一样，上面标着数字——不过不是分值，而是嘌呤含量。我想，胡吃海塞的日子终于走到了尽头，从此我要精心规划自己的食谱，信念坚定地戒掉啤酒，我得小心翼翼地走，才能不与钻心的疼痛撞个正着。

痛风急性发作，吃一天药就好了，但我从此在吃饭问题上变得战战兢兢。酒不能喝，推不掉的聚会因此成了纯粹的“饭”局，其实能吃的菜也没几样，往往吃不了几口就开始听其他人说车轱辘话。酒兴上来，每个人都变成了话痨，而只有我，被苏打水洗空了肠胃，冷却了脑筋，除了盼着主食赶紧来，别无他想，要不然一次聚会下来，就饿得前心贴后背了。

每周五我在电台有一档节目，这一周最热的话题是一位摇滚乐手和他的保温杯。我的搭档神不知鬼不觉地就成功地把话题从保温杯引到了脚趾痛。她说，当一个人开始惜命的时候，他的中年就来到了。我觉得她说得可真有道理。

渺小的字节

文◎中读用户@李车车　图◎谢驭飞

10年前，我的运气还算不错，计算机系毕业的我，找工作时很快就被两家公司录用了，一家是做数字电视机顶盒里面的人机交互网页的，另一家是做报表软件的，作为程序员，工作内容上的差别对我来说无关紧要，开出的薪水也相差无几，于是我就选择了后面一家公司，原因是离家更远些，我方便以此作为借口在外面租房住。

我从西直门坐上城铁来到上地，心里正疑惑着这里的人是否都皈依了耶稣（上地跟上帝谐音），而所换乘的公共汽车，则拉着我连续路过了上地三街、四街、五街、六街，一直来到七街，才到达自己的公司。这些顶着简单粗暴名字的斜街，让我感到莫名寒冷，以前只是听说过没文化的美国殖民者，喜欢用数字命名公路和街区，没想到在自己所居城市的西北角，也存在这种做法。第一次见到这些街名时，我的情绪不可避免地变差了，这些街道的两旁，分布着外形相似的大厦或者软件园，一幢幢楼的名字同样是用数字命名的。但是我想这些命名，恰恰提醒着我认清现状，不要对未来抱有太高奢望，在这里，卑微的程序员无非是这些大厦里的一块砖头而已，在这庞大的二进制帝国里，自己不过是一个小小的字节上的一个比特罢了。

第一天上班时，我发现这家公司还真是不小，几百平方米的办公室里，摆着数不清楚的电脑桌。坐在一群胡子拉碴、穿着大裤衩和凉鞋的同事中间，脚蹬新皮鞋，衬衫扣到领口的我，活像一个傻瓜。这些同事在上班时间过后，才陆陆续续地来到公司，有的甚至上午10点钟才到，这给我了一种感觉，这家公司的工作并不繁忙。不过几个小时后，这感觉就被证明是个错觉，这里的人在晚6点下班后都不着急回家，而是继续对着电脑屏幕，稳如磐石般地坐在座位上。因为公司规定，前一晚加班过8点的，第二天可以采取弹性工作制，10点前到即可，只有晚上要和朋友聚会的那些人，才会“早”在8点钟就离开。

午休时分，无数穿着五花八门的人走到楼房外面闲逛，他们唯一的共同点，就是胸前都挂着同时作为门禁的工卡，手里都攥着一个小小的香蕉或者皱巴巴的橘子——楼里食堂统一配给的水果。上班时间冷冷清清的楼下，此时无比热闹，有的人在绕着楼房一圈圈散步，有的三三两两聚在一起抽烟聊天，还有一些精力旺盛的家伙，围在一起踢着毽子，挂在他们脖子上的随着踢腿上下翻飞的工卡，标识着他们是某公司的程序员。这景象让我不禁想到了监狱里定时放风的犯人，又想到了《生化危机》中那些倒霉市民——被病毒感染，变成在街上无序游走的僵尸。

好容易熬过了第一个月，工资发了下来，刨去保险和公积金一共到手3500块，交过房租后，几乎只剩下了饭钱和交通费。不过这也正好，回到住处洗漱后，时间已经是9点开外了，实在没空去消费.而到了周末，我总是感到疲惫，很少出门，只是待在房间里看书和电影，于是日子过得倒也不显拮据。这期间看了大量的小说和电影，倒是为自己日后创作小说打下了基础。

《银河系漫游指南》

文◎二公子　图◎谢驭飞

很久以前我看过一部科幻小说《银河系漫游指南》，几年前它被改编成了电影。书中讲述如何在银河系搭顺风车旅行：在远古时间的迷雾中，在银河系帝国最光辉的那段日子里，生命是野性、富足而且是不用缴税的。那个时候，男人是真正的男人、女人是真正的女人、半人马座阿尔法星上的长毛小动物是真正的半人马座阿尔法星上的长毛小动物。

但银河系有很多问题至今没有答案，比如生命意义在哪里？为什么生存？为什么会死亡？为什么大家把这么多时间花在电子产品，比如手机上？亿万年前，银河系里某种高等智慧生物厌倦了在生命意义这个问题上不断争吵，他们决定一劳永逸地解决这个问题。于是一个巨大的超级计算机被他们设计了出来，取名“深思”。“深思”的智慧令人吃惊，在没有数据支持的情况下它凭空想出了“我思故我在”，并大胆预测了布丁和所得税在当今世界的出现。

据“深思”的反复计算，生命、宇宙和万物存在这个问题的终极答案是42！对，数字42，六七四十二的那个42！

“深思”有了自己的思想后，觉得还有追剧等更重要的事情要做，就设计了一个无限细微复杂、部分操作系统由有机生命体组成的计算体系，对42进行进一步求证。这个体系被其命名为“地球”。

地球在银河系中，除了名字土里土气，还被公认是渺小而且无害的，因为其对任何其他星系都构不成一点儿威胁。地球的问题——一个相当大的问题——就是：在大部分时间，生活在上面的人类本身并不快乐。他们觉得自己很可怜，甚至买得起手机的人也这么想。越来越多的人觉得他们的远祖从树上下来就是个错误。还有人说他们干脆就不应该离开海洋。

地球上的海洋被海豚等高等生物而不是人类主导，虽然在地球人类认为自己是比海豚高等的生物。人类觉得自己高等的原因无外乎是他们发明了轮子、曼哈顿和战争等。而海豚除了在水里游荡猎食什么都不会。其实，海豚觉得它们比人类高等——跟人类认为自己高等出于同样的原因。

以上观点都出自伟大的《银河系漫游指南》——它在地球上用不着，因此在地球上就没有出版过，所以也没有地球人看过或者听说过。但是这不妨碍《银河系漫游指南》成为一本伟大的著作。

在银河系的很多地方，尤其是边远而比较宽松惬意的文明中，《银河系漫游指南》已经取代了《银河百科全书》成为最畅销作品。尽管前者仍然有很多非常不精确的地方，但是跟缺乏想象力的后者相比有两个最大的优势：第一，它便宜！第二，《银河系漫游指南》的封面印有非常实用、设计也非常赏心悦目的一句银河系旅行忠告——Don't Panic！（无需惊慌！）

1000年前的朋友圈聚会规则

文◎朱国洋　图◎谢驭飞

《资治通鉴》的主编司马光不仅小时候是个砸缸的熊孩子，成年后还是一个有趣的社交达人，越活越精彩，和他严谨保守的政治风格完全不同，和他楷书的规矩端正也截然不同。

他出生在北宋1019年，算算差不多有1000年了。这哥们在64岁那年，一边在家里写书，一边和一群七老八十的老同事老哥们发起了一个朋友圈，取名“洛阳耆英会”，人数也就十二三个，他自己还亲自撰写了《会序》，交代了搞这个聚会的缘由，是仰慕白居易此前搞的“香山九老会”。文人雅集玩点曲水流觞诗酒唱和，本来这也没啥太大的新意，但他接着写的《会约》，约定了游戏规则，就有趣了：

一、序齿不序官：大家都是场面上的，有潞国公，有节度使，都是当过省部级和大区正的老同志，司农少卿、太常少卿等等好歹也是财政部、文化部的副职，司马光自己资历浅一点，但端明殿大学士是皇帝核心秘书班子成员，地位也不低。类似这些人物聚会，轮流做东，怎么排顺序，饭桌怎么排位置，向来是个难题。好在这帮哥们都是明白人，干脆约定好，既然是朋友聚会，就一律按年龄排序，只有老张老王，千万别再拿单位职务来端着了，免俗。

二、为具务简素：场地、餐具，简单朴素一点，有能力的也收着点，不要竞相攀比，否则影响不好，土豪。

三、朝夕食不过五味，菜果脯醢之类，共不过二十器：安排主菜大菜别超过五道，小菜水果瓜子甜品不要超过二十道。上要封顶。大家是来聚会聊天的，不是来胡吃海喝的。

四、酒巡无算，深浅自斟，主人不劝，客亦不辞：这个规矩实在太好了。酒有管够敞开喝，墙边一溜搁着，要喝多少自己倒，别争满杯半杯了。不劝酒，自己来。

五、逐巡无下酒时，作菜羹不禁：喝到兴头上，下酒菜没了，可以加个西湖莼菜汤、五香花生米，不算超标。

六、召客共用一简，客注可否于字下，不别作简：聚会轮值主办方和大家确认意向时，只发出一张会签单，派人各家跑一圈，能够来的签个字打个钩，这就OK了。

七、会日早赴，不待速：聚会那天得早点到，别拖拖拉拉抱怨堵车啥的，反正到点开饭，过时不候。想想古代没有手机也有没手机的好处。

八、有违约者，每事罚一巨觥：大家如果违规了，可不是认罚三杯，而且是满满一大壶——令狐冲原来也是有历史的。

和1000年前的古人比比，我们今天的朋友圈文明程度未必更先进。无论高铁飞机计算机怎么演进，人类的生活底色总是不变的。
