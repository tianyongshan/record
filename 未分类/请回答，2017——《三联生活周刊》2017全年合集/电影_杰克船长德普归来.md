# 电影_杰克船长德普归来

杰克船长德普归来

电影

杰克船长德普归来

记者◎宋诗婷

电影《加勒比海盗5：死无对证》剧照

对于忠实“粉丝”来说，《加勒比海盗5：死无对证》里有他们见证过的岁月。时隔六年，杰克船长虽然老了些，但这个角色与约翰尼·德普本人一样，有个不老的灵魂。

海盗归来

“船的意义，就是想去哪儿就去哪儿，就是自由。”时隔6年，杰克船长约翰尼·德普与他那象征自由的“黑珍珠号”又回来了。

一亮相就是熟悉的味道。杰克醉着酒从装满金币的保险箱里挣扎起来，边上还躺着个官太太。大副一声令下，几匹骏马齐发力。马车的另一头拴着保险箱，按计划，保险箱该被拖出银行，但杰克船长的偷抢拐骗计划向来不靠谱，马车不仅拖动了保险箱，还拖走了整幢房子。一路的追逐，一路的化险为夷。

开场戏一出，我就有种预感，这部《加勒比海盗5：死无对证》很可能是最接近首部作品的续集。它不再是第四部里杰克船长的个人秀，而是重新找回了群戏，和第二部、第三部相比，节奏上也更为松弛。

对于拍摄“加勒比海盗”系列，迪士尼和制片人杰瑞·布鲁克海默一直感情复杂。2003年，《加勒比海盗：黑珍珠的诅咒》意外走红，成为当年好莱坞最卖座的电影之一。从那时起，“加勒比海盗”系列就成了迪士尼独立开发的最成功的真人电影，票房更是屡创新高。

与票房同步增长的是这一系列电影的预算。到了第三部《加勒比海盗：世界的尽头》，电影的总投资已经超过3亿美元，是当时成本最高的电影。迪士尼不得不在第四部开拍时控制成本，尽管电影的全球票房依然超过10亿美元，但这一经典IP的口碑却被《加勒比海盗4：惊涛怪浪》拉低了一大截。

对于任何系列电影来说，拍到第五部，再出点新花招都不容易，更何况是个性过分鲜明的“加勒比海盗”系列。

“我们都知道，要把第五部拍出新意将会有很大难度，但这个系列电影在全球各地有如此多的影迷，我们感到责任重大。”导演之一乔阿吉姆·罗恩尼说，他和另一位联合导演艾斯彭·山德伯格都来自挪威，两人自小相识，结伴到好莱坞发展。他们年纪不大，是《加勒比海盗》和杰克船长的第一批观众，因此，对这个系列作品有迷影情结。

因为情怀，更因为责任重大，《加勒比海盗5》让观众等了6年。杰瑞·布鲁克海默说，剧本筹备工作从四五年前就开始了。“今天的观众和14年前的当然不同，但本质上，他们和农耕时代围在篝火边听故事的听众也没有什么区别——需要的依然只是一个好故事。”

《加勒比海盗5》的故事当然延续了以往风格。漂泊多年的杰克船长依然死性不改，依然麻烦缠身。哈维尔·巴登饰演的萨拉查船长率领一群亡灵水手逃出了百慕大，誓言杀尽所有海盗，而杰克船长就是头号目标。大难临头，他被迫与企图解除父亲诅咒的亨利、女天文家卡琳娜组成同盟，共赴大海深处，寻找能够解除诅咒，打败萨拉查船长的宝物——三叉戟。

对于新观众来说，这是一部“大叔”杰克船长带领一对俊男美女对抗恶势力的电影。对于忠实“粉丝”来说，这部《加勒比海盗5》里有他们见证过的岁月。在14年前的首部电影中，杰克船长与铁匠威尔、美人伊丽莎白并肩作战。14年之后，杰克船长依然是那个杰克船长，但与他相爱相杀的已经变成了威尔的儿子和死对头巴博萨的女儿。

电影中的人物关系似乎又回到了原点，完成了一个轮回。到海上寻找爱与自由的男女配角总是年轻的模样，而杰克船长虽然老了些，但这个角色与约翰尼·德普本人一样，有个不老的灵魂。

颠覆传统

《加勒比海盗5》的全球首映在上海举行，这是迪士尼第一次在中国举行全球首映活动。除了出镜率极高的超级英雄们，杰克船长或许是中国观众最熟悉的迪士尼电影人物。活动当天，很多“粉丝”打扮成杰克船长的模样，争着与约翰尼·德普合影。

14年前，迪士尼完全没有料到，《加勒比海盗》会被如此追捧。“这是一个意外，这是一个意外。”杰瑞·布鲁克海默有着华丽的制片人履历，是好莱坞权力排行榜上很靠前的人物，连他都没能预见这一系列电影的成功。

当年，迪士尼准备以旗下的三个大型游戏为概念，打造三部电影。《加勒比海盗》的灵感来源于加州迪士尼乐园里一个备受欢迎的游戏项目。但在三个电影计划中，《加勒比海盗》是最不被看好的一个。

欧美电影史中不乏经典的海盗形象。1935年，埃罗尔·弗林在《铁血船长》中塑造了风流倜傥、劫富济贫的海盗皮特·布拉德。这个角色一度让“海盗”这一形象流行起来。大银幕上的“海盗”虽多，类型却乏善可陈：偷淫抢掠、无恶不作的坏海盗.个人英雄主义、英俊潇洒的皮特·布拉德式海盗。几乎只有这两种角色设置。

这个定式被《加勒比海盗》打破了。

尽管已经过去十几年，杰克船长的出场依然令人记忆犹新。一个画着烟熏妆、邋里邋遢的男人，走起路来一扭一扭，娘里娘气，刚亮相就被抓，气场全无。这样一个不靠谱的海盗自称船长，手头却没有一艘船。

按照传统的电影模式，杰克船长不可能是主角，这根本就是个丑角，男二号、男三号的命。但在《加勒比海盗》中，杰克船长是绝对的男一号，男二号是一身正气的铁匠威尔，饰演这个角色的还是帅得与德普不相上下的“精灵王子”奥兰多·布鲁姆。

传统的英雄形象和英雄模式被《加勒比海盗》瓦解了。在我看来，14年前的“杰克船长”就是今天的超级英雄“死侍”，他们的存在都是对已有英雄形象的解构。

德普塑造的杰克船长具备一切大反派的造型和性格特质。他冷漠、自私、胆小、神经质、娘娘腔，时而懦弱，时而疯疯癫癫。他代表的是颓废和自我放逐，是失败者。但就是这样一个不着调的人，内心却有善良的一面，他对自由的渴望远远走在了17世纪那个保守时代的前面。

2000年初，早已厌倦“超人”的观众急需新的角色来开启大银幕新的英雄时代。而由德普塑造的这个满身缺点的海盗就符合新世纪年轻人对于新英雄的想象。

杰克船长的复杂和不可捉摸让他魅力倍增。十几年之后，这个反英雄形象早已深入人心，他甚至可以脱离《加勒比海盗》独立存在。哪里有杰克船长，哪里就有《加勒比海盗》。

魔幻元素的引入也是“加勒比海盗”系列成功的重要因素。2000年前后，《哈利·波特》《魔戒》等魔幻题材电影先后上映，都取得了空前成功。《加勒比海盗》并非构建于魔幻世界之内，而是在现实主义的基础上加入了魔幻元素，让这部电影成了魔幻现实主义题材。

这符合当时的潮流趋势，魔幻与海盗题材的结合也足够有新意。更难得的是，当年的《加勒比海盗：黑珍珠的诅咒》并非为了魔幻而魔幻。其中的魔幻元素都有现实的指向。

杰克船长手中的罗盘从不指向明确的物理方向，它指向杰克船长的内心，指向他的自由之路。电影中的恶势力巴博萨船长受到诅咒，在月光下会变成骷髅，诅咒的背后也指向人性——贪婪和背叛，这正是魔幻元素与现实主义能够相融合的衔接点。

杰瑞·布鲁克海默在接受采访时对我说，《加勒比海盗5》的剧本一直在修改，除了结构和桥段上的反复琢磨，他们花费时间和精力最多的部分是“如何让这部电影更幽默”。

“幽默”是“加勒比海盗”系列另一重要元素。这幽默不仅体现在德普塑造的杰克船长身上，电影从台词设计、故事桥段安排到动作场面设计，处处都透露着幽默。当然，这个特色在《加勒比海盗》第一部中表现尤为突出，效果随二、三、四部递减，但到了第五部，《加勒比海盗》似乎又找回了最初的幽默功力。

永远的杰克船长

“如果约翰尼·德普不再出演杰克船长了，‘加勒比海盗’系列还有存在的价值吗？”我问制片人杰瑞·布鲁克海默。

“我也有些怀疑，希望这一天不会太早到来。”杰瑞·布鲁克海默说。

德普塑造的杰克船长不仅是他个人演艺生涯中最重要的角色，这个亦正亦邪的怪人也必将载入历史，成为影史中经典的角色之一。

按照德普在很多场合的说法，杰克船长的灵感来源于两个形象：摇滚明星凯斯·理查德斯和卡通角色Pepe——一只法国臭鼬。在热爱摇滚乐的德普看来，剧本中的海盗就是17世纪的摇滚明星，他们一样的离经叛道，一样的渴望自由，一样的“臭名昭著”。而那只臭鼬Pepe永远幻想着母猫爱上了自己，这种毫无来由的妄想和自信被德普用在了杰克船长对伊丽莎白的感情上。

德普对角色的想象脑洞很大，这符合他的一贯风格。但在当时，和《加勒比海盗》不被看好一样，德普由着性子打造的杰克船长也不被迪士尼看好，他还险些被替换掉。

他是每天喝大了才来片场吗？他演的是个同性恋吗？他在耍什么把戏？《加勒比海盗》拍摄之初，德普每天都收到来自片方的质疑。

接受媒体采访时，他提到了当时双方的博弈：“最开始谁都知道迪士尼担心到喘不过气。我告诉他们，我理解、明白他们的担忧。但他们雇我来演这个角色，难道没看过我之前演的片子吗？如果没看过又为何来找我？当时，杰瑞给了我非常大的支持。戈尔导演也有先见之明，基本只有我们三个这样坚持，其他人都不喜欢。但我觉得，冒险是很重要的，你不尝试点还没被玩烂的新创意，就无法给一部电影增加任何有趣、超出预料甚至是荒唐的东西。”

正像德普所说，从第一部《加勒比海盗》起，杰瑞·布鲁克海默就是这系列电影的制片人，过去14年，他一直牢牢把握着这个系列电影的方向。布鲁克海默是个谨慎的人，你很难从他口中撬出什么“如何让电影大卖”的秘诀，但他愿意透露自己的一个原则——只与有极高天赋的人合作。

什么是天赋，见仁见智。但永远能找到有天赋的人，他们的“天赋”又能让观众满意，这就是布鲁克海默最出众的能力。在过去的几十年里，他靠自己找人的天赋，捧红了尼古拉斯·凯奇、汤姆·克鲁斯等明星和一众导演，当然，约翰尼·德普是他最得意的案例之一。

德普能演出如此不正经的杰克船长也并非偶然，这和他的个性，以及表演经历息息相关。在成为杰克船长之前，他是怪才导演蒂姆·波顿的御用男演员。1990年，他出演了蒂姆·波顿的电影《剪刀手爱德华》，在其中饰演那个有一双剪刀手的机器人爱德华。这个角色深入人心，德普也由此开始了自己怪诞风的银幕形象，他那张帅气的脸也基本都隐藏在化着浓妆的角色背后了。

如果说《剪刀手爱德华》让德普成了著名演员，那《加勒比海盗》就把他捧成了风靡全球的大明星。杰克船长的部分个性来源于摇滚歌手，德普的部分人格也是被摇滚乐所塑造的，这是德普和杰克船长重合的部分。

12岁抽烟，13岁破处，14岁吸毒，15岁文身，16岁盗窃……德普年轻时的糊涂账一大堆，不输杰克船长。最近几年，离婚、情感纠葛、财产风波，再加上主演的几部电影都票房惨淡，很多人调侃说，离开杰克船长，德普的生活似乎不太顺利。

他在塑造杰克船长，杰克船长似乎也在塑造他。演员和他饰演的角色之间形成了某种互文性，这让德普和杰克船长都平添了几分魅力。

像很多被一个角色所成就的演员一样，德普曾抗拒杰克船长这个角色，试图逃离“黑珍珠号”。显然，他没能成功，如今看来，他好像也放弃了逃离的念头，打算与这个角色和谐共处了。

“我曾经也想过要摆脱这个角色，希望迪士尼能把我炒掉，但这很难，最终我还是回来了。”德普说，“躲在这个角色里我总是感到安全。”
