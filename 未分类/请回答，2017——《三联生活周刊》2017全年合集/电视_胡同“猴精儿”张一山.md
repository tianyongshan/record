# 电视_胡同“猴精儿”张一山

电视：胡同“猴精儿”张一山

电视

胡同“猴精儿”张一山

记者◎张月寒

优酷剧集《春风十里，不如你》剧照

《春风十里，不如你》正式上线后，张一山因在其间饰演男主角秋水而再度成为热议的网红。

采访在北京望京一间宾馆内，张一山的穿着就是十分简单的T恤、短裤，给人扑面而来的印象是和《余罪》中的余罪以及《春风十里，不如你》中的秋水非常类似，这种气质，在随后的采访中被渐渐冲淡了，可某一时刻，又不经意地冒出些许。窝在宾馆靠窗的沙发凳上，他暴露了“多动”的本性。一个多小时的采访过程中腾挪反转了多种坐姿，或盘腿，或瘫着，或斜坐，或点一根烟。

胡同男孩

张一山是从小在胡同里长大的北京人。新街口的大四条胡同。如今，新街口的店铺换了几番，道路也拓宽了几次，张一山怀念的“胡同生涯”也没有了。不过，他至今仍和父母住在胡同原址上重盖的回迁房内，不拍戏时，他说自己最大的爱好是光着膀子闲逛。

“现在光着膀子不会给人认出来吗？”我问。

“认出再说认出的事吧。”他说，“不过如今也收敛很多了，但是在比较私密的胡同里或者是人比较少的地方还是喜欢光膀子。”

所以，他说，看冯唐的《北京，北京》会特别有感触。因为“那叙述的就是他们年轻时候我们小时候才有的事，看上去分外亲切、分外有共鸣”。

张一山住的新街口大四条胡同自明代就有，原是东西走向，东起新街口北大街，西至西教场胡同。1911年二段合并，为区别北边小四条胡同，称大四条胡同。上世纪90年代初的时候，胡同里有修鞋摊、副食店、奶站，家家户户门前挂着报箱、奶箱。心情好的时候，穿耐克蓝黑色大衣（90年代曾风靡一时）的成年男人，就相约去“老西安饭庄”吃泡馍。

在这条胡同里，小时候的张一山“天天追跑打闹，和典型的北京小孩儿一样，玩弹球，踢足球，逮人，在胡同里上蹿下跳”。不过，与他作品中一贯呈现的形象有别，他说自己皮虽皮，但却是一个“不出格”的孩子，和余罪或刘星（《家有儿女》）的带头捣蛋类型不同。可聊开了以后，他又透露出小时候经常挨打的形象。“手、筷子、盘子……都是‘武器’。只要一惹我妈生气，她一觉得不对劲就打。”

演员张一山

10岁时开始了剧组生涯，用张一山自己的话说，“剧组也是个小社会”。比同龄人更早地和成年人开始相处，虽然是孩子大家都让着他，但也会更早熟。“其他同龄人都是在学校面对同学，回家面对家长，我又要面对同学老师，又要面对家长，还有摄制组。摄制组其实就是一个社会，什么人都有。从这一点也可以说我比较早地步入了社会，所以比同龄普通学生能更早适应这个社会的人情世故和怎么与人相处。”张一山在采访中颇为坦诚。

10岁时，他被当时学武术的隔壁班老师推荐去了《小兵张嘎》剧组试镜，被导演徐耿相中在剧里饰演乐乐。12岁时，出演《家有儿女》刘星一角，被广大观众熟知和喜爱。从此，他开始了长达十几年的半剧组、半普通孩子的生活。

这段双重生活状态，对于一个孩童的影响是显然的。或许他从此以后不会再那么怯场，不会再那么惧生，演戏状态也更自然。但是，从一个演员长久的发展角度来考虑，或许也是一种限制，因为童年时被塑造的印迹太深，导致他成年后的每一个角色，都难逃最初那个影子。

采访中，张一山并不否认这一点。他承认自己迄今为止较为成功的角色或许都有些雷同，也多多少少都会有自己身上的影子。只要“观众喜欢我那样”，他也觉得无可厚非。“人缘、脸缘作为演员来讲都是很重要的，有人天生招观众喜欢，有人天生不招观众喜欢。没有绝对的演技，只有观众心中的感觉。有的人能接受你，有的人不能接受你。……我不会刻意摆脱个人印迹。”他说。

过早走向舞台，很多同龄人也许都非常容易经历的一些事情——早恋、学业压力、同性之间的竞争、父母的期望和反叛，也许张一山都没有切身的体验。这些缺失的人生，他会觉得遗憾吗？

问及这个问题时，张一山眼睛一亮，似乎有种说到他心里去的感觉。“比较小的时候大家都认识我，我去哪里也不方便，有点丢失了一点童年生活。”他这样说。可下一秒他又话锋一转：“但也没有什么特别遗憾的，我不是那种苦大仇深要求很多的人，我不觉得这个是对我一生影响很大的事。……我很小的时候就觉得这个事或许对我以后有好处，所以我没有觉得丢失了什么。”较早地成为公众人物，他坦言和周围同学、朋友的相处方式并没有什么改变，因为没太把自己当回事。“这个世界上有很多人，会老是觉得别人另眼相看自己。我觉得是他们内心作祟，和别人没有关系。如果一个人老是觉得别人对自己和对别人不一样，那不是别人的原因，是那个人自己的原因。”

《家有儿女》中刘星的机灵劲儿，曾在暑假的晚饭餐桌上一次次让无数孩子和大人捧腹，堪称佐饭佳品。“一个小孩，那时候不懂什么是表演也不懂什么是演技，别人让我怎么演就怎么演，加上自己的小想法，按照自己怎么舒服的方式。也许，自己觉得对的那个劲儿也是大人眼里的还不错的标准。”

他这样评价童年时的演技。

北京味儿

同为北京人，冯唐的作品，一直有种直男式的温暖自恋、自以为是。其中最具特点的或许是装不来的“北京味”。那种“北京味”，是真实的、回忆视角的，因此，某种程度上它能打动人。

张一山说他很喜欢《北京，北京》小说中描述的，只有老北京人才知道的燕京啤酒的细节。“现在燕京啤酒都是绿瓶了，但我们小时候是有紫瓶的。紫瓶是那批酒的酒头，更浓更好喝。这个冯唐小说里写到过。”

张一山印象中的北京，就是由胡同组成。有杂院，有四合院，然后有“平房，公共厕所，菜站，烙饼”。菜站是当时卖菜的地方，张一山之前在胡同里的家，旁边就有一个菜站。冬天的早上起来，他穿着大棉袄“哆哆嗦嗦地跑到胡同上公厕”。从小住胡同，张一山说自己非常习惯去公共厕所的日子。冯唐在《北京，北京》里描述过一种每天早晨需要去公厕倒马桶的胡同细节，张一山说他没有经历过。但至今仍保留了胡同生活的一种特有印迹——蹲式如厕。

“以前胡同很长，差不多有半公里、500米，甚至更长。一眼可以从胡同这头看到那头，站在胡同的西口能够望到东口过的车和人。现在根本看不见了，停满了汽车，扩建。”谈起胡同，他至今仍很是神往。

在糖葫芦和北冰洋汽水的橙子香味中，张一山从一个有点皮但不出格的孩子，变成了如今号称“90后演技扛把子”的佼佼者。这条道路，有生命中的偶然，当然也有偶然背后的努力。

《家有儿女4》结束后，他进入了一段时间的沉寂期。这段时间他也出演过一些作品，《妙想天开》《生于80后》《扣篮对决》《寻找成龙》《不想长大》，有电视剧也有电影，但“都不是什么很让大家记得住的角色”。这段日子或许是他由童星转变为青少年的尴尬期，他说自己也并没什么落寞或惋惜，就好好上学而已。

高考时他的志愿只填了一所学校，北京电影学院，因为不想自己有退路。至于不报考中戏的原因，他说他以为中戏只喜欢招那种“高高大大有气场的，演话剧”的孩子。可是，就连他报考北影当时也有很多人不看好，说艺术院校不愿意招演过戏的人，他们喜欢一张白纸。

后来还是考上了。张一山坦言大学时光是他人生中最喜欢的一段日子，类似《春风十里，不如你》，也会有“对女生的喜爱和性的冲动”。戏中的许多细节，他的真实人生也有不少经历过。在采访中他说他大学时期没有谈恋爱，因为觉得如果交个女朋友，隔三岔五就在网上被公开，被别人知道，他不喜欢那样的生活。

“可是，感情这种事情，有时也是没办法控制的吧？”我问。

“可以控制啊，”他说，“用理性控制。”

成熟《余罪》

《余罪》毫无疑问是他成年后的一个巅峰。对于如何达到这一巅峰，张一山至今回想起来仍没有预料到当时这部剧能受到这么大的欢迎。“当时拍完戏以后，因为涉及很多东西担心不让播，我就想，就这样自己留着看也很好的。”

剧中人物余罪的爆发力，尤其是几个几乎狰狞着暴青筋的近景演绎，让人看到了这个“90后”演员内心积攒的很多东西。张一山却说自己当时演余罪的时候并没有做什么特别的心理建设，只是常规地按照科班的方式准备人物。“不过，我确实是一个很有爆发力的人。”他说，“每个角色都不能平平淡淡，每个角色都是有高低起伏的。我当时是完全按照那个人物的心理去感受每个阶段的想法和情绪。”同样，“演得很过瘾”的感觉，在《柒个我》里他觉得也有。这部剧还未开播，他在其间饰演一个具有七重人格的人，相当于有七个不同的角色挑战。

7月21日，优酷的“超级剧集”《春风十里，不如你》正式上线。张一山因在其间饰演男主角秋水而再度成为被热议的网红。如今，剧集播出一半，有些网友评论军训情节过长。张一山说：“1992年是北京全部高校罕见的军训了整整一年的时期，冯唐的这一记述其实非常符合。当时医学院和清华、北大是在一起军训的。”

之前《家有儿女》末段有一个剧情，刘星因为机灵和不怯场，被选为文艺兵。心理学上普遍认为，一般从事演艺行业的人，属于那种“外放型”人格，既不会怯场，且十分享受那种在舞台、众人目光中的那种感觉。问及张一山，他却对此呈现出一种全然否定的状态。“我从小就是那种不愿意往舞台站，不愿意所有人都看着我的人。”

演戏十几年，他说自己所有的片酬都交给母亲保管。“身上也没卡，出门都带现金。”也没有什么特别的物质迷恋，所有二十几岁男生都会感兴趣的譬如说靓车、波鞋、电动游戏，他好像都没有什么特别的癖好。他最喜欢穿的衣服是“裤衩背心”。除了演戏，他说自己私下生活也挺无趣，运动，打篮球，然后就待着。“不爱旅游，就喜欢在自己的家乡北京待着，没事的时候在各种地方走。……喜欢去人多的地方，体验生活。”

他说自己现阶段接戏的标准，就是不要太累，“从早拍到晚，身体真盯不住”。他在第20届上海电影节上接受关于《血战湘江》的采访时，也蹦出过“累”的字眼。“我是很喜欢演戏的。”他说，“但我也是比较随性的人。有时一拍就拍三四个月，每天重复同样的事情，也会觉得有点疲惫。演员的压力很大，每创造一个人物，要让自己变成另一个人，让观众觉得你就是那个人，这是一件不那么容易的事。”
