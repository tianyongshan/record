基于风格的设计方法总是围绕着造型、潮流、限量、潮酷以及欲望的对象，也许，现在是采取更全面视角的时候了。

上世纪90年代初，益力地质勘探队在深圳寻找矿泉水源的时候，在南山区塘朗山附近发现了两处泉眼，由此在水源附近建造了益力矿泉水二厂，之后还诞生了深圳第一个合资矿泉水品牌“达能益力”。作为2017年深圳设计周的重要组成内容之一，由TOPYS与盈投置地联合主办的Mindpark创意大会就在如今已经废弃的矿泉水生产厂区中举办的。

如同昔日厂房在城市更新过程中面临着重塑的问题，Mindpark创意大会将重点聚焦在创意思维对各个领域的改变与重塑。来自七个城市十几位顶尖创意人以“创意的可能性”为主题，围绕品牌表达、创新体验、产品设计、互动创意等内容进行分享，讨论未来可能影响行业与社会的创意动能。

类似的重塑过程也发生在今年33岁的英国设计师本杰明·休伯特（Benjamin Hubert）身上，作为受邀发表主题演讲的创意人之一，近年来他被视为英国新一代设计师中的佼佼者。2015年9月，他将他的工作室名字从本杰明·休伯特公司改为“层次Layer”——这不仅仅是减少自我意识、重塑品牌的问题，而是一种重新出发，从设计本身寻求新的定义与价值。

基于风格的设计方法总是围绕着造型、潮流、限量、潮酷以及欲望的对象，也许，现在是采取更全面视角的时候了，经济、社会、伦理和环境问题等不可避免地成为整个设计过程的一部分。

层次工作室在伦敦V&A博物馆展出的动力雕塑装置“铝箔”

全面的视角可以是显示生态友好、减少个人碳足迹的可穿戴设备，一件名为WorldBeing的多彩腕带以及应用程序。纸屑图案的腕带由注塑回收电子废物制成，这是层次工作室与碳信托基金会的合作产品。

与身体健康监测器相似的原理，WorldBeing能够实时跟踪你的碳消耗，通过计算穿的衣服、吃的食物和旅行方式等，创建一个“碳云”来帮助设想你的碳足迹。云的大小因人而异，随着一天的流逝，碳云从冷变暖，以这种可视化的直观方式，提醒你的行为对自然环境和全球变暖的影响。

为了创造出“以有意义的方式改变人们现在及未来生活方式”的产品，“层次”在工作室内部设立了一个新部门LayerLAB，它基本上是围绕研究展开的，集中于能够提高生活质量的新兴技术及材料。它研发的前瞻性产品可以是任何类型的东西，也许是应用程序，也许是一种服务。

可以满足残疾人个性化需求的3D打印轮椅GO

GO轮椅特殊的轮圈护套和手套增加了手与推动边框之间的握力

层次工作室为Moroso品牌设计的“网格桌”系列，2013年

“在LayerLAB，我们不断研究和思考人们的工作和生活方式，接下来会如何生活，世界会如何变化，以及我们使用的工具如何使得这些变化和过渡更容易、更快乐和更健康一些。”休伯特对本刊解释说。

LayerLAB的首个项目是GO，一把可以满足残疾人个性化需求的3D打印轮椅，座位部分和耐用烧结铝的脚托采用定制形式。它不仅仅是可以被3D打印的设计，工作室与3D设计公司Materialize合作创建了一个端到端的定制过程。

使用者的身体被扫描和评估，所得到的3D数据被合成到定制的轮椅中。例如，后背中部脊柱损伤的人可能需要更多的支撑和更高的座椅靠背，而失去一条腿的人可能要将他的偏心平衡纳入计算范围。考虑到用户在轮椅上度过的漫长时间，这样的定制轮椅能够准确适应个人的体重、身体形态和残疾状况，提供更多舒适度和灵活性。

收集生物识别信息后，轮椅的创建过程由用户在应用程序中完成，附带的GO程序允许用户在下订单之前，通过指定图案、色彩等可选元素参与设计流程。

“全世界每年生产200多万把轮椅，英国总人口的2%在使用轮椅。轮椅设计似乎是一个被忽视的小行业，事实上存在着很多运用技术向前推动的可能性。”休伯特说，“我们的目标是以同一大小和规格的产品及可变化的解决方案，来满足和适合不同使用者的需求。”

在椅子开发过程中，层次工作室采访了众多轮椅使用者和医疗专业人员，发现使用者的主要担忧在于自我推动时的扭伤。轮椅的推动轮通常只依靠一圈金属，简单耐用但可能会导致应力损伤。LayerLAB为此设计了一个特殊的轮圈护套和手套，原理与越野自行车的握把纹理相似，目的是为了增加手和轮圈的推动边框之间的握力。

去年9月的伦敦设计节上，层次工作室曾在V&A博物馆陈列着中世纪挂毯的细长展厅中，展出了一件名为“铝箔”的动力雕塑。整个装置被20米长的乳胶条覆盖，上面点缀着5万个具有独立铰接的三角形金属箔片，构成波浪起伏的条带。

由一端的电机供电，箔片以平滑的波纹移动着，这是从中心轴的200个分支产生的运动。排列在装置两侧的定制LED照明系统对准金属箔片，将不断变化的斑驳光影投射到展厅墙面与天花板上，与墙上挂毯的中世纪图案不分彼此地重叠在一起。恒定的波纹运动散射着光线，改变了传统的画廊空间。如休伯特所说，整体效果有点像“把迪斯科舞会球灯的表面像橙子皮一样剥下来，然后平摊在地上”。

你可以把“铝箔”视为一件艺术装置，也可以当作德国剃须刀制造商博朗的一个广告工具。事实上，装置的三角形金属箔片原型来自博朗电动剃须刀箔中的微型穿孔结构，制作精良的德国产品中一个微小的工业设计被放大体量和规模之后，产生了截然不同的艺术与空间体验。

自动扣紧表带

LABB/Layer X Noomoon，2016年

“当瑞士创业公司Noomoon请我们为新一代智能手表设计一款表带时，我们决定把重点放在寻找有趣、创新的固定方法上。结果就是LABB，一款注塑成型的表带，不需要搭扣就能自动扣紧固定。

“LABB是由一种名为氟橡胶的柔软材料制作，表带上一系列纵横交错的洞孔构成它的图案，可以与相对应的紧固元件牢牢地锁定在一起，同时也易于解开。注塑胶带目前有五种混合搭配颜色。当它在Kickstarter推出后，获得的筹款比它的2万瑞士法郎目标超出了1.1万瑞士法郎。”

帐篷椅

Tent Chair/Layer X Moroso，2017年

“作为世界上第一把3D针织座椅，为意大利家具品牌Moroso设计的帐篷椅在今年的米兰设计周上首次发布。它是我们正在进行的智能纺织品结构研究内容之一，探索用新的数字针织技术来改进家具的制造过程。

“帐篷椅由一整块3D针织尼龙制作，总共5万米纱线和200万个结节组成无缝连接的椅盖，通过钢架上的导绳拉紧。100%可回收纱线制作的3D织物具有高强度和低密度，不仅符合使用者的身体形状，而且为身体提供必要的人体工程学支持。流水可以通过它的细小网格，这意味着椅子能够对应室内外各种不同的天气条件。”

配对椅

Pair Chair/Layer X Fritz Hansen，2016年

“我们用三年时间为弗里斯·汉森（Fritz Hansen）设计了一款模块化椅子，拥有超过8000种不同组合，我们也是第一家与这一丹麦著名家具品牌合作产品的英国设计机构。

“模压成型胶合板椅是弗里斯·汉森的DNA，我们采用这一根植于20世纪中期的设计语言，把具有流畅几何感的胶合板外壳与注塑聚碳酸酯靠背进行配对，形成新旧材料之间的张力。注塑靠背中包含10%的透明度，给它一种玻璃般的质感。

“除此之外，我们构思了一个智能、灵活的系统，允许用户创建自定制的产品。混合搭配的组件包括带扶手或者不带扶手的四种颜色靠背，五种色彩的底座可以选择雪橇形或四条金属椅腿，还有40种色彩的衬垫可选，这样的自定义系统可以满足市场越来越注重个性以及不同使用环境的需求。”

捐助盒

Charity Box Layer X Maggie's，2015年

“Maggie's是英国一家癌症慈善机构，是全英国十几万家这样的慈善机构之一。但是，常见的慈善捐助箱相当呆板乏味，也是一个从未被设计师考虑过的物品。

“我们想要设计一个温暖的、可识别的捐助盒，拥有平易近人的姿态。它是25厘米高的硅胶体，在颈部逐渐变细，顶端微微向前倾斜。这样的改进试图以更加受欢迎的形式鼓励人们的捐赠，似乎在无言中向对方表示感谢。通常志愿者需要托住捐助箱底部，包括了硬币的重量。现在我们把硅胶带放置在顶部，可以穿过佩戴者的手腕，这样也就减轻了它的载重。

“一个设计上的小小变化可以带来很大的差异，使用这个零钱捐赠箱之后，Maggie's收到的捐款总额比前一年增加了83%。”

英国新一代设计师的代表者本杰明·休伯特

我对设计的理解发生了转变

三联生活周刊：

当你在2015年把工作室的名字从本杰明·休伯特改为“层次Layer”时，是否意味着整个设计策略的改变？

休伯特：

是的，之前工作室是以我的名字命名的，我们做了些不错的项目，主要是家具。如今我们还做一些这样的工作，但是试图将重点转移到战略性工业设计上，因此需要改变日常工作的框架。我的名字只是表达了我的观点、风格和品位，当你为大型策略性项目工作时，这一点不再重要。“层次”是比本杰明·休伯特公司更加民主的平台，可以说我的个人作用更像是一个连接器，而不是独断的人。

三联生活周刊：

重组工作室之后，你们是如何做出接受或不接受某项设计委托决定的？

休伯特：

通常我们会寻找某个挑战，它必须有新增加的价值。我们可以改进之前的产品、开拓新的市场机会或者帮助成长中的企业等等，让我们的设计技能有所作为。在改名为“层次”之前，我们总共6个人，现在已经发展到20多人，其中一个主要原因就是我们引入了超越风格和品味的新价值观，产品面向更多的人，提高生活质量的同时不损害自然环境，这对年轻设计师很有吸引力。另外我们也越来越重视应用程序、软件、网站等数字产品设计，与需要制造、出售、使用和再循环利用的硬件产品相比，软件的碳足迹要小得多，它们只是被删除和替换。

三联生活周刊：

这些软件产品的设计关注点有什么不同？

休伯特：

从创作过程来说，硬件和软件产品都有原始开发的阶段，你要快速做出原型，不断地改进和测试。在我们工作室，数字设计师是和硬件设计师密切合作的，需要打破物理和数字的不同分层。事实上，这也是“层次”这一名字的由来，我们把世界视为各种不同的分层，其中包括人群、社区、文化、价值、观念等，它们代表了我们生活其中的这个极其密集复杂的世界。

三联生活周刊：

你们设计的3D打印轮椅会尽快投放市场吗？

休伯特：

我们已经在这个项目上工作了两年多，我想它是一个5到7年之后颇具竞争力的产品。目前3D打印由于专利因素成本高昂，很多材料都涉及到专利。当这些专利开始到期之后，价格将会下滑，在市场上更具竞争力。目前GO轮椅还在原型阶段，面临国家卫生服务部门的审核等，但是已经有医疗机构和轮椅厂商对这一设计颇感兴趣。

三联生活周刊：

你们近几年的改变是否也代表了欧洲设计领域正在发生的变化？

休伯特：

我不知道，我觉得对我来说更多是一种责任，设计师有责任切实地使用他们拥有的技能和工具。如今，我们生活中越来越多的东西在受到质疑，比如过度生产、过度风格、过度个性等，所以我对设计的理解发生了转变。但我认为仍然有许多欧洲设计师并没有回应社会的变化，设计一把漂亮的椅子总是更容易找到发布平台。假设你对他们说，未来两年你不允许设计一件家具，而是要去解决某个世界问题，我很想知道，如果这样重新聚集资源的话将会发生什么。

三联生活周刊：

在你看来，年轻的设计师应该怎样更好地构建自己的职业生涯？

休伯特：

我是2007年开始成为设计师的，说实话从那以后世界发生了很大变化。借助新的工具，像Kickstarter或Indiegogo这样的众筹平台，年轻设计师可以迅速向前突破。但另一方面，世界变得更复杂了，如今的设计过程不可能是单一维度的，设计师必须了解所有不同的驱动因素，产品必须在复杂的广告和传媒世界中竞争。我认为设计师应该更多考虑他们正在开展的产品类型，专注于那些可能在人们生活中产生更多影响的产品，而不仅仅是好看的东西。

三联生活周刊：

你们工作室有众筹的项目吗？

休伯特：

我们是使用Thunderclap平台与人们沟通，激发公众的兴趣。支持者们不必提供资金，而是通过他们的社交媒体帮助传播新项目的内容，让更多的人谈论它，创造足够的声势之后再把它带到投资的层面上。比如荷兰设计师戴夫·哈肯斯的模块手机Phonebloks项目就用Thunderclap引起了很多人的兴趣，其实它不是一个已经准备好的产品，但是有3亿人使用他们的社会影响力在谈论它。

三联生活周刊：

所以，众筹切实地改变着设计产品从概念到市场的传统进程？

休伯特：

我认为众筹绝对是时代的产物，人群的力量驱动着新的产品和服务，将想法推向市场的能力被民主化了。地点和本地的基础设施不再是设计师成功的决定因素了，重要的是好的想法。不过，现在产生一个原创的、独特的想法是很困难的，大部分事情都已经被人以这种或那种方式做过了。
