# 话题_《十三个原因》：美式校园青春的哀伤与冲击

《十三个原因》：美式校园青春的哀伤与冲击

话题

《十三个原因》：美式校园青春的哀伤与冲击

文◎悦涵

美剧《十三个原因》剧照

近来青春剧似乎呈现出一个新趋势，揭露种种校园阴暗面。

4月10日，美国加州小学发生枪击事件，一名教师和一名8岁的小学生被击身亡，枪手在行凶后自杀。

一直以来，美国校园枪击事件频发是一个严重、众所周知的社会问题。这一问题背后，涉及美国学生心理以及学校管理层面的种种问题，还有流行文化对于青少年群体的非健康影响。Netflix在3月底播出的一部新剧《十三个原因》（Thirteen Reasons Why），一周内就在推特上获得了比Netflix以往任何剧更多的转发率。该剧探讨校园霸凌对个体产生的影响，大胆揭露美式校园种种阴暗面。可以说，《十三个原因》以一种悲剧的浓墨重彩，探讨种种校园问题。

剧集开头以一个美丽女高中生汉娜的自杀作为开端。自杀消息传遍校园后，所有人都在为她哀悼。她的相片被悬挂在走廊里她的置物柜上，同学们自发为她组织各种纪念活动，在她柜子前献花。这些人里最为难受的是一个叫格雷（Clay Jensen）的男生，他一直对她有小小的暗恋，怎么都无法相信这样一个如花生命会如此结束自己。

是日晚间，他收到了一个神秘的盒子。13盘老式磁带，被精心放置在一团裹着的彩纸中间。格雷辗转找来设备，开始播放磁带。刹那间，死去汉娜的声音在他的地下车库内响起。原来，汉娜在自杀之前录了13盘磁带，每盘磁带围绕一个主题、一个人物，分别叙述一个她自杀的理由。并且她言明，磁带上的13个主角，每一个，都为她自杀负有不可推卸的责任。

故事由此展开了。

一直以来，负面化的青春期问题是美国影视剧领域不曾错过的主题。这一主题，随着真实社会校园枪击案、霸凌案的不断升温而成为流行文化的真正聚焦。2003年的校园枪击案经典之作《大象》、探讨校园强奸和学生等级制度的《美眉校探》、BBC心理学暗灰力作《我们需要谈谈凯文》（We Need to Talk About Kevin）、阴沉美女代表《美丽坏宝贝》，还有前些年颇受欢迎的美剧《美少女的谎言》，甚至在非青春类题材的“美恐”第一季，都有一条完整的故事线叙述校园枪击案以及这种现象背后的青少年心理。

“青春期的痛苦”也是文学领域热衷刻画的主题。《少年维特之烦恼》中无法实现的恋爱，《麦田里的守望者》那种迷惘和呐喊，《小妇人》中令人心痛的成长瞬间，萨冈的《你好，忧愁》更是从书名就直指人生这一特殊阶段的痛苦。英国皇家精神病研究学院的一篇文章指出，青少年时期，个体面临来自各方的冲击，很多人会觉得青春期是人生最难渡过的阶段，是因为“快速的心理发展和深度的情绪变化”都恰好发生在这一时期，青春期的荷尔蒙变化会导致喜怒无常、焦躁不安的情绪产生。

这种情绪上的多变性，使得青春期题材故事变得非常适合文艺性改编，因为它本身就足够跌宕起伏、具有戏剧张力。

《十三个原因》以少女自杀事件为导火索，探讨霸凌、过度早熟、对少数族群的排挤隔膜、学校心理咨询制度的粗略……相较于以往类似作品，《十三个原因》的进步或许就在于它的大胆、极端。每一集都是站在批判的角度，大声声讨学校对霸凌事件的视而不见、学校心理医生对一个面临精神崩溃的学生是如何草草了事。在没有看结局之前，你或许会觉得《十三个原因》夸大了人性中“恶”的一面，因为每个人似乎都是那么自私、邪恶，只会伤害汉娜、保全自己。而最后一集却揭示出有时一个人的主观会让自己对于这个世界的判断产生多么大的偏差，很多汉娜最心寒的瞬间，有时却是她自己的误会。美国最受青少年欢迎的杂志《少年时尚》（Teen Vogue）总结，整部电视剧最大的警示是：校园中的很多问题，只要参与者当时更大声一些、更努力一些、对身边的人更关心一些，很多悲剧，最终可以避免。这部剧不仅从正面批判了校园霸凌，也暗示有时忽视别人，甚至懦弱导致的逃避，最终都可能引发悲剧。

除此之外，《十三个原因》继续探讨社交媒体的过度泛滥对校园生活的负面影响。剧集中，汉娜刚死，就有两个女生在她的纪念鲜花前拍照，格雷感到愤怒，因为这两个女生平时甚至根本没和汉娜说过话。“上瘾型更新社交媒体状态”，使新世代的年轻人更冷漠、麻木、以自我为中心。

第一盘磁带，刚转学过来的汉娜和学校最帅的帅哥贾斯汀约会，地点在一处街角公园的小游乐场。当时快乐、阳光、对生活充满希望的汉娜以为这是她美好爱情的开端，他们开心地从滑梯上滑下，她奉上了自己的初吻。

谁知第二天，贾斯汀将拍到她滑滑梯时裙子走光的照片群发给学校中的每一个人。一瞬间，仅仅是一瞬间，汉娜的生活开始由天堂走向地狱。所有人都以为她是一个放浪、随便的女生，男生开始轻贱她，女生不喜欢她、嫉妒她、排挤她。美国《现在17岁》网站指出，社交媒体的过度便捷，使得“恶者行恶更加容易”。校园霸凌演变出更轻易的“网络霸凌”，不需身强力壮，任何人，都可以不费吹灰之力毁了别人的生活。

除预计中的青少年观众群体，《十三个原因》更引起了成年群体的关注。“这是一部学校老师应该看的剧。”《华盛顿邮报》这样认为。英国《每日电讯报》撰文说这部剧也尤其推荐给青少年的父母观看，思考怎样走入孩子们的世界。因为，磁带中虽没有明确指出，但汉娜自杀的一个间接原因是她父母因为生意上的事焦头烂额，已经很久没有跟她真正交流过。

不过，也有不少外媒认为，这是一部自私、自以为是的剧。剧中汉娜在死后将13盘磁带寄给13个同学并要求他们每个人必须听下去，这种行为本身就很“自私、自负”（《华盛顿邮报》）。剧中关于割腕自杀场景过于真实的刻画，也是该剧饱受争议的一个点。《纽约时报》采访的一个心理学家认为，该剧是教导如何避免自杀，但仍间接提示了如何自杀。

相反地，英国媒体却对这部剧评价不低，或许是因为早在2007年，该国就出现了阴暗青春剧的先锋作《皮囊》（Skins）。这部剧当时在英国青春剧领域形成了一股绝大的风潮，其受欢迎程度一直持续至今。该剧主创布莱恩·艾斯利（Bryan Elsley）被誉为英式青春剧教父，目前也正在和Netflix合作一部新的校园剧《先吻我》（Kiss Me First）。

《每日电讯报》认为，《十三个原因》在2017年引领了青少年影剧的一个新风潮，超越了吸血鬼系列的肤浅莫名，和一些“过于裹上糖衣”的甜美之作，开始关注更深层、更严肃的问题。《卫报》也认为《十三个原因》丰富了之前阴暗青春剧的小众流派，并达到了一种新高度。过往力作《皮囊》的尝试更具美学性，它的很多镜头、故事线、人物设置、台词，有时更像一件艺术品，具有独特的英式审美趣味.而且，《皮囊》的整个观剧过程是愉悦的，可《十三个原因》的观赏过程却“不那么让人舒适”——从倒叙角度讲述一个少女的自杀，剧集越深入，观众的情绪就越惋惜、越愤怒。

拍摄《十三个原因》的主演也时常觉得一种情绪的过于紧绷。饰演格雷的迪兰·米内特（Dylan Minnette）坦言自己拍摄时，一件比较幸运的事是家离得近，可以经常回家休息，“逃离24小时”，和家人在一起，缓解深陷剧中的负面情绪。但饰演汉娜的凯瑟琳·朗福德（Katherine Langford）就没有那么幸运了，她的家不在加州，于是只能回到自己租住的公寓中，没有家人在旁边。“我搬进公寓里的第一件事就是为自己买一架钢琴。”凯瑟琳·朗福德在一次采访中说，“它成为我拍摄《十三个原因》时缓解压力的一个重要道具。我经常一回公寓，就开始不停地弹奏。”

2016年以后的青春剧，开始出现一些新特点。如今，在西方，2000年以后出生的人被称为“Z世代”（Generation Z），现在众多的校园青春剧，事实上都是为他们所拍摄的。除去《十三个原因》，BBC 今年推出的《女校风波》（Clique）、2017年美剧《河谷镇》（Riverdale）、2016年的《甜蜜邪恶》（Sweet/Vicious），这些新一波青春剧的一个共同特点就是阴暗性。这一方面，正如《卫报》所说，青少年影视剧开始显露出前所未有的真实。另一方面，当丑陋被过度揭露，这对于正在成长中的心智，是否是一件好事？现在的青少年正比以往任何一代更加接触现实，但似乎，一种孩子特有的享受纯真的时期，在全球范围内被流行文化过度缩短了。

“如今，成人剧情类电视剧反而大多倾向于逃离现实——过于迷恋犯罪类、历史类题材，这是一种对于如今这个危机时代的一种典型应对。相反的，青少年电视剧却显现出一种前所未有的真实。”布莱恩·艾斯利说。

《卫报》的汉娜·戴维斯（Hannah Davies）认为，如今这些青春剧在揭露阴暗面的同时又不引起青少年过度郁闷之间，一直在绞尽脑汁做着努力。主创们试图将真正的现实展现出来，又不忘加入一点点理想主义色彩：世界还是有很多美好的东西存在。

《河谷镇》的主创罗伯托·阿吉雷－萨卡萨（Roberto Aguirre-Sacasa）说：“在剧本中我时常提及《大象》这部电影，是我个人的一个偏爱。但是尽管如此，我还是难以想象将《大象》中惨烈的结局用于《河谷镇》的人物。那实在是太悲惨了。”

当然，这些揭露阴暗面的剧也有不时闪现的积极色彩。《女校风波》于此的聚焦是“女权主义”。布莱恩·艾斯利介绍：“如今，在大学，女生们似乎已经忘了上世纪六七十年代女性为之争取的女权主义。校园强奸在如今的年轻群体中被过度弱化或忽视了，大多数女孩遭遇此类事件后，不会为自己抗争。”

在跨越了一个大西洋的美国，《甜蜜邪恶》同样聚焦校园强奸，女主角用暴力手段报复那些强奸女生的男性。该剧主创詹妮弗·凯汀·罗宾森（Jennifer Kaytin Robinson）坦言，如今，随着特朗普执政，美国文化中一直存在的“保护男孩”父权制思想，将被进一步煽动、加浓。因此，现在有必要在自己的作品中，积极发射出抵制的讯号。

《十三个原因》有一个非常微妙的结尾。在剧集中偷窥汉娜，拍到她和另一女生亲密照片并使之流传的校报摄影师泰勒，他平时在学校也是饱受欺凌的个体。结尾处镜头掠过他已经偷偷购置了一箱重型武器，并将平时霸凌他的人的照片，一张张洗出来。似乎下一场校园枪击案，已经迫在眉睫。这也让很多剧迷猜想这是否会是第二季的内容。

美式校园问题，之前虽已有无数影视剧进行探讨，但似乎仍在不断发生。无论如何，《十三个原因》将此类事件再次置于聚光灯下，从现实意义来讲是一件好事。正如剧中讲述的那样，如何让悲剧不再发生？从源头杜绝，才是最好的方式。
