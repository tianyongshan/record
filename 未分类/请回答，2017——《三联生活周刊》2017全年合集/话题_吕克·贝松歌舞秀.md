# 话题_吕克·贝松歌舞秀

话题：吕克·贝松歌舞秀

话题

吕克·贝松歌舞秀

记者◎驳静

科幻电影《星际特工：千星之城》剧照

这位爱拍科幻片的法国人，让人记住的却总是他电影里的歌舞桥段。

15年前，麦当娜盛极之时，吕克·贝松曾邀请她参与他当时想制作的一部音乐剧，麦当娜一看300多页的剧本就拒绝了他。若干年后，导演的追索对象变成了蕾哈娜。

《星际特工：千星之城》算得上今年入夏以来视效与奇观结合得最出色的一部科幻电影。故事发生在28世纪一个叫作阿尔法城的地方——戈达尔曾拍过一部《阿尔法城》，这个地理概念几乎代表着对未来世界高度乌托邦化的一个最美好的期待，有上千个星球的种族与文化在这里共处。

对科幻迷来说，吕克·贝松早在《第五元素》就赢得了他们的心。不过在这次奇观特质浓厚的新片里，最让人在意的不是科幻，而是蕾哈娜的变装舞片段。美国这位超级流行歌手，在该片里扮演的是位叫Bubble（泡泡）的外星人，她的能力是可以任意变幻外形。所以她第一次现身，就是在男主角无意闯入的一家红灯区夜总会里。也正是在这家复古舞厅里，上演了这场颇具经典流传气质的变装秀片段。

片中夜总会的整体氛围设计就是活脱脱的“芝加哥”，它把这部百老汇经典剧目里的张扬而神秘的气氛学了个全。舞台不大，也就够蕾哈娜一个人在上面折腾。灯光亮起，出现的便是黑衣黑帽深色口红的蕾哈娜。这是丽莎·明奈利（Liza Minnelli）在上世纪70年代歌舞片《歌厅》（Cabaret）里的经典造型：黑裙、黑色绕颈露背上衣、超短裤、过膝靴，以及最重要的硬礼帽，设计师添加了刺绣和亮片以增加现代感。

之后还有诸如跳着钢管舞的护士、耍着跳绳的啦啦队员、女仆、猫女，都是万圣节变装派对里的常见形象。镶嵌在充满宇宙想象力奇观的科幻电影里，这段舞台表演就因为复古而出挑。并且，正是因为复古与28世纪这个设定形成对立，再加上夜总会氛围里的某种情欲火花，交织出一种人类荷尔蒙，给这部科幻太空片添了一笔“烟火气”。

当然，少不了梦露。蕾哈娜下了舞台，一眨眼就以梦露金色卷曲头发的形象出现在特工面前。而模仿梦露的桥段，早在1994年吕克·贝松赖以成名的《这个杀手不太冷》中已经出现过一次。虽然娜塔莉·波特曼扮演的玛蒂尔德与杀手莱昂的“闺阁游戏”看上去并不是什么正经的舞台表演，变装和歌舞这两项元素，却让这个小小的房间活色生香起来。20多年后看到高级繁复版的夜总会变装，仍不由得想起大叔萝莉对手戏里的浪漫。该片拍摄时，波特曼才12岁，稚气未脱地模仿麦当娜，唱的是《宛若处女》（Like A Virgin）。从门后再出来，又穿上白色露肩绕颈连衣裙，俯下腰，抬起臀部，唱《生日快乐》，不用想，这是梦露了.接着，还有卓别林.最后模仿的是《雨中曲》里的吉恩·凯利（Gene Kelly）。

到了《星际特工》里的泡泡，除了天赋异禀可以变幻外形，也自称从4岁就开始练习变装这项技能。在欧美变装文化里，变装者有时候可以以艺术家的身份存在。泡泡熟知莎士比亚和莫里哀，能应景地引用兰波和拜伦，从这个角度，这大概是变装世界里的终极艺术家。

表演者蕾哈娜本人，在现实世界里也是某种程度上的“终极艺术家”，毕竟在流行歌曲这个领域内，多次蝉联格莱美的蕾哈娜说实在的已经到达名气的顶峰。也正因如此，吕克·贝松提出要请她来试镜时，制作人的第一反应是：“吕克，你的第二人选是谁呀？”毕竟蕾哈娜此前与电影的交集总共才两次。但吕克·贝松不作第二人选，毕竟，这位美国流行歌手这几年来最为人所知的已逐渐不是她的音乐，而是她的造型。所以当这个选角消息一出来，网上粉丝的反应是某种理所当然，这个角色跟蕾哈娜的公众形象无疑相得益彰。实际上，从2014年休整回归后，蕾哈娜（中国网友爱叫她“日日”）在造型方面就更加用了心。无论是美国时装设计师协会（CFDA）颁奖典礼上的施华洛世奇水晶透视装，还是2015年Met Gala上那一袭“煎蛋饼”，在敢穿这件事上，蕾哈娜是歌手里面的翘楚，“整个世界就是她的舞台”。

同样堪称翘楚的还有吕克·贝松的想象力。对法国人来说，如今50多岁的吕克·贝松大概都长成一个符号了：艺术电影的叛逃者。另一方面，法国人又默默对这个符号抱以极大期许，于是又有了他身上的另一个符号：法国商业电影的代言人。所以围绕他的新作《星际特工：千星之城》展开的讨论，多半集中在这部电影的制作成本上，以及他能收得回成本吗？1.8亿美元的制作成本使它成为法国乃至欧洲历史上最贵的电影，而更重要的是，它是游离在好莱坞制片厂体系外却又是制作如此庞大的电影。

跟英国人诺兰一样，吕克·贝松自己拍电影，妻子作制片。不同的是，从1999年成立EuropaCorp公司后，吕克·贝松的电影基本都是通过该公司“独立”制作，这也是导演直到两年前才着手开拍这部高成本科幻电影的原因之一。电影故事缘自影响了法国好几代人的漫画《瓦莱里安与劳拉林》（Valérian et Laureline），漫画作者梅齐耶尔（Jean-Claude Mézières）曾经参与设计吕克·贝松的《第五元素》，这部电影几乎代表了科幻电影实景拍摄时代的末章，《第五元素》以后，CG（视觉特效）时代就来了。

梅齐耶尔开始跟吕克·贝松合作后，就时常问他，什么时候拍“瓦莱里安”。导演说这是他从10岁就开始追的漫画，一度完成了一稿剧本，那是在2007年左右。之后卡梅隆邀请吕克·贝松去《阿凡达》拍摄现场探班，这为吕克·贝松决定了两件事，一是自己的剧本太烂需要毁掉，二是原本以为不可能完成的视觉特效正在逐渐成为可能，因为“摄影机可以运动后拍摄维度一下广了”。最终完成的《星际特工：千星之城》有多达2700个视效镜头——以“视效驱动电影”为宣传口号之一的《魔兽》，其视效镜头数目不过2000个左右，而当年的《第五元素》，这类镜头才200个。即便这200个镜头，当年也让吕克·贝松苦不堪言，所以尽管影迷对《第五元素》的评价是“又将科幻电影提升了一个台阶”，当被问及是否会怀念当年拍摄科幻电影时传统的“现场特效”时，吕克·贝松的态度很鲜明，他说自己是以欣喜的态度“站在科技发展的延伸线上”。

有意思的是，无论宣传片中的科幻脑洞如何，观众的嗨点却有点不跟拍。外星女伶演唱著名歌剧《拉美莫尔的露琪亚》，并穿插着女主角打斗回忆的这场戏，是《第五元素》最为人津津乐道的经典片段。如今“蕾哈娜变装秀”又是《星际特工》的议论重点，至少对非科幻迷来说是如此。吕克·贝松这位爱拍科幻片的法国人，大概终究还是没能“隐藏好”他的艺术基因。
