# 话题_让他们娱乐我们

让他们娱乐我们

话题

让他们娱乐我们

文◎范庭略

几年前我们出版过一本很有趣的英国小说，叫作《如何众叛友离》，当时根据国外出版社发过来的新书梗概，让我最感兴趣的一点其实就是《穿PRADA的女魔头》的作者是看了这本书之后才写出了自己的这本处女作。众所周知，女魔头的小说不但大卖，而且电影也屡获好评。其实英国人写的这本书说的并不是时装天桥上的霓裳风云，而是根据这位英国作家自己在纽约《名利场》杂志所遭遇到的各种感想。原本伦敦的文艺工作者对时尚大都会纽约的东岸知识分子就抱着一肚子的偏见，再加上作为好莱坞的好朋友以及老朋友，美国《名利场》杂志可以一直50年做奥斯卡颁奖典礼的after party的主办方，显然这本杂志也算是好莱坞娱乐行业的一个中坚力量吧！中国互联网在十几年前开始大热的时候，一个著名的国内城市生活网站就把《名利场》里面关于互联网行业的权利排行榜的群像给扫描下来，自说自话地翻译成中文印成了一本企业内刊，当然《名利场》里面的美国互联网大神几乎悉数出镜，看着那本印制精美的中国互联网企业内刊，我知道美国《名利场》杂志是不可能在中国出版的，因为这不是新钱和老钱的区别，那是乡绅和互联网新贵的区别。后来无数的国内社交娱乐杂志都信心满满地要做中国版的《名利场》杂志，也是跨页的群像，也是高端生活的细节，描述也是谈人生谈理想喝鸡汤，但是十几年过去了，当时的somebody都变成了后来的nobody，各路本土名利场都尘归尘土归土，所有的八卦娱乐新闻都归了手机客户端，连我最爱看的某大明白的公众号都有了孟山都的广告赞助，这不亚于前门楼子和大萝卜头子的差异。20年前戴安娜王妃猛然离世，完全是因为狗仔队的偷拍而命丧巴黎，那个夺命隧道里的奔驰汽车残骸和范思哲遇刺以及香港回归等重大事件成了那一年难以忘记的时光记忆。这么多年过去之后，每次当朋友圈里面出现怒转各种明星出轨劈腿图片的时候，我都会想起美国的《名利场》杂志，虽然我已经很久很久不看各种电视直播的综艺歌唱比赛了。媒体爱挑事儿，娱乐明星要上位，吃瓜群众要看戏，我们的生活还要继续，这一切本身都是鲤鱼跳龙门的过程，跳得过就跳，跳不过就只好慢慢蹦跶了。

名人一直都有，但是古往今来的各种名人要么是歌唱得好，要么是小说写得好，或者要么是戏拍得好，名人也是因为才艺出众而有名而变得众人皆知，但是互联网时代的摧枯拉朽让各种没有任何名作的普通人也是忽如一日梨花开一样地变成了路人皆知的网红。其实在柯达相机关门的时候，我还庆幸数码相机可以躲过一劫，但是到了连数码相机也要砸钱去拼命做市场推广的时候，我才明白不是胶卷相机败给了数码相机，也不是数码相机得了天下，而是我们的生活都被智能手机占领了，正所谓名人不是人而是有图有真相，对那些当红的名人来说，无论是好事还是坏事，无论是有婚姻没有婚姻，无论是有孩子还是没有孩子，也无论是有广告还是没有广告，有偷拍还是没有偷拍，任何新闻，无论好事还是坏事，都是娱乐明星的好新闻，见不见报、上不上新闻不重要，最重要的是上头条，只要有了头条，就好像著名的克里斯托弗·希钦斯（Christopher Hitchens）说的一样：“今天的美国，再无毁誉，更无臭名远扬，只有名气！”什么叫名气？就是英文单词Celebrity啊！你说夫妇两个可以向公众隐瞒离婚多年的事实，你也可以说关你屁事。但是谁让你做名人代言呢？粤语里面的名言叫“吃得咸鱼抵得渴”，这不就是看香港电影《古惑仔》长大的一代人最熟悉的一句名言——“出来混总要还的”嘛！名人去召开记者招待会、去访问、去假装被狗仔队偷拍，这些都是假装的事件。在电视镜头前面的滔滔不绝、能言善辩以及口若悬河是不是一个好的政治家的标准呢？也许在尼克松与肯尼迪辩论的时候可以说是尼克松倒霉，但是时代都已经发展到特朗普这样用“英语九百句”的前三百句来演讲的时代，你还可以说这是一个假装当选的时代吗？50年之后，电视加上手机视频的传播速度，不但让娱乐八卦成为传播最快的视频，也让政治选举成了一场大秀。微博谴责、微博发律师函、微博捉奸、微博告白，如果没有微博，公关公司的营业额是不是还应该有不少的增长呢！美国的科学家甚至都嘲笑坐在轮椅上的英国科学家霍金，他们觉得不要太理会霍金说什么，因为他已经是名人了，而不是基本粒子理论的专家了。

娱乐新闻做到今天真的是挺可悲的，看看那些电视台的烂招，观众们就要用娱乐的心态来看待节目的发展，泛娱乐的一种节目高潮的安排，其实说到底就是电视这种媒体的娱乐精神所在：那就是让这些明星娱乐我们这些电视观众，但是你千万不可以当真，必须不能比演员还要入戏，因为一认真你就输了。娱乐明星通过各种各样的娱乐形式来取悦观众，以此增加知名度以及关注度，然后向广告商兑现，换取广告的代言机会、娱乐头条、锦衣玉食以及等等围观的产品。各种耸人听闻的标题，死了、撕了、豁了，这些娱乐圈的荒唐乱象，不正是一个众人狂欢的场景纪实吗？

这个时候我不由想到十几年前我供职的那份橘黄色的娱乐报纸上，沈公子写在报头的广告语，也是最让我难忘的神来之笔：“让他们娱乐我们。”这其实就是娱乐新闻最大的社会效益。
