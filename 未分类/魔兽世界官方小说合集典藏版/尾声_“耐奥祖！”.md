# 尾声_“耐奥祖！”

未知

尾声

“耐奥祖！”

听到有人呼唤他的名字，兽人萨满、部落大酋长大喊了一声，睁开了眼睛，目光闪烁着。他的感官马上就被包围在他身周的一种奇怪的旋涡状的虚无侵袭了，逼得他差点儿发疯。他紧紧闭上了眼睛，试图驱散这种混乱感。紧接着，在敲击、号叫和爆裂声中，他又听到了。

“耐奥祖！”

他眨了眨眼睛，向四周看去。在很近的地方——或者只是看起来很近，因为过了一会儿他又发誓说在几里外的远处——耐奥祖看见了一个黑影。从身形上看，像是一个兽人，他仔细端详了一阵，确认了这点，绿色的皮肤，长着獠牙，还有长长的辫子。绝对是一个兽人，耐奥祖认出来了，那是影月氏族里的一名战士。那个兽人一动不动，尽管耐奥祖认为他看到了那个兽人的胸部在上下起伏，但在这个鬼地方，他对任何事都没有十足的把握。

其他的人影也散布在这个由光和影组成的奇怪旋涡中。凡是跟着他穿过裂隙的人，好像都在这里。

问题是，这是什么地方？为什么裂隙没有把他们带到另一个世界呢？不管这个地方是哪里，耐奥祖可以肯定，这不是一个正常的世界。发生了什么事？为什么其他的人都困在睡梦中，而他却醒着，意识清晰？

一束光从他身边扫过，那一瞬间，耐奥祖看到周围每个兽人的身上都反射出了微弱的光芒——包括他自己。他睁大了眼睛，然后又迅速闭上，刚刚看到的景象对他造成了强烈的冲击。他明白自己看到了什么。他们被困住了——什么东西把他们绑在了这里！

“耐奥祖！”他的名字又一次在这个陌生的虚空中飘荡，但这一次，耐奥祖感到有什么东西在拽着他的胸口和四肢。而其他的兽人在迅速后退，或许只是他在移动，而他们被锁在原地——到底是怎样，在这里不可能分辨得出来。几分钟后，周围只剩下耐奥祖一个人，其他的兽人变成了遥远的身影。

然后，一个巨大的、深色的阴影逼近了他，他抬起头来——

一张愤怒的脸。

在耐奥祖面前的是一个巨大的生物，他身上披挂着血红色的金属战甲。这个人的脸长得很像德莱尼人，看上去充满智慧，但却有着鲜红色的皮肤和恶魔的面孔。两只弯弯的短角从他高耸的额角两边伸出来，他下巴上留着短须，还有两条触须一般的东西垂落下去。几枚耳环闪耀着，他的眼睛里闪烁着深黄色的光。

耐奥祖立刻认出了他。

“大人！”耐奥祖喘着粗气，挣扎着鞠了一躬，尽管他的四肢仍被束缚着。

“哦，耐奥祖，你这个不忠的小仆人。”燃烧军团的恶魔领主基尔加丹回答道，“你以为我忘记你了吗？”

“不，大人，当然不是。”但事实上，这确实是耐奥祖所希望的，在最初的几年过去之后，他开始以为基尔加丹真的把自己遗忘了。但现在，他的心沉了下去。

“哦，我可一直都在密切地关注着你，耐奥祖。”基尔加丹向他确认道，“你自己心里清楚，你浪费了我太多的东西。”恶魔领主笑了，发出一种令人恐惧的、刺耳的声音，“现在，你要为你的失败付出代价了！”

“我——”耐奥祖准备辩解，但他的大脑一片空白。

“你这是弄巧成拙。”基尔加丹替他把话说完，“我知道，你终究会再次去尝试施法，但是，这种法术你根本没有掌握，也没能理解。我就等着，我就知道总有一天，你的傲慢会把你带到我面前的。”他张开了戴着手套的双手，眼睛眯成了一条缝，“现在，我们都在这里了！你梦见了死亡。你想躲开它。但是现在，我的小木偶，死亡是你唯一的选择。”

简短的画面在耐奥祖的脑中闪过：血肉被从他身上生生撕下的极致痛楚.死去的人们围绕着他，他手上沾满他们的血，而他自己的血则溅在他们身上，这是一种由死亡、生命和极度的痛苦组成的恐怖联合。

“不！”耐奥祖喊道，翻来滚去，试图通过一切办法从看不见的束缚中挣脱，“我的人民还需要我！”

恶魔大笑起来，他那巨大的身躯随之颤抖，发出一阵恐怖的、怪异的声音，令耐奥祖的心脏开始痉挛。

“我很清楚，对于你来说，他们无足轻重。所以不要担心。”恶魔领主轻声说道，用一根长长的手指戳进了耐奥祖的脸颊，将钻心的酷热与剧痛送入耐奥祖体内。“你救不了他们。你还不明白吗？小木偶，你连自己都救不了。”

然后他扭动手指，张开了手掌，覆盖了耐奥祖的脸，兽人萨满的脑袋向后仰去，颤抖的双唇间爆发出一声惨厉而恐怖的尖叫。

他知道，这只是噩梦的开始。
