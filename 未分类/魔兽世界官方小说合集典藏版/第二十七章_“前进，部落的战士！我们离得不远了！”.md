# 第二十七章_“前进，部落的战士！我们离得不远了！”

未知

第二十七章

“前进，部落的战士！我们离得不远了！”

格罗玛什·地狱咆哮的声音穿透了战场的喧嚣，听到的人都振奋起来。雷克萨旋转着，左手的战斧划过一个联盟战士的脖子，右手的战斧砍下来，把另一个战士从肩膀劈到了腰。在他的身旁，他的战狼哈拉萨咆哮着冲了出去，巨大的嘴咬住了另一个战士的前臂。雷克萨听到了牙齿咬裂骨头的那种特别的嘎吱声，那人大叫起来，剑从手中掉落。哈拉萨松开了那只受伤的手臂，以闪电般的速度跳了起来，咬紧了那人的喉咙。他们是一对致命的组合。

雷克萨望向一边，看到格罗玛什·地狱咆哮，战歌氏族的酋长，正挥舞着尖啸着的血吼不断把敌人劈开。战歌的勇士们与他们的领袖并肩作战，他们的吟唱和战斗的呐喊混合在一起，组成了死亡和毁灭的诡异旋律。雷克萨是少数不属于这个氏族的人之一，但这对他来说再寻常不过。他没有真正的家族。至少，他的族人，莫克纳萨人，没有一个加入部落，一直固执地保持着独立。他们的数量很少，一直生活得很艰难，他们把主要的精力放在守住他们位于刀锋山的那片传统的土地上，保护它不受食人魔的侵犯。雷克萨曾试图告诉他的父亲莱欧洛克斯，兽人们正在建造黑暗之门.他认为这是为四面楚歌的莫克纳萨人找到一个新的世界的机会。但是莱欧洛克斯看到的只是他的儿子没有留在他出生的地方、为保卫家园而战斗。他们的目标都是帮助他们的人民.最后，他的选择被否定了。雷克萨是跟着部落走了，现在，部落是他唯一的家。

然而，他又总是与众不同。

又一个人类倒了下去。雷克萨抬起头看了一眼，他的身高让他能够居高临下地看到其他战士。格罗玛什是对的——他们离黑暗之门不远了。在他和他的故乡之间，大约有一百个多个人类。雷克萨咧嘴一笑，举起两把斧子，准备进行一场屠杀。

在过去的几个月里，幸运的天平总是在联盟与部落间摇摆不定。最开始的时候，有很短的一段时间，联盟把他们围在和这个山谷相邻的一个小山谷里，但对方的困阻并未持续太久。人类战士低估了被逼入绝境的兽人的意志，也低估了他们作战的凶猛程度，格罗玛什带领着他的人马冲杀出来，获得了自由。他们在北方一个名叫斯通纳德的地方重新聚集起来。这是部落最初穿越黑暗之门后，建立的第一个哨所。那片沼泽虽然充满了恶臭，令人不快，但至少有生命和水源，格罗玛什率领兽人们摆脱了绝望。他们建立了斯通纳德营地，并通过突袭抢夺了联盟的供给物资，对其进行扩建，最终重新控制了传送门。

后来，部落和联盟对这里展开了往复争夺。但是现在，这个小游戏结束了。格罗玛什已经决定，是时候该回去了。一直到现在，都没有其他氏族过来援助他们。而且，尽管联盟已经意识到，他们仍然是一支不可小觑的战斗力量，但他们的数量正在持续减少，相反，联盟的人数却在渐渐增多。同时，还有那个关于奇怪装置的谜团。术士们试图激活它。他们告诉格罗玛什，这将产生一个护盾，以保护他们免受攻击，让他们更容易守护黑暗之门。但实际上那东西的设计初衷就是要毁灭，根本不是为了保护。有人打算抛弃他们，格罗玛什·地狱咆哮不会让他的人民因为别人的背叛而牺牲。当格罗玛什准备回去和策划这个阴谋的人当面对质的时候，雷克萨选择留在他的身边，

一个人类骑着马向他冲来，他高高地举起剑，盾牌横在他身前，但那个战士没有考虑到雷克萨的身高。雷克萨用一把斧头重重地砸在盾牌上，把它砸进那个人的身体里，同时用另一把斧头把剑荡开。当骑手在马鞍上摇晃时，雷克萨举起两把斧头，让那人径自被斧刃穿透。雷克萨咧嘴一笑，发出一声凶狠的战争怒吼，猛地拔出战斧，从死去的士兵身上跨了过去，那匹失去了主人的战马掉转头去落荒而逃，躲开了哈拉萨那张牙舞爪的撕咬。

有时候做个半食人魔还是不错的。

在他视线的余光中，黑暗之门里有什么东西在闪烁。虽然只是惊鸿一瞥，但他看到的景象深深地印在了他的脑海里，闪电、翻腾滚滚的尘埃云、猛烈拍打的海浪和分崩离析的大地。黑暗之门的帷幕中总是会显示一些门另一边的景象，所以他能在战斗中看见德拉诺。但他刚刚看到的——那不是他的家乡。那是一个噩梦之地。

这时，另一名联盟士兵向他发起了进攻，立刻将雷克萨的注意力带回到战场上。他轻松地干掉了那个战士，但是离他一两步远的另一个兽人就没有那么幸运了。那兽人身穿术士的长袍，有着多数部落成员的绿色皮肤——不像雷克萨自己，他直到入侵艾泽拉斯之前才加入部落。这里有好几个术士，他们中有些人非常强大，但是，他们的暗影魔法需要施法时间，而战局中机会总是转瞬即逝。

两个战士一起袭击了术士，那兽人施法使其中一个人在恐惧中奔逃，但另一个战士刺穿了术士的胸膛，附近的一个战歌勇士用一支尖啸着的棍棒敲碎了人类的头骨。现在，这个术士踉踉跄跄，一只手压住自己胸前的血迹，他的皮肤已经开始变得苍白，额头上冒出了汗珠。雷克萨只是咕噜了一声，摇了摇头。他可帮不上术士们什么忙，而这个术士显然没有为战斗做好准备。

这个动作引起了术士的注意，受伤的兽人盯着雷克萨，他脸上现出厌恶和轻蔑的神情。然后他踉踉跄跄地走上前来，另一只手伸了过来。

“你！”术士喊道，“狗杂种！你不是真正的部落，也不是真正的兽人。但你马上就是了。你给我过来！”

雷克萨盯着术士，惊讶得说不出话。什么？这个术士侮辱了他，还指望他去帮忙？他彻底疯了吗？

但是，当术士靠近的时候，雷克萨看到了兽人的手指散发出绿色光芒，一种罕见的恐惧感涌上心头，他倒吸一口凉气。不，术士不需要他的帮助，他想要的是雷克萨的命。术士可以通过吸取别人的生命能量来治疗自己。这个过程的代价极高，治疗一个严重的伤口需要搭上一条健康的兽人的性命。

这个术士的伤口是致命的。雷克萨试图往后退，但是他被困住了，他身后的兽人和人类挤得太紧，他完全无法移动。他大声咆哮着，举起两把斧子，决定砍倒术士，而不是让自己白白送死，但是兽人做了个手势，雷克萨便突然跪倒在地，无法言述的痛苦流过了他的身体。

“怎么，你对自己没信心了？”术士轻声地嘲笑道，走近了雷克萨，他们离得如此之近，以致他的呼吸让雷克萨的皮肤一阵发痒。雷克萨痛苦地瘫倒在地，挣扎着。“疼吗？不要担心。很快疼痛就会消失。”他缓缓地举起手，故意把那一刻拉得很长。雷克萨看到，那具绿色的躯体离得更近了。他已经感觉到自己的能量被从身上慢慢抽走，疲倦感席卷了他全身。

突然，一声怒号刺透了痛苦的阴霾，一个巨大的黑影撞向了术士。

“哈拉萨，不！”

术士有一瞬的分神，魔咒被打破了，雷克萨重新恢复了行动力。但为时已晚。他忠诚的战狼伙伴推开了术士，但是在这个过程中，兽人的手碰到了哈拉萨那厚厚的皮毛。雷克萨惊恐地瞪大眼睛，眼睁睁地看着他的朋友在他眼前枯萎，那只强大的战狼在一瞬间缩成了一团，瘫倒在地，它的身体化为尘埃，很快便被风吹散了。

“啊，感觉好多了。”术士说着，站起身来，掸了掸身上的袍子。血迹还在，但他现在行动自如，伤口已经好了。“你的宠物刚刚救了你的命。”他险恶地笑着，对雷克萨说。

“是的，它救了我。”雷克萨轻声回答，上下转动着两把斧头，“可是谁来救你呢？”

雷克萨肩膀用力、轻扭手腕，啪的一下，斧头划了一道弧线劈了下来，从术士的脖颈两侧深深地砍进了他的胸膛。这一击用力相当猛烈，术士一下子被砍翻了，跪倒在地，斧头穿过了术士的身体将他劈成了碎片，他瘫倒在血迹斑斑的地面上。

雷克萨盯着尸体，不住地喘息，然后转过身看向战狼死去的地方，怒火仍在他心中奔涌。他跪了下来，把双手浸在术士那和满尘土的湿漉漉的血液里，停了一会儿。

“我为你报仇了，我的朋友。”他轻声说道，“尽管我希望你还在我身边。”他深深地吸了一口气，站起身来，把满腔的悲伤和愤怒化作行动，开始呼唤战歌的酋长。

格罗玛什抬起头来，看见了雷克萨，挥动斧头向半兽人致意。雷克萨一直都很喜欢战歌氏族的这位领袖，尽管战歌酋长是如此的野蛮和暴力。格罗玛什一视同仁地尊重所有战士，对他也是如此，他也因此对格罗玛什报之以应有的敬意。但现在，结果比礼貌更重要。

“传送门！”雷克萨指着传送门喊道，“有点不对劲！”

格罗玛什朝传送门瞥了一眼，有几个兽人踉踉跄跄地从里面走了出来。一开始，雷克萨受到了鼓舞，以为部落总算是派人过来帮忙了。但接下来他发现这些兽人遍体鳞伤，正在流血，他们不是在行军，而是在逃跑——好像在逃离什么东西。德拉诺那边的东西。

“快跑！”其中一名兽人大声喊道，他飞速地奔跑着，撞到了一个联盟士兵，强大的冲击力直接把这个人类撞翻在地，但他继续向前跑去，甚至没有停下来攻击那个被撞趴在地的目标。“快跑！”

“发生了什么事？”格罗玛什问道，雷克萨也同样很困惑，他耸了耸肩。他们都盯着黑暗之门，它所展现的画面从刚才的疯狂景象变成了一个旋转着的彩色大旋涡，接着，完全暗了下来。

然后，它消失了。

很快，支撑着连接两个世界的时空裂隙——黑暗之门的石头框架开始嘎吱嘎吱地响了起来。声音越来越大，然后，它从中心断裂了，两半石顶向内倒了下去，撞在一起，发出巨大的响声，激起了一片尘烟，变成了一堆石头碎片。紧接着，支柱也倒了下来，刚才的撞击使它失去了平衡。雷克萨低下头，把兜帽的边缘拉过来罩住嘴，以免被滚滚而来的灰尘呛着。兽人和人类都在四散奔逃，尽力远离这片混乱和充满残骸的地方。

“不！”有人在尖叫，有人在呻吟，有人在哭泣，空气中充满了这些声音。雷克萨惊得目瞪口呆，盯着那些曾经连接着世界的传送门的碎石。黑暗之门消失了吗？难道说他们永远都不能回家了吗？他们的将来会怎样呢？

幸运的是，有一个兽人仍然保持着头脑清醒。“我们要重组队伍！”格罗玛什喊道，拍了拍雷克萨的肩膀，“你把召集那边的人，我去把这边的人都找回来！告诉大家在山谷的入口集合！”雷克萨从呆滞中惊醒过来，他点了点头，迅速去执行战歌酋长的命令。摆脱了纷飞的尘埃后，他再次把兜帽放了下来。他的内心仍然充满恐慌，但他得把精力集中在格罗玛什分配给他的任务上，迫使自己重新振作起来。他每看到一个兽人，就指引他们前往谷口。不管是由于他的体型，还是因为他挥舞着的斧头，或是仅仅因为他们迫切地需要得到指令，兽人们都毫无争议地服从了。当雷克萨自己也抵达了入口的时候，格罗玛什也回来了，所有仍然留在艾泽拉斯的部落成员都和他们在一起。他们大多数看上去都像雷克萨一样不知所措。

“格罗玛什！传送门消失了！”其中一个兽人哭了起来。

“我们怎么办？”

“是的。黑暗之门已经不见了。联盟已经重新集合了。”格罗玛什大声宣布道，指着原来传送门所在的地方，人类正在那里聚集。“他们认为我们很好对付。他们认为我们会迷失，会因为传送门的消失而恐惧，但他们错了，我们是部落！”

他用那双发光的红眼睛扫视着面前的人群，举起了血吼，“我们向北走，回斯通纳德。我们要去弄明白我们的世界究竟发生了什么。先包扎一下伤口吧。我们会活下来的。”

“然后，我们将重整队伍，只有这样，我们才能够决定自己的命运，而不是任由人类宰割。”他咆哮道，“联盟逼近了。他们能带走我们吗？”

一声响亮的“不”从兽人的人群中爆发了出来，雷克萨暗自担心，这些就是兽人部落残剩的全部人马了吧？格罗玛什咧嘴一笑，仰起头，张开布满黑色文身的下巴，呐喊着他的战斗口号，紧接着冲了出去，他的人跟在后面。

就是那个人。

当晚他们在斯通纳德宿营时，格罗玛什朝着那个蜷缩在篝火旁的兽人走了过去。他身上既没有尘土，也没有流血，而且，格罗玛什认识他所有的战士，但这个人不是。格罗玛什抓住那个兽人的肩膀，把他猛地向后一拽，俯视着他，旁边站着高塔般的雷克萨。兽人惊呆了，眼睛睁得大大的。

就像举起一个小孩一样，格罗玛什轻松地将兽人举到空中。兽人的双脚又踢又蹬。

战歌酋长凑近了他。“现在，你告诉我，”格罗玛什面色阴沉，平静地说道，“以先祖之名，那边到底发生了什么事？”

兽人颤抖着，急切地说出了他所知道的一切。其他的兽人在一边安静地听着。唯一的声音就是那兽人急促的说话声、火焰的噼啪声，以及夜晚沼泽中无处不在的声音。他讲完后，所有人都沉默了，呆呆地看着他，震惊得说不出话来。

几分钟后，格罗玛什终于摇了摇头。“所以——”他咆哮着，怒视着其他人，半是羞愧，半是恐吓，逼着他们把目光移开，站起身，挪动脚步。“现在，我们去准备。”

“准备？”雷克萨喊道，格罗玛什转过身来面对着半兽人、半食人魔战士。“准备什么，地狱咆哮？我们的整个世界已经死了，我们的人民也死了，我们将永远被孤零零地困在这里。以先祖之名，我们还要准备什么？”雷克萨紧紧地抓住他的斧头，格罗玛什觉得他似乎听到了石头斧柄被捏得吱吱作响。

“我们准备为死者报仇！”格罗玛什厉声说道，加尔鲁什的模样再一次浮现在他的脑海中。他的儿子和继承人。我的孩子，他想到，我的孩子死了，和其他所有人一样。“我们是仅有的幸存者！”他坚持道，扭头对着其他兽人，“我们现在就是部落！如果我们放弃，意味着我们所知道的一切，我们所关心的一切都到了尽头！我们的种族是不会灭亡的，除非我们像胆小的懦夫那样躺下来接受死亡！如果耐奥祖的计划——”

“耐奥祖！”雷克萨喊道，他俯下身子，这样他的脸就凑到了格罗玛什的脸旁。“这一定是他的错！还有谁能让世界破碎成这个样子？他背叛了我们所有人！他说他要拯救德拉诺，但却把它毁了！”

“我们并不了解事情经过！”格罗玛什坚持道，“我们只知道他是在用极其强大的魔法打开通往其他世界的传送门。或许这中间发生了一些什么错误。”

“或许对他来说，这一切都十分完美！”雷克萨激烈地反驳道，“也许他只是在利用我们，我们所有人，我们的整个世界，来实现他自己的野心。古尔丹就是这样做的，不是吗？”兽人们聚集在一起，咕哝着、低声议论着，或者咆哮着表示赞同——大家都知道古尔丹的背叛，以及他是如何让他们付出了第二次战争的惨痛代价的。“谁训练了古尔丹？”雷克萨继续说道，“谁教了他？是耐奥祖！上梁不正下梁歪。”

人们的抱怨声越来越大，越来越愤怒，格罗玛什心里清楚，在这群勇士变成愤怒的暴徒之前，他必须得阻止他们。

“你不觉得这无关紧要吗？”他说道，用一种特别的平静打断了雷克萨的愤怒，“我们难道要根据谣言和担忧来决定我们该做什么吗？我们真的要为事情可能是怎样而痛苦，或者为可能发生了什么而担忧吗？这是强大的部落的行为吗？”他的目光在兽人之间扫来扫去，将所有人囊括进这段对话，他很高兴他们的窃窃私语消失了，都在等着听他还会继续说什么。

“我们已经活了下来！我们在艾泽拉斯，一个充满生命、食物、土地和战斗的世界！我们可以重建部落，再一次横扫这个世界！”

一些兽人为他的说法喝起彩来，格罗玛什用这种能量点燃了自己的激情，他将血吼高举过头挥舞着，血吼发出的尖啸声烘托起激昂热烈的气氛。

“是的，联盟正在追捕我们，”他喊道，“的确，我们今天不是他们的对手。但总有一天——而且这一天会很快到来，我们将战胜他们！在这里，我们可以休息、恢复和制定战略。在这里，我们将发动袭击，就像我们在过去的几个月里所做的那样。我们将再次变得强大。我们将再次成为掠食者，人类会因为恐惧而瑟瑟发抖！”他猛地停下了斧头，让它悬停在他的头顶，周围突然安静下来，他压低了声音，话语在一片沉寂中渐渐平息。“有一天，我们，部落，将会站起来，以真正和最终的胜利来完成对人类的复仇！”

战士们欢呼雀跃，呼喊着，高举着自己的武器，格罗玛什高兴地点了点头。他们在他的身后，再次团结起来。

所有人，只有一个除外。

“你已经被人出卖了很多次，每次都是被自称为首领的兽人出卖，但你还是要沿着同样的路走下去。”雷克萨轻声说道，尽管他的眼中燃烧着愤怒，“你没有理由再去战斗了！以前，我们为保护我们的人民而战，为他们争取这个世界。但是他们都不见了！我们不再需要这个世界！带着剩下的这些人，你可以找到一个人类从未踏足过的地方，不流一滴血就能得到它！”

“那里会得到荣耀吗？”另一个兽人喊道。

格罗玛什点点头。“没有战斗的生活会是什么样？”他问雷克萨，“你是个战士——你明白这一点！战斗使我们坚强，使我们敏锐！”

“也许吧。”混血儿承认，“但是为什么要在没有必要的时候去战斗呢？为什么仅仅为了战斗而战斗呢？不是为了拯救任何人，赢得任何东西，甚至不是为了荣耀而战斗。它是为了纯粹的嗜血欲，仅仅出于对暴力的热爱。我对此感到厌倦。我不想参与其中。”

“胆小鬼！”有人喊道，雷克萨的眼睛眯成一条缝，直起身子，将两把斧头高举过肩。

“走上前来，再说一遍。”他挑战道，声音里充满了不祥的隆隆声，“从人群中走出来，让我能清楚地看到你，当着我的面叫我胆小鬼！然后看看我是否会退缩！”

没有人动，等了一会儿，雷克萨摇了摇头，他那粗壮的面孔上露出嘲笑的神色。“你才是懦夫。”他宣布，把话啐到他们身上，“你们非常害怕，不敢真实地生活。在谎言与承诺的阴影之外，你们的灵魂已经被收买。你们没有勇气，也没有荣誉。这就是为什么你们不能被信任。”半兽人的两臂颓然垂下。“从今以后，我只相信野兽。”

格罗玛什看着这位高大的战士离去时，内心非常复杂。雷克萨怎么敢在他们最需要抱团取暖的时候抛弃他们呢？但同时，谁又能责怪他呢？他甚至都不是正常意义上的部落的一分子，因为莫克纳萨人从来都不愿意离开刀锋山。据格罗玛什所知，只有雷克萨自己对部落的请求做出了回应，他参加了第一次战争，也参加了第二次战争，但这些给他带来了什么好处呢？他失去了他的世界，他的人民，甚至他的战狼伙伴。这个半兽人觉得被出卖了，这有什么可奇怪的吗？

“没有人可以离开部落！”有人坚持道，“我们应该揪着他的耳朵把他拖回来，否则就把他杀了！”

“他侮辱了我们所有人！”另一个指出，“他应该为他的傲慢而死！”

“我们需要他的力量，”第三个人反驳道，“我们不能失去他！”

“够了！”格罗玛什喊道，怒视着他们。持异议者不敢再说，陷入了沉默。“让他走吧。”他命令道，“雷克萨已为部落献出了自己的力量，现在让他拥有自己的平静吧。”

“那我们怎么办？”一个战士问，“我们现在该做什么？”

“我们知道自己该做什么。”格罗玛什回答，“这个世界现在就是我们的家。让我们彻底地融入其中吧。”兽人们点了点头，重新回到了篝火旁，轻声细语地谈论着计划、胜利和物资供应等问题。但是，即使眼前的一切都按部就班，雷克萨的话仍然在格罗玛什的脑海中萦绕，在他的内心深处，有一个声音在问，他们是否还能找回他们在很久之前就失去了的东西：平静？
