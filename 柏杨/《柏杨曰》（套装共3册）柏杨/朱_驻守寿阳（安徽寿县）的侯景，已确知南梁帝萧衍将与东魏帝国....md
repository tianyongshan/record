# 朱_驻守寿阳（安徽寿县）的侯景，已确知南梁帝萧衍将与东魏帝国...

朱

朱

驻守寿阳（安徽寿县）的侯景，已确知南梁帝萧衍将与东魏帝国和解，把他遣返，于是上书陈情，十分哀切。又写信给萧衍的宠臣朱

，赠送黄金三百两。朱

收下黄金，但仍搁置这份奏章，不代他转呈。

俗话说：“拿人钱财，给人消灾。”朱

收了贿赂，竟连奏章都不肯转呈，是吃定了侯景，认为侯景不过一个游魂，有苦也无处伸诉，两眼只看到黄金，没有看到灾祸。事实上，不是朱

看不到灾祸，而是他认为根本就没有灾祸。

不过，朱

的罪恶却不在受贿，而在他隔绝上下，使下情不能上达，使在上位的人面前一片漆黑，作出错误的决策。萧衍固是一只猪，但朱

却在一旁用鞭子抽打这只猪，把它赶下悬崖。
