# 江平：警惕国家权力膨胀  





\ 
    【财新网】（记者 杜珂）“只有改变权力行使的观念，推进民主宪政改革，中国才能有真正的进步。”

11月20日，法学家、中国政法大学终身教授江平在“蔡定剑先生逝世周年纪念会”上如此强调。

江平说，当前中国有一种强调国家权力至上的倾向。“这是很危险的信号，会使整个社会失去弹性。中国还是要坚定不移地向着宽松、扩大民主宪政的方向前进。”

在江平看来，当前社会矛盾多发。在“稳定压倒一切”的理念下，有些人会得出结论：越是政权不稳定，越是应该强调国家控制。但这会阻碍真正的改革。

有学者指出，当前，改革有停滞不前的危险。社会学者李楯在会上指出，当今，有的人主张回到新民主主义，有人主张回到文革，有的人甚至主张为国家统一而不惜动武。

“这些都说明，当前中国利益严重失衡，社会各阶层缺乏共识。这与集权控制的社会结构互为因果。中国必须建立基于人权、民主、平等的社会治理构架。”

北京大学法学院教授王锡锌提出，“宪政的希望在于每个人”。中国宪政进展有限，很大程度上与民众太麻木、太势利、太在乎眼前利益有关。

他指出，面对权力越界，每个公民至少要做到不同流合污。“限制权力，需要民众共同来限制。这是为自己负责。如果不能对自己的行为负责，中国就没有希望”。
