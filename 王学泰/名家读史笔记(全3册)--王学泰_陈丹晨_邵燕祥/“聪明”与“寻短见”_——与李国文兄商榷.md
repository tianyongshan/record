# “聪明”与“寻短见”_——与李国文兄商榷

风雨微尘

“聪明”与“寻短见”

——与李国文兄商榷

《随笔》2002年第6期载李国文兄的大作《王国维之死》，对王国维自沉昆明湖作了新的解释。王国维的死因历来众说纷纭，国文兄将诸说加在一起并加评论，不失为一家之言。但其中有一段涉及1949年后的历史的话，似乎与事实相去太远。国文兄是这样说的：

所以，五十年代的政治运动，六十年代的文化革命，多少学者名流，教授权威，文化大师，学术泰斗，被折腾得痛不欲生，可很少听说有人起而效仿王国维，毅然跳湖。看来，大家都聪明起来，“观堂已随屈子去，此地空余昆明湖”，少有人寻短见了。

读了这段话，使我很为国文兄的健忘而惊讶。现就我的记忆所及信口可列举的就有：北京解放初，沈从文自杀未遂。反胡风时，田间跳“海”未死。反右时，戴着教条主义帽子自杀的有马寒冰。曾被誉为“浩烈之徒”的杨刚竟也自杀身亡。我当时在北大。已调东北人民大学历史系的丁则良教授就是自沉未名湖的。我的老师朱家玉先生则是跳海自杀的。我的邻楼有一位女生跳楼自杀未遂。到了“文革”期间，自杀几乎成“风”。亦官亦学的知识分子中，有田家英、邓拓、姚溱等。著名的作家学者记者演员中，有海默、老舍（请注意，老舍就是跳湖的，只不过跳的是太平湖，不是昆明湖）、叶以群、杨朔、李广田、闻捷、傅雷夫妇、翦伯赞夫妇、范长江、孟秋江、蒋荫恩、金仲华、李平心、王科一、言慧珠、上官云珠等。袁水拍、萧乾、杨承芳则是自杀未遂。谢晋的父母因儿子受迫害，不堪忍受，或服药、或跳楼，先后自杀。再以上海音乐学院一个单位为例，就有著名钢琴家顾圣婴和她的母亲、弟弟一起开煤气殉难。指挥系主任杨嘉仁和夫人程卓如也是开煤气双双自尽。管弦乐系主任陈又新跳楼自杀。钢琴系主任李翠贞开煤气自杀。民乐理论系主任沈知白也是自杀的……我所在的外文出版发行局，在造反派和军代表先后主持的揪斗、隔离审查、严刑逼供的情况下，自杀者多达三四十人，大多是从事编辑翻译工作的知识分子。以上仅仅是我个人有限的见闻或记忆所及，“文革”期间，全国自杀的人中有许多是相当著名的作家学者，他们的死已为人们所熟知。无论如何，说是五六十年代以来的历次政治运动中，“大家都聪明起来”，“少有人寻短见了”，是完全不符合历史事实的。

十年前，我曾写过一篇讨论中外作家自杀的文章《生命的歧路》，载于《钟山》。我在文中列举了许多事例，证明日本、西方作家中颇多自杀的现象，恰恰中国很少有这种事情。翻开中国古代文学史，自杀的，只有一个开头的屈原，中间有一个唐代卢照邻，一个结尾的王国维。民国以来，大概只有一个诗人朱湘是自杀的。此外，不管社会有多少黑暗，个人生存有多少艰难，却无别的作家自杀的。古代作家、诗人、学者，遇到社会腐败黑暗不公抗争无效时，最消极的办法就是隐遁山林，逃亡于社会之外，而决不会选择自杀之路。其原因与中国传统文化密切相关。儒家思想中“身体发肤，受之父母，不敢毁伤”的教导历来为人们所遵奉。“士可杀，不可辱”，“杀身成仁”，都是表示可以壮烈迎接死亡，但接受的是他杀，不是自杀。因为他们认为生命不是属于自己的，而是父母给的，是属于君王的。（所谓“君要臣死，臣不得不死.父要子亡，子不得不亡”）。自己是没有权利随意处置的。与中国这种传统文化观念不同的是，西方文化思想中，一般是把生命看作属于自己的，可以由自己支配的。早在罗马时期，就有学者普列尼认为，“‘自然’所赋予人类最优厚的财宝是‘在适当的时机而死’，尤其是‘得以自杀’”。希腊的斯多葛学派还赞美自杀是高贵的、勇敢的行为。后来的休谟、叔本华都曾反复阐释了这些思想。当然，这还与宗教文化思想有关。一般来说，信奉天主教的有选择自杀的，而儒、佛、道等则不太主张自杀，这与中国较少自杀是有很大关系的。这些问题说来话长，这里就不赘述了。

问题是，一向不选择自杀的中国知识分子，何以在1949年后，特别是在“文革”十年中，却出现了如此多的自杀者。这完全是一种反中国传统文化的行为。我们可以因此想象当时这种严酷野蛮的政治环境下，人们的尊严被践踏，人格被羞辱，人身自由被剥夺，处于欲生不能，欲死不许，且又生不如死的险恶境地，人们不得不选择了几千年来中国人不愿选择的路。可见其当时处境，真是“史无前例”、空前未有的了。

上述列举的这些自杀的知识分子中，相当一部分是为了维护人类的尊严而慷慨赴难的。傅雷在“文革”风暴初起时，就对友人说过：“如果再来一次1957年那样的情况，我是不准备再活的。”可见，他们的死都不是偶然的，是非常严肃庄重的。我们尽可以不那么同意他们的做法，可以深深哀惜他们的无辜牺牲，但对他们这种悲壮的死不能不认为是高贵的勇敢的行为，致以崇高的敬意，而没有理由健忘，更不应该酷评嘲讽。若说“聪明”，倒是那些把往事忘得一干二净的，既能过得轻松愉快，也许还能讨好某些人。

我多么希望将来有人能够对“文革”中自杀现象作为一个专门的政治文化课题进行调查研究探讨，弄清其确实人数、具体死因、自杀的方式……由此可以窥见那个历史时代的某些真相、特征和教训，让生者和后来者永远深深记取，而不致忘却。我深信这是一个非常有意义的工作。

我的意见如有不当处，愿聆听国文兄和读者的教诲。

2002年12月3日写于北京，2003年5月改定
