# 三言五语_揎臂挥拳，“包打天下”，天下可是这样打下来的吗？

三言五语

三言五语

揎臂挥拳，“包打天下”，天下可是这样打下来的吗？

拍拍胸脯，“包治天下”，怕也不那样容易。

以前的大话已变为笑谈，后来的豪言又酿成悲剧。

天下是天下人的天下，治乱当然系于天下人。

时贤有诗：“从此莫谈天下事。”可慨也已。

潮流好，浩浩荡荡，莫之能御。

砥柱好，凛立中流，巍峨千古。

潮流行时自夸潮流，砥柱吃香自夸砥柱。

自夸潮流时，可以指责别人都是逆潮流而动。

自夸砥柱时，可以指责别人都是同流合污，沉渣泛起。

总之，唯我正确。

不过看怎么划算，怎么卖价高而已。

“九天闾阖开宫殿，万国衣冠拜冕旒”：这是昔日的梦，上国的梦，中央帝国之梦。

在今日世界潮流中，各国以平等往来为国际关系的准则之一。所谓“有朋自远方来”，所谓“我有嘉宾，鼓瑟吹笙”便是。

宾主之间，自应平等相待，平等交谈。

旧有“喧宾夺主”之说，这种情况不该复见于今日。

电视新闻中，却常常只见主方侃侃而谈，客方一笔带过，给人“喧主夺宾”的印象。不知到底是怎么回事。
