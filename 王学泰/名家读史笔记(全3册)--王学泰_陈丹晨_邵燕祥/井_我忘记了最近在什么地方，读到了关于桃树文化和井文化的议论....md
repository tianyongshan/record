# 井_我忘记了最近在什么地方，读到了关于桃树文化和井文化的议论...

井

井

我忘记了最近在什么地方，读到了关于桃树文化和井文化的议论。——读后也只记得这两个题目，却忘了出处。记忆是人的生命，记忆力的衰退竟这样急遽而不容商量，急不得恼不得，呜呼，老之将至！老之已至！——忘记了也没有关系，我从来不是有考据癖的人，更早厌倦了案牍功夫。我只愿意随心所欲地遐想罢了。我完全像少年何其芳那样，从小留意的不是那意思和概念，而只是一些颜色，一些图案，一些姿态，或者索性只是氤氲的一种气氛。

长大读书，知道有离乡背井的话，按照经典的解释，那井是指井田而不是水井，几十年后还有人用许多史证表明中国从来没有过井田制，井田制只是理想，聚讼纷纭。那么，远扬的人们告别村舍庐墓时，只是离弃了理想中的井田么？我才不管这些，宁愿相信有井水处就有人歌柳词，浪迹天涯的游子，吟唱着“问今宵酒醒何处”，走出里门，凡有人烟的地方，一路能得到柳永的流行歌曲相伴。

我生在城市，在没有自来水的时候，喝的是轧轧的水车送来的甜水，车过后洒下的水渍有如蜗牛的行迹。那水是从“水局”的井里汲来，但我未曾亲见过。大甜水井、王府井，也都只剩个地名而已.辘轳把儿胡同，其实与井无关，只因为地形像辘轳把儿得名。

“井底引银瓶，银瓶欲上丝绳绝……”白居易说给我的却是一个始乱终弃的故事，“寄言痴小人家女，慎勿将身轻许人！”我关注的不是对女孩子才有用的教训.什么时候我能邂逅这样纯真而痴情的女孩子，像戴望舒写的村姑在井边，或在桃花照人、秋千出墙的墙头上？

然而，不仅罗逖的《冰岛渔夫》供认了八国联军在古城里的罪行，母亲告诉我，单是她童年居住过的齐化门里老君堂，光绪二十六年就有十几个姑娘媳妇跳井……城内并不都是眢井。一处处水井，也许后来就都掩埋了，于是北京城里才不见处处井水了，我想。

到了乡村，我也要吃井水了，我也要担井水了。有把十字锚掉落井底的惶恐，也有窥井如对镜的欢愉，在玻璃般清明的水面倒映着靛蓝天上的星光，比坐井之蛙看到的一锭蓝天更玲珑，更深沉，更幽邈。

每一口井都是“海眼”吗？平静的井水都通向那烟波万顷吗？惊涛骇浪下又是什么？洞庭湖心的君山有一口井，跟平常的井无异，石碑上写着一个幸福地遇到牧羊的龙女的读书人的名字：柳毅。我怀疑所有这一切，连同《搜神后记》里的田螺，都只是聊以自慰的幻影吧。

但是有一口井，从里面升起的不是美人胎子，不是窈窕淑女或小家碧玉：万丈深浅的地穴里，刮喇喇一声响亮，滚出一道黑气，直冲到半天，散作百十道金光，往四方八面去了。井底镇锁多年的天罡星地煞星共一百单八将从此出世了。这是公元十一世纪在江西信州贵溪县龙虎山下。说话是前年了，一个秋天的日子，我来到这昔日的地穴边，已经砌出井台，上面覆着一层青石板，这里既没有什么黑洞洞的伏魔之殿，也没有石龟驮着龙章凤篆、天书符箓的石碣。四望空阔，草茎摇风而已。一切朕兆都已过去，预言都已成为历史，梁山的故事曾经使我们神往，今天的孩子对那里面的人、事、风物和恩怨都感到陌生了。

1990年1月2日
