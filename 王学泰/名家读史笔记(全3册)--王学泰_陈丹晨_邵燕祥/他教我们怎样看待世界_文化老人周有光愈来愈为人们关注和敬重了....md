# 他教我们怎样看待世界_文化老人周有光愈来愈为人们关注和敬重了...

风雨微尘

他教我们怎样看待世界

文化老人周有光愈来愈为人们关注和敬重了，首先当然是因为他健康长寿，已经是百岁人瑞，正在继续创造生命的奇迹，向着109、110、111……更高更长的生命记录快乐地前进.还因为他仍然充满着生命的活力和智慧，对人类历史发展规律，对中国政治思想文化等进行独到的思考和研究，提出了一系列睿智深刻、极具启示性的创见.当然还因为他那浅显直白而又机锋深邃的语言表述，使人感到亲和易解，欣然豁然而恍悟，原来事情的真相、历史的规律是这么一回事！这些关乎历史政治文化等严肃沉重的话题的随想式简短文字竟包含着如此巨量的信息和道理，读来愉悦轻快，却又发人深思.真使那些满嘴新词佶屈聱牙却又不知所云的长篇高头讲章陈腐套话愧煞羞煞也！

周老说他“是研究人类历史发展规律的”。

注1

已有的各种有关思想学说共同点都是希望人类建设一个美好的理想社会，这是“人类文明的原动力”，“推动了社会发展”。但他认为只强调高远不可及的理想而鄙视甚至破坏现实建设既违背了社会发展规律，也走进了历史误区。以儒家学说为例，他阐释的孔子的“世界大同”理想和脚踏实地地建设“小康”社会的关系简约明确而令人信服。重要的是，他指出“没有国家能够长期违背社会演进的历史轨道”，这个道理对我们回顾认识近百年以至更长时间的中外历史大有裨益。

周老对过去关于五种社会形态即人类经历的五个阶段提出了质疑，例如他认为“把共产主义作为社会发展的最后阶段也不妥当。社会发展没有尽头，科学的历史观不预设最后阶段”。显然这是颇有见地的看法，就二十世纪至今为例，科学发展出现了多少无法想象的创造发明，大大改变了人类历史的进程。人们完全可以绘制未来的美好图画，但谁也不能确切设置人类的“最后”。

在《朝闻道集》《拾贝集》中，周老对人类历史发展几个阶段的分期，是从不同层面概括的：从政治上说，是神权统治、君权统治、民权统治.从经济上说，是农业化、工业化、信息化.从文化上说，是神学思维、玄学思维、科学思维。这样的分期是否合适，可以进一步研究.至少拓宽并打破了我们习惯的旧思路，启发人们从更具体存在的历史事实出发去验证去寻找历史的必由之路。对此，周老着重对中国、美国、苏联的历史发展和特点作了精辟的分析。尤其对于俄苏的历史和苏联的最后解体评析得切中肯綮，无可辩驳。

他认为“苏联瓦解，基本原因不在某一个人的行为，要到俄罗斯的历史背景中去找答案”，其“深层原因是专制制度”。他对苏联历史追溯到最早九世纪基辅罗斯大公国，历经伊凡雷帝、彼得大帝、叶卡捷琳娜二世这些曾使俄罗斯辉煌过的年代，但也是血腥残暴的记录累累，沿袭形成暴政统治的传统。

对于苏联，周老并不只说其罪错，他很公正地历数其科学、文化，以及卫国战争胜利等方面的功绩和成就，但归根结底是致命的专制体制使他不可能有更好的命运。周老将他阅读掌握的有关苏联解体的第一手报道、评述资料选摘附于文内，再加周老的总结和分析，确证了他所指出的人类历史发展规律之不可逆转，才会有莫斯科的“没有哭泣的葬礼”。

周老为苏联解体专门写了《苏联历史札记》一组文章，编辑收集在《朝闻道集》《拾贝集》两书中，大致相仿。他所以重复编入，正说明他特别重视，因为他知道总结苏联的经验教训对中国有特别重要意义。如今，多么需要对苏联解体这样沉重的历史教训进行回顾和总结，从中寻找我们新的道路。

说到这里，就涉及一个对待现代先进文化的问题。周老梳理了世界几种重要文化体系，认为经过交融互补后，“好的东西大家用，就慢慢地产生一个新的文化，这个新的文化就叫作现代文化”.是全人类的共同创造和宝贵财富，是造福全人类的，“聚合成‘共创、共有、共享’的国际现代文化”，绝非为一家一阶级所特有。这样既有原来的传统文化又有现代新文化互补共存共发展，“它自然地变成一种双文化现象”。这个崭新的解释和论断完全符合历史实际及其发展规律，是周老对人类文化学的一个极大的贡献。他解决了长期以来争论不休的难题，破解了过去所谓“两种文化”论的陷阱。事实上，许多外来先进物质文明的成果常常是统治者最先接受的。因为他们有一般人不可能有的条件。例如钟表、照相机、自行车、火车、汽车……最早就是出现在清代皇宫中。对于享受物质文明，统治者从来是不拒绝的，只不过对先进的思想文化是恐惧的，拒绝的。

周老不仅学养渊博精深，而且胸怀也博大开阔，期颐之年还对人类历史发展规律这样宏观的课题进行深思研究，提出的高远卓识具体而微，明白切实，是人们日常所见所闻所感。真的是举重若轻，谈笑间把一个深奥重大的问题解说清楚了。想到他当年主创的“拼音方案”，看是几个字母却造福国人，造福后世.从识字、发展文化、到今天电脑手机所用的文字输入，莫不仰赖这个“拼音”。就这个贡献来说，足以流传子孙后代。如今，这位百岁文化老人没有显赫的身份，不求闻达于诸侯，以一介平民，身居陋室，旁无助手，全靠他单枪匹马坚持广泛阅读掌握大量国内外最新的书报资料信息，手抄打字电脑，孜孜不倦，乐此不疲……这些本应由青壮年学人干的事，他一个不能少地静静地快乐地干着.经过他的睿智的思考，一篇一篇随想式的重量级的文字产生出来。他知天下事，谈天下的过去、现在和未来，讲真话，求真理，这是继“拼音”后又一造福国人的大贡献。他的这些随想文字也是继巴金的《随想录》后中国当代文化史上又一反思历史的杰作。不同的是巴金以个人视角反思“文革”，周老则以客观的角度把握探讨人类历史变迁的规律，两者有异曲同工之妙，都是耄耋、期颐之年的老人还在为后人费心，谆谆教导、告诫我们，值得我们深长思之。

读书至此，衷心祝福周老快乐自由，祝福他的智慧人生将创造新的纪录！
