# 北大的另类另事_——《开花或不开花的年代》札记

风雨微尘

北大的另类另事

——《开花或不开花的年代》札记

近些年，在书店里看见过好几种北大人写北大的书，都是前贤或是年轻些的师弟师妹们写的文章，都很有意思，很开心，把北大与众不同的学习学术生活相当生动传神地写了出来。那年北大百年纪念活动时，报刊上也有大量文章颂扬北大的通体光明，更是热闹非凡。但我常常觉得似乎缺少了些什么。有一天，老同学聚会时，有人倡议写写我们自己时，我才忽然恍悟，原来正是五六十年代的北大，以至“文革”时的北大，也就是百年中的近三十年的北大还少有人涉及。

当然这是最难写的，但也是北大历史中不能割断的一部分，恐怕也还是前无古人，后无来者，非常难得的。写出来，不仅立此存照，也还有以史为鉴的重要意义。使后来者从此可以看到，原来北大人还有这样的一页，北大是这么走过来的，北大应该怎样走向未来。我现在要说的正是指中文系1955级同学写的《开花或不开花的年代》开始补了这方面的空白。它没有那么欢乐，那么开心，那么畅快，而是在表达青春狂欢的同时，混合着惨烈的苦痛和凄楚。尽管它远远没有写出历史的全貌，但也可以说是表露了冰山的一角，是值得人们一读的。

北大中文系1955级是一个很有名的年级。它之所以出名，主要是因为在1958年“大跃进”时，集体写了一部“红色文学史”。由此，在全国引发了一股学生集体写教科书的风。这个55级就是始作俑者。这部“红色文学史”就是在拔白旗、插红旗运动中应运而生的，恰中了政治运动领导者的下怀，被利用为批判打击所谓“资产阶级知识分子”即高校中的老教授们的一块砖头，一个工具，因而被大肆宣传渲染，树为先进标兵，名闻全国。当时的北大党委书记在全校大会上就曾说：“现在学生都能写书了，资产阶级教授们上不了课堂，只好进编译所了。”

回忆这段往事，该是怎样一种心情呢？

有些同学还像当年青春年少，热情可爱，描写那个“如歌的岁月”“难以忘怀的纯真”，在春寒料峭、满天晨星里，打着红旗，高歌行进在、战斗在十三陵水库工地上.或是日夜奋战，在三十多天的暑假里写出八十万字的“红色文学史”，“成为全国文教战线‘大跃进’的标志性产物”。对于这样付出青春、献身理想的时刻，使有些同学“终生难忘”，甚至是“终身受用不尽”。然而，如果想到十三陵水库不久成为荒洼，“红色文学史”转眼即被废弃，千万人的热情、创造力和理想不仅被虚掷、戏弄、践踏，还要齐声歌颂这样的荒唐，还要当作攻击真理的子弹。那样的悲剧，那样的灾难，竟是北大人的参与或创举，也是北大人的天真无邪、豪情壮志的结晶！多么难堪的历史！多么尴尬的回忆！

何况，就在这样高歌猛进之时，也还有“一些受难的同学，有的带着‘悔改’的沉重走在行进的行列里，有的已经无辜被迫离开了55级的队伍”。毕业以后，在几十年的灾难中，吃的苦头受的难，更是我们所难以想象的。其中《往事历历漫诉说》《毕业前后》《一封写于四十年后的信》《老后，你先走了》等几篇，或是本人的直接回忆，或是同学的追记，都详略不同地记述了怎样由一个单纯的大学生变成“敌人”，优秀的高材生变成“处理品”的过程。这是过去很少能够窥见的真实心态。但却像一面镜子映照着历史，也无情地映照着当年部分同学的生活面影。这也都是发生在北大校园里的故事。多么残酷的岁月!多么沉重的往事！

这本书像别的校园记事一样，也记述了许多日常学习课余生活的细事，参加剧艺社、办文学刊物、搞科研、到农村劳动等，虽然时隔四十年，却仍然充满着青春余息、美好的情怀。但也时时流露出一丝伤感和遗憾，烙印着那个时代的痕迹。譬如，在那些记述师生情谊部分。最突出的是三篇关于老校长马寅初遭受迫害的记事（也还散见在其他文章中），人们把当年积压下来的不平和愤怒终于抒发了出来。五十年代北大学生已有万人左右，且我所在的又是中文系，与马老直接接触的机会并不太多，所以不存在个人亲疏恩怨的关系.只是因为马老以他特有的学识、气度、风采和人格魅力吸引着同学们，有着崇高的威望.一旦遭遇大规模的粗暴批判和围攻，真理被践踏，说话权利被剥夺，这是学生们无法接受不能容忍的。即使在当时政治高压气氛下，也用不同形式表现了自己的反感和抵抗。一位刚刚幸免于右派的同学在班里开会时说：“这样对待我们的马老，我感情上不能接受。”说着几乎落下泪来。真可谓不计个人安危了。

马老被批判了，被迫辞职了，但他在嘈杂批判声中还大声坚持说：“我没有在课堂上教学，影响不了学生，但我要用自己文章教育北大学生坚持真理！”后来他在文章中又强调说：“我虽然年近八十，明知寡不敌众，自当单枪匹马，出来应战，直到战死为止，决不向专以力压服，不以理说服的那种批判者们投降。”多么响亮的声音，多么刚毅不屈的形象，深深刻印在学生们的内心深处，融化在学生们的血液之中，教育着他们怎样做一个北大人。有同学写到当时他听到这样“豪气冲天、掷地有声的铿锵誓言传来的那阵子，我只觉得自己的血液凝固了，呼吸止息了。我想起了屈原的遭遇和诗句……我除了扼腕长叹之外，还能有什么作为呢？”

还有对系主任杨晦先生的记事：看到他因为“性格倔强，直面人生，仗义执言为学生们拥护.也因为直面人生，仗义执言而被某些人忌恨。终于在1965年被校长宣布为‘敌我矛盾’……罪状却没有宣布，属‘莫须有’的逻辑。然而，杨先生依然直面人生，仗义执言.这又要加重批判……”不知道这校长何以有如此大的权力可以任意判处教授是“敌人”？北大什么时候竟然代行了“法院”的权责？这是北大的耻辱和荒谬。相对照的是，从马老、杨晦……身上看到了北大的传统和光荣。

这本书里，还记述了同学间朝夕相处，互相关怀，切磋学问，充满情趣的细节。有的同学被打入另册时，仍还得到室友的温暖和鼓励，这时的情谊就更胜于平日了。也有怀着反思和忏悔的心情追述当年参与批判同窗的情景，这也是中外校园生活所没有的独特的一幕。

也许有人好奇：在这样特殊困厄的背景下，又怎么出了那么多的人才，又这么团结友好，有那么强的凝聚力？难道是1958年“大跃进”所赐？难道是当年“青春、理想、进步”的结晶？这本书里将会解答这些疑问。譬如《小毛》中记述的是当年被某些学生干部视为落后的小毛，平日学习刻苦、做任何事踏踏实实、不夸张、不炫耀.毕业时被惩罚分配到最边远地区教书。那是一个地处荒野、狼群出没、极为艰苦的地方。但是，小毛并没有被困难压倒，经过艰苦奋斗，做出了优秀成绩，成为全省先进教师.后来回乡继续任教，又被评为全国优秀教师。数十年来，一身布衣，站在基层教育讲台上，贡献了自己的全部心血。又如另一篇《毕业前后》，是记述自己发奋学习、成绩优秀，反倒被视为个人主义的白专典型，“一贯和党的方针对立”，加以批判，分配到京郊一个拖拉机手培训班教书。但他在逆境中依然不能忘情于学术，经过孜孜不倦的奋斗成了著作丰富的民国史专家。像这样当年“轰轰烈烈”战斗中的“落伍者”“对立面”“处理品”，如今卓有成就，在55级同学中不在少数。那么其奥秘又何在呢？

这就不能不说到北大这块神奇而膏腴的土壤了。本来这些同学就是从普天之下选拔出来的优秀人才，到了这个特殊历史传统的人文环境里学习、生活，耳濡目染，熏陶滋润，哺育营养。人们以她的历史为荣，接受先贤师长们传递的薪火：自由的思想，民主的精神，张扬的个性和独立思考的风气，宽容，大度，追求科学创造和学术传统。无论怎样的气候和风向，怎样的曲折和灾难，邪恶和荒谬，都不能把她植根很深的历史精神摧毁或动摇。只要读一读这些风风雨雨里走过来的人们写的经历和思考，尽管有着不同的视角、心态和体验，但都可以感受到反思历史时特有的冷静、严谨、宽容，以至有点矜持、雍容，充满自信。

有一位同学没有来得及完成学业就被当作“敌人”罚去劳改，后来回乡种田，饱受人间的苦难，从此不再回北大。不久前，他在给同学信中说：“四十年来我受的苦实在太多，但我活得很坦然，我可以正视任何人的目光，我认为我是正直的中国人，因而这罪就该我受，我个人不算什么，好在我们的民族在逐渐觉醒，并且渐渐走上振兴，这该是我们每个人的大高兴……”这就是那个时代过来的北大人的精神和胸怀。

我们的老师、诗人林庚先生在给55级题词中说：“那过去的岁月，仿佛是无言之美。”这位同学则感慨说：“那岁月，不可说，不能说，谁能活出味道来，认出了真善美与假丑恶，那也就是‘美’的享受了。”我不知道人们长久沉默，究竟是在享受那“美”不可言，还是因为不可说、不能说而独自在辨味。如果说，这是美，也是惨烈的。无论如何，作为北大人，是无法忘却的，于是就有了这本书的出现。

2001年12月2日
