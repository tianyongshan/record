# 厚殓·祭祀·谥号_近日报载，称秦始皇陵兵马俑二号坑将启动第二...

风雨微尘

厚殓·祭祀·谥号

近日报载，称秦始皇陵兵马俑二号坑将启动第二次发掘。关于发掘秦陵文物考古界一直存有分歧，主要在于发掘后的文物能否善加保护问题。我非业界人士，无资格置喙，只是想到21年前写的一篇小文，今天似乎还可录以备考于下。

2015年5月4日

为秦始皇守陵的兵马俑——秦俑二号坑于今年（1994年）3月1日正式动铲发掘。据查内有陶俑陶马1300余件、战车80余辆，以及众多的青铜兵器。这些兵马俑之高大逼真据说将是历来墓葬俑殉之最。发掘秦陵是一个重大的工程，已经发掘的秦陵一号坑、三号坑中出土的兵马俑，震惊了世界，被誉为“世界第八奇迹”。这次发掘秦俑二号坑将需五年到七年，甚至更长时间才能完成。内藏的文物比前掘出的也将更加丰富重要珍贵。而这还仅仅是为秦始皇守陵的部分。考古学家还已试掘了秦陵附近十七座殉葬墓中的八座，发现殉者多为贵族大臣，棺椁考究，陪葬丰富。那么，令人难以想象的是，当秦陵全部发掘时又能看到怎样一个千奇百怪的地下博物世界呢！

秦陵的修建和秦始皇的葬埋，在当时是在极端秘密的情况下进行的。秦始皇的儿子胡亥下令将后宫没有生育孩子的宫女统统殉葬，然后还将参与工程知悉机密的工匠也全部活埋于陵墓内。所以，秦陵的情况，后来的史书文献几无记载，只有司马迁在《史记》中有一段粗略的记叙：秦始皇即位以后，就在骊山给自己修造坟墓。他征发了七十多万劳改犯去服修墓的劳役。墓高一百多米，掘地极深，灌注铜液。墓中有宫殿和百官位次，藏满了奇珍异宝.同时还布满了机关，如有人盗墓，就会有箭矢自动劲射.水银流注成百川大海，机械运转出现天象地貌。用人鱼膏做烛，照明墓内，经久不息。

仅仅这样一小段记录，人们也可以想象这陵墓该有多么壮观。我们也还可以由此推想，秦始皇之流的心里该是恨不得把整个人间财富权势统统带到地下阴间才能填满他的贪得无厌的欲海。

咸阳西安一带古墓葬极多，已发掘的和未发掘的帝王宗室陵墓如武则天墓和章怀太子墓等。全国各地发掘的战国、汉代、六朝、隋、唐墓葬也很多。中国地下文物相当数量出自于墓葬。即以嘉峪关附近的魏晋墓为例，人们怎么也不会想到在那茫茫无边的大戈壁地下竟集中埋有一千四百多座墓葬。从已发掘的十余座中就已发现了大量彩绘、砖画和砖雕等，被称为“宝贵的地下画廊”。最近，在四川丰都汇南乡一处也发现了汉至六朝的古墓一千余座。凡此，真是多得不可胜数。

丰富的墓葬起因于厚殓。厚殓在中国流传了数千年，也可算是一种中国特色。商殷时代迷信鬼神，国王死了，用活人殉葬。多的时候竟有千百人。到了周代，慢慢改用草、木、陶制的人和器皿来殉葬，所谓“始作俑者”。但是，史实证明，到了千古一帝的秦始皇时，不只还用活人殉葬，而且其中还有许多王公宗室.由于科技、工业发展，器皿也愈来愈丰富多样珍贵。这种厚殓的流韵遗风直到明清以至近代久盛不衰。

北京以东一百多公里的昌瑞山峰下的马兰峪，一派龙盘虎踞的非凡气势，被皇家看中，建有规模巨大的清代五个皇帝和许多后妃的陵园。北京以西一百多公里，在群峦叠嶂的永宁山峪也建有另一个陵墓建筑群，葬有清代四个皇帝和许多后妃。这就是所谓清东陵和清西陵。你若看到那些壮丽精致、规模宏大的宫殿、墓穴、地宫、神道、石像、牌坊，一定会很费解，为什么一个死者要让生者付出如此巨大的人力物力，只是为了埋葬自己的一副皮囊骸骨。更可怪的是，当年秦始皇称帝之初，就给自己建造葬身之地。后来那些皇帝竟也继续这个遗风。像清代乾隆年纪大了，退位做太上皇，传位给儿子嘉庆的同时，却还要给他选定一处陵址。也就是说，他把帝位和坟墓一起传给了他的接班人，可见他们对于死的重视。因此，道光即位后就忙着造陵，七年后修建完工，因地宫浸水，又全部拆毁，改在西陵重建。建成后的隆恩殿的材料主要是用金丝楠木，雕成大量游龙、蟠龙，造成“万龙聚会，龙口喷香”的气势。至今，人们走近大殿，那缕缕浓郁的楠木清香依然扑鼻。慈禧因为寿长，早就建成的陵园也就一修再修，建了又建，至死方休。

当年那些贪婪的帝王贵族的厚殓给后来的贪婪的盗墓贼创造了条件。“盗墓贼”大概可以算得上中国特有的一种传统“行当”。近几年盗墓之风更加疯狂，遍及各地。有记者说，关中地区盗墓已成社会公害，仅陕西省近两年至少已有三千多座古墓被盗，大量珍贵文物被破坏、走私、外流。有的地方甚至出现全家全村合伙盗掘古墓，倒卖文物，坐地分赃。

人们历来把儒家思想看作中国传统文化核心。儒家学说虽被独尊而行于世，但这种厚殓以及祭祀、谥号等一套文化形态却与孔老夫子并不相干。一则，如厚殓、祭祀早在孔子之前至少已经相传千年。二则，孔子是不信、也不谈鬼神的。他是反对厚殓的，对祭祀也没有兴趣。他的得意门生颜渊死了，孔子很悲伤，别的一些弟子要将颜渊厚葬，孔子却以为不可。但是，弟子们不听他的，弄得他毫无办法。孔子总认为生的道理都还不容易弄清楚，哪还弄得清死的问题。他自己病重时，弟子子路祷告上天保佑。孔子怀疑说：“有那么回事吗？”子路说：“有啊！祷告文里不是说向天地神祇祈求吗？”孔子反讽说：“要那么说，我早就祈祷过了。”正因为他不信鬼神，所以他从根本上是否定祭祀的。所谓“非其鬼而祭之，谄也”“祭如在，祭神如神在”“吾不与祭，如不祭”都是人们熟知的论述。

奇怪的、令人不解的是，后来一些尊孔的人们常常打着孔子的旗号搞厚殓，搞祭祀，而且还要演祭孔大典，那些反孔的人们也远不如孔子那么开明，常常喜欢修大陵墓。总之，在这点上，这些后代子孙不管以什么先进文化的面目出现，其实是很不如孔子的。

鲁迅当年曾经对人们搞祭孔闹剧颇多尖锐讽刺。那时，民国初期，几乎年年闹祭孔。一些军阀头子却照样装扮成一副道德君子模样，大吹大擂祭孔。袁世凯还下令制作仿古的衣冠，让人们穿了这种古里古怪的行头到国子监（孔庙）举行祭孔大典。当时的教育总长率领教育部官员们行跪拜大礼。山东军阀张宗昌虽是混世魔王，却也提倡尊孔祭孔，自己也跑到孔庙叩头跪拜。最使人惊讶的是，近几年北京、山东等地也连续“上演”过热闹非凡的戴着古代衣冠的祭孔大典。参加者官员学者皆有，盛况远胜于民初。想到鲁迅当年说有些人“把孔子当砖头用”，因而“带累孔子也更加陷入了悲境”，这实在也是很值得人们玩味的。

还有一种对死者的谥号，也很离奇古怪。唐代张守节写过一篇《谥法解》，考证谥法最早起源是周公制定的。谥是指生平事迹，号是指贡献，并且列举所规定使用于谥号的字表。其中褒贬都有。帝王的谥号是由大臣们会同司仪官员，由新皇帝审批。官员们的谥号则由皇帝赐给。但是，到了后来渐渐演变成为只讲好话，成了又一次集中的评功摆好，愈演愈烈，愈来愈肉麻。皇帝生前威风得还不过瘾，就像想把世上的好东西都带进坟墓一样，也想把所有好听的“马屁”都贴在自己身上同归于尽。譬如唐高祖死后，唐太宗追谥其父为“太武”.唐高宗时又改谥号为“神尧皇帝”.唐玄宗时又加码改谥为“神尧大圣皇帝”.过了几年，又增谥号为“神尧大圣大光孝皇帝”。但这都还不算什么。到了明清时，就变本加厉，把各种各样的长长的一连串的至高无上歌功颂德但却谁也记不得的谥号追封死者。

譬如清朝入关后第一个皇帝清世祖（顺治）被追谥为“体天隆运定统建极英睿钦文显武大德弘功智仁纯孝章皇帝”。到了清高宗（乾隆）被加码追谥为“法天隆运至诚先觉体元立极敷文奋武钦明孝慈神圣纯皇帝”。至于这些人生前到底是个什么样子，已经没有人会认真去核对，因为谁都没有把这看作真实的评价，都知道不过是一场表演（秀）而已。

当然也不只是谥号，也不只是帝王家搞这一套，就是一般王公贵族官僚也一样狂热追求死后那些辉煌空洞的颂词和身份。唐代韩愈给人家写墓志铭、碑文收取高稿酬。长安城中一些高官家里死了人争相求他撰稿。刘禹锡形容说：“公鼎侯碑，志隧表阡.一字之阶，辇金如山。”可见当时官场风气。《红楼梦》里写秦可卿之死，贾珍为了面子，丧事办得风光些，就贿赂内监给儿子捐了一个官衔，使死了的儿媳妇身份一下子提高了。铭旌、牌位上就大书“奉天洪建兆年不易之朝诰封一等宁国公冢孙妇防护内廷紫金道御前侍值龙禁尉享强寿贾门秦氏恭人”。看了这一大堆曲里拐弯的东西，人们可能会觉得可笑。然而，在当事人眼里却是看得非常重的。

现代社会这些习俗风气当然有了很大变化。厚葬、祭祀、谥号等好像成了陈规陋习不大时兴了。人们辞世后用火葬，已成了通行的习俗。不过，在一个等级分明的社会里，人死了也仍还维持生前的级别享受待遇，就如什么级别领多少工资住什么房子坐什么车子一样，骨灰盒存在第几室第几架第几层都是有讲究的。有一位朋友讲到他的老丈人灵盒是存放在第一室的某架某层时，脸上泛出一种光彩，得意满足之情油然而生。还常看到一种讣告上的名字后面强调说明死者是“享受XX级待遇”，不禁使人默然无语。至于对死者评价性的形容词也是很受重视的。如是“伟大的”“优秀的”“杰出的”，还是什么“家”什么“战士”等，有时也会列上一二十个，绝不比古人的谥号逊色。有时为了评语，生者煞费苦心，聚会商讨，句斟字酌。有时因此讣告迟迟不能发出。有时单位与家属形成拉锯，议个不休。逝者只好躺在那里听候处理。凡此种种，都说明中国传统文化之深厚，之影响深远，之了不起，之顽强，之深入人心……

至于给大人物修建规模宏大豪华的陵墓棺椁的厚葬现象在近代也时有发生。且不止中国，还有某些号称“先进文化代表的社会主义国家”。这种现象在一些为我们排斥的西方价值观的国家里倒是看不到的，其中什么原因也颇可令人深思。

不久前，文学前辈葛洛谢世，留下的遗言竟使我惊愕，久久不能平息。他说：他去世后，“不发讣告，不印生平事迹，不举行告别仪式和任何悼念活动”，“骨灰长久保留，可用简便方式抛撒在中华大地上，使我回归大自然。如果条件允许，也可找一个地方，把骨灰埋下，上面种一棵松柏或泡桐，让我的生命转化成枝叶繁茂的树木，继续挺立在大地上”。

这遗言几乎像首诗，表现了诗人丰富的想象、高贵的诗情、豁达的胸怀，从从容容，清清爽爽。我没有想到，这位温文儒雅宽厚的老人对于旧文化竟有这样一种反叛的性格。平日他含蓄不露，临去一顾，竟是那样强烈，是多么难得！
