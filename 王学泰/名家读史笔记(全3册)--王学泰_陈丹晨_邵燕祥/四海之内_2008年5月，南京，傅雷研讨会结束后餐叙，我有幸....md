# 四海之内_2008年5月，南京，傅雷研讨会结束后餐叙，我有幸...

风雨微尘

四海之内

2008年5月，南京，傅雷研讨会结束后餐叙，我有幸和《译林》杂志创始人李景端先生比肩而坐。我和他早在二十多年前有过一面之缘，这次竟是相逢不相识了。然而，惊叹之后就是老友重叙。我常读到他写的文章，篇篇都有冲劲。这会，在餐桌上，也是他最活跃健谈，声如洪钟，豪爽亲和。

坐在我们对面的三位来自台湾的学者，都是留学法国的博士，研究法国文学、历史的专家，但都很年轻，说话儒雅谦和，虽是初识，却无陌生感。叙谈间，李兄与对面一位女学者发现他们竟是同乡，于是就用闽南语隔着桌子开讲起来，十分热络。俗话说：“老乡遇老乡，两眼泪汪汪。”这是乡情的自然流露。李兄还讲到去台湾访问的观感，与台湾文艺界交往的情况。他还说到他曾陪同林青霞去拜望过季羡林、杨绛等老人，寄赠过许多图书给林，林大为开心……旁边另一位女学者对他们的乡情洋溢微笑着说：“我都嫉妒你们了！”

两岸来往已经二十年了，人们一点感觉不到什么间隔。就像现在，因为傅雷先生的文化遗泽，四面八方的学者会聚一堂，切磋学问，畅叙心得，那融融的情意，使人无法想象同胞手足怎么会视若仇敌，兵戎相见呢！

现在，餐叙中那种和煦温暖的氛围，那种浓浓的同胞情、同乡情、同行情……与厮杀、争斗完全是两个世界，这才是人性向善的本源。这时，傅雷先生的哲嗣傅敏夫妇来敬酒，李景端看着傅敏太太陈哲明一副陌生的样子，我开玩笑说：“这是我小妹。”李兄有点将信将疑。我说可问傅敏。傅敏在一旁早就幽默地笑着说“是”。再过一会儿，华东师大陈子善教授过来碰杯，他和李兄熟悉，我又开玩笑说：“他是我的小弟。”子善也幽默地说：“是啊！”这次李兄就不当真了。当然这只是玩笑，我仗着虚长他们许多岁，倚老卖老而已。但是中国人习惯讲同姓同宗，五百年前是一家。所以我的玩笑也算是事出有因。我和傅敏夫妇是近些年相识相知的。我和子善认识二十多年，往来不多，偶然相见就能倾心相谈。人类心灵的沃野，在正常的阳光雨露的滋养下，生长的一定是善、爱、和平……之花，相互之间理应是容易沟通理解友好相处的。

在我左边邻座是一位老外，大概三十多岁，蓄着一部浓密的大胡子，戴着眼镜，也是非常儒雅文气。我看他的酒杯空了，好像他意犹未尽似的。我说：“再要一瓶吧！”说着我正要站起身，他把我阻止了，轻声说：“我自己来！”他像在自己家里一样熟络，找了服务员要了一瓶，给桌上人们一一加满，然后再给自己斟满。他是比利时鲁汶大学雷纳特·比赫特教授。我问他的中文是哪里学的？“大陆？”“不是。”“台湾？”“不是。”“香港？”“不是。”那是哪里呢？他说：“就在比利时。”啊！学的真不错，汉语讲的那么好，一点洋腔都没有。他很高兴，很有礼貌地谢我的夸奖。再深谈下去，知道他的太太原籍是中国丹东。我说：“我去过，是个好地方。在鸭绿江边，有一座大铁桥，桥那边就是朝鲜。”他点头说：“是，是！”显然他已经非常熟悉那地方了。忽然，我想起说：“啊！怪不得，你的汉语说的那么好，因为你有一位家庭教师，一对一的教你……”他幽默地点头笑称“是”，还说，会后还要去上海看望他岳父。他笑得那么灿烂，一脸幸福的样子。

餐叙后，回到房间，电视里正播放着四川抗震救灾的新闻。今天5月17日，地震第五天，全中国，全世界，都伸出了救援之手……我忽然想到现在流行“国学热”，不知人们是否记得孔老夫子的弟子子夏曾说过：“四海之内，皆兄弟也。”这是中国传统文化思想的精粹，两千多年前的古人就有这样宽阔的胸怀。当然，西方人也有。贝多芬的第九交响乐最后大合唱，用了席勒的诗《欢乐颂》作歌词，傅雷在《贝多芬传》将其译成“拥抱啊，千千万万的生灵……”。我也看到过另一种译法，索性译成“四海之内皆兄弟”，意思想是一样的。
