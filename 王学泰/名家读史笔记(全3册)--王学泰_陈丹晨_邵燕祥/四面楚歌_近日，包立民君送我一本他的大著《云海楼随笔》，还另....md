# 四面楚歌_近日，包立民君送我一本他的大著《云海楼随笔》，还另...

风雨微尘

四面楚歌

近日，包立民君送我一本他的大著《云海楼随笔》，还另送了一篇他的新作，是讲关于徐悲鸿与孙多慈的轶事的，颇多新意和材料，读来饶有趣味。老包是专门研究绘画文物的专家，与我曾同事十多年，著有《张大千传》《百美图》等名作。当下次再见面时，他很认真地问我有什么意见。我说：“就是有很多错字，我替你改在稿子边上了。”他听了，惊呼道：“怎么还有错字？我已经校了好几遍了！”老包落后，不会打电脑，至今仍然手写稿，写好了，送到街上的文印店去打印，然后自己校对。他是深度近视眼，戴的眼镜片像啤酒瓶底那么厚。我近年视力严重下降，几近半盲，看书读报，很是费劲，但旧习不改，每每发现错别字，或是过于文理不通的文句，就喜欢在报刊上乱改。有时惬意地躺坐在沙发上看书享受，但仍还忍不住站起来找笔，非要把那个错字勾出来才肯罢休。其实，看完那张报后，也就随手丢到一边去了，并不真的去和作者较什么劲。等到下次看报时，仍然如此。这次，老包看了我的校改，连连说：“稿子已经寄走了，我得赶紧再去信改正。哎呀，还是你的眼睛比我好。”我说：“我的眼睛也快瞎了，我这是当了三十年编辑的职业病，看什么书报稿子，有错就忍不住想动手改。”

最近读了听了许多关于纪念抗战胜利六十周年的书报电视片，看着看着，听着听着，又旧习不改。这次，想了想，把它写出来可以引起大家注意吧！

有一次，看介绍武汉保卫战的电视片。说武汉在日寇合围下，陷于“四面楚歌，危在旦夕”。这“四面楚歌”的成语用在我军被围，用在日寇围困我军，显然是极不恰当的。“楚歌”是什么，谁唱“楚歌”？这么一深想，岂不觉得荒唐！这是什么问题，是修辞学上的问题，用词不当，因辞害意。在这同一部片中，还有另一些用词不当例子，如把日寇侵占我们的城市，却用“攻克”，等等。

还看到一家报纸介绍南京大屠杀的元凶松井石根在东京大审判后被处绞刑。报道全文写得好好的，前面却加了一段“编者按”说：“……他们曾耀武扬威，风光一时，但结局也是悲惨的。”松井石根只是“耀武扬威、风光一时”吗？处死他是“悲惨”的吗？把这两者怎么能连着说成因果关系？用词不当，以至于此，真是不知让人说什么好！

问题是这样的不当出在一些高级文化人身上：编者、资深著名的主持人、撰稿人……记得前些年，有一家大报用整版篇幅请四位专家谈论“家书”，并且组织了一场音乐朗诵会，由著名演员朗诵。结果列举的范文例子中，有朋友写的，有情人写的，统统算到“家书”中去了。我很奇怪，经过作者、编辑、报社终审的领导、专家、名演员……竟都没有弄清楚“家书”是为何物！近日又见一位作家写的文章中，引一位文化老人对他说的话，说某人“深得家父遗风”。本意是想说某人的父亲，应该说是“乃父”。如今这么一来，就乱了套，真不知这位“家父”指的是谁的父亲（我深信绝不是这位老人的原话）。如果，我们的高级文化人士们，连“家父”“家书”都搞不清楚，我们当今的语文水平又该如何了呢？

2005年8月
