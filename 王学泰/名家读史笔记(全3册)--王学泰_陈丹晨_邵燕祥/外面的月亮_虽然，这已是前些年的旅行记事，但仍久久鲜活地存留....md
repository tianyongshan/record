# 外面的月亮_虽然，这已是前些年的旅行记事，但仍久久鲜活地存留...

风雨微尘

外面的月亮

虽然，这已是前些年的旅行记事，但仍久久鲜活地存留在我心中.虽然，这只是匆匆短暂的一瞥，但却成了我人生旅途中难以忘却的一个驿站.虽然，这只是一个走出铁屋子后初次看到外面的世界，但无论它是多么肤浅、微不足道，毕竟是睁开了眼睛，呼吸了一点新的空气。

在那些日子里，我总喜欢与那里的人们聚谈，尽可能多地知道他们的生活习俗，他们的心境、情绪、思索。说得直白一点，我想知道他们是怎样生活的？他们是怎样做人的？他们是怎样看世界的？其实，我很明白，这只是一种奢想。想在这么仓促的时间里，了解另一个国家、民族的人们的生活和文化心理是不可能的。就像维也纳的奥利太太对我说的那样：“人们不会把真实情况全部告诉你的！”于是，记下的，将是自己的观察、感受、印象，也就是所谓亲见亲闻，所感所思，点滴断片，掠影浮光。它一定是肤浅的，表面的，但却确确实实曾激动过我，使我的视野豁然开朗，灵魂骚动不安，精神变得丰富充实起来，对于人生有了一些新的奇怪的感悟。

其实，我走过的地方很有限，我接触的外面世界的人们也很少。正像有人说过的那样：“到国外三个月的人，可以写一本书.到了国外三年，甚至时间更长的人，连一篇短文章都不敢写了。”不知那是因为前者感觉新鲜敏感，后者司空见惯习以为常有点麻木了，还是因为前者不知高低深浅，贸然敢于下车伊始妄发议论，后者深思熟虑，颇知其中奥秘而不率尔下笔……但我还是记下了自己的影像。

当我行走在雄浑神秘的阿尔卑斯山，苍翠欲滴的维也纳森林，悠悠流淌的蓝色的多瑙河，或是奔驶在原始荒凉的落基山，蒙蒙茫茫的太平洋畔，北纬五十多度的严寒冰封的城市.当我日日看到的是，温和善良满含笑容的奥国少女，傻大白粗的纽约警察，紧张忙碌的上班族，匆匆赶路去度假放怀纵情欢乐的人们……我会想起这里是海顿、莫扎特、舒伯特、施特劳斯的家乡，那里是马克·吐温、海明威、杰克伦敦的祖国，还有贝多芬，还有爱因斯坦……满眼是鲜花、阳光、草坪、鸽子、雕塑和各种风格迥异的美轮美奂的宫殿、教堂、剧院、博物馆……

安详，平和，从容，宁静，闲适，或是紧张，忙碌，跃动，快节奏，或是健康，快乐，富有色彩，充满灵韵……都给你一种新鲜感，亲切感。

这是一个美好的时光，美好的享受。多么想使更多的朋友来分享这样美好的人生情致，就像从大花篮里采撷一束鲜花献给你。这就是我写下这些一鳞半爪的笔记的初衷。

有一次，和同事们闲聊。人们问我外出的感想。我随意讲了一些观感。旁边有一位仁兄冷冷地正色说：“看你说的，外国月亮就比中国圆？”对这样陈腐颟顸的质问，我不想躲让。我迎着他的“爱国”情结，据实据理答道：“是的，就是比我们的清晰，明亮！连人口众多的纽约的上空，都是蓝天白云。在加拿大，我还看到那云，蓝得碧绿，我才悟到古代诗词中的‘碧云天’是怎么一回事了！因为我们的空气污染太厉害。要想进步，就不要闭着眼睛讳言这样朦胧的月亮。”

当然，我多么希望看到我们的月亮，也会愈来愈皎洁、美好。我深信这一天将会到来。

2004年2月
