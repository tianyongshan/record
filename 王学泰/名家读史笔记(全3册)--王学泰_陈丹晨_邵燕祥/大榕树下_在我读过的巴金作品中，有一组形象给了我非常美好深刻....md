# 大榕树下_在我读过的巴金作品中，有一组形象给了我非常美好深刻...

风雨微尘

大榕树下

在我读过的巴金作品中，有一组形象给了我非常美好深刻的印象：明亮的红土地，耀眼盛开的龙眼花，枝叶繁茂的大榕树，新绿亮丽的含笑花叶……这是多么富有生命力的图画！三十年代初，巴金曾经先后三次来到这里小住，看望朋友。有时，在浓黑的夜晚，打着火把，走过青石板铺成的窄巷，去参加聚会，围坐在微弱的煤油灯前.有时，到散发出牛粪味和草香的小村镇去会见友人，兴奋地谈论着那些使人热血沸腾的问题和未来的美丽梦景。

这个地方就是古老僻远的福建泉州。那正是革命高潮退落以后，一些怀抱着美好理想的有志之士来此办学校。学生中有许多是侨胞和清寒家庭的子弟，得到了学习、磨炼的机会。教师们踏踏实实做教育和科研工作，只求有益于社会的改造和进步。巴金赞叹说：“在这里每个人都不会为他个人的事情烦心，每个人都没有一点顾虑。我们的目标是‘群’，是‘事业’.我们的口号是‘坦白’。”像陈范予、叶非英……许多志同道合的朋友就是在这里相识的。以至在他离开以后，他还深情地说：“我一生中最快乐的日子就是在那样的土地上度过的。”因为那里仿佛有他长久追求的乌托邦的影子。巴金后来写的大量知名的散文、小说《南国的梦》《爱情的三部曲》《春天里的秋天》等，都是以这里的生活为背景的，都是充满着青春、热情，而又带着浓厚的闽南地域色调的优秀作品。

于是，我也因此有了一个南国的梦，多么想去看看这个耀眼的红土地，古老美丽而又有一点神秘的小城。

直到最近，有机会匆匆来到泉州。但是，我看到的已是一座繁华恢弘的现代化新城市。火把、窄巷、颓垣、煤油灯……当然早已消失得无影无踪。龙眼花、含笑花、大榕树，风韵依旧，更加繁荣茂盛，庇荫绿遍了全城，满眼都是一片新兴气象，充满着生气和活力。整个城市建设，不同于时下流行的求大贪高，平庸笨重.而是红白相间的墙体、飞檐建瓴、精巧别致、独具闽南风貌的新建筑.即使是普通的民居也是美观不俗，而有地域特色。这使我不能不为这个城市主人的深远的文化眼光和情趣而惊讶赞叹。当我听说这里曾是海上丝绸之路的出发点，唐宋时曾极一时之盛，被誉为“世界第一商港”.曾通行过七种文字语言，至今还有八大宗教的寺庙.国家一级的文物保护单位，全省二十多处，泉州就占有其中十四处……当地朋友说得眉飞色舞，充满自豪感.我们听的人更是惊叹折服，庆幸此行收获丰盛，大开眼界.更深深感到这里既是开放的先行者，又是深厚传统文化的传承者，也因此更加懂得两者兼收并蓄的好处。

这时，我才开始理解，为什么三十年代初这里会聚集一大批优秀的知识分子，默默耕耘奉献.因为这是一块有着深厚文化根基的沃土。我们到黎明中学旧址访问时，周围楼群完全是新建的，只剩下一株有二百七十年历史的古榕树作为历史的见证。那老榕树枝干苍劲如虬须，枝叶高耸参天，一直延伸到两旁四层楼的房顶上。我们站在树下，体验昔日巴金在此与学生座谈时的拘谨.或与好友们谈天，看着夜色渐浓，动情的话语在晚风中追逐，使他感到自己变得年轻了，这里的生活像一个美丽的梦幻。

想到那时这些知识分子，远离繁华生活，自愿来此备尝清寒艰难，竭尽心力奉献给社会，却从中感受到了人生最大的乐趣，可谓甘之如饴。与如今俯拾皆是的腐败风气完全是两个世界、两种人生。这到底是怎么一回事呢！

现在，这里已是黎明大学的旧校区。新校区在去年刚刚完成，十几栋规模宏大、色彩明丽的新大楼已在市郊风景秀丽的宝觉山上耸立建成，现代化设施完善配套，不久即将成为有八个系，三十五个专业，五千名学生的规模。看到这样依山而筑的雄伟气势，又面向烟雨迷蒙的晋江和大海，这里的学生能在这样优美的环境里学习，真是有福啊！

令人吃惊的是，校领导告诉我们说，这个学校只有两百位教职员工，而现在学生人数已经达到近四千人，也就是将近1∶20的比例，但是，很多高校是在1∶3的比例上下。我们到黎明大学所属巴金研究所参观时，看到所长方航仙教授是兼职的，既做所里的研究、编辑工作，又负担很重的教学任务。另一位在职的工作人员在做研究、整理资料工作同时，还与另一位兼职教师共同编辑一份定期出版的季刊《巴金研究》。这是全国唯一有关巴金的专门性刊物，已经坚持出版了十三年了，受到学术界专家们的重视和好评。如果这个学校的教职员工没有奉献精神、敬业精神，那样蒸蒸日上、欣欣向荣的兴旺局面是无法想象的。真是老“黎明”的薪火传人啊！

我不能不赞叹：这耀眼的红土地！在这古老的文化沃土上盛开的美丽的生命之花！
