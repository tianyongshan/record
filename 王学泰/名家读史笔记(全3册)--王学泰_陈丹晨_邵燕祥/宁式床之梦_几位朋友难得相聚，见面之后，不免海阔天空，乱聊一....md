# 宁式床之梦_几位朋友难得相聚，见面之后，不免海阔天空，乱聊一...

风雨微尘

宁式床之梦

几位朋友难得相聚，见面之后，不免海阔天空，乱聊一通。朋友甲是位影迷，困惑地讲述了一则新闻：“有一次读一张大报，读娱乐版，读到一条报道，斗大的字作标题：《成龙痛打美国人》。我吃一惊，细读下去，才知说的是电影《红番区》。这部电影相当卖座，许多人都看过。成龙在这部影片中饰演一个武功超群的小伙子到美国遇到当地一群流氓的骚扰和欺负，被他打得落花流水。故若说是‘成龙痛打美国人’，似是，因‘痛打者’的的确确是成龙，挨打者美国人也.而非，英雄乃剧中一角色，被痛打的是一群流氓恶棍。如说做这标题是为了生意经，吸引读者眼球使用的耸人听闻的手法，又好像不甚高明，不甚贴切。当然，人们可以付之一笑，不必太认真对待。何况如今人们都讲‘玩一把’，调侃，娱乐至死……认真了反倒被人讥为迂腐，不懂幽默。但是，平心而论，这里面就没有透露出一点某种说不清的情绪吗？以前，美国电影里出现的中国人总是留着辫子的下三滥，我们很反感，认为是辱华。陈冲在《大班》里演一个英国佬的情妇，还有半裸的镜头。许多媒体奋起抨击。我不知道这里面有无什么相通的东西。”

朋友乙是位小说迷，抢过话题说：“我倒也想起另一件事。前几年，有一位周励女士到美国成了成功人士，还写了一本小说《曼哈顿的中国女人》在中国畅销走红。出版者和一些评论家郑重其事说是一部弘扬爱国主义的作品。后来又发生当地华裔商人和传媒对她的书和商业行为的批评，周女士因此声称要诉诸官司。后来怎样我就不甚了了。就在这时，我还看到这位周女士写文章讲她怎样在美国发了财，出人头地，与书中表现的踌躇满志一样。有一次，她在酒店与别人晤面，看见一群金发碧眼的白人女服务员来侍应她，使她感到那么自豪，一种成功感、幸福感、不凡感、得意感、胜利感、优越感……油然而生。这些我都能理解。这部书正是迎合和满足了那些渴望去国外淘金，一夜之间变成富翁的人们的欲望和心理。就像还有一本书的名字就叫《娶个外国女人做太太》一样。但我却弄不明白，这种情绪与爱国主义有什么关系？是过于强烈的‘民族自豪感’，还是过于卑微的‘民族自卑感’？”

甲、乙两位朋友一席话讲得大家惘然，期期艾艾不知所云。这时，朋友丙开口了。他是一位电视迷，说：“这有什么不好懂的。你们说的这些事又有什么新鲜。就说不久前晚会上的一个节目吧。一个装上了电话的农村老大娘与在大洋彼岸留学读书的儿子通话，听说儿子学习之余还要打工赚钱，十分怜惜，愤愤地且又豪情十足地说：‘等你学好了，把咱们国家建设好了，让他们也给咱们洗盘子，洗碗来！’你看，她这么一说，大家都乐了。这有什么不好，乐一乐就得了。你又何必较真呢？有一位著名学者在一篇论述传统文化的论文里引证了这个故事。说是一种粗鄙的民族主义文化观念。是不是说得太严重了。不过，话又说回来。咱们讲的这些故事怎么都有点类似的相通的叫人难以说清楚的这个那个……”

于是，朋友甲、乙、丙……都看着我：“算了，该轮着你讲点什么新闻了，就是别说这个题目了。”

我哪里来什么新闻呢！每天闭门枯坐，两耳不闻窗外事。有一次，有朋友来访，说到有一暴徒枪击行人的事情。我茫然不知，他惊讶得瞠目结舌，说：“这已是一年前的旧闻了，街谈巷议，家喻户晓，你倒真的与世隔绝了！”这会，我拗不过朋友们的厮缠，只好说：“我没有新闻，就讲一个老掉牙的故事吧，不过请你们不要厌烦。”

我讲的是阿Q的故事。我讲阿Q从“中兴”到“大团圆”的故事：阿Q从城里回到未庄，穿着打扮与先前大不同：腰间挂着的大褡裢沉甸甸的，伸出手来满把是银的和铜的，开口就是畅谈城里和乡间的孰优孰劣。据说阿Q曾在城里举人老爷家里帮忙，“那当然是可敬的”。阿Q发财而身价十倍的消息传遍了未庄，曾经和他打过架的王鬍也开始敬畏他，认为阿Q不配姓赵的赵太爷称他“阿Q”，那些见了他躲得远远的良家妇女也都争先恐后找他买便宜货。城里闹革命啦，阿Q也想造反，赵太爷这次见到他时怯生生地唤他“老Q”，赵白眼索性叫他“阿Q哥”。阿Q自己呢！他正在吐谷祠做着一场美梦，梦见未庄的鸟男女都跪在他面前求饶，赵家的财宝任其取用，女人呢随他挑选，特别是秀才娘子的那张宁式床一定要先搬过来。“土豪劣绅的小姐少奶奶的绣床上，也可以踏上去滚一滚。”……

我忽然发现朋友们都在打呵欠，或闭目养神，或低眉敛眼，似听非听。我很不高兴。我觉得他们太不尊重我了，就猛然大咳一声：“不讲了！”

甲、乙、丙……被我的吆喝声惊吓了，愣然，面面相觑，并且带着一点怒气，怨我不该惊醒他们。他们好像正在甜蜜的梦中，是梦见了自己在宁式床上打滚，还是……

1995年4月
