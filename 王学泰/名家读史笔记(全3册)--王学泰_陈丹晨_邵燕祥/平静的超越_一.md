# 平静的超越_一

风雨微尘

平静的超越

一

一轮明月当空悬挂，晴朗澄澈的银光遍洒在田野里。乡间的土路像一条白色的带子逶迤曲折延伸在远方。初春的麦田散发出诱人的清香。这样美好的天气，这样美好的夜晚，本该是青年们在田野村口嬉玩、谈情说爱的时候，也是往返城乡忙于生计的乡民们赶路回家的时光。然而，这时却是异样的死寂，没有一点声息。只有我一个人匆匆行走。好像这个清明世界只有我一个人还生存着，而我却想到了死。

前几天，就在这条土路上发生过一宗谋杀案。据说，仅仅为了抢劫行人唯一的财产——口袋中仅存的二十元钱。这些日子，这类恐怖的故事在农人之间流传很广。但在我心里却并没有引起一丝波澜，我也许有点麻木了。今天，当我独自在这条路上夜行，这个故事自然地在我脑中映现。我身上没有带什么钱，但是有一块比二十元更贵重些的旧表。如果这时有人从沟壑中突然跳出来剪径打劫，我决不会抵抗：“把这表拿去吧！”如果他还嫌不够，用凶器向我刺来，“那么，你就来吧！把我的性命一起拿去吧！”

这是七十年代初的一个夜晚。我正三十多岁，年富力强，生命之花最成熟辉耀的时期。我在少年时代就参加了革命。我曾经怀着献身文学的热情到大学里深造。我全身心投入工作，兢兢业业，尽责尽心。但是，我永远不能明白：从年轻时起就遇到过一场又一场激烈的政治运动不知到底是为了什么?每次都会看到许多同学、同事、老党员、文化界前辈……一个一个倒了下去。每次都要求我们这些青年人上阵参与凌辱、诬陷、践踏他们的活动.稍有迟疑就会被斥为右倾，甚至一起打入其中。我就是这样每次成了不是运动对象的“对象”。如今，终于轮到我自己也被当作敌人了，天天遭受“斗争”、逼供和侮辱。我感到愤怒，困惑，我觉得一个正常的社会生活不应该是这样的。

六十年代末，中央机关成千上万的干部、知识分子被强制到所谓“干校”进行变相的劳动改造。我所在的单位也在河南原是劳改农场的地方围了一大块地，建立一个以连、排、班等军事建制起来的干校。单身的“学员”住在干校里面，带家属的住在干校周围的农村里，但仍归属干校管理。我和妻、两个幼儿就栖身在一个村子的村口平房里。那房子倒是新盖的，是用土坯搭建起来的。室内墙壁凹凸不平，夹着杂草的土坯龇牙咧嘴似的到处漏风。晚上鼠群在梁椽上乱串跑动，大大小小的土坷垃就像下雨似的掉下来。我们只得蒙被而睡。两个孩子在本村小学上一、二年级。我们跟着农民出工干活：锄草、平地、麦收、打场、浇田、拉车、送粪、起圈……一年四季，什么活都干。开始一段时间倒也相安无事。

过了不到一年，又搞起了什么“清查”运动。每天要从村里到干校步行往返二十多里路，参加连队里的所谓学习、运动。我在没有任何说法的情况下被“勒令”关在干校隔离审查，不准回家，日日夜夜审讯逼供恐吓辱骂诬陷围攻。我扪心自问，清白无愧，面对张牙舞爪的攻击，口水直喷溅到脸上，手指直指到鼻梁上，虽然厌恶而愤怒，但却能保持平静，常常带着轻蔑的冷眼看着他们声嘶力竭的喊叫。我不明白，为什么要无缘无故地来迫害我呢？“文化大革命”至今，我没有参加任何造反派、战斗队的任何活动，是否因此引起他们的嫉恨，如今趁机来打击我？是否因为“文革”以来，我一直被认定是“保皇派”“修正主义路线黑干将”“修正主义苗子”等，在新一轮的运动中就要被狠斗一下？这还是号称中央派来的军宣队主持下发生的。

这样的逼供审讯持续了一个多月，当然毫无结果。那些一副君临天下的军宣队、造反派终于失去了兴趣，但又不甘心放掉我，就勒令我做最重最脏的体力劳动：拉粪车、背粮包、装运石灰包……到现在我还清晰记得那次背扛粮包的情景：拖拉机驶行到仓库前，旁边是一群年轻力壮的汉子监工，我们这些准罪犯到拖拉机的车斗旁，背起200斤重的粮包，我们像只虾似的压弯了腰。但是怎么也得咬着牙拼着全身力气，迈着踉跄软弱摇摇晃晃的步子，一步一步背进了仓库里。一天下来，骨头像散了架似的，倒在床上不能动弹。

我虽被隔离审查，但始终没有一个明确的罪名，没有一顶政治黑帽子，像那时正在穷追猛打的对象都被唤作“516”“杀人犯”“黑高参”……诸如此类的罪名，像是要安到我头上但又不明确说是这样或是那样。至于那些被认为问题重大的、日日夜夜有人看守监视，与外界绝对隔绝，连吃饭上厕所都有人跟踪寸步不离。我的被隔离却就是限制在干校范围内还可以自由活动，但没有人敢与我说话，唯恐粘连都避之不及。我也懒得说什么。所以很长时间里，我每天只说三次同样的话，就是到食堂里打饭时说一声“一个馒头一个菜”。此外，我就像哑巴似的终日沉默不语。

有一次从田里干活回来，在路上遇到一位原来的同事，他与我打了一个招呼，我也回应了他一下。旁边有一位与我也是极熟悉的毫无恩怨的人回到连队就去报告说：“他们两人有黑串联。”结果，在下次斗争会上又来逼供一番。又一次，我被派到县城拉石灰。石灰包装上车后，带队的让大家自己上街吃饭，然后集合回干校。我在街上遇到一位平时十分相知的同事，现在就住在同村紧邻。我正想托她带个口信给家里，她却一脸惊慌紧拉着儿子的手转身逃离，就像遇到鬼似的可怖。

我就生活在这种恐怖的环境里，几乎让人活活窒息。它要让人处于绝对孤立状态。如果这个人精神上比较脆弱一些的话，很可能会走上疯狂的绝路。

在家里，每天清早，还没有一丝曙色，妻就起身到井边担水，把水缸里的水灌满。然后，赶做早饭，同时也把孩子的午饭一起做好，让他们中午放学回来自己热一下吃。接着，她才招呼两个孩子起床，一个八岁，一个六岁，早早就把他们打发出门去学校。她自己则快步赶到连队去参加运动，她的编制所在连队离我们住的村子相隔十多里路，必定要经过干校本部。我知道这个情况后，常常提前到干校路口等候，佯作漫步，陪她走一小段路，说点家里孩子的情况，也说点审查我的情况。

那时，那帮军代表和造反派天天正逼迫我交代如何参加反动组织“516”，甚至诱供说：“也许你自己不知道，糊里糊涂填了一张表格，这个表格就是参加那个组织的。”在持续轮番硬逼软诱的情况下，我真的有点迷糊了，自己都怀疑起自己会不会在那些乱哄哄的日子里无意中填写了一个什么表格，我就要妻帮我回忆，有没有这回事？她一听就急了，斩钉截铁地说：“你别犯糊涂。你从来就没有填过什么表，也没有参加过什么组织。这都是骗你的。你不要上当！”

她这几句话使我一下子清醒过来。真是当局者迷，旁观者清。一个人在孤立无援的情况下是多么需要别人的帮助。那时，哪怕一句善意的鼓励，一个友好的眼神，都会把人从死亡的边缘挽救过来，在苦闷的绝境中重新坚定求生的意志。我曾经多次这样去对待过与我一样遭遇的难友，也曾接受过别人对我这样的回报。

我与妻悄悄见面的事情有过几次，侥幸没有被发现。她每晚从连队赶回家时已是夜色浓重。她匆匆推开家门，见两个孩子趴在炕桌上困倦得睡着了。炉火也灭了。孩子们那时只上半天学，白天大部分时间是在没有大人照料下度过的。因为我关在干校，也常有邻家孩子嘲骂他们是“反革命家属”。这时又冷又饿，一见母亲回来，正忙着点火烧水做饭，才使这家有了一点生气和温暖。等到一切弄妥齐，妻躺倒床上睡下已是近半夜了。

那时使我唯一牵挂的，难以放心的也就是这个家，这个被残酷的变态的政治运动搅得东倒西歪的家！

有一次，妻对我说：“小二病了！”农村里什么吃的菜都没有。我想起干校食堂有咸鸭蛋卖。那时物质匮乏，何况在乡间，连咸鸭蛋都是稀罕的限量供应之物。食堂管理员原是我手下的一个实习编辑，一个老实善良的青年，我就去与他商量卖两个给我。他沉吟了半晌，坚定地拒绝说：“不行，不能卖给你！”

这样，当然更加增添了我对家里的思念。我非常非常想念已经几个月没有看到的两个孩子。有一天晚上，漆黑的夜空中飘洒着点点的雪花，干校却在拖拉机车库放映老掉牙的电影《地道战》。平日人们根本没有什么文娱生活，这时都挤集在一起观看。四乡也有乡民闻讯赶来的。我却一个人在宿舍里发愣，看看这个有十几个通铺的房间里空荡荡的，没有一个人影。一个念头像溅闪的火星在我心底掠过：电影至少要放映一个多小时，我不正可以偷偷回一趟家吗？想到这里，我毫不犹豫地一跃而起，悄悄地走出宿舍很快穿插到田埂的小路上。

沉沉的黑夜，弥漫着风和雪。地下已很湿滑，但我却走得异常疾快。当走近村子已能看到那间孤立简陋的平房里漏泄出来的昏黄黯淡的灯光，我忽然颤战起来，猛一下推开房门，一个浑身又湿又白的“怪物”突然出现在妻和孩子们面前，他们都惊骇得喊了起来，然后又很快省悟过来，说着笑着围拢在我身边。

这时，我的身上不只棉袄前前后后都是白花花的，湿漉漉的，而且眉目脸颊上也都沾满了雪珠。他们正在吃饭：玉米粥、咸菜，倒也热气腾腾。我的到来使这间陋室一下子热闹起来，妻赶紧叫孩子们轻声点，唯恐被邻居听见。她叫我吃饭，我说吃过了。她还是盛了一碗玉米粥叫我喝，暖暖身子。我们一家人就这样面面相觑，兴奋地注视着。孩子们喜悦中还流露着一丝陌生的神情，因为有两三个月没有见到了。

那次，我们几乎没有来得及诉说什么，似乎又无需诉说什么，也不知从何说起。就这样坐了十分钟不到我就起身离去了。当他们还沉浸在喜悦中没有省悟过来时，我却已经走出屋门，仅仅几步路，就消失在漆黑漫天飞舞的雪雾中。留给他们的是：担心我路上是否平安？回到干校是否被发现受罚？

二

有一晚，干校又放映电影，是在场地上。我又像上次那样悄悄地踏上了归路。和两个月前那次风雪夜归人不同的是：如今已是春天，天气晴朗温和。但我的心情却更加消沉、沮丧。时光过得很快，半幽禁的“准囚徒”生活仍在继续。干校里照样搞运动，哪怕势头已经大不如前。一部分人出入专案组、政工组、连队队部，嘻嘻哈哈，打打闹闹，游手好闲，却又变着法整人.一部分人日出而作，日落而息，默默地受辱挨批挨斗。这样的日子什么时候才是尽头呢？

这里地处黄河故道，是一望无际的平原，却又多是白花花的盐碱荒芜的洼地，几乎与世隔绝完全听不到外面世界的一点声音。历史的钟摆好像断了发条停滞了似的不再走动。在晴朗的月光下，好像有生命的生物也为躲藏灾祸蛰伏起来，使你感觉不到一点生的气息。在这空旷的田间小路上，只有我在独自行走。我的思绪渐渐地飞舞起来。

我感到自己的生命像一团轻絮在空中飘荡，像一叶浮萍在水中流逐，没有着落，没有目标，没有意义，也没有价值，是完全不为这个社会所需要的。虽然我从未有过用自己的双手结束自己的生命的念头.但我确实想过，这时不管是什么人用什么方式来索取我的生命，我都会毫不犹豫地双手递奉过去：“拿去吧！这个毫无用处的东西！”

因此，我又是平静的。在这样厮杀鼓噪、折磨虐害声中，我是那样看轻自己的生命。我没有太多的愁苦，也没有太多的烦恼，很少激动和怨愤。这样异常的平静连我自己都难以相信。因为我自信，我活得清清白白，我没有伤害过别人，更没有做过什么坏事，我也因此有了面对各种打击和迫害的勇气。在我眼里。看那些恶狠狠、凶兮兮、好像很威风的人，可恨亦可怜，可恶亦可笑。我没有力量抵抗你们，但你们也压服不了我.你们可以强制我做这做那，我不能用语言和行动反击，但我会在心底深处嘲笑你、诅咒你、憎恨你。这就是“腹诽”，口不能言而心非之，这就是腹诽而心谤，你钳制得了我的嘴，你不能诛我的心！

我想起小时候练毛笔字，写大楷，临用的字帖有一种是录文天祥的《正气歌》：“天地有正气，杂然赋流形……於人曰浩然，沛乎塞苍冥。”这也是孟子讲的“我善养吾浩然之气……塞于天地之间”。有了这样的气，就“至大至刚”“生死安足论”，没有这样的气，就会馁乏无力。这是中国传统文化中的精华，是历来倡导的士应有的操守。我们普通人也许还谈不上那样浩然的正气，但是要求自己清白无愧怍总还是可以做到的。即使面对逆境，就有可能平静地从容地跨越过去。即使不幸倒下了，也不至掉落在污泥浊水中，也就无怨无悔。

那天晚上，在那条乡间小路上，我就这样想着想着……在途中，我并没有遇到盗贼。我又一次突然出现在家人面前。这次仍然只停留了十分钟。在妻和孩子们脸上流溢的欢愉神色中，我清晰地感到自己对生命的依恋。我还不能让它轻易地随意流失。我要好好地活下去。

三

天气渐渐暖和了。我被派到牛棚里干活。那里原来有一个劳力，是从部队转业来的一个很壮实的小伙子，现在头上也给戴了一顶“现行反革命”的帽子。牛棚里有三头牛。每天上午铡草，早晚饲喂它们吃草。天气好的时候，牵着它们到田边去放牧。我们没有饲养的经验，也没有人来教我们，只是让我们自己琢磨着做。譬如牛鼻和颈脖子上吊挂了很长很长的绳子，那是前一拨饲养员干的，我们接手后仍然照旧没想到要去改变什么。铡草是比较累的活，我们两个人轮着做，一个铡草，一个续草。铡半个小时就换手做。还担水、清扫、起粪出圈，都是每天要做的事。因为只有我们两个人，没有连部的人监工，我们心情倒也还放松。

有一天上午，我牵着牛到田里去，想把它拴在树边。因为绳子太长，绕来绕去正觉得有点为难时，那牛突然狂奔起来，垂绕在我脚下的绳子随着把我拖翻在地，绳子绕在我脚脖子上，牛把我拖过田埂，沟渠……大约跑了一百多米路才停了下来。这场面很像电影《农奴》里的一个镜头。四周的人早已被惊吓得喊叫起来。待到人们把我扶起来时，右边着地的衣衫已经破烂，胳膊、腰腿也都已被擦破，血肉模糊。到医务室上了药，包扎妥齐。医生开了假条。连部看我已无法干活，总算开恩准我回家养伤。

过了七个月的“准囚徒”的生活，如今和家人团聚在一起，分外感到温暖和欣喜。虽然栖身陋室破屋，农村医疗条件很差，再加上创伤面积大，化脓发炎，开始时还发了几天烧……如此折腾了半个多月，伤势终于渐渐减轻。这些日子不再出工干活，更不去干校参加学习和批斗会，村里干部和本机关下放干部也都不来过问我的事。所以躺在床上竟过上了一种意外的平静生活，是“文革”以来五年中不曾有过的。那时，脑子空空如也，家徒四壁，想看一本书都不可得。只有“毛选”和马恩选集，这时也实在不想再读这些“天天读”的东西了。

创伤痊愈后，我已能下地活动，假期也已超过，还要不要回干校去继续过那种幽禁受辱的生活呢？我想，我是无辜的，为什么还要自投罗网去受罪呢？原来是让我全家下放农村劳动的，现在我还是在村里随生产队干活就是。

有了主意，心里踏实平和多了。过了些日子，连队派了两个造反派来通知我回干校继续接受审查。其中一个是现任连队的头头，惯于对被审查者出手毒打的打手，又是一副凶神恶煞似的样子。

“你告诉我，我是什么问题，为什么要到干校去审查？已经关了七个月了，有了结论没有？你说清楚了，我该去就去，不该去就不去。”

“你还是回去接受革命群众的审查，对你有好处。什么问题到时候自会弄清楚。你不去，一切后果由你自己负责，你不会有好下场的！”

这时，我积蓄已久的怨愤突然爆发了出来，往日的平静、柔弱、满不在乎等全被他的无理威胁引发成一腔怒火：

“我不要你们的好处。我也绝不会再回干校连队去。你们要开我的批斗会，我随叫随到。你们给我走吧！”

两个人纠缠不清，一个做红脸，大吼大叫，几乎要动手了。一个做白脸，在一旁劝说我，说是为我好为我着想。我还是把他们赶了出去。他们像舞台上《打渔杀家》中的教师爷和家丁，一边退走，一边还回过头来指指点点嘟嘟囔囔：“你等着！你等着！”

看着他们那种气势汹汹的样子，我也曾担心会再来报复，用野蛮手段对付我。那个年头，无法无天，不讲道理，不认事实，什么残忍肮脏的事情都能做出来。我像等待什么似的，一天、两天、三天……许多天都没有声息。

过了一些日子，住在村里的下放干部接到通知都去干校听传达报告。我是有问题的“敌人”，当然没有通知我。中午时分，妻回来了，神色仓皇，进了屋子还要瞻前顾后，看看周围没有别人，才附着我的耳朵轻声说了一句话。不知是否因为她太紧张，说得太含糊声音太轻，我一点也没有听清楚，只是问：“你说什么？！”她又轻声说了第二遍：“林彪叛逃了！”我听得莫名其妙：“林彪！哪个林彪？什么出逃了？”我自说自话糊里糊涂地反问。直到她说了第三遍，并且稍稍解释了一下，我才听懂，震惊得愣怔在那里，久久说不出话来。这简直是不可思议的事情：这位伟大领袖最亲密的战友，怎么会与“叛逃”两个字联系在一起呢？几年来每天开大小会都要举行仪式，高声祝贺领袖万岁的同时，不还要祝贺这位副统帅永远健康吗？

妻对我说：“传达前先宣布了纪律：听了传达以后不许对任何人透露半点，上不告父母，下不告妻儿。如果泄密，要受严厉处分。”但是，她一进门就对我泄密违反了上面规定的纪律，可见谁也不再信那一套。但她和我还是很紧张的，因为这个事件本身太惊人，与日常宣传灌输的反差太大了，初听时总觉得有点匪夷所思！

我在豫北这个荒僻的小乡村里，终于开始感受到了外面世界的风暴、雷雨和闪电，是那样急骤、狂乱、强劲！历史老人并没有昏睡。他正在悄然前行。一个时代将要结束了！这个想法是不是有点太乐观，太简单……毕竟又跌跌撞撞勉强挣扎了五个年头。但这确是一个转折点，“史无前例”的“大革命”从此从顶峰上跌落下来，开始式微走下坡路了！

多少年被阶级斗争的各种“帽子”即罪名压得喘不过气来的人们，有了一点喘息的可能，有了一点消极的隐形的抵抗，在无形中，在不知不觉中，慢慢地……干校里的斗争、学习、劳动等愈来愈松弛散漫，运动好像无疾而终，初期有过的轰轰烈烈气势衰颓了：这个人返京治病，那个人回家探亲，有的索性请了事假离去不再回来……走的人渐渐地多起来，那些革命头头们像是被霜打了似的蔫了，有点无精打采。干校连队一片萧索景象。

当然也就没有人再来揪斗我，过问我的事。1972年底，我在干校待了整整三年后离去前，曾经逐一问过干校当时的领导：我的问题的结论应该是什么？他们都不承认审查过我，也不知道我有什么问题。经过我再三追问，他们茫然地说：“你是不是回去问问北京原单位是不是对你有什么说法。”回到北京，问原单位领导，答称：“干校发生的事，我们不清楚。我们从来没有对你有过什么审查。”

这事似乎有点荒诞，像是世界上从来没有发生过，仅仅是一场噩梦而已。不过，后来我在北京办公室的废纸堆里发现过一摞打印材料，是关于我的“罪行录”三册，署名本单位革委会编印的，是我以前见所未见、闻所未闻的。我留下了一份，但我从未对人再说起此事，我也就当作没有发生过一样，不想再追究也无从追究事情的原委。我觉得自己的生命已是经历了一次超越，感悟到劫后余生是多么可贵，幻想以后将不再会有这样的荒谬。
