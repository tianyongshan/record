# 放天灯_1987年春在芒市。

放天灯

放天灯

1987年春在芒市。

泼水节的晚上，人们又在向中心广场移动。知道那里要跳舞，人声中仿佛已有高音喇叭在播送舞曲。“放天灯啰！”这一声喊才引我跟着人群涌去。

白天在广场上，人们用柳枝点水，用桶泼水，都比不上天空那一块云，甩下麻线粗的阵雨。此刻地面已干，天作深靛色。仰头不见天灯的影子。不知天灯到底什么样儿？1951年冬我在皋兰县水川区参加土改，黄河南岸属定西县，夜间隔河望南山，有点点灯火缓缓移动，人们说是孔明灯，我也不曾深问。现在听说天灯又叫孔明灯，抱着孩子般的好奇心，挤进人堆，一圈一圈大人孩子，围着一间房子大的圆顶白帐篷似的玩意儿，有人正在下面点火，那就是所谓天灯了。

须臾，那白帐篷腾空而起，以很迅捷的速度，无声地斜刺里向青空飞去。很快就只剩帽盔大小，那摇曳的火苗，远望缩小，像一盏普通的灯了。天已黑下来，三三五五的疏星逐渐散开，这时在星的棋局中有一颗款款挪动的星，就是我们瞩望的天灯。

也许就是热气球的原理吧？遗憾我来迟一步，没看到它上天以前人们的一系列操作。这时爆出一声欢呼，第二盏天灯又凌空而上。我这时忙找第三盏天灯，想仔细看看准备活动，却没有第三盏了。

直到两盏天灯都消失在夜空深处，人们才回首人间歌舞。

我久久地陷入关于天灯的缅想中。

四十年前就听冀中解放区有点天灯之说，与我今日所见有相同之处么？

两年半以后，翻阅孙犁的《耕堂散文》，有1947年新春写的《天灯》：

……西头财主家的天灯不见了。却看见东头立起一个天灯，真是高与天齐，闪亮的灯光同新月和星斗争辉。灯笼纸上，用红纸剪贴着四个大字：“穷人翻身！”……

原先穷人家只能在地皮上点些香烛祈福，财主家才有高高立起一盏天灯来的威势。从孙犁此文来看，河北的天灯也只是立，并不飞起来，名为天灯，只是言其高耳。甘肃的孔明灯，我印象中不是固定的，而是游走，也许接近云南的孔明灯即天灯？

在云南一带，诸葛孔明的威望已经延续了一千多年，似乎不仅由于他的权变之术或所谓政治手腕或美称为政治智慧的缘故。如果甘肃的天灯竟同于滇西的天灯，那真许是诸葛亮五月渡泸，以诸如此类的奇术震慑边民之后，又带着它六出祁山，谁知有没有军事讯号方面的意义呢？反正终归难免“秋风五丈原”的结局，空留下浩叹和纪念。

像端午对屈原的纪念，龙舟、米粽都转化为欢乐的节目.孔明灯倘有纪念诸葛先生的意义，今天也融入节日的余兴中了。
