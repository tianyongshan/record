# 林荫道上_那天，我走过这个城市的著名闹市，忽然觉得一种异样的...

风雨微尘

林荫道上

那天，我走过这个城市的著名闹市，忽然觉得一种异样的寂静，这才意识到无意中走进了一条林荫道，几乎像是转换了一个世界似的。

人们都说，游山玩水，却不曾听说有玩道路的：山水游记，一般记的是山，是水，感受的是湖山之美。即使写到山径，也在山的胜景之中偶然一提。我曾经读过几本记叙描写杭州西湖的书，都是广为收集了古今名家的有关文章，竟没有找到一篇专门写苏堤、白堤的。写到它们的美，也还是离不开那湖水，不怎么注意那条道有什么引人之胜。倒是《世说新语》中，留下了这样一笔：“从山阴道上行，山川自相映发，使人应接不暇。”成为传诵的名句。

但是，城里人喜欢“逛街”，上海人叫“荡马路”。“逛”“荡”两个字都很有意味，颇有轻松闲散随意的意思，但也都包含了“逛商店”“购物”的内容，而非纯粹游逛街道。这次，我走在林荫道时，才意外感悟到了一种与此完全不同的意境和韵味。

林荫道可说是城市里的一个特殊产物，是人工修建的，是为了保护环境、美化城市，也还可以供人憩息通行的地方。当然，乡村里也有林荫道。那年，在阳关附近，大沙漠中，走访一个乡村，就看见村中央有一条宽阔的林荫大道，高耸矗立的钻天杨遮天蔽日，道旁的清溪汩汩然畅快地流淌着，使人感到这是生命之泉，养育着生命之路。相比之下，今天城市中的林荫道则显得简要小巧，别有一番情趣了。

这时，你不再听到繁杂喧闹的锣鼓声，叫卖声，小贩拉客的吆喝声，车声……你不再看到炫目的彩色的霓虹灯影，灯红酒绿的声色欢场，惨白银色的商厦……一切忽然都远离了，是那样清净、平和、安详。除了匆匆行走的过客，在这里散步的人，心境一定是特别的宁静，整个情绪必是闲散的、松弛的。

看着地上树影密匝，枝丫摇曳隐约.看着满目绿色，薄雾轻笼.隐隐还能感到幽幽清香袭来。这里没有茂盛色浓的繁花，只是一片清绿，一片清新的气氛，一片清寂的意味。似乎平平淡淡，却又是那样清清爽爽，安安静静，从中去感受愉悦，领略幸运，萌生爱意，竟足以使人微醉了。

最好的时光，是在晚春初夏，天色蒙蒙昏昧未暗之时，迷离恍惚，更多了一层浪漫的情调。树木葱茏掩映中，比道外更添了几分黯淡和朦胧。微风轻飏，似乎飘飘欲仙起来，使你生出几多幻想。你可以缓缓悠悠地漫步，默默地让思想的翅膀飞翔，让感情的河流奔泻，甚至让时间停滞，什么也不想。这时你的孤独，你的寂寞，在这个绿色的迷蒙的空间都会诗化，品尝了新的意味。

如果有一位挚友偕行，当是另一番情趣。也许轻声交谈，也许什么也不须说，已是感到心融神会。原来这个奇异隽巧的大道上，使灵魂也能自由畅游。我想起一位诗人写过这样动人的诗句：“夕阳西下/留下最后一抹余晖/涂抹在暮色苍茫中/情人相依/漫步在晚风吹来/摇曳的绿树荫里/这是一幅美丽的画面/这是春风沉醉的夜晚/渴望路有无尽长/渴望将爱到永远/在这宁静的林荫路上/逃离尘世的纷扰/今天的夜晚格外迷人/只因有爱相随/才有我欲诉无语的情怀。”

诗人走在这样的道上，自然会引发诗兴。因为这里可以感受到自然天地，听到悠远轻细的天籁，呼吸清新的正气。至少暂时让你似乎远离了污浊的尘俗。这不也是值得欣幸的吗？

2001年10月
