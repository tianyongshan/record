# 柳字第三笔_用双钩字写《九九消寒歌》：“庭前垂柳珍重待春风”...

柳字第三笔

柳字第三笔

用双钩字写《九九消寒歌》：“庭前垂柳珍重待春风”，今天是四九第三天，轮到填充“柳”字第三笔了。

人们总认为三九是最冷，才有“夏练中伏，冬练三九”之说。今年是暖冬，三九过去，四九却骤寒，午间温度也在零度以下了。隔窗外望，阳光明丽，但是可以听到风声如啸。

这时候我愿意到室外去走走。我曾经诅咒过北京的风沙，而近些年来，汽车排放废气造成的空气污染，常使我在街头走路不敢深呼吸，这种教人窒息的污染，已经使大小烟囱的煤尘小巫见大巫，风沙更不在话下。我倒宁愿冬天多刮西北风，刮得更猛些，只有这样的日子，天空不再混混蒙翳，从石景山的楼上能望见玉泉山的宝塔。走在人行道上，看行道树在风中俯仰，我以为树木也跟我一块畅快地呼吸。

黄昏，无论在居民楼区还是多平房的小胡同里，扑鼻而来的不是废气，而是千家万户晚炊的气息。我的嗅觉记忆最感温暖的，是葱花煸锅的香味了。这是熟悉的，习惯的，几十年来它告诉我的是晚饭桌上的温馨。

在海外的商业区、排档、餐馆、面包房，我也闻到过炊食的气味，那是另一种气息，多半带着浓浓的奶油味道。而在一些中国城或叫唐人街里，因为一家挨着一家，京广混杂，也总觉得有失亲切，走了几分家乡味。我想起林海音在台湾写的散文里，饱含深情地忆起了北京城里冬日薄暮从西单回家路上的糖炒栗子、烤肉宛……的香味，我想，再加上烤白薯，那确是催人“闻香下马”的，林海音凭她的嗅觉记忆，也可算深知“个中三味”（开个玩笑，不是“三昧”）了。

不知从什么时候起，虎坊桥街头也在卖“美式热狗”，也许年轻人会慢慢适应、习惯和喜欢，但我还是偏爱煎饼、锅贴。不过，我不会去攻击热狗或炸鸡，只是觉得中国的饮食业不会做生意而已。
