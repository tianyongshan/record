# 梦与痴_梦到过去的，是记忆。

梦与痴

梦与痴

梦到过去的，是记忆。

记忆中有无限温馨，值得留恋的旧梦，所以有“鸳梦重温”。

记忆中不堪回首，犹有余悸的是噩梦，所以说“噩梦醒来是早晨”。

梦到未来的，是预兆。

谁不希望梦中都是吉兆？

香港有节目，曰《好梦成真》。

人们常说，痴人说梦。说梦的未必痴。

说旧梦时，噩梦也无妨，自然有个分寸。

说梦到未来，如同说希望，谁不有所遐思？

1949年10月，畅想畅谈，谁不描摹美好的未来如梦中画卷？

假设当时，有人预言，他梦见40年后的北京城，将在戒严中度过十月佳节，谁能相信他？岂不会被目为诅咒？他会落到什么下场？

究竟谁是痴人？哪个梦是真梦？

原来痴人之痴，有的是痴傻的痴，有的是痴情的痴。两种痴，时合时分。痴傻的未必痴情，甚至无所谓情，即使痴于情，那情未必可贵.痴情的未必痴傻，痴情于中而痴傻于外，傻到不计一切，执著的倒是真情。

“侬今葬花人笑痴，他年葬侬知是谁？”知情痴为人所笑者，是痴于情而不至痴傻.但从局外人看，痴情处处流露，企求世俗中不可能的事情，则无论林黛玉还是贾宝玉，都又透出一派痴傻，为精明的世故人所不取也。

说好梦以自欺欺人，蒙痴人说梦之名而面不变色的，往往是这类精明人。

指出美梦的虚妄，自以为清醒的，其实倒多是痴人.句句说的是真话，本人却还在梦中未醒吧。
