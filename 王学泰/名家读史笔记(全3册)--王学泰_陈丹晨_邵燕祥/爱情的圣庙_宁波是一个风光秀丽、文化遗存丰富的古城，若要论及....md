# 爱情的圣庙_宁波是一个风光秀丽、文化遗存丰富的古城，若要论及...

风雨微尘

爱情的圣庙

宁波是一个风光秀丽、文化遗存丰富的古城，若要论及他的景物真是美不胜收。若去旅游，三两天内不能尽兴的。真可谓十步之内，必有芳草，游人常因此目不暇接。譬如鄞县东乡，始建于西晋，迄今已有一千七百多年的天童禅寺就是天下闻名的佛教圣地，也正是我的祖辈生息之地。我常常为此感到自豪，但在宁波却只是许多胜景之一而已。

2001年10月，我因开会之便，在家乡又走访了几个景点。与五年前来时相比，宁波的面貌已经焕然一新，变化之快之美几乎像是神话一样。但当我到梁祝文化公园一游时，我却有点怀疑，心想，会不会像有的地方那样是今人新造的古迹？但到了那里一看，却是意外的惊喜。

梁祝文化公园在宁波鄞县的西北高桥镇。它是由两个部分组成的。一部分是新建的人工公园，依据有关梁祝民间传说、故事、戏曲的内容设计布置构筑的种种场景，诸如读书院、十八相送时经过的凤凰山、小河、独木桥、鸳鸯池、夫妻井、观音堂、长亭等等。令人惊叹的是，虽说是城市公园，却有浓厚的乡土气息和农家情趣。另一部分是在原有的基础上重新修葺的梁庙和梁墓。两者竟很自然地融为一体。

大概是梁祝故事实在太感动世人了，据称当年“梁祝读书处”遗址在全国就有七处.梁山伯墓遗址就有十个.梁祝故事更是传遍各地。但是，梁山伯庙却只此鄞县一家。正宗的梁祝故事的发源地到底在哪里？鄞县古代地方志颇多这方面的记载。史称东晋时鄞县有县令梁山伯勤政爱民，兴修水利，积劳成疾，病故在姚江边。乡民因为纪念他，建庙祭祀，距今竟有1600多年了。唐宋典籍中也有提到梁山伯和祝英台同葬的“义妇冢”。前些年，鄞县发现并发掘了梁墓。无论梁墓的遗址所在地，还是墓中发掘所得的文物，与史书记载的年代都是相符的。因此，这是一个实实在在的可珍重的古文物遗址，如今把他修复扩建成一个文化公园，实在是一个很好的创意。

浩渺的姚江从公园身边犹如群龙嘘吟而过，我们站立的地方正是有名的“九龙嘘”。一边，极目远眺，竟是苍穹乡野，使人胸臆为之开阔舒畅.另一边，绿草如茵的广场上，是座高大的梁祝雕塑，正在翩翩起舞化蝶，且伴有婉转动人的戏曲演唱。这也是公园里唯一突出的有现代意味的音乐喷泉。修复后的梁庙门上的匾额书有“梁圣君庙”，庙内颇有一些旧时传存的文物.也有今人书写的碑林，赞叹梁祝的爱情.有晋代的梁山伯县令功绩图.还有后殿的寝宫布置得像洞房似的，充满着爱的气氛，这是一般庙宇绝无的。庙的西邻是梁祝墓，一墓双碑，分置前后，称为“蝴蝶碑”。但是前一块“敕封梁圣君山伯之墓”的碑，中间却是断裂的，正是传说中祝英台在雷电交加、悲痛欲绝时，跃入崩裂的坟中，留下的裂痕。

显然，这个文化公园把真实的史料文物和传说故事结合起来了。这种结合已是长久以来的事。据说梁庙几经毁损，又几经修复，香火颇盛而不绝。人们祭祀他已不仅因为他的政绩，而是因为他们的爱情。传说故事如此长久广泛流传，为世人赞叹，也是因为他们的爱情。从来拜神求佛的，或求富贵，或求延年，或求子嗣……立庙的，或显威权至高无上，让人们匍匐听命，或是慈悲为怀，普度众生……现在城市公园遍及天下，但以爱情为主题的公园，似乎还是头一遭，这也是因为他们的爱情。

上个世纪的中国人经历了挣脱封建专制，争取婚姻自主的人性、自由的启蒙，然后是以政治标准第一（出身、成分、政治面目）作为爱情婚姻的前提。接着是世纪末到新世纪风行爱情婚姻商品化……总之，功夫在情外（仿陆放翁诗句：“功夫在诗外”），就是没有爱，没有情，甚至嘲笑“真情”，不知“真情”为何物，不信天下有“真情”。其实，人们从心底深处又是那么尊重爱情，追求爱情，歌唱爱情，喜爱梁祝故事，从中寄寓对爱的渴望。
