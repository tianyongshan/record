# 英雄观_于右任1924年《读史三首》，其一云：

英雄观

英雄观

于右任1924年《读史三首》，其一云：

风虎云龙亦偶然，欺人青史话连篇。

中原代有英雄出，各苦生民数十年。

或以为诗中的英雄是反讽，我以为正是直陈。见诸史籍的，最早以英雄自称和听人这样恭维就默认下来的，是青梅煮酒的曹操和刘备。“英雄割据今已矣”，把割据一方的军阀叫作英雄，才是这个词的正义。

在欧洲，不管是Hero也好，什么也好，历史要早一些，希腊罗马神话中就有英雄。而我们古代神话中与之相当的形象，则除了神仙，便是皇帝，要不就等而下之，近于怪异。在神仙以外的尘界，在皇帝以下的民间，有所谓英俊，智过万人者为英，智过千人者为俊，这是孟轲的话，也许因为这位夫子是文人，虽能利辩，却是动口不动手的，他所谓“英”，也“雄”不起来。

看来中国的英雄多草莽气、血腥气.欧洲的英雄也不是吃素的，但笼罩着宗教气氛。

这是就原来的意义而言的。后来英雄就有了引申义.角度不同，解释各异，于是不但杀人的权力者是英雄，被杀的牺牲者也可能是英雄了。

杨绛为《傅译传记五种》写的代序中，说到《贝多芬传》《米开朗基罗传》《托尔斯泰传》同出罗曼·罗兰之手：

……贝多芬供大家享受的音乐，是他“用痛苦换来的欢乐”。米开朗基罗留给后世的不朽杰作，是他一生血泪的凝聚。托尔斯泰在他的小说里，描述了万千生灵的渺小与伟大，描述了他们的痛苦和痛苦中得到的和谐，借以播送爱的种子，传达自己的信仰：“一切不是为自己，而是为了上帝生存的人”.“当一切人都实现了幸福的时候，尘世才能有幸福存在”。

杨绛说：“罗曼·罗兰把这三位伟大的天才称为‘英雄’。他所谓英雄，不是通常所称道的英雄人物。那种人凭借强力，在虚荣或个人野心的驱策下，能为人类酿造巨大的灾害。罗曼·罗兰所指的英雄，只不过是‘人类的忠仆’，只因为具有伟大的品格.他们之所以伟大，是因为能倾心为公众服务。”

这里从罗曼·罗兰对英雄的解释，提出了英雄的道德标准。

依照中国一些马克思主义者对唯心史观即英雄史观的批判看来，这里所谓英雄是指“杰出人物”或“卓越人物”，实指活跃在政治舞台第一线上的人物。近年来，则在反对“英雄创造历史”的史观的一派中，又出现了对“人民群众创造历史”质疑，认为应是“人民群众和英雄人物共同创造历史”的说法。争鸣气氛一度有所活跃，是好事不是坏事。

我以为在争论各样史观之前，也许先谈谈各自的英雄观是必要的。

除了历史学家以外，最好也请语言学家介入：弄弄清楚各种语系的哲学、历史著作中的，包括我们四十多年前奉为马克思主义中心教材的《联共（布）党史简明教程》中的，汉译为“英雄”两字的，到底都指的是什么东西。

记得有过一个“爱国主义”宣传提纲，历数几十位中华民族历史上的杰出人物，除了有些科学家、文学家以外，也还颇有几位皇帝，大概都是所谓“英雄”了吧。给我的感觉是像安排政治协商会议的名单，着眼于代表性者多，难免口径不一了。
