# 赵树理的两本书及其他_一

风雨微尘

赵树理的两本书及其他

一

我对赵树理没有研究，但我的藏书中有两本赵树理作品值得一说。这就是香港海洋书屋印行的《李有才板话》和华东新华书店印行的《李家庄的变迁》。

《李有才板话》是由作家周而复主编的“北方文丛”第二辑之一，初版于民国三十五年（1946年）12月，我手里的这本是第三版，印于1947年10月，印了一千册，连同前两版不到一年之间共印了四千册。里面共收赵树理小说四篇：《孟祥英翻身》《地板》《小二黑结婚》《李有才板话》。还有附录：周扬的《论赵树理的创作》、郭沫若的《读了〈李家庄的变迁〉》、茅盾的《关于〈李有才板话〉》和主编周而复写的《后记》。

那时，香港有一批左翼文化人聚集在那里从事党的文化宣传，如编辑出版《大众文艺丛刊》等，宣传毛泽东的《讲话》，批判胡风、沈从文、萧乾、朱光潜等。周而复编印的这些书则是从正面宣传，都是解放区的新作如李季的《王贵与李香香》、孙犁的《荷花淀》、柯蓝的《洋铁桶的故事》、艾青的《释新民主主义的文学》等，前后印了数十种，用香港海洋书屋名义出版，然后发行到国统区公开发卖。

那时上海的大小书店集中在四马路（福州路），但是霞飞路（今淮海中路）和吕班路（今重庆南路）之间曾经有过一段书店热。抗战胜利不久，上海中共地下党就在培文公寓的东侧开了一家生活书店门市部，两开间门脸是开放式的，图书刊物多得摆到店门口。店堂里读者人头攒动，热闹非凡，可谓盛况空前。那里卖的许多书是别的书店买不到的，其中就有地下党编辑出版的政治、时事、文化等图书和刊物。八十年代有一次我与中国社科院文研所长许觉民开玩笑说：“那时我在书店看见你，你不理我。”他瞪大着眼睛好奇地反问我：“真的！”其实我只是一个中学生，因家住在附近，常去书店倚着书架白看。书店里的人从来不会干涉。觉民是生活书店老店员，抗战初期十六岁时就参加地下党的老党员，长我十岁左右，怎么会认识我这样的小孩呢？我是说着玩的，他是老实人就认真了，听我说穿了就大笑不止。

生活书店吕班路门店维持了两年左右就关闭了。过了一些时候，就在生活书店原址的马路对面，吕班路东开了一间单开间的书店。重庆北路口霞飞路上也开了一家小书店。两家书店相距仅一个十字路口，遥遥相对。它们的名字我忘了，门脸不大都不起眼，有什么背景我迄今也不知道。但是我发现他们也卖一些外面不太容易买到的左翼的书，包括郭大力、王亚南合译的《资本论》等。我的这本《李有才板话》就是在这里买到的。

我那时爱好文学，已经读过许多中外名著，看到《李有才板话》觉得非常新鲜。即使像《小二黑结婚》似乎就是写自由恋爱，但与我平日读过的小说完全不同的是，那浓厚的乡土气息和生活情味，清新简洁生动的语言，展示了一幅幽默开朗、别样的社会风貌，令人耳目一新。这也许因为我个人思想已经左倾之故，所以容易接受。但也说明赵树理的作品当时就能征服城市青年的心。与此同时，也还可以看到那时党的文化渗透相当厉害，像这样的书包括有周扬、艾青、赵树理等中共文化高干、名人的名字竟然可以公然摆在书店的书架柜台上。香港海洋书店等的左翼书刊都设法或秘密进入国统区，载有郭沫若的《斥反动文艺》的《大众文艺丛刊》，我也是在这书店里读到的。可见国民党的文化统治是很不严密的。赵树理的作品虽然是正宗的文学小说类，但在党的领导心目中是把它当作宣传党的形象、党的政策思想的工具，即毛泽东所说的文化军队，是用来“团结人民、教育人民、打击敌人、消灭敌人”的武器，这才是被称为“赵树理方向”最基本的缘由。

我买到另一本《李家庄的变迁》时，上海刚刚解放。此书版权页上只有出版印行的华东新华书店名字，没有出版的地名，时间是1949年5月，印数是11000—41000，也就是说这是第三版印了三万册，数量很大。是用土纸印刷的，很粗糙，前后页几乎可以透视印刷油墨的黑影。那时上海战役正在进行，那些接管上海各行各业的人员集中在京沪线上的丹阳，新闻出版方面也都是军事化的编队待发，华东新华书店也是随军行动的一部分，在那里突击印刷各种图书刊物宣传材料，赵树理的作品也就成为其中的一部。这本《李家庄的变迁》随着大军进入上海出现在书店里，我就是在那时买到的。虽然纸质较差，但作为一种在特殊情况下印刷的版本来说，还是很稀罕珍贵的。

二

对于自己的文学作品被作为政治宣传读物进入读者手里，赵树理本人显然也是高兴的，因为这与他的创作初始意愿是一致的。如他所说，早年就是做“农村宣传员”工作的.后来“转业”写作，他对自己的要求一直是：“我写作品的目的就是要政治上起作用，人民大众能看得懂。”这句话包含了两个内容：一是起政治作用，一是老百姓能看懂。他不像别的作家喜欢强调作品的文学性而不太愿意说成政治宣传品。恰恰在这两点上，赵树理的创作符合了毛泽东在1942年延安文艺座谈会上的讲话精神，他叙写表现的四十年代太行山边区农村生活面貌正符合那时政治需要，但他也非常强调作品的艺术形象和艺术感染力，是与一般政治宣传不同之处。

近几年，我常读到《中国赵树理研究》，是中国赵树理研究会和山西省作协主办的刊物.对赵树理生平和创作有深入持续的研究，颇多新意和创见。譬如历来认为赵树理是在学习了毛泽东的《讲话》后，才写出了如《小二黑结婚》等作品的。他们经过充分查证历史事实，发现“事实上，赵树理是在不知道《讲话》的情况下创作出了《小二黑结婚》”，是两者不谋而合，不期而遇的成果。赵树理的创作是从生活实际出发，那时他正在家乡太行山一带活动，对晋冀鲁豫边区的民风世情变迁十分熟悉。他又立志要写农民都能接受的通俗文学作品，以一个党员作家的政治思想观念观察社会人情。所以他说的政治作用与那时党的政治宣传要求是相一致的，他的作品被当作宣传读物也就一点不奇怪了。

我在文学界打杂多年，却无缘见到过赵树理。只听说1949年进城后，赵树理在创作和文坛上开始有点磕磕绊绊、起起落落，不像之前那么顺遂。1956年2月至3月间，正是政治空气相对宽松的时候，中国作协举行第二次理事会，意在纠正创作中的公式化概念化等倾向，鼓励和发展文学的繁荣。周扬在会上作报告宣称茅盾、老舍、巴金、曹禺、赵树理是“当代五位语言艺术大师”。当时人们对此说盛传引为美谈。因为这是一次很罕见的也是唯一的一次对部分作家冠以如此高的赞誉。对赵树理来说尤为重要，与以前被称为“方向”不同的是这次是从文学艺术角度的肯定和赞扬。而且，前四位都是比赵的资历老一辈，且又都是在国统区活动创作的.丁玲已在半年前受到严厉批判而被废黜，这样赵树理就成了解放区来的贯彻毛泽东路线的唯一代表，这就非同寻常了。我以为此事实际上使赵树理在文艺界的地位到达了顶点。后来人们习惯称“鲁郭茅、巴老曹”正是从此而来的。那是被看成党的权威评价和编排的名次座位。但这个“顺口溜”对赵却又忽略了，不知实际原因为何？

我想可能是赵树理后来的政治思想和创作愈来愈跟不上趟。赵树理有他自己的政治追求，就是从生活实际出发而不是依傍上级政策理论图解生活。他熟悉农民的生活和思想，热心关注农民的命运，他相信自己眼睛所看到的真实图景，他常常把自己的作品称作“问题小说”是因为从农村实际工作中发现体验到的。他写作的成功在于此，受到责难也在于此。尤其是1958年“大跃进”及其所造成的灾难性后果使他完全不能接受。他写的作品与愈来愈高调、愈来愈极端的文艺指导思想不相协调：如要求把与农民间的矛盾都写成阶级斗争，要歌颂光明不许触及所谓阴暗面，要写英雄人物发展到不许写缺点以至要写高大全，要粉饰颂扬浮夸风共产风等诸如此类，使他的思想和创作渐行渐远。1962年大连会议十几位作家聚集一堂对农村曾经发生过的灾难有很严肃的揭露和抨击，为农民的苦难发出强烈的呼唤，表现了中国作家们的良知和勇气，在人民疾苦面前没有闭上眼睛。否则作家整体的沉默失语将使中国文学蒙羞留下耻辱的记录。所以，对这次会议似应给予高度的评价，是中国现代文学史上一次具有历史意义的重要活动而载入史书。也正因此，作家们后来受到严重的追查和迫害。唐达成作为工作人员在会上只是做记录而非正式代表，也被作协书记刘白羽指控会议吸收有政治问题的人参加的根据之一，因唐曾是右派尽管已摘了帽，也为此被逐出京城去了山西。赵树理后来也为此离开了北京回到家乡。

唐达成前后与我多次共事，我听他讲过许多关于这次会议中的事。1978年和1979年之交，我为了写大连会议的翻案文章（刊于《文艺报》1979年第3期）曾从中国作协借阅过有关会议的档案材料，看到会议记录都出于唐达成和涂光群两人之手，记得非常详细，密密麻麻的字迹记了好几本。与会的作家们都说到赵树理在这几年里不为左倾浮夸之风所影响，而是坚持求实精神，创作写实的作品，是作家中不多见的。后来许多有关传记和文章也都记述了赵在会上的主要发言内容，此处略去不赘。只想摘引周扬在会上的话：“中国作家中真正熟悉农民、熟悉农村的，没有一个能超过赵树理。他对农村有自己的见解，敢于坚持，你贴大字报也不动摇。”主持会议的中国作协党组书记邵荃麟则说：“这个同志是不会刮‘五风’的。在别人头脑发热时，他很苦闷，我们还批评了他。现在看来他是看得更深刻一些，这是现实主义的胜利。”两位文艺界主要领导讲的话不是虚与委蛇的官话，而是由衷地从政治、生活和文学不同角度赞扬肯定了他。但是，这个会议是在1962年8月进行的，就在这同一个月，毛泽东在北戴河发出了“千万不要忘记阶级斗争”的号召。两个会真是南辕北辙了！我那时听到这些不同的传达声音颇为困惑不解。

1966年4月“文革”的锣鼓已经敲响，中宣部分工领导全国文艺批判的林默涵副部长在文联礼堂对文艺界做了一个很长的动员报告，刘白羽和张光年主持会议。笔者有幸叨陪末座，聆听了那个疾言厉色的重要讲话。林大谈国内外严重的阶级斗争形势，反复号召矛头对着夏衍、田汉、阳翰笙、齐燕铭、邵荃麟等一大批“反动的资产阶级的修正主义”的文艺界头面人物进行彻底的大批判.此外就是点了赵树理的名，批评说：“专业作家思想落后于生活现实，赵树理就是一个例子。解决的方法就是突出政治，突出阶级斗争，兴无灭资。专业作家应该积极参加阶级斗争，反映、推动阶级斗争……”所幸那时对赵树理还是被当作“人民内部”来要求的，不过赵已不能像当年那样成为符合上面要求的宣传工具了！

“文革”时，原来领导文艺批判者如林默涵后来也成了被批判者。原来早已被揪出批判的关的关了，死的死了，残的残了……包括赵树理惨死，等等，这里也都略而不说了。且说“文革”后，我写的为大连会议、“中间人物”论翻案的文章初稿，因为有不同意见，《文艺报》于1979年3月1日郑重其地事召开编委会专门讨论拙稿。已经复出回到领导岗位的林默涵在会上再次坚持认为当年的批判还是对的：说那时确实听说邵荃麟提倡写“中间人物”，反对写英雄人物，至今认为还是错误的，当时批判没有错.同时又说到赵树理：“大连会议说赵树理是最好的榜样。赵树理在文学上有不可抹杀的成绩，但确实在新的历史时期跟不上。在社会主义时期，他不是最好的榜样。《三里湾》写了很多落后人物，但写先进人物就苍白无力。这正是他的缺点。为什么大连会议要以他为榜样呢？”这些话比在“文革”前夕大会上说的似乎稍微平和些，但意思是完全一样的，仍然认为赵的政治和文艺思想跟不上极左的政治思想政策要求，为赵不能成为当年一样的宣传工具而悻悻然。

林默涵的批评是很有代表性的。这种意见在1949年后一直困扰和纠结着赵树理以及所有作家。赵树理的兴衰荣辱也都与此有关。赵树理已经作古不能为自己辩解。但是，他生前常说的：凡是规定必须写的政治性大话，“我是不写的。这明明是假话，就冲淡了！”“我没有胆量在创作中更多加一点理想，我还是相信自己的眼睛。”文学创作不能脱离实际，“生活中没有的东西，你硬要编造出来，就失真了。”这说明无论是被当作宣传品的前期作品，还是1949年后被认为落后了，赵树理的创作态度是一贯的，坚持自己的独立思考，不随大流不媚俗，不为权势不为钱，正是一位优秀作家最可贵的精神品格，在那时不多见，在今天又何尝不仍然有着重要现实意义，值得人们深思。
