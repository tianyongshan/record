# 青青的，清清的_很久以来，有一个童年的梦，一直长存在我的记忆...

风雨微尘

青青的，清清的

很久以来，有一个童年的梦，一直长存在我的记忆中。那是一个关于西湖的梦，一个美丽的梦。

抗战前夕，世道还比较平静的年代。我还是五六岁的孩子，曾经随年轻的父母到杭州春游。我们泛舟在西湖，坐在一只张着遮阳布篷的小船上，悠闲地慢慢地随着湖水荡漾，漂浮……虽然是春光明媚的日子，游人似乎也不少，但浩渺的湖波与辽阔的云天连成白茫茫的一片，我们似乎正拥抱着这整个空蒙的世界。我看着这水，那样清澈，透明，像是晶莹剔透的玻璃，连湖底的卵石都隐约可见。这清清的水，这清清的天地，这清清的空气……在我眼里变得那么神幻美好，从此深深地烙印在我那幼稚无知的心中。

在后来的岁月里，我无数次听到人们谈论西湖，从诗文里看到对西湖的赞美，但我却没有机会再来看访她。那个清清的静静的世界，只是活在我的记忆中，成了一种期待，一种梦幻，一种追寻。直到年届半百时，我忽然有了自由出行的时候，才重到这个儿时流连过的地方。我像要去觐见一位公主，不！更准确地说，像是去见一位青梅竹马但又分别已久的女孩似的，那种紧张、忐忑、激动的心情，做着种种设想和可能，那份不安连我自己都感到怪怪的。

到达杭州那天，入住华侨饭店，已是天黑。我略安顿一下行李，就急急匆匆出门先去到六公园湖畔，没想到湖面上竟是一片漆黑，夜雾笼锁，浓得像幕墙一样密不透风，连星星点点的灯火都没有……奇怪的是，我不仅没有懊恼，反倒更增添了一份神秘感，更多了几许幻想。

然后几天里，我当然踏遍了环湖的山山水水，饱览了仙境似的美景秀色，真的迷醉了：是在我儿时的梦耶？还是老来的现实耶？

那天，我独自漫步到苏堤，天上正潇潇洒洒地飘拂着蒙蒙的细雨，我竟毫无想打伞的意思，不愿让任何东西挡住视线和兴致，觉得在雨帘中走在这绿色葱茏遮天，垂杨烟柳夹道的堤岸上，润物细无声，是再美妙惬意不过的了。也许是我的偏爱，眼中的绿总是那么年轻娇嫩，一年四季好像永远都是充满着青春生机。无论浅草垂柳，还是雪松、广玉兰等常青树，碧桃、海棠、月季……这些春花，在粉红艳黄色彩缤纷的花团锦簇中，却夺不去这青青翠翠的绿色，它是这里的主色调，且还是柔情绵绵，温润汩汩，新鲜而生意盎然，像一条碧玉如意，又像翡翠珠链静卧在西湖碧波上，不，用任何珠宝来形容她都是不合适的，她没有半点浮华富贵世俗之气，让人感受到的是幽雅恬静，藏在深闺，但又是天生丽质，自然隽永，一边是里西湖如庭院深深，宁静得沉稳慵懒，一边是白浪堆雪推涌着三潭印月。她就那样安详轻盈地走着自己的路。

晚钟，残照，莺啼，鸟鸣，碧云天，芳草地，青青湖畔草，郁郁堤上柳，烟雨空濛，层林尽染，疏影横斜，暗香浮动……慢慢地尽收眼底，缓缓地呼吸回味，清爽纯净温润幽香的空气，即使不醉人人也要醉了。在这绿色长廊里，最好是独行，没有烟火尘嚣，没有车马喧闹，没有人声嘈杂，这里是最清寂的了，只有你自己，遨游在这无声平和的天地里，随你怎样想象，怎样诉说，怎样舞动……你尽可放浪于这美的世界里，尽情地享受。你踏着青苔斑斑的湖石，你走过映波、锁澜……石拱桥，你从“花港观鱼”身边翩然走过，你又悠悠地走到“曲院风荷”的门前，游兴正浓余韵满怀却也顾不上旁骛了。

从此，每次来杭州，即使再匆促，无论春秋，也一定来翠堤，似乎是让自己的灵魂憩息，浸润神游在如诗如画的意境中，畅快地作一次美的享受。每次都是在雨丝风片的拂抚中悠然漫步，目不暇接。有过一次，有车送许多朋友到跨虹桥边，天晴朗，欢声笑语，高谈阔论，一时间惊扰得翠堤花容失色，至少我已意趣全无。也还有一次，我和内子一起赏玩，相约彼此默默不语，近在咫尺，却兴趣盎然，别有意味。游苏堤这样的清寂纯美的胜景，是要用心去体验，要用一颗虔诚的除却尘俗的心去欣赏，与游岳庙、灵隐是完全不一样的，更与西湖周遭的市廛商贾权势功利无关，与那所谓“销金锅”样的生活更不相干。

我到西湖还总会有重温桑梓之情的感受，就像在苏堤游走那样自然从容。最使我感动难忘的是二十年前，我正遇到人生困惑苦闷之际，西湖向我伸出了亲切的友谊的手，尽管有很多友人关心我，但唤我来静心舒散，寻找精神皈依的，是西湖。就像我把西湖长存在我记忆中一样，我永远记得那次的赐予比什么都更珍贵。尽管它一点也不张扬喧嚣。

我的翠堤我的梦，永远都是清清的，青青的！愿人情友谊也是永远青青的，清清的！

2009年2月
