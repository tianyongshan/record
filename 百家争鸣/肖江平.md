# 肖江平  





\ 
北大法学博士，清华管理科学与工程博士后。现为北京大学竞争法研究中心主任。兼任国家工商总局市场监管专家委员会委员，中国法学会经济法学研究会秘书长，ICC中国竞争政策委员。

主持国家市场监管总局、国家发改委、商务部、农业农村部、原国家工商总局、原国家质检总局等主管机关反垄断法、反不正当竞争法以及市场监管等领域课题20多项，直接参与反垄断法、反不正当竞争法立法或配套立法工作，参与竞争法领域大量的立法、项目论证，参与最高法院、国家发改委、市场监管总局、商务部重大疑难案件咨询讨论等工作。■
