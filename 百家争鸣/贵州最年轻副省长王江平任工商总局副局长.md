# 贵州最年轻副省长王江平任工商总局副局长  





\ 
【财新网】（记者 林韵诗）再有一名地方干部上调中央部门任职。国家工商总局网站更新显示，今年51岁的贵州副省长王江平已任该局副局长，在五名副局长中排名第四。

2015年12月，已有至少六名地方副省部级干部进京履新。除了王江平，福建、吉林两省原省委组织部长姜信治、齐玉转任中组部副部长，黑龙江高院原院长张述元转任最高法副院长，西藏自治区政府原副主席王双全任中央政法委副秘书长，江苏原副省长徐南平任科技部副部长。

王江平是四川渠县人，曾任职于国有磷化工企业瓮福集团（前身为贵州宏福实业开发有限总公司）超过20年.2011年由商入仕，从瓮福集团总经理转任贵州省经信委主任。

2013年初，49岁的王江平升任贵州省副省长，成为副省部级干部，分管工业经济和信息化建设、国有资产管理、工商、质量技术监督、安全生产、民营经济发展等，在八名副省长中排名第六，且是该省领导层中最年轻的一员。

12月29日晚的贵州新闻联播还播出了对王江平的采访，由工信部主管的《中国电子报》12月25日也刊发了对王江平的专访，他在两个采访中均重点谈及贵州“百千万”工程。

王江平表示，全国经济进入新常态，工业经济产能过剩比较突出，贵州的情况也是这样。传统产业面临过剩的，面临结构调整和转型升级，但是新的产业还没有培育起来，这些矛盾迫切需要政府力量推动解决。

他认为，政府在优化要素配置、淘汰落后产能等方面可以有所作为，政府要通过精准调度、服务企业，降低企业的商业成本，“‘百千万’工程能很好地解决这些问题”。

12月20日，王江平还出席了“国酒茅台2015年度经销商联谊会”，对茅台2016年的工作提出“加大研发投入，提升自主创新能力”等建议。

目前，国家工商总局领导层为一正五副，局长张茅，副局长刘玉亭、马正其、甘霖、王江平、刘俊臣.纪检组长刘实。其中甘霖、王江平、刘俊臣均为“60后”，王江平是其中最年轻的一员。
