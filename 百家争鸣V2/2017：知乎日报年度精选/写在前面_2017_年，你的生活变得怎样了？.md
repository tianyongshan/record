# 写在前面_2017_年，你的生活变得怎样了？

知乎周刊 200

写在前面

2017 年，你的生活变得怎样了？

这一年，共享经济热潮来了又散去，勒索病毒爆发，乐视王国分崩离析.城市中产们忙着消费升级，直播平台里却在上演着另一个世界的荒诞剧.佛系、油腻与「中危」之中，年轻人的身份焦虑被一遍又一遍掀起.而儿童读本与性骚扰事件，终于让公众开始对性教育脱敏……

也有一些或平凡或耀眼的生命在令人不安的昼夜中消失。

和一首适逢其时的老歌一样，我们也相信，一篇文章有锚定记忆的力道。从 2017 一整年知乎日报发布的近 8000 篇文章中，我们选出了最值得回顾的 12 篇，愿你再次重温时，还能想起当时的震动和思索。

现实并不完满。好在，身处队伍之中，仍能保持自己声调的人始终存在。此刻，我们把他们的视角和声音转交与你，让被错失的提醒复位，也让尚未冷却的讨论继续辐射出余温。

2018 年，希望再与你一起观看这个处于剧烈变动中的时代。
