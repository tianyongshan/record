最近在学博弈论，我也来讲一个很简单的问题：为什么肯德基麦当劳总是一窝蜂地出现在市中心，郊区却被一些杂牌军占领。

博弈论里面有一个很经典的

豪泰林（Hotelling）价格竞争模型

，这里介绍其简化模型。不想看推导过程的直接 pass，后面有解释以及一些推论。

假设有一个长度为 1 的线性城市，消费者在区间[0,1]里均匀分布，且密度为 1。商店 A 开在 x=0 处，某商品售价为 p_{1}.商店 B 开在 X=1 处，某商品售价为 p_{2} ，两者的成本都是 c。消费者购买商品付出价格成本 p 和旅行成本，旅行成本单位为 t，与距离成正比。

于是，在 0 和 1 中间可以找到一点 x，使 x 左边的都去商店 A 购买，x 右边的都去商店 B 购买。

简单地说，我们假设人们买东西只要投入购买成本和旅行成本。于是，当 A 和 B 卖出的商品一模一样时，人们会选择就近的商店购买。以城市中点为分界线，左边的人（阴影部分）到 A 消费，右边的人到 B 消费。

在这个时候如果 A 要赚更多的钱，它的占优策略是把商店往右边移动。

商店 A 往右移动，位于中间的人发现他们离商店 A 的距离更近了，便更倾向于前往 A 消费，均衡点 x 也会往右移动。于是在双方价格相等的情况下会有更多的人买 A 的商品。

商店 B 在 A 的竞争压力下的反制措施便是向左移动，以争取更多的顾客，

最后的均衡情况便是两者都搬到 x=0.5 处。

当 A 和 B 都在 0.5 处时，所有人前往 A 和 B 所花费的旅行成本相等，此时两者的客户总数相等，都为总人数的一半。

而在实际中，一个城市多半是市中心人口多，城乡接合部人口少。不妨用正态分布进行拟合。

明显可以看出，越往中间人口越多，市场越大，商家越有动力往中间跑。

A 也可以通过开连锁店夺取主动权。（以下都为基于豪泰林模型的推论）

A 在 B 的旁边开了家分店，把 B 包在里面，此时绝大多数的人离 A 更近，倾向于前往 A 消费，B 的经理哭晕在厕所。

商店 A 和商店 B 的博弈显示了在同一个维度下进行竞争的两个企业最终会走向同质化。

在案例中 A 和 B 的竞争维度是空间，最终两家店会开到一起。在现实中，我们会发现肯德基麦当劳总是挤在市中心，因为相较而言，靠中间的位置能辐射更广的区域，吸引更多消费者。而连锁店则能挡在对方店面前面，更便捷地服务消费者。两者在不断的博弈之下，最终便会发展为「贴身肉搏」的形式。（啊咧？你在这里开了家店？那我在马路对面也开一家）

于是，在 0 和 0.5 中间可以找到一点 y，使 y 左边的都去商店 C 购买，y 右边的都去商店 A 购买（为简便起见移去商店 B）。

C 的成本高，价格高，但它选择偏远的地方，使得附近的人宁愿花更高的价格，也不愿走更远的路。

」这个案例中，人们也可以跑到山下去吃 5 块钱一份的盒饭然后再跑上来呀，但考虑到那么远的路，不还是乖乖掏腰包了。

而商店 C 的占优策略是把商店往右移，但它不可能移到城市中心处，那时它过高的成本使其完全处于竞争劣势。

这个时候如果有规模更小、成本更高的商店开张，它们也将选择更偏远的地方，靠节约顾客旅行成本弥补价格劣势。

商店 A 和商店 C 的博弈显示了在有市场先行者的情况下新的竞争者需要靠面向小众获得市场，也就是我们通常说的利基市场。

在案例中 A 和 C 的竞争维度是空间，于是 C 靠降低周围居民的旅行成本弥补在自身产品成本上的劣势。在现实中，我们在小县城往往能看到诸如「麦德豪」之类的没名气、口味差的汉堡店，它们靠距离弥补在竞争上的劣势。

这个模型还能解释很多其他方面的问题。

豪泰林模型的一大特点就是除了空间维度以外，还能被赋予其他维度，而且衍生之后可以多维度共同进行博弈。

比如口味，你会发现无论什么菜系的菜肴进入其他地区之后口味都会向当地的口味产生偏移，很多饮食连锁点在进入新市场时也多有本土化的努力。

比如在苏南，人们相较于辣味，更偏好甜食。
