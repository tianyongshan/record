# 导读：_营养师小吕指出，大豆是一种营养价值极高的食材，其营养...

知乎周刊

导读：

营养师小吕指出，大豆是一种营养价值极高的食材，其营养价值不仅体现在蛋白质方面，大豆的膳食纤维含量也非常高。可无奈大豆存在多种抗营养因子影响其在体内的消化吸收，所以人们发明了针对大豆的加工工艺。

——————

舌尖上的中国第一季第三集，名字叫做「转化的灵感」。第一个讲的就是大豆，其中有一段解说词是这么说的：

「在已知的豆科食物中，大豆是蛋白质最丰富，也是最廉价的食物来源。可它早期的境遇一度十分尴尬。煮熟的大豆，无法引起人们的食欲，而且使肠胃大量胀气，迫切需要寻找进食大豆的最佳方式。」

图片来源于：舌尖上的中国第一季第三集

舌尖上的中国对于大豆的这段描述突出了这个「天然食材」未经加工时非常严重的两个短板：不好吃、难消化。分分钟打我开篇说到的「天然、加工程度低」食材更健康的脸。

那大豆到底是怎么回事呢？它特殊在哪呢？

通过和大米、猪肉的对比，我们会发现黄豆的营养价值不仅体现在蛋白质方面（远远高于大米等谷类，与猪肉相比都毫不逊色），其膳食纤维含量也非常高，另外虽然脂肪含量较高，但主要是较为健康的多不饱和脂肪酸（猪肉是饱和脂肪，而饱和脂肪被认为是会增加心血管疾病的，需要限制）。这样看来，黄豆确实是一种营养价值非常高的食材。

不幸的是，黄豆本身虽然营养价值较高，同时也存在多种抗营养因子影响其在体内的消化吸收（煮黄豆蛋白质吸收率只有 60%，而加工成豆腐后吸收率能到 95% 以上）。不仅如此，大豆中的低聚糖、植酸、脂肪酶等还会有胀气、豆腥味、影响其他营养素的吸收等问题。

关键是，煮黄豆还那么难吃。

为了解决大豆上面存在的种种问题，我们智慧的祖先发明了豆浆和豆腐的加工工艺。通过浸泡、物理研磨、高温煮制和卤水点化等工艺实现了抗营养因子的消除，并创造性地通过石膏、卤水添加了非常重要的钙。自此以后，又难吃又不好消化的大豆摇身一变，变成了既营养又美味的豆浆、豆腐脑、豆腐干、豆腐皮、毛豆腐、臭豆腐....

这绝对是所有食材中，最华丽的营养和美味兼得的双重「变身」。

那豆浆、牛奶和现在市面上出的新品「豆乳」，哪个更营养？

之前就一直有人问豆浆和牛奶到底喝哪个好？最近市场上又出了个新品「豆乳」，那这个又怎么样呢？这里给大家解释一下。

在比较这三种饮品之前，需要先说明两点基本营养学知识，一个是动物蛋白和植物蛋白，一个是饱和脂肪和不饱和脂肪。

蛋白质根据其氨基酸的组成不同分为完全蛋白（也称优质蛋白）和不完全/半完全蛋白（非优质蛋白），而一般来说植物类蛋白属于半完全或者不完全蛋白，而动物蛋白属于优质蛋白。也就是说像一般的米饭、馒头里的蛋白质消化吸收利用率是不如猪肉、牛奶里的蛋白质的。但是大豆蛋白是特殊，虽然它属于植物蛋白，但是它的氨基酸组成很合理，属于优质蛋白。也就是说，豆浆和豆乳里的蛋白质是可以和牛奶相媲美的。

其次说脂肪。上文中也提到过，脂肪分饱和和不饱和两种，一般植物脂肪大多为不饱和脂肪酸，而动物脂肪多为饱和脂肪。考虑到对健康的影响，一般不建议摄入过多的饱和脂肪。而回到三种饮品的对比中，牛奶里的脂肪是饱和脂肪，而豆浆和豆乳属于不饱和脂肪，相比之下，豆浆和豆乳要略胜一筹。

除了蛋白质和脂肪，牛奶最大的优势是钙。豆浆和豆腐不同，没有卤水点化的豆浆含钙量是不高的，豆乳要看是否钙强化，否则也是不如牛奶的。

综上，

（1）牛奶在钙和蛋白质上占优势，但在脂肪上占劣势.

（2）豆乳在蛋白质和脂肪方面相比占优势，无钙强化的话在钙上面相比牛奶占劣势.

（3）豆浆除了脂肪偏低之外，在其他方面均不占优势。

因此，豆乳和牛奶（脱脂奶更优）相对营养价值更高，豆浆略逊。
