卢周来 农产品涨价与无锡水危机的背后

2007-06-12 作者： 卢周来

 近几天来，两则表面看似没有关联的经济新闻让我忧心忡忡。

    一则新闻是说，包括猪肉在内的农产品近期涨价明显。在广东地区，猪肉已涨至八到十元一斤。不仅普通消费者承受起来吃力，而且因为在政府价格控制下，销肉户也越来越难以消化成本。中央对此很重视，一方面要求组织充足的货源供应，另一方面要求物价部门加大价格监控力度。 ( http://www.tecn.cn )

    另一则新闻是，因为太湖水域蓝澡大暴发而影响水质，无锡市民无法饮用自来水，只能在超市中抢购纯净水。政府对此也很重视，一方面采取措施引入长江水对受污染的太湖水加以冲刷和稀释，争取早日使水质有所恢复，另一方面也在积极组织周边市场纯净水货源以最大限度满足居民饮用水需要。 ( http://www.tecn.cn )

    在我看来，这两则新闻的背后都透露出同样一个信息：中国经济面临的结构性矛盾远没有得到解决。

    我们先看农产品涨价。我留意到，对于此轮农产品涨价，经济学家的说法很多，我也大抵可以附会这些说法。但在这里我想给出我最近在农村调研时听到的农民的说法。一位平时喜爱看报纸的养猪农民就对我说："城里人赚钱真有招，只要在电脑上动动键盘，炒炒股，就可以用钱生钱。我们乡下人觉得那钱来得太快了。一边是不干活，出来这么多钱，另一边养猪种菜那是有季节的，想多种多收还不行。你说，价格能不涨吗？再说，我们养猪种菜在乡下又没有办法去炒股，干的活又脏又累，你再不让我们涨点价，我们还有活路吗？"这位农民还说："猪肉涨点价，城里人就觉得受不了，政府就说要控制农产品价格.那楼市与股市疯了似的涨，不正经干活就可以坐收大钱，我们农民看着都心惊肉跳，政府为什么不好好管管？"尽管这位农民的话在我们看来肯定很土，很不中听，甚至会得罪城里人，但我觉得其中某些道理其实还是很实在的。这个道理就是，中国作为一个有着十几亿人的发展中大国，经济发展必须建立在实体经济的发展上，只有好好发展农业，发展现代制造业以及高技术产业，那才是真正的发展。如果在现阶段就把房地产与资本市场定位为"支柱产业"，整个经济发展将不可避免面临泡沫化的危险，最后的结果可能是有了钱，但物价涨得你同样受不了。 ( http://www.tecn.cn )

    无锡自古就是江南鱼米之乡，现在竟然发展到无锡市民守着中国第三大淡水湖没有水喝，这真正是对中国经济发展过程中长期以来不计环境代价的一种报复。实际上，不仅仅在太湖流域，整个江浙地区作为中国经济发展最具活力的地区，却也是环境代价最为高昂的地区。早在2004年，人民日报所属《经济》杂志记者王信川、张朋等曾发表过题为《浙江经济模式面临全面危机》的报道。报道透露，由于其经济飞速增长的背后是对水资源的掠夺式利用,自2002年以来，浙江陷入大规模的水短缺危机。由于大量地面水被污染，地处江南水乡的浙江也不得不对地下水大量开采，由此直接导致地面下沉。到2003年底，宁波市的沉降区域总面积达175平方公里，整个宁波市区已成了一个巨大的碟形洼地。而我们原来认为只有少雨水的北方才会发生这样的事。然而可怕的是，在少雨的西北地区，以水资源污染为代价发展地方经济的事并没有被遏制的迹象：在陕西，在甘肃，在宁夏，我最近都见识了当地百姓面对水资源被污染后生活的无奈。宁夏一个农民对我说：报上说中央拨了多少亿的抗旱资金，但干净水是买不到的！除非我们迁走。所以，无锡太湖水污染事件再次以一种特别的方式告诉我们，要维持中国经济可持续发展，必须重视环境保护。 ( http://www.tecn.cn )

    除此之外，农产品涨价以及无锡水污染事件的背后，还透露出另外一个令人担忧的事实：政府执行力与公信力不够。楼市调控这么多年，商品房价格仍然在一个劲地涨，老百姓对政府调控能力已很怀疑。正因此，当政府用一般的市场手段想放缓股市时，老百姓因为已不相信政府调控能力而并未很好地领会政府意图。完全可以说，楼市与股市发展的情势已经让政府执行力与公信力大打折扣。太湖流域治理因为涉及到跨省协调，更是体现政府治理能力的一个重要方面，在当年轰轰烈烈的"零点行动"之后，太湖污染再度反弹，也证明政府调控能力远远不够。所以，两则新闻还告诉我们另一个同样重要的道理：要夯实中国经济可持续增长的根基，必须认真加强和改善政府的宏观调控，特别是要增强政府的执行力与公信力。 ( htt
