易宪容 住房是中国人最大的民生

2007-12-17 作者： 易宪容

住房是中国人最大的民生

易宪容

据报道，中国国务院总理温家宝19日在出访新加坡，在新加坡国立大学发表演讲表示，“如果提起人民生活，我最为关注的是住房问题。”也就是说，从温家宝总理来看，住房是最大的民生。 ( http://www.tecn.cn )

    温家宝总理认为，中国改革开放30年来居民住房的条件有很大的改善，现在城乡居民住房面积人均超过20平方米，但是分布不均，特别是近些年来房价上涨较快，人民有很大的意见。 ( http://www.tecn.cn )

    而解决民众的住房问题，要从四个方面来考虑。首先，政府的职责最重要的是要搞好廉租房，让那些买不起房或者进城打工的农民工能够租得起房、住得上房。为此，中国政府今年将安排49亿元用于廉租房建设，再加上地方财政，投入将达到几百亿。他说，明年我们将加大廉租房建设的力度。 ( http://www.tecn.cn )

    其次，是建设经济适用房。他说，经济适用房大多数是面向中产阶级的，他们中有许多工资也不是很高。在中国，经济适用房户均面积90平方米是比较适当的，因为中国人多地少，“对廉租房和经济适用房，政府应提供和保障土地的需要，当然（这些）土地也要节约和集约使用”。 ( http://www.tecn.cn )

    对于高档住房，主要靠市场调节，“但是也必须有国家的宏观调控，防止利用房地产进行炒作造成市场混乱”。

    要让全国人民懂得中国的国情，中国是有13亿人口的国家，要守住18亿亩耕地的“红线”才能养活中国人。中国每年新生人口要超过1000万，每年由农村进入城市需要解决住房的进城人员达到1800万，还有2100万流动农民工，也需要居住条件。所有这些问题，政府都要统筹兼顾。 ( http://www.tecn.cn )

    温家宝总理对住房的讲话，其实是对当前房地产市场一个清楚明确的表态。而在温总理的讲话中，主要表达了以下几个意思。一是，住房问题是当前国内民众最大的民生。大家可能看到，近几年来政府在关于民生的方面下了大功夫，作足了努力。比如医疗体制的改革、社会保障体系的改革、教育的改革、劳动收入分配制度的改革、农民问题的改革，而且这些方面都取得了相当大的进展。这不仅写进了党的十七大报告中，也落实在政府的各项政策中。但是，住房问题政府面对的困难则越来越多。特别是房价在高企时，民众对房地产市场的质疑也越来越多，就目前的情况来说，绝大多数人都在问，中国的房地产市场改革成功了吗？正因为，房地产市场的问题丛生，总理也就最为关注了。 ( http://www.tecn.cn )

    房地产市场为什么问题多？房地产为什么是最大的民生？最为重要的是，对购买住房居住的民众来说，住房消费是他们一生中最大的消费。就目前的高房价下，民众不仅要把其个人一生的全部收入都投入到购买住房这一消费中，而且还可能让自己的父母一生的积蓄也投入到购买住房中。这不仅加大广大民众购买的困难，弱化的许多居民的老年基本生活条件，也制造了绝大多数居民无法让自己其他投入增长，更是让许多居民家庭及个人成长永远限制在一个小小的房子的消费上（限购买一个高价的住房，其他消费都低到极限）。这种现象的出现，对他们是相当不公平的，也是根本上不利于未来整个社会经济的发展。因此，解决住房的问题，不仅是绝大多数居民最大的民生，也是我们总理所关注的最大的民生。总理有这样的观念，相信中国的房地产市场的政策会出现大的变化。 ( http://www.tecn.cn )

    二是，从总理的讲话中，总理既看到这几年住房市场得到快速的发展，住房市场的发展改善也一些居民的基本居住条件。同时，他也看到，尽管这几年房地产市场快速发展，生产了不少住房。但是，大量住房生产仍然存在很大的持有不公。也就是说，这几年的住房市场是得到快速的发展，但是这些生产住房有可能并没有公平让更为广大的民众持有。住房持有的不公是目前中国的房地产市场存在严重的问题。这个问题既有我们制度缺陷，也有这种制度缺陷所导致利益问题。因此，在我看来，如何来调整这种住房持有的不公，政府会采取相应政策。这也预示着中国房地产许多政策会发生大的变化。 ( http://www.tecn.cn )

    三是，中国的房地产市场问题的解决，总理把住房保障体系建立作为一个重要的方面来做。无论是廉租屋还是经济适用房都是这样一个思路。而且总理特别去看到新加坡的祖屋，也就是政府希望多学一点新加坡的经验。但是就目前中国情况与条件来看，新加坡的经验是很好，它们的85%以上的居民可以住祖屋，但是中国来看则没有这种经济实力，更何况新加坡祖屋制度已经发展了几十年。现在，政府投入49亿建廉租房与2006年总销售达达2万亿房地产市场来说，是杯水车薪。 ( http://www.tecn.cn )

    当然，在中国住房保障体系建立中，总理已经把他的整个视野放在全国，而且不仅仅是城市。在总理看来，农民进城，无论是进城务工，还是迁移到城市，都得把他们的住房问题，都把他们考虑在整个住房保障体系内。比如，每年由农村进入城市需要解决住房的进城人员达到1800万，还有2100万流动农民工，也需要居住条件。总理这样一个视野，不仅是我们中国房地产市场发展之方向，也是中国农民、中国经济持续发展的方向。因此，这也预示着未来中国的房地产市场应该是一个平民化的市场，而不是目前所推崇的豪宅市场。而一个平民化的住房市场，就得让我们房地产市场的价格保证绝大多数民众都有支付能力来购买得起。如果说，就目前的房地产市场的价格来看，城市里的居民工作了一辈子，都购不起，那么农民进城购买住房更是没有这种能力。因此，从总理的讲话中，也就是预示着中国的房地产市场发展方向，预示着中国的房地产市场要成为一个平民化的市场。结合最近中国的房地产市场发生一些变化，也就预示着中国的房地产市场要走向哪里。 ( http://www.tecn.cn )

    三是，从总理的讲话中，特别注意到要把房地产市场的投资与消费区分开。对于房地产市场的投资，则由市场来决定。投资者在房地产市场投资不仅要承担投资房地产的成本，也得要承担房地产投资的风险。这就是目前359号文件的基本精神。同时，对于房地产的投机炒作，政府是严格限制或是要制止的。 ( http://www.tecn.cn )

    其实，温家宝总理对房地产市场的讲话，其包括的内容十分丰富，我这里只是简单谈一些个人思考。
