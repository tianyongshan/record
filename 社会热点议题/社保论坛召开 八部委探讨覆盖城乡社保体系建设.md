# 社保论坛召开 八部委探讨覆盖城乡社保体系建设

社会热点议题

社保论坛召开 八部委探讨覆盖城乡社保体系建设

2007-09-23 作者：

人民网北京9月22日讯　今天，“中国社会保障论坛”2007年年会开幕式在钓鱼台举行。八部委从政府角度阐述了建立覆盖城乡社会保障体系的有关问题。



中国所面临的最重要的挑战之一



联合国驻华协调员、联合国开发规划署驻华代表马和励在会上表示，社会保障体系已经成为时下中国所面临的最重要的挑战之一。在目前的发展阶段，社会保障体制在中国现阶段的极度重要性，源于其在消除持续的贫困以及在促进社会公平等多方面所发挥的作用。



劳动和社会保障部部长田成平指出，要清醒地认识到，我国社会保障领域还有不少人民群众和社会关注的热点和难点问题没有解决好，也将面对人口老龄化、就业形式多样化、城市化等严峻的挑战。我们必须紧紧抓住有利时机，充分利用有利条件，深刻认识建立覆盖城乡居民社会保障体系的紧迫性和艰巨性，立足于我国的基本国情，坚持"广覆盖、保基本、多层次、可持续"的方针，不断健全保障项目，稳步提高待遇水平，进一步拓宽筹资渠道，加快规范管理服务，逐步建立起覆盖城乡居民的社会保障体系，让广大群众享受到国家发展进步的成果。



全国人大常委会副委员长、中国社会保障论坛顾问成思危还在会上发表了题为《加强社会保障理论和战略研究，建立中国特色的社会保障体系》的演讲。对于中国社会保障理论和战略研究的重要性，成思危指出这个重要性在于：建立覆盖城乡居民的社会保障体系是全面建设小康社会的必然要求，也是构建社会主义和谐社会的重要保证。



不断扩大社会保障的覆盖范围



国务委员兼国务院秘书长华建敏指出，不断扩大社会保障的覆盖范围，特别是实行“两个确保”、“三条社会保障线”政策，建立基本医疗保险制度，较好地保障了职工群众的基本生活和基本医疗，有力地促进了国有企业改革和经济体制转轨。



全国总工会副主席董力认为，随着我国经济体制改革的进一步深入，市场经济条件下的各种社会风险逐渐凸显，作为应对社会风险主要手段以及保持社会稳定和谐的社会保障制度，必须尽快扩大社会保障覆盖面。



董力说，当前，养老、医疗和工伤保险应列入优先实现全覆盖的项目。在"扩面"工作中，养老保险应以非公有制企业、城镇个体工商户从业人员和灵活就业人员为重点.医疗保险应以有缴费能力但尚未参保的用人单位、国有和集体困难企业退休人员、灵活就业人员和农民工，城镇非就业居民为重点.工伤保险应以农民工为重点人群、以高风险企业为重点行业，以大中型城市为重点区域。同时，要辅之以城市社会医疗救助制度、流动人口社会保障及相关福利保障制度等，确保全体社会成员在遭遇困难时免于陷入生活无着落。



应强化政府责任



劳动和社会保障部部长田成平还在会上强调说，建立覆盖城乡居民的社会保障体系，必须既强化政府责任又履行单位和个人义务，突出社会保障的公平效应，兼顾制度运行的效率。我国的社会保障运营和管理制度还不健全，覆盖范围窄，统筹层次低，社会保险关系转移接续难，政府、用人单位、个人三方筹资机制还有待进一步强化。我们必须进一步加强政府对社会保障的管理，充分发挥政府调节收入分配、维护社会公平、提供均等公共服务的职能，切实履行政府职责。



北京大学教授王思斌从社会学的角度指出，要建立一个个人、集体和政府三家组成的一个筹资结构，无论对于城市来讲，还是对农村来讲，都需要政府的支持。



会上，有关部门领导、专家还从政府、社会保障的理论、法制、财政的投入以及体系结构等不同的角度，对于如何建立覆盖城乡的社会保障体系进行了深刻的论述。
