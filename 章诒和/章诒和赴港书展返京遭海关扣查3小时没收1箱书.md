# 章诒和赴港书展返京遭海关扣查3小时没收1箱书

章诒和赴港书展返京遭海关扣查3小时没收1箱书

发表于 2019年7月22日 编辑 Jenn

    以《往事并不如烟》、《伶人往事》等书闻名的中国作家章诒和（左）。（本报资料照片）



赴港参加香港书展签名售书的中国知名作家章诒和21日透露，她19日返回北京时一箱书被内地海关没收，就连她在台湾出版的成名作《往事并不如烟》也被当非法出版物，她还被扣三小时。



明报报导，对于海关人员没收书籍，章诒和表示他们「碰上了一个不要命的老妇」。章诒和说，自己坐过十年牢，什么都不怕，她表示准备打听海关总署署长家庭地址，每晚到他家小坐，说书。



章诒和现年77岁，是中国民主同盟及中国农工民主党创始人章伯钧二女。香港书展之际，章诒和赴港签售她的《伶人往事》等著作，并与香港作家董桥、倪匡等人会面。



章诒和21日在微信朋友圈说：「此前无数次从境外带书入内（地），均畅通无阻。这次，我香港归来（7月19日）一箱书全部扣下，人被扣3小时，愤懑且屈辱。百思不得其解。读了阎准（阎淮）文章，终于明白了一点点——你就写男旦，写小说，别公开支持这个，质问那个！」



据报导，阎淮是前中共中央组织部干部，是「红二代」，现旅居美国。本月初阎淮曾披露北京边防警察禁止持有美国绿卡的他返回美国，原因是他发起声援被清华大学停职调查的法学教授许章润。



今年3月，许章润被清华大学停职调查时，章诒和也曾发表言论称，「听说不让我的老乡许章润上课，是吗？问清华。」



百度百科的资料显示，章诒和儿时曾居于香港三年，建政前随家迁到北京定居。大学时期（1963年）被下放到四川川剧团艺术室工作.因为在日记中写道：一人得道鸡犬升天，而得罪了江青。



1970年，章诒和被当时的四川省革命委员会、四川省公检法军事管制委员会宣布为现行反革命罪犯并判监20年.1979年秋天，章被判无罪释放并回北京。1979年起章在中国艺术研究院戏曲研究所任助理研究员、副研究员、研究员，硕士生和博士生导师。2001年退休并开始写作。



另据南方人物周刊报导，章诒和众多作品中，最富盛名的当属《往事并不如烟》、《伶人往事》，这些沉痛之作，记录一个时代的人，也记录他们的光鲜与屈辱、闪耀与消亡。



「什么叫文字，读一读章诒和就知道了」，作家毕飞宇说。



北大法学院教授贺卫方也赞誉章诒和的作品，「在汉语世界里树立一种独特的文学风格。」
