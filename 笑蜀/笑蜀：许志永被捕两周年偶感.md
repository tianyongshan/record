# 笑蜀：许志永被捕两周年偶感

笑蜀：许志永被捕两周年偶感

发表于 2015年7月17日 编辑 bj

治史出身，知道群众运动咋回事儿。群众往往是群盲，往往是愚民加暴民，群众运动很容易流于暴民运动，从五四、三一八，到号称反饥饿反内战反迫害的国统区学生运动尤其沈崇事件，再到文革，群众运动没有不被阴谋家操纵的。我天然地惧怕群众运动，惧怕群众运动背后形形色色的操盘手。

但新公民运动不是群众运动，恰恰相反，它是要从群众转型到公民，从群众运动转型到公民运动，在宪法和法律范围内争取公民权利，推动整个国家和平有序的转型。这符合我的理念。所以我完全认同许志永，认同新公民运动，认同自由、公义、爱。我佩服许志永，他不仅有理念，而且敢于践行，敢于牺牲，敢于为最小成本转型付出个人代价。

我只在最后一刻才不得不站出来。整整两年过去，我不后悔自己的选择，哪怕这条路走不通。如果真的走不通，那是整个国家的悲剧，历史最终会再次证明：坑灰未冷山东乱，刘项原来不读书。
