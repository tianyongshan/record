# 茅于轼：公平促进效率 ——评自由农转非

茅于轼：公平促进效率 ——评自由农转非
http://www.newdu.com 2018/3/9 爱思想 茅于轼 参加讨论

    我们常说，公平与效率是互相矛盾的，增加公平往往会降低效率。但是取消户口对人口流动的限制却对公平和效率都有利。
    解放以前对人口流动没有什么限制，户口管理也不严格，有的地方根本就没有。解放后，逐渐实施了户籍制度，而且越来越严。最初的目的并不是想制造农民和城市居民之间的不公平．但是不以人们的意志为转移，计划经济必然会制造出人为的不公平，首先因为有计划的制定者和计划的执行者，他们之间是不平等的.其次因为计划所规定的分工不是建立在公平竞争基础之上的，而且一旦划定，再也改变不了。
    现在四川，湖南，河南，山东等地规定了农民可以自己选择农村户口或者城市户口。这是一个伟大的进步。其中特别规定对进城农民不收取城市增容费，进城农民享受城镇居民同等待遇。在一定期限内还保留农村的自留地和宅基地，这些规定为农民进城进一步扫清了障碍。
    市场经济能最有效地配置资源，在各种经济制度中是比较好的一种制度。但是市场的有效性是有条件的。这个条件就是公平竞争和自由选择．WTO运行的基本规则也就是这两条。其中规定的所谓国民待遇，就是指外国人和本国人一样，都有同样的选择权利和竞争条件。过去中国自己人中间还做不到这两条，岂非笑话。现在我们在这方面前进了一步，值得庆贺．希望有更多的城市也能做到。
    从计划经济改变到市场经济是一个渐变的过程。取消户籍对公平竞争的限制也不能毕其功于一役，而要逐步完成。因为城市忽然接受大量农民是有困难的，需要一步步来。问题是城市政府要明确，大量农民进城是一个必然的过程，在自己的政策中必须将接受农民作为一个基本方针，在住房，交通，环境，公安等方面下功夫扩大容量，特别是要以平等态度对待农民。态度的改变是立刻就可以做到的。
