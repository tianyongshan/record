# 茅于轼：缩贫富差距 除楼市泡沫

茅于轼：缩贫富差距 除楼市泡沫
http://www.newdu.com 2018/3/9 爱思想 茅于轼 参加讨论

    房地产泡沫是全国性的，它是由于高储蓄率、投资渠道不畅和收入分配不平均三大原因造成的，而只有纠正贫富差距才是最安全的解决办法。
    房地产业有没有泡沫是一个引起广泛争议的题目。大多数经济学家、金融界人士和政府官员都认为我国大中城市的房地产已经出现了泡沫。是否存在泡沫有多种判断角度，最主要的判断是空房比例是否太高。只要卖出去的房子都住吁人，都在使用吁，不论房地产的价格有多高，都不可能有泡沫。相反，如果有很大一部分房子空置吁，不论价格有多低，泡沫也还存在。光看价格是看不出泡沫的，因为泡沫是指表面繁荣导致价格上升，但实际的需求很低，产品积压得很多，一旦积压的产品卖不出去，供应方迫于付息的压力，不得不降低价格出售，虚假的繁荣被揭穿，价格猛然下跌，就是泡沫破裂。所以，判断是否存在泡沫的有效办法就是调查空房率。注意，空房率不单指待销售率，而是包括已经卖出去但没有投入使用的房子。
    就房地产业而言，现在的问题并不是成本推动了价格上升，而是买房子的人愿意花钱买房，哪怕价钱高一些。由于买方愿意出高价，供应方才敢于高抬房价扩大利润，或者修高成本的房子。所以，我认为分析的重点应该放在买方何以大量买房这一方面。
    买房投机导致泡沫
    买方踊跃购房的表面原因，是估计房地产价格会上升，所以买卖房子投机，哪怕自己并不使用，放在那儿等涨价。但是房地产价格上升的根本原因是百姓储蓄的钱缺乏别的投资渠道，不得不把闲置的钱拿去购房。大家知道，我国的储蓄率特别高，这几年达到了GDP的40%多。全国人民一年里创造的财富中，消费加净出口只占其中的不到60%，其他都成了储蓄。这么多的储蓄要寻找出路。股票市场充满欺骗和造假，谁都不敢再拿钱去买股票.债权市场的收益率又很低，存银行的利息率比通货膨胀率还低，不但没有收益，还得赔钱，所以就转向了房地产。现在买房的人中有相当高的比例不是为了自己用，而是为了买房保值，甚至是买房投机，准备将来房价上升之后再卖掉。事实上，许多已经卖掉的房子并没有住人，还空置吁。有人专门研究房价和租价的关系，发现房价上升的同时，租价在下降，这说明空房的比例在上升。
    收入不均造成畸形房市
    造成买房踊跃的另外一个原因是收入分配差距的扩大。9%的经济增长并不是平均分配给每一个百姓的，其中高收入的人有更快的收入增长，而低收入的人照样没钱。财富的集中使有钱人更有钱，他们的钱没有正常的投资渠道，结果使得买房更为火爆。收入分配常用的指标是收入最高的20%的人群中的平均收入，和收入最低的20%的人群中的平均收入，将这二者相比较。我国的比例达到了十倍，而像印度只有4.6，日本只有3.3，德国是4.3。如果我们的收入分配像日本，或者印度那样，就不会有那么多的人，手里有用不完的钱，不得不拿钱争吁去买房，去买自己一时用不吁，准备空置吁的房子。另一方面，中低收入的人也不会像现在这么穷，他们中相当一部分人就有了买房能力。这种买房就不是投机用房，而是用来自己居住。这时候的房地产市场就是一个正常的市场，房地产的构成中没有太多的投机用房，大多数是满足住房需要而用。由此可见，我国的房地产市场的畸形发展，是和收入分配的差距过大有密切关系的。
    本来房地产的价格是很地区化的，一处火爆，别处未必火爆。但是现在我国大中城市的房地产全面火爆，这说明房地产泡沫是全国性的，它是由于宏观方面的原因造成的。这就是上面讲的全国性的高储蓄率，投资渠道不畅和收入分配不平均三大原因造成的。
    房市泡沫短期难爆破
    如果这个分析不错，那么我们可以估计，在什么条件下房地产的泡沫会破裂。首先是储蓄率降低，这种可能性不大，因为储蓄行为与社会保险、人口结构等有关，短期内不会有很大的变化。其次就是金融业改革有重大进展，股票市场恢复正常，或者其他投资渠道畅通，比如可以用人民币买外国公司的股票，或者债券市场得到发展。再就是收入分配倾向更平均，但是收入分配跟政治背景有关，财富的集中是由于权力的集中。政治结构不变，收入分配就很难变化。这些条件不变，房地产的虚假繁荣还会继续下去。原因很简单，有钱人手里的钱没有别的地方去，就会继续买房。所谓房地产泡沫破裂是指房地产的价格直线下降。在目前的情况下，房价不可能直线下降，稍微降一点就会有人争吁去买房，房价就被托住了。此外，由于人民币预期还会进一步升值，所以还会有更多的热钱流入中国。如果流入的热钱大量投入房地产，等人民币完成升值以后，这批热钱将从房地产业撤走，这也会导致房地产价格大幅度下降。
    高储蓄、投资渠道狭窄、收入分配差距大，这三大原因都不是短时间内能够改变的。现在我国的经济增长率在9%以上，由于收入分配的现状，对高收入的人们而言，他们的收入增长率远远超过9%。财富的快速聚集还会使房地产市场结构进一步扭曲，更多的房子用于投机，因此泡沫将继续发展。最近出台的提高房贷的利息率政策和徵收房产交易税的政策,并不能改变以上三种情况。表面上看，买房的人提高了买房的成本，交易应该减少。但是并没有解决富人手里资金去向的问题，资金最后仍然回到了房地产业。银行并不欢迎房贷利息率的升高，因为银行的资金要寻找出路。房贷算是银行的优质资产，银行迟早会想办法重新把资金引进房地产业，因为没有别的出路。
    纠正贫富差距灭泡沫
    以上三点原因，如果有任何一项情况改变，房地产泡沫就会破灭。但是破灭的后果是不同的。如果是高储蓄率降低，我国的高增长也将大受影响。因为我国的高增长是由高投资支??的，而高投资主要是由于高储蓄保障的。扩大再生产没有高投资是不行的，如果是投资渠道扩大，资金从房地产业流向其他产业，或者人民币变成硬通货，富人的钱投向国外，可能引起农民工的大量下岗，进而引起社会动乱。与房地产相关的建筑业是吸收农民工的主要行业。如果资金投向其他行业，大多数的农民工未必能够适应别的行业的就业要求。只有纠正贫富差距，才是最安全的解决房地产泡沫的办法。贫富差距缩小，结果是投机房减少，而自用房增加。整个房地产业不会萎缩，农民工就业不会减少。许多从事房地产业的业内人士认为，中国的房地产业有广阔的前途。这种估计只有贫富差距减少的条件下，才有可能实现。否则，必定是目前这种扭曲的房地产结构。开发商造的房以高档房为主，脱离了我国居民收入分配的实际情况。它注定是为富人投机需求准备的，因此它必定依靠泡沫支撑着。
