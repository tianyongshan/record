# 一九九六_纪事

一九九六

一九九六

纪事

多年来，我一直不满意高度意识形态化的近代史主流叙事，自20世纪90年代中期开始，我开始研究清末变革的历史。该年发表的关于戊戌变法的激进主义研究，注重于分析支配中国的激进变革家的深层心理因素。在专制政治下，变革者的思维方式中已经不自觉地潜含着类似于革命者的激进思维与心态。我对戊戌变法的新反思，体现在三年后完成的《危机中的变革：清末现代化中的激进与保守》一书的最后一章。

《危机中的变革》出版后，虽然并没有产生当年的《儒家文化的困境》的社会轰动效应，但其内在的思想深度则远远过之。这本书广泛运用发展政治学、政治社会学与现代化理论等方法，来考察从戊戌变法到清末新政的清王朝如何在丧失权威合法性以后，不得不进行真诚的变革，但却一步一步陷入进退两难的“急诊室效应”的。它揭示了一个因深度危机感煎熬，而不得不采取激进改革的清王朝，因真诚改革而“搬起石头砸自己脚”地走向崩溃的过程。这本书特别着力于考察变革者的文化心态、深层心理与思维方式如何影响了他们的激进的政治选择，并最终导致变法失败的。这一独特视角应该说与作者经历了“八九风波”后的反思有关，以下这段文字选自该书。
