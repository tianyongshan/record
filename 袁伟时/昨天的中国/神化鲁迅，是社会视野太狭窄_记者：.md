# 神化鲁迅，是社会视野太狭窄_记者：

揭示历史真相，我义不容辞——答《华商报》记者问

神化鲁迅，是社会视野太狭窄

记者：

鲁迅应该是近代中国公共知识分子的代表之一，最近，因为中学教材选录他作品的问题，曾引起了一场风波。网络上有人认为之所以有传闻称教材减少鲁迅作品，是作品生涩难懂，而语文老师水平不高也难辞其咎。“五四”之后，鲁迅文章一路风行，伴随几代人成长，今日出现这样的风波，以您对“五四”的深入研究，能否谈谈对此的看法？

袁伟时：

我认为鲁迅在中国现代文学史上，有很高地位，开创了中国文学史的新篇章。他的小说和杂文都很好，语文教材完全应该收录，这是中国珍贵的文化遗产，是我们应该珍惜、并引以为傲的东西。至于选哪些恰当，哪些不恰当，是学术问题，不该炒作那么厉害。

另外一方面，我们编教材应该胸怀更宽一些，在中国当代思想史上，胡适的成就也很高。他是白话文的开创者之一，也应该收进他的作品。

总之，要将语文课本编得丰富多彩，适合传承中国文化，不要那么狭隘。前几天我才跟记者谈到，中学语文里，文言文比重应该提高.台湾地区的文言文比重占到国文课45%，我们现在的比重应该适当提高一点，有利于中国文化传承。

记者：

您觉得这场风波是媒体炒作？

袁伟时：

首先，鲁迅文章除掉少数文言作品，我想没有什么读不懂的，不会比选入教材的文言文更难懂，说难懂是炒作。第二，这次的风波里面有公众的逆反心理。

多年来，我们将鲁迅神化了，把他定位为一个神，他讲的都对，鲁迅是中国文化革命的主将。对历史研究来说，这些都不准确。现在中年人、老年人都是在神化鲁迅的环境下成长起来的，提起鲁迅都很敏感。作为业余爱好者，他的全部文章我最少读过三遍，我感觉他成就很高。但20世纪，除了鲁迅，还有一些成就很高的人，甚至有比鲁迅成就更高的人，比如，在思想文化领域，我认为胡适的成就超过鲁迅，而沈从文的小说也很好。这应该作为学术问题，让大家发表意见，我们现在的视野太窄，有些问题又不恰当地意识形态化了。
