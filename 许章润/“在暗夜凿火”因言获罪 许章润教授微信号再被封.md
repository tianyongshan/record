# “在暗夜凿火”因言获罪 许章润教授微信号再被封

“在暗夜凿火”因言获罪 许章润教授微信号再被封

发表于 2020年7月28日 编辑 Jenn

中国公民运动网2020年7月28日报道：因言获罪的著名法学家、原清华大学教授许章润的微信号再次被封杀，多位友人无法与之联系担忧其安危。这并非许章润第一次被封号，今年春节后许章润自安徽老家返回北京即以“隔离”为由遭到软禁，其家中的网络也被阻断，同时个人微信帐号被冻结，微博被注销。

图片来源于“鲁难推特”

据鲁难（@lunanweiyi）透露:仅仅半天，许章润教授的微信号又被封了，目前处于失联状态，令人担忧。昨天，许教授在朋友圈赋诗铭志：读史早知庚子事，飘蓬又是戊戌年。六章辗转心血在，万象豪端一线天。刚看完许教授的朋友圈，发现我刚刚注册的微信号也被封了，千言万语汇成一句话：草。

在中国言论管控日趋严厉的局面下，无惧政治迫害、多次公开批评中共的许章润教授于7月6日被四川警方带走，被关押一周获释后，7月15日被清华大学以“道德败坏”开除教职，其中清华大学作出“关于给许章润开除处分的决定”中包括“经查实，自2018年7月以来许章润多次发表文章，严重违反『新时代高校教师职业行为十项准则』有关规定”。随即清华部分校友发起募款活动，以示对许教授的支持。许章润于7月19日在致清华校友的公开信中婉拒了大家的好意，写下“极权必败，自由终将降临吾土，天意人心，如日月昭昭矣。吾人一日不死，便一日呼唤”，并言“天快亮了……”

今年2月武汉封城新冠肺炎疫情在中国大地肆虐之时，许章润公开发文《愤怒的人民已不再恐惧》，文中写道：“兑现《宪法》第35条，解除报禁，解除对于网络的特务式管控，实现公民言论自由和良心自由，坐实公民游行示威和包括结社在内的各项自组织权利，尊重全体国民的普遍人权，特别是政治普选的权利，而且，对于病毒的来源、隐瞒疫情的责任人及其体制性根源，启动独立追责机制……”文章发出后外界非常担忧许章润先生的安危，但他似乎对执笔为剑所产生的后果已有心理准备，文中还写道：“戊戌以来，在下因言获罪，降级停职，留校察看，行止困限。此番作文，预感必有新罚降身，抑或竟为笔者此生最后一文，亦未可知。但大疫当前，前有沟壑，则言责在身，不可推诿，无所逃遁。否则，不如杀猪卖肉。”

许章润在今年接连发表的三篇文章《许章润：愤怒的人民已不再恐惧》、《许章润：世界文明大洋上的中国孤舟》、《许章润：践踏斯文 必驱致一邪魅人间》传播甚广。今年6月底在纽约出版了许章润的文章合集《戊戌六章》，成为其被警方抓捕的导火索。

被中共禁言的公共知识分子不在少数，在中国每天都在重复着因言获罪的人权侵害案例。不久前，遭腾讯“以莫须有的罪名”屡屡封杀微信号和微信公众号的荣剑指责腾讯严重违反了公民的言论自由权利和社交权利，突破了一个国际性公司的应有道德底线.早于2010年就开通了新浪微博的郭于华教授，曾在今年6月向外界透露“我的微博被封了80个号，微信被封了五个” 。
