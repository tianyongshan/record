# 中国法学家贺卫方言论封锁下被噤声

中国法学家贺卫方言论封锁下被噤声

发表于 2017年5月28日 编辑 Timothy Lin



中国法学家、自由派标杆性学者贺卫方星期五表示，由于当局对言论自由的打压达到几十年来空前严厉的程度，他个人多个社交媒体账号屡屡被封，因此未来无法再在社媒上发声，也不会在广大中国读者无法接触到的海外社媒平台上撰文。

美联社星期五报道，近年仍坚持不断发声的北京大学法学院教授贺卫方表示，当局不断地关闭他的个人博客、新浪微博和两个微信账户，让他“感到彻底的无助”，感觉“发出任何声音都是不被允许的”。贺卫方表示，在过去40年里，知识分子的言论从未像今天这样受到如此严重的限制，“这令人感到愤怒”。

贺卫方是中国当局近年对言论采取空前严厉打压下，最新被迫放弃发声的公共知识分子。外界普遍认为，中国过去5年在言论和公民自由等方面大幅度倒退，而崇拜毛泽东的极左思潮却大行其道。

曾拥有几千万读者和粉丝的贺卫方，文章数量多且涉猎广，尤其是有关法治和时政的散文，时常针砭社会弊端。2015年1月，中国共产党机关的求是网发表文章《高校宣传思想工作难在哪里？》，点名批评贺卫方和另外一位广受尊敬的艺术家和学者陈丹青，称他们影响青年人，“不断地抹黑中国”。

不断公开呼吁法治和宪政民主的贺卫方2008年因签署主张宪政的“零八宪章”，一度被北大法学院派到新疆支教。回到北大后的贺卫方，近年来仍始终坚持倡导司法独立和宪政民主。贺卫方2011年曾被美国外交政策杂志列为“全球百大思想家”之一。
