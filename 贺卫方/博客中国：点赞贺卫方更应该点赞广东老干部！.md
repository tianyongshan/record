# 博客中国：点赞贺卫方更应该点赞广东老干部！

博客中国：点赞贺卫方更应该点赞广东老干部！

发表于 2014年12月15日 编辑 凌江峰









    真是巧舌如簧。在眼睁睁看着开风气之先的广东的老干部们聆听贺卫方讲解党的中央全会精神时，环球时报竟然发出妒妇一样的怨恨声：我们不太相信广东省委老干局有发出这种信号的意愿，也很难相信该局领导有意蹚舆论场斗争的浑水。或许更大的可能是，这被当成了一场普通的讲座，负责最终确定这场讲座者人选的领导并不了解贺的背景，或者想不到这件事将会造成很大影响，事情“很简单”地发生了。



    这是掩耳盗铃。难道广东的老干部个个老眼昏花，闭目塞听，一个不知道，个个不知道，全体都不知道贺卫方被毛左、自干五诬称为“砸锅党人”？老干部不知道，包括广东新闻联播在内的广大主流媒体，以及全国知名的《南方日报》、《信息时报》的老总、记者、编辑们也不知道？他们的报道称，12月10日上午，广东省委老干部局、广州市委老干部局联合举办广东省暨广州市老干部学习贯彻党的十八届四中全会精神报告会，引导全省百万离退休干部为全面推进依法治国、依法治省和开创广东法治工作新局面增添正能量。于是有五毛质问“广东想干什么？”可见他们惊慌失措到什么地步，对此，用他们进水的脑袋是无法想象的。仿佛当初，邓大人的南巡讲话，让四人帮的残渣余孽和顽固派感到末日来临一样。



    既得利益集团的头等要务就是阻遏改革，让一切保持原样，保持权力的最大化，也就保证了利益最大化，他们把持舆论阵地，扼杀真实的声音，曲解依法治国，以达到维持法治不彰的局面。但出过习仲勋等一大批改革开放先驱的广东又一次使他们大跌眼镜。再说，贺卫方是魔鬼、是妖人吗？这位有美国密执安大学、哈佛法学院访问学者背景的北大司法研究中心主任，全国外国法制史学会副会长，中国法学会比较法学研究会副会长的博士生导师，没资格诠释什么才是依法治国吗？希望党带领中国转向北欧模式，并把自己改造成为一个社会民主党，承认一种竞争性政治的合理性，难道不是一种理性的劝告？这至少为政治体制改革提供了一个可能的选项，难道执政者只愿听腐败分子害党败国的阿谀之声，而听不进有利于推进治理体系和治理能力现代化的谏言吗？



    贺卫方在12月10日转载了一篇赞美自己声音的微博称：为离退休老干部们讲课是我的第一次.听众说这类报告的讲者站着做报告的也是第一次。我结合一些事例，讲解了怎样把四中全会决定中的几个重大事项落到实处，尤其是怎样维护宪法权威，人大如何在守护宪法方面更有所作为，司法公正需要怎样的体制安排。老干部们聚精会神，让我感动。从媒体报道和贺卫方个人的感言看来，讲座是成功的，这个不遗余力鼓吹宪政的学者对依法治国的理解必有过人之处，并非那些照本宣科的宣传干部以及谎话连篇的自干五可以望其项背的。这也证明，广东的老干部对依法治国的期待和愿景没有停留在口号上，而是希望切实推进，反映了体制内外建设法治国家的意愿有多么一致。



    明知山有虎，偏向虎山行。这是在发扬30年前改革开放的敢闯敢干的大无畏精神。所谓吃共产党的饭，砸共产党的锅的说法，不过是仇视改革、仇视依法治国的既得利益集团攻击党内改革派的黑话。这类江湖流氓用语竟然在党媒上堂而皇之的出现，表明某些宣传机构及其从业者已经堕落到何种不可求药的地步。只要敢于说真话，不说假话，就给你戴上砸锅党的帽子，企图吓阻真实的表达。贺卫方与广东老干部的成功互动表明，绝大多数党内同志早已不在乎他们的恐吓，并用实际行动向他们证实，旧的意识形态枷锁已经锁不住人心所向，流氓黑帮的一套更暴露了利益集团的没落衰败的晚景，他们无力阻挡时代前进的步伐。这里不但要点赞贺卫方，更应该点赞襟怀坦白的广东的老干部们。

  (作者：南京龙)
