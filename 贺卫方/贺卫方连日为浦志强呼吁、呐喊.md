# 贺卫方连日为浦志强呼吁、呐喊

贺卫方连日为浦志强呼吁、呐喊

发表于 2015年5月19日 编辑 bj

【道理】一个人，对于发生民族冲突，直言批评本民族主导政府的某些决策，为那些少数民族说话，事实上他就是在为弱势族群说话，有利于族群关系的和谐。相反，如果他口口声声地为政府辩护，那就是让民族之间的隔阂愈发加深。这个道理显而易见。所以打压这样的人，才是煽动民族分裂的行为！

【人性相通，但】古来中国一直没有发展出对皇权的有效约束机制，封建制早废，连州县官也由遥远的朝廷任免，只向上负责。民这方面，不能形成超越亲族以及狭小地域的利益或信仰联合，权力更加恣意。西方的教权与王权的抗衡也无从发生。官家先抓人，再找罪证，明明无罪，也要判刑，即一例。

【法官要当心】十八届四中全会决定：“任何党政机关和领导干部都不得让司法机关做违反法定职责、有碍司法公正的事情，任何司法机关都不得执行党政机关和领导干部违法干预司法活动的要求。”法官们，不要再以什么上方指令做托辞推卸司法责任了。今后，冤狱判决书的署名者就要负直接责任。

【罪与非罪】依法治国在刑法上最基本的要求是区别罪与非罪。过去几年，最高司法机关不断强调坚决杜绝冤假错案，也在纠正冤案方面作出了相当努力。如今，司法又面临一个大考验：对于国家政策提出批评，哪怕是措辞激烈的批评，也是言论自由的体现。我们不可一边纠正错案，一边制造新的冤狱！
