# 今天也是一个重要的节日

今天也是一个重要的节日

上一篇 / 下一篇  2010-04-01 14:52:12 / 个人分类：

/ /

今天是你的节日，我的中国，

清晨我放飞一只古鸽，

为你衔来一枚亚克蜥，

只可惜墙高它飞不过。

我们祝福你的节日，我的中国，

愿你持续不断折腾，永不消停.

我们祝福你的节日，我的中国，

这是草泥马在圈里的诉说。
