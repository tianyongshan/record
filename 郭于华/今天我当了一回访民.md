# 今天我当了一回访民

今天我当了一回访民

上一篇 / 下一篇  2010-06-18 22:29:26 / 个人分类：

/ /

 　　学校原来还有信访办，我今天第一次知道。年初时因参加一位老师的告别第一次知道学校有殡仪馆，真是个部门齐全的衙门啊。

言归正传。今天因为对“博士生毕业前必须在国家权威期刊与核心期刊发表论文方可获得学位毕业”之不合理规定提出异议而求见校长，在美丽如画的工字厅院内校长办公室总值班室被告知未事先预约不能见校长，在本人坚持下被同意先由信访办老师接待听取情况。于是我向和蔼可亲的信访办李老师表达了三条质疑：

1、博士生毕业前必须发表两篇论文(原来为4篇在我们强烈抗议下改为2篇)的规定是何人所定?依据何在?学校有什么权力为了建世界一流大学而把提升发表数量的任务交由学生完成?一流大学是由发表数量决定的么?进而，一流大学是靠培养人才、学术研究的优质而得世人公认的还是靠自封、自吹出来的?校领导究竟要建一流大学还是一流丢人大学?本校十分重视与国际接轨为什么唯在此事罔顾国际知名大学均无此项规定?

2、不仅博士生包括海外留学生都必按照上述规定执行，学校把留学生也当作学术劳工么?算个账：一个留学生一年要向学校交纳三万元学费，除了利用图书馆和上网之外其上课、研究指导等全部工作都是由系里和导师承担的，其田野调查、论文打印、评审、答辩的所有费用亦都由导师承担，而学校每年返给导师的费用是多少?猜猜看，七百五十元!四五年读下来，论文完成了，答辩通过了，文章没发够不能毕业!我靠(请谅解实在忍不住了)!学校难道是留学生贩子么?我们不愿招收留学生学校鼓动你招，原来奥秘在此!

3、作为社会学系却不能自主安排研究生的社会实践工作，而被强迫参加学校统一安排的社会实践，把学生宝贵的暑期做田野工作的时间浪费在不相干且无意义“社会实践”上。行政可以如此干预学术么?

以上三项已经向学校反映报告多年，全系教师都签过名的，然而全如泥牛入海，各级“职能部门”都说自己负不了这个责，“这是学校的规定”。我真不知道这个“学校”是个什么东西!怎么才能找到它?

表达完上述三条意见，李老师说她会向有关“职能部门”反映情况。我坚持要求约见校长或副校长面谈，回答那是不行的。我就奇了怪了：教授不能见校长是何道理?敢情本朝副部级的校长是皇上么?平头百姓是不能随便见的?

访民日记到此结束。

补记：我会坚持上访，直到见着校长。
