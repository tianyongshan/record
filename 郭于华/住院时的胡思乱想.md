# 住院时的胡思乱想

住院时的胡思乱想

上一篇 / 下一篇  2008-09-30 23:56:04 / 个人分类：

/ /

        古人的智慧真是不可不听，“欲速则不达”，“心急吃不了热豆腐”。在最需要时间、精力、体力和智力的五月，我却因“欲速”而心不在焉，在楼梯上一脚踩空，摔成骨折。结果是什么都不得不慢下来――课程、论文评阅、日常生活，……哎，传统的红五月对我却成了黑色的五月。

        在住院等待手术的当晚（ 5月7日），我无意间看到病房墙上挂着一张近乎广告的牌子，名为“TSH大学医院病房入院宣教”，穷极无聊的我于是开始琢磨这个牌子，上面所写无非提醒患者注意的事项：如作息时间，生活、医疗、安全和卫生等方面的要求。可为什么是“宣教”呢？完全可以写“患者须知”，或者“病房注意事项”之类的。“宣教”，顾名思义，当然是宣传教育，难道病患住医院也要对其进行宣传教育吗？病人也不能成为宣传教育的漏网之鱼吗？

        也许是无所事事中特别矫情，也许是儿时经历致使对一些字眼、画面、话语、腔调特别敏感。看到这种令人头皮发麻的熟悉，心中不免惴惴然，想到例如最近一些海外爱国人士在各国大搞红海洋的情景。

        看到一则文革时期的笑话，引在这里，聊供一哂。

        三改婚联：

        特殊年代什么都有它的印记，就连结婚也不例外。这件事发生在“文革”期间的1974年4月尾，有一老贫农的儿子准备“五·一”结婚。那时候讲究破“四旧”，立“四新”，但几个亲戚一合计，觉得还是要贴的，于是拟了一副： 两个节约能手  一对勤俭夫妻  ——勤俭持家

        生产队的批林批孔小组长见了，说：“你们不关心集体生产，只顾勤俭持家，这不是搞资本主义自发吗？”老贫农听了，只得将对联改成： 两个生产能手　一对劳动夫妻——劳动光荣

        真不巧，大队大批判组长下来布置任务，看见这副对联，说：“现在天天大讲继续革命，这副对子宣传唯生产力论，不行，得改！”老贫农又将对联改为： 两个革命能手　一对团结夫妻——相亲相爱

        恰好，公社“大批判办公室”主任下来检查运动，见到这副对联说：“阶级斗争是你死我活，要团结就要先斗争，相亲相爱不是调和矛盾吗？” 老贫农听了，吓得连忙找人商量，于是改为： 两个斗争能手　一对矛盾夫妻——你死我活

        对联改成这样，老贫农的儿子很不服气，他打听到公社大批判办公室主任也是“五·一”结婚，于是连夜将对联贴在主任家门口。

        第二天，主任的新娘一下自行车，见到这副对联，就昏了过去。(黄显环)
