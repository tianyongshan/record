# 别拿豆包不当干粮，别拿学生不当百姓

别拿豆包不当干粮，别拿学生不当百姓

上一篇 / 下一篇  2009-07-06 18:20:18 / 个人分类：

查看( 801 ) / 评论( 0 )





        临近暑假，年复一年的研究生社会实践工作又提上日程。今年的情况有些不同，研究生被要求参加“国庆游行方阵”队列训练来代替原来的暑期社会实践。据有关通知：“社会实践可以提前、推迟或取消，地方政府和接待单位会充分谅解，毕竟国庆是举国大事之一。希望各位同学从和政治的高度看待这项活动。TH和BD 将各自独立组成方阵参加国庆游行，机会难得”。虽说不是强制参加，但上边有话，如果不参加就等于没有社会实践成绩，到时将不能获得毕业资格。当然，参加方阵队列训练也不是没有利处：大约十五天的训练、合练、彩排时间，“可以替代所有必修或选修的社会实践活动”，这比要求的六周社会实践时间要短，对研究生来说比较“划算”.同时还有若干待遇：包括一千元补助费，训练服、表演服、正式服装及鞋帽等。



        我想这一活动若是让学生自愿参与、自主选择倒也不失为好事——可以磨练意志品质、增强集体观念，也能就此了解国家庆典的运作方式…….但如果以上述棍棒加甜枣的方式动员组织，就不免让人有些晕：学生是有自主选择权利的主体吗？走队列与深入生活了解社会的实践可以等同吗？不知道这样的“规定”和安排出自于上边还是中间还是下边，不难看到的是，一项政策、任务或者活动常常在落实过程中被层层加码，过度操作，这或许出于压力不得不为之，亦或许出于“政绩动力”有意而为之。曾经有人说，中国是一个从上往下搧耳光、从下往上磕头的社会，正可谓“上有所好，下必甚焉”.“楚王好细腰，宫中多饿死”。在对中国农村社会生活口述历史的研究中，我屡屡发现在各种政治运动和政策实施的运作过程中存在这一现象，并将其概括为“楚王好细腰”逻辑。



        除此之外，同时让人感到困惑的还有，国庆典礼的内涵是什么？谁为谁来庆祝？检阅这件事又是怎么回事？曾经两次见识过其他国家的国庆庆典，其主要的过程就是阅兵，由国家首脑检阅陆海空部队，这个不难理解，因为军队是国家机器的重要部分，承担着保卫国土和国家平安的重要职能，在国庆之时展示国防力量和国家的强大以增强人民的自信和国家认同和是理所当然的。阅兵时国家领导人是检阅者和观看者，同时普通的民众也是检阅者和观看者，看到精彩时他们会自发地对受阅部队发出欢呼之声。由此让人感到有些不解的是，只见过国庆大典阅兵的仪式，没见过阅老百姓的。



        我们从党的教导中一再获知：人民是国家的主人，执政党、政府和领导者是全心全意“为人民服务”的公仆，“公仆意识，是共产党人立党为公、执政为民的具体体现”。举行庆典就好比一个大家庭要举行成家××周年的庆祝活动，主人高兴出了不少钱，来个主仆同庆、皆大欢喜也很不错，怎么就非得是仆人站在高台之上，主人却要列成方队、步伐整齐地从台下走过，同时还得向仆人挥动花束、欢呼致意？



        谁能解释下这是为什么呀？
