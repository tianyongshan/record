# 到底以什么为本啊？

到底以什么为本啊？

上一篇 / 下一篇  2008-10-06 23:56:04 / 个人分类：

/ /

        四川大地震发生已经两天了，通向震中汶川的道路还未打通，那里没有确切的消息传出来，周边的情况已是惨不忍睹，令人心痛、心焦，震中的严重情况更让人不敢去想象。只感觉回天无力，施救无计。

        今天已经是手术之后第六天，感觉不错，自认为去上课没问题，于是按照医生的建议向病房护士借轮椅，但得到的回答却多少有点出人意料：病房的轮椅只能在医院范围内使用，不可外用。我进一步说明：不到外面用，就是去一下教室，用后马上归还。护士说“这是规定”，并建议我“只能到门诊服务台去借用或租用他们的轮椅”。于是我让学生前往门诊办理此事，经服务台、××部门、××部门，一直到请示院办的曲折过程，得到的结果更出人意料：“无此先例，不予使用”。我无语。

        当然了，“离了张屠夫，也不一定非得吃混毛猪”，WHO怕WHO啊？最后课也上了，不过是临时调换了教室，我拄着拐去而矣。但这件事还是得说叨说叨，我老是想找人问问：

        一个轮椅，究竟是摆设？还是供品？不就是一个为人所用的东西吗？我就是本院的住院病人，我脚骨折了需要使用一下为什么就不行呢？那个东西存在的意义究竟应该是“符合规定”？“遵循先例”？还是为人的需要服务？

        这件小事确乎微不足道，但是让人联想到许多问题的相同逻辑。为什么要制定规定（制度）？制定的依据是什么或者说以什么为本来制定？例如最近我们被告知：每个导师每年只能招收0.6个博士生，也不知道这是谁定的？为什么是0.6？有什么根据这样定？又如何操作？是招缺胳膊的还是招短腿儿的？是不是学院以后该改名为TSH大学“残疾学院”？

        sigh，在面临全民族的大灾大难之际，还是温总理说出了人们的心声：首要的是救人。
