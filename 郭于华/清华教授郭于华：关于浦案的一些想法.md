# 清华教授郭于华：关于浦案的一些想法

清华教授郭于华：关于浦案的一些想法

发表于 2015年12月24日 编辑 Timothy Lin



他以血肉之躯对抗钢铁机器，他已经做了许多，做得很好.他没有弯曲，他选择了伏法却没有认罪，他仍然是、一直是顶天立地的公民，正直公义的律师。超过19个月的羁押、查证，只有7条微博，指控两项重罪，判三缓三，轰动法律界，震惊全世界，这个案子无论怎么判，他都赢了！输了的是中国法治。而他付出的代价也巨大：消瘦的身体，花白的头发，衰退的记忆力，令人担忧的糖尿病……

自他入狱到他被判缓刑出来，泼向他的污泥浊水就没有断过.清者自清，毋庸多言。我只想说，不要在中国呼唤曼德拉、寻找昂山素季，我们面对的是质地不同的高墙，是特殊材料的体制。血肉之躯何能长久地经历钢铁机器的碾压？生存、健康地生活是人的基本条件，是各种权利的基础。不做律师仍是公民，能做的事还很多。那些在大个子先行者身后享有遮风挡雨好处却不知感恩的人，那些指责他“认罪”妥协、鼓动他人做英雄的人，我连鄙视都懒得表达。

浦志强，无论你如何选择，你还是你，你就是你，你永远是我们敬重的大个子律师。

2015年12月23日
