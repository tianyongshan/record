# 让信息透明、让民众出声，其实没那么可怕

让信息透明、让民众出声，其实没那么可怕

上一篇 / 下一篇  2008-10-05 23:56:04 / 个人分类：

/ /

       大灾大难之时，最能够看出政体、制度的良弊，最能体现出一个的运行是否正常，也最能表达出民情民意，而这些都有赖于一个通畅、透明的信息渠道。围绕这次汶川大地震及抗震救灾的新闻报导清楚地证明了这一点。较之以往各种天灾人祸的信息传达状况，这次的相关报导可以说是前所未有的相对开放、及时、全面、细致，当然并非无可指摘。

       人们通过正式媒体在第一时间得知了灾难的发生和严重性.中央的、地方的、国内的、国外的各类媒体比较及时且未受限制地进入灾区开展工作.有关灾情和救灾的数字、图像、文字跟踪性地持续不断地播出.信息内容既包括党和国家领导人、军队、武警、各政府相关部门指挥和参与救灾，也包括灾民和普通人民的抗灾救灾的方方面面情况.……。不难看出，信息的相对通畅带来的都是正面积极的效果：得知毁灭性的灾难后果，全体人民的悲悯之心、仁爱之情被唤醒，形成捐款献血、合力援助灾区的局面.港澳台同胞、国际社会的大量援助源源不断地到来.救灾中政府的指挥调度、军队的全力以赴得到人民的充分肯定和海内外的高度赞扬，一些不得力的地方也因人们通过媒体了解了困难所在而被充分谅解.少数挪用救灾物品、失职、腐败、发国难财等不良现象因曝光而得到及时惩处和加以防范.民众的声音得以传出、网友的合理建议被采纳.……。

       凡此种种都在瓦解“防民之口”的统治思路。让、让民众出声，其实真的没那么可怕.非但不可怕还大有益处。它能够动员更大规模的抗灾救灾，造成齐心协力、众志成城的形势.它也使建立起良好的国际形象.它更使救灾变得有效。与稍早发生在缅甸的风灾比较，结论更不言而喻：重大灾难发生后，缅甸灾区被多重封锁，国际救援难以进入，物资发放缓慢.缅甸军政府不但拒绝外援进入，而且不许外国媒体进入报道。“把灾区变成了一个监狱”。还竟然组织国际外交人员按照计划好的路线参观表演了一圈。这种“真相不可抵达”，酿成更大的“人道灾难”。灾区的幸存者们，面临食物饮水短缺、伤痛、疫病等极为严峻的生存威胁，死亡人数无疑会大规模上升。这样的独裁政府只能用“邪恶”来形容。

       专制独裁的体制、“不可抵达”的信息会使天灾变成人祸，会把灾难加倍地放大。从我们自身和他人身上可以看到极权统治面对灾难时的几种常见做法：一是把人祸归因为天灾（不用举例大家也明白）.二是把丧事办成喜事――不报导关于灾难的实情，只宣传救灾的“好人好事”而且着力突显领导重视、英模涌现.三是贪天功为己有――把因自然之力获得的好转说成是人为之力.这些都是通过控制信息和宣传而实现的。

       问题是，公众和国际社会“不可抵达”真相就会相信假相吗？再说了，以现代科学技术的高度发展，有卫星、遥感、航拍、互联网等等的利用，想完全隐蔽事实、掩盖真相，能做得到吗？相形之下，开放信息、公开真相实在是良善之举，而且也是聪明之举。不是吗？
