# 这样的人民配有什么样的政府？

这样的人民配有什么样的政府？

上一篇 / 下一篇  2008-10-06 23:56:04 / 个人分类：

/ /

        汶川大地震至今已超过半个月时间了，我们与灾区一道经历了震动、感动、行动还有触动：为巨大的灾难造成的生命折损.为人们失去亲人尤其是许多孩子的惨痛.为全国人民被灾难唤起的悲悯、关爱和齐心援助的行动.为全力以赴在救援一线奋战的人民子弟兵.为港澳台同胞和各国的大力支援.为国家领导人亲临现场指挥救灾慰问灾民的亲民形象.与更为灾区人民不屈不挠、坚韧勇敢的表情.也为政府工作和整个点点滴滴的进步、改善.……。

        近日来我们从各类媒体频频看到听到对这次政府的快速反应、全力动员和救灾效果的“高度赞扬”，例如，“联合国秘书长潘基文高度赞扬中国政府抗震救灾迅速有效.中国领导人在抗震救灾中展现了超凡的领导能力”.“外国媒体纷纷发表文章高度赞扬中国政府和人民抗震救灾工作积极有效，信息报道公开透明.中国政府有能力以高速发展让国家焕然一新，这次地震检验了中国领导层的能力”.“国际社会高度赞扬中国政府抗震救灾工作，中国政府的营救行动透明、迅速、得力”.连刚刚被痛斥为“披着袈裟的豺狼”的达赖喇嘛也高度赞扬中国政府努力救灾。这类赞扬听的多了，不由得让人担心又出现以往的“把丧事变成喜事”的情形。在这一巨大的灾难和堪称伟大的救灾过程中，其实更应该得到高度赞扬的是中国的普通人民――那些顽强自救和救人的灾区人民，那些永不放弃、全力救援的解放军官兵、武警战士、医生护士，那些满怀公益心和奉献精神的志愿者，还有无数默默无闻却以捐款献血等各种力所能及的方式支援灾区的普通民众。山崩地裂的灾难可谓惊天动地，而众志成城的救助同样感天动地。

        甚至那些身处社会边缘的人们、那些因各种原因沦落至社会底层的乞丐、残疾人、病人都在此时迸发出美丽的人性光芒。看到衣着破旧的乞讨老人捐出以毛票分票组成的多年积蓄，看到只有半截身体的残障人士出现在捐助现场，谁还能说中国人素质差？来自艾滋病救助组织的消息告诉我们：四川地震受灾以来，爱源（Http://www.aids-care.org）接到了很多电话，他们一直在询问我们，想寻找一个好的途径帮助这些地震的受难者。今天中午，一个河南艾滋病感染者给我打来电话，他说：“四川地震了，你知道吗？”我说：“我也正关注呢”。“这几天我天天看电视，死了好些人，很可怜的，比咱河南（艾滋病人）还可怜”他说。“我们几个老艾（当地艾滋病人的自称）能不能做点事？”他问我。但同样没有给我回答的时间。“电视上那么多人都在献血，我们是老艾，都是坏血，献血人家肯定不要！”他感叹。我还真不知该怎么和他说。沉默，也许我只能以此表达对地震的悲痛与对他们那份特殊情感的理解。片刻，他接着说，“我们能不能捐钱？”“怎么想着捐钱，哪来钱呢？”因为彼此的了解，也随便问了出来。他说：“没事，捐个三百五百没有大问题，我们十多个老艾都商量着想捐点钱，但不知道捐给谁牢靠？你们那边收不收？”我一时不知道该怎样回答。让他们捐？我知道给我电话的是个有“职业”的人――一个开三轮车拉人谋生的艾滋病感染者。我在家的时候，他也经常的给我讲他开三轮车的，因为没有驾驶证，和缴不起城管每月30元的卫生费，经常遭受被罚款和扣车的“待遇”。还是不让他们捐？电话那头的他也许是无法感知我心理的矛盾，他接着说：“咱这些人受难了（艾滋），还不是外面那么多人帮着我们的？不管是公家的钱还是私人的钱，都在帮着咱，现在四川地震，也受难了，天天看着电视，揪心，俺也想捐点钱”。看到这样的，谁又能说中国人觉悟低？

        中国有着最好素质的人民，中国有着最高觉悟的百姓。这些时日我们每每听到“感谢党感谢政府”的话语，受到救助人们的感恩之心是真诚的，而中国百姓也是最容易感恩戴德的。但我认为最应该心存感激的不是人民，而是执政者，是政府，应该为有这样的人民而感动，为有这样的人民而庆幸。写到这不由联想起长久以来关于人民与政府关系的讨论，当听到人们对制度特别是对政府工作有所不满或批评时，有人喜欢说：有什么样的人民就有什么样的政府。意为这样素质低下的民众只能享有这样的政府。在探讨中国民主化进程的问题上，也不时听到“民主不适合中国”，或者“中国人不适合民主”的判断，所含之义也无非中国人民缺少现代的、民主的素养，不适合也不喜欢民主制度。对此，正如杨恒均所尖锐指出的，在对人民这种“妖魔化之下，百年来连民主的影子都没有见过的中国普通百姓一听到民主几乎就条件反射，自然而然地把民主与自己生活中一切美好的东西对立起来了。在民众们所受的教育使他们把自己归队到不适应、不喜欢民主的行列里同时，洋洋得意的统治者又找到了一个阻止民主的借口：这些民众素质比较低，不适合民主”。

        5.12大地震让中国和世界看到了中国普通人民的力量和素质，他们的坚强、善良、团结和爱心。人民才是真正的英雄。这样的人民配有什么样政府，难道答案还不清楚吗？

        大灾大难应该为制度建设和改善提供经验教训。温家宝总理曾经在视察灾区时对学校的孩子们说“地震应该成为你们人生中生动的一课”，这一课其实又何止于学生，这次大地震同样应该成为给全体人民更是给执政者们上的一课：它让会敬畏自然、善待自然，也让权力学会悲悯众生、以人为本。
