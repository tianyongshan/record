# 郭于华老师：《柔弱但却有温度》

郭于华老师：《柔弱但却有温度》

发表于 2014年12月1日 编辑 凌江峰

知名作家、财新传媒编辑徐晓被失踪（被带走24小时之后亲朋好友才知道消息），引发国内国外、线上线下轩然大波.一时间微信微博满屏尽徐晓。作为好友当然不可能置若罔闻，但现实中除了呼吁找人却又束手无策，除了心痛几乎什么也做不了。旋即又传我也被带走，于是各种方式的询问关怀汹涌而至。一人失联，一次误传，让人感到冬日暖阳一般的温度，内心的感动、感激无法以语言表达。这是人性的温度，也是人心的力量。



虽是一次误传，却不难从中感受到一种普遍的心态——紧张、恐惧。能不恐惧吗？专政的利剑随时可能落下，对任何人，以莫须有的罪名，今天是别人，明天就可能是你我。而这种普遍的恐惧其实也来自于统治权力的恐惧——视所有人为敌，自己难免生存于敌人的包围中，当然就会草木皆兵、杯弓蛇影。在这种结构状态下，没有人是不恐惧、不紧张的。



我早就说过，真正的改革开放，真正的社会和谐，其实很简单，作为执政者不是一定要做什么，而是不做什么：放松紧绷的神经，放开紧握的拳头，给社会以正常的空间，还人民以正当的权利，实在是良善之举，也是明智之举。须知，依法治国的根本是法律面前人人平等，尤其是要限制政府权力，而不是用法治民，更不能依想法治国。



徐晓，一位姣小柔弱的女性，但她的作品她的思想她的作为中却有着一种强韧而持久的力量，有时你无法想象她弱小的身躯里哪来如此的强大，但细思量却可以得知，力量来自于人性的温度，来自于内心的博爱。忽然觉得，那句已经著名的话“谁爱得最多，谁就注定了是个弱者”似乎可以变换一下理解，柔弱却有温度的心灵，最有力量，无可战胜。

          2014年11月30日
