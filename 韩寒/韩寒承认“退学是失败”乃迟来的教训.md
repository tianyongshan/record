# 韩寒承认“退学是失败”乃迟来的教训

韩寒承认“退学是失败”乃迟来的教训
http://www.newdu.com 2018/1/11 红网 何勇 参加讨论
8日，韩寒在微博发表长文谈到退学往事，韩寒感慨退学是一件很失败的事情，说明在一项挑战里不能胜任，只能退出，这不值得学习。（1月9日 中国日报网）
现在的韩寒，已经当上了CEO，迎娶了白富美，走上人生巅峰。按照当下成功学标准，韩寒无疑是成功人士，人生赢家，是80后群体的杰出代表。当韩寒走向成功的时候，公开否定18年前的年轻自己，承认当年退学是一件很失败的事情，不值得学习。韩寒的这份勇气值得肯定，应当给予掌声，也反映了已过而立之年的韩寒在思想上的成熟。要知道，当年的韩寒在退学之后，曾经意气风发地叫嚣说，“去当中小学老师的，都是当年成绩不好，考不上大学，又没有一技之长，然后想当兵又没门路，于是只好上个中专，去当个老师混日子。”
在这将近20年里，韩寒一直是很多年轻人的偶像，不少学习成绩不理想的中学生把他当成学习的榜样。尤其是每年的高考季，总是有人拿韩寒当年退学说事，劝说那些高考成绩不理想的学生，指出成绩不理想并不会不影响一个人成才、成功，完全可以像韩寒一样。甚至有人列举出成功人士比尔·盖茨、乔布斯、韩寒都有退学经历，误导一些大中学生把退学当成走向成功的第一步，没有坚持好好学习，而是选择退学，坑害了不少人。
韩寒在退学18年后，勇敢承认当年的退学是一件很失败的事情，公开呼吁青年人不要学习。这无疑是一份迟来的教训，迟来的提醒，有助于当下的一些青年人重新认识和正确认识读书和学习的重要性，而不是盲目觉得不学习照样能够走向成功。实际上有过退学经历的世界首富比尔·盖茨，不止一次表示，虽然他个人的例子证明了当年的退学决定是个正确的选择，但他并不鼓励别人退学。
有过退学经历而走向成功的盖茨和韩寒，一个不鼓励别人退学，一个承认退学是失败的事情，其实都是在告诫青年人，并不是每一个人都适合退学，更不是每一个退学的人都会走向成功。每个人的条件和具体情况完全不一样没有可比性，退学从来都不是也不会成为一个人走向成功的必要条件。最重要的是，不管一个人有没有退学的经历，要想取得成功，时刻不能放弃和停止学习，必须时时处于学习之中，提高自己的学识和能力，这才是走向成功的基础。
总之，每个人都有自己的人生之路，不要迷信他人的成功之路。我们应当清醒认识到，绝大多数人不可能成为第二个韩寒、第二个比尔·盖茨、第二个乔布斯，但我们可以成为第一个、独一无二的自己。
