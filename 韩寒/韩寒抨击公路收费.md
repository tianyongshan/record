# 韩寒抨击公路收费

韩寒抨击公路收费
http://www.sina.com.cn  2009年05月31日11:40  南方周末
■名人谈民生
口述 韩寒（青年作家、赛车手）
韩寒个性张扬独特，喜欢评点时事，惹来争议不断。图/CFP
“不管去哪里，没走多远就掏过路费”
公路收费一直是我们国家的一个毛病。因为我常年在外比赛，所以对此很有感触。只要出了上海，不管去哪里，没走多远就掏过路费。
上海市范围内不用掏，不是因为上海大方，而是因为从1993年起，上海已经收了一个车辆通行费，大概相当于过200次收费站的钱。外地都是在国道上砍一截砍一截那么收费的，直辖市不能在高架桥和环线上砍一截，但又不甘心收不到钱，所以，就有了直辖市独创的每年交一次的通行费。
在费改税以后，养路费终于被取消了。作为一个上海车主，我以为每年3000元的养路费不用交了，结果朋友告诉我，不是的，养路费是每个月100元，也就是1200元，其余的1800元是道路建设贷款费。这个费是要继续收下去的。
同为直辖市的重庆和天津，也单独收取每年数千元的汽车买路钱。到底哪个合算，这很难说。
北京作为直辖市，没收车辆通行费，所以相关部门在奥运会后，创造出了个“车辆排污费”，准备开始征收。没想到运气不好，遇到经济疲软暂时停止。你收取了汽车排污费，环境和空气就好转了吗？那么多的工厂排放污染，你怎么不收费呢？
早在1998年，杭州、郑州、吉林三个城市就搞过机动车排污收费试点，因为2002年《国务院办公厅关于治理向机动车辆乱收费和整顿道路站点有关问题的通知》，2003年停止试点，把试点期间征收的机动车排污费又分别做了退款处理。我估计，如果经济变好，没准“车辆排污费”又回来了。
有人问我，这些费应该不应该收取？我的回答是：应该，但不要费尽心思想那么多的花样，上海的贷款道路建设费，天津的道路使用费，重庆的道路建设费，北京的排污费，太麻烦，索性直接叫“直辖市牛X费”好了。
有些人说，你不交买路钱，国家的公路桥梁什么的怎么造啊？这是不成立的，我们国家的税收是很高的，和道路相关的汽车车价中也至少包含了百分之五十的各类税，这些基础设施的建设当然不应该再另外收费，修点路是政府最基本的职责和国德。
二级公路收费奇闻很多
从去年末开始，交通部开始在全国搞“费改税”，交通部要在全国取消二级公路收费站。
我当时就想，为什么不同步取消一级公路、高速公路、过桥收费呢？那样，才是真正的“费改税”。现在只取消二级公路收费，许多地方马上把二级公路加了个分道行驶的隔离带，继续收费。这好比软件升级，在升级过程中遇到了很多麻烦，技术没升级，二级收费站先升级了，成了一级收费站。
我从报纸上看到一些奇闻，作为5个首批撤站省份的山东，记者调查发现，一些二级路还在继续收费，收费的管理人员把自己的路称为“超二级路”。《黑龙江晨报》1月份的报道说，该省内二级公路收费近期将全部取消，又说“届时部分二级公路将按照建设规划，转成一级或高速公路。”
新疆更是胆大，交通厅2月份下发通告，把原本免费的阿拉尔至和田沙漠二级公路变成收费公路；兰州居然已经把2010年路桥费收到手了，对老百姓的批评，兰州市建委和路桥费收费管理处把原因归于“工作失误”。
更多的省份在打擦边球，以制订方案为名维持二级公路收费。发改委的同志说，广东、浙江、山西三省的二级公路得到2012年年底之前才能分期分批取消。
朋友对我说，不能相信他们，到时候他们一定会换个名目收钱的。在“费改税”的大旗下，他们好比做商场促销，以退为进，表面上给消费者让利，其实还是赚到了钱。
